# <a name="local-partitions"></a>Particiones locales

Una manera de usar particiones es localmente en un servidor de aplicaciones. En este escenario, un administrador puede crear una partición mediante la herramienta administrativa Servicios de componentes. Las particiones implementadas localmente y no dentro de Active Directory solo son accesibles para los usuarios locales, no para los usuarios de otros equipos del dominio. Después de crear una partición y de instalar una aplicación COM+ en ella, el administrador puede asignar esa partición como una partición predeterminada para uno o varios usuarios de ese servidor. Para asignar una partición predeterminada a un usuario, el administrador puede usar la herramienta administrativa Servicios de componentes.
