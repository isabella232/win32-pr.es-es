# <a name="general-design-tips-for-using-com"></a>Diseño general Sugerencias para usar COM+

La clave para un buen diseño es el planeamiento. Si no está familiarizado con el diseño de una aplicación de tres niveles, consulte la sección titulada Diseño de la aplicación [COM+ mediante UML.](designing-the-com--application-using-uml.md)

La forma en que se construyen los componentes COM que crean el nivel intermedio de la aplicación COM+ puede afectar considerablemente al rendimiento, la escalabilidad y la facilidad de implementación. En los temas siguientes se proporcionan consideraciones de diseño y sugerencias de implementación que debe conocer al diseñar componentes COM.



| Tema                                                                   | Descripción                                                                                                                |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [Diseño para la escalabilidad](designing-for-scalability.md)<br/>   | Sugiere formas de aumentar la escalabilidad de la aplicación COM+.<br/>                                             |
| [Diseño para disponibilidad](designing-for-availability.md)<br/> | Describe los servicios que puede usar para mejorar la disponibilidad de la funcionalidad de nivel intermedio en aplicaciones COM+.<br/> |
| [Diseño para la seguridad](designing-for-security.md)<br/>         | Describe las maneras de configurar la seguridad de forma más eficaz para una aplicación COM+.<br/>                                      |
| [Diseño para la implementación](designing-for-deployment.md)<br/>     | Describe formas de minimizar los problemas de implementación al diseñar una aplicación distribuida.<br/>                       |



 

Además, asegúrese de ver Mejorar el rendimiento con la agrupación [de](improving-performance-with-object-pooling.md) objetos para ver las formas de usar de forma más eficaz la agrupación de objetos.
