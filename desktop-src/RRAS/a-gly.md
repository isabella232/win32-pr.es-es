Además, *está fuera de la memoria caché.* En RRAS, una ruta se arrasa cuando se quita de la tabla de enrutamiento después de un período de tiempo especificado.

</dd> <dt>

<span id="_rras_authentication_gly"></span><span id="_RRAS_AUTHENTICATION_GLY"></span>**Autenticación**
</dt> <dd>

La autenticación es el proceso de comprobar que un usuario es quien dice ser. La autenticación se realiza normalmente mediante un nombre de usuario y una contraseña.

</dd> <dt>

<span id="_rras_autonomous_system_gly"></span><span id="_RRAS_AUTONOMOUS_SYSTEM_GLY"></span>**sistema autónomo (AS)**
</dt> <dd>

Un sistema autónomo es un grupo de enrutadores que comparten una única directiva administrativa. Todos estos enrutadores usan el mismo protocolo de enrutamiento para comunicarse (denominado protocolo [*de puerta de enlace interior).*](i-gly.md) Consulte también protocolo [*de puerta de enlace exterior*](e-gly.md).

</dd> <dt>

<span id="_rras_autostatic_route_gly"></span><span id="_RRAS_AUTOSTATIC_ROUTE_GLY"></span>**ruta estática automática**
</dt> <dd>

Una ruta estática automática es una ruta que se obtiene dinámicamente de un protocolo de enrutamiento. Una vez que se obtiene una ruta de esta manera, se comporta como [*una ruta estática.*](s-gly.md)

</dd> </dl>
