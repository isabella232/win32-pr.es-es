# <a name="client"></a>Cliente

Un cliente de RTMv2 es una instancia de un protocolo de enrutamiento o un componente de administración que registra e interactúa con el administrador de tablas de enrutamiento mediante la API RTMv2. A menos que se especifique lo contrario, cualquier referencia a un cliente de administrador de tablas de enrutamiento hace referencia a un cliente de RTMv2.

Un cliente debe ejecutarse en el mismo proceso que el administrador de enrutadores.
