# <a name="maintaining-client-specific-lists"></a>Mantenimiento de listas de Client-Specific

RTMv2 proporciona funciones que permiten a los clientes crear una lista privada de rutas almacenadas en la tabla de enrutamiento. El uso de esta lista puede ser más eficaz que enumerar las rutas de la tabla de enrutamiento.

Para ver el código de ejemplo que muestra cómo usar estas funciones, consulte [usar una lista de rutas Client-Specific](use-a-client-specific-route-list.md).
