# <a name="retrieving-route-information"></a>Recuperar información de ruta

Hay tres métodos que se usan para obtener información de ruta del administrador de tablas de enrutamiento.

-   Enumerar rutas (descritas [en Enumeración de entradas de tabla de enrutamiento).](enumerating-routing-table-entries.md)
-   Buscar rutas específicas (descritas en [Búsqueda de información específica en la tabla de enrutamiento).](finding-specific-information-in-the-routing-table.md)
-   Recuperación de destinos modificados (descrito en [Recepción de notificaciones de cambios).](receiving-notification-of-changes.md)
