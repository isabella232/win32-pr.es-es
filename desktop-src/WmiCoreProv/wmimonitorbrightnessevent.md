# <a name="wmimonitorbrightnessevent-class"></a>Clase WmiMonitorBrightnessEvent

La clase de eventos WMI **Wmi WmiMonitorBrightnessEvent** representa un cambio en el brillo de un monitor.

## <a name="syntax"></a>Sintaxis

``` syntax
class WmiMonitorBrightnessEvent : WMIEvent
{
  boolean Active;
  uint8   Brightness;
  string  InstanceName;
};
```
