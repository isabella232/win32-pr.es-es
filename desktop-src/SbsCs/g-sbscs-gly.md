# <a name="g-isolated-applications-and-side-by-side-assemblies"></a>G (Aplicaciones aisladas y ensamblados en paralelo)

[A](a-sbscs-gly.md) B C [D](d-sbscs-gly.md) E F G H [I](i-sbscs-gly.md) J K L [M N](m-sbscs-gly.md) O [P P](p-sbscs-gly.md) Q [R](r-sbscs-gly.md) [S](s-sbscs-gly.md) T U V W X Y Z

<dl> <dt>

<span id="_win32_global_application_configuration_gly"></span><span id="_WIN32_GLOBAL_APPLICATION_CONFIGURATION_GLY"></span>**configuración global de la aplicación**
</dt> <dd>

Especificación de que todas las aplicaciones del sistema usan la misma versión de ensamblado en paralelo. La configuración global se especifica por ensamblado en un archivo de manifiesto de ensamblado global instalado en la carpeta Winsxs. La configuración global se puede usar cuando un desarrollador o administrador de ensamblados necesita configurar todas las aplicaciones del sistema para usar una versión de ensamblado más reciente. En este caso, el nuevo ensamblado debe ser compatible con versiones anteriores. Una configuración por aplicación invalida la configuración de aplicación predeterminada o global para aplicaciones específicas.

</dd> <dt>

<span id="_win32_global_assembly_manifest_gly"></span><span id="_WIN32_GLOBAL_ASSEMBLY_MANIFEST_GLY"></span>**manifiesto de ensamblado global**
</dt> <dd>

Archivo que define la configuración global de la aplicación de un ensamblado en paralelo. Los archivos de manifiesto de ensamblado global se instalan en el almacén de ensamblados del sistema en la carpeta Winsxs.

</dd> </dl>
