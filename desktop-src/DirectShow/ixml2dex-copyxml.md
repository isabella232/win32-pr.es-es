> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

## <a name="syntax"></a>Sintaxis


```C++
HRESULT CopyXML(
   IUnknown *pTimeline,
   double   dStart,
   double   dEnd
);
```
