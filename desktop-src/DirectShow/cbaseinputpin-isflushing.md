# <a name="cbaseinputpinisflushing-method"></a>CBaseInputPin. IsFlushing, método

El `IsFlushing` método consulta si el filtro se está vaciando actualmente.

## <a name="syntax"></a>Sintaxis


```C++
BOOL IsFlushing();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve la variable miembro [**CBaseInputPin:: m \_ bFlushing**](cbaseinputpin-m-bflushing.md) .
