# <a name="cimagepalettem_pdrawimage-member"></a>Miembro CImagePalette::m \_ pDrawImage

La `m_pDrawImage` variable miembro es un puntero al objeto [**CDrawImage**](cdrawimage.md) que dibuja la imagen de vídeo.

## <a name="syntax"></a>Sintaxis


```C++
CDrawImage *m_pDrawImage;
```
