# <a name="playcap-sample"></a>Ejemplo de PlayCap

## <a name="description"></a>Descripción

Aplicación de captura simple.

Esta aplicación crea una ventana de vista previa para el primer dispositivo de captura de vídeo que encuentra en el sistema del usuario (si lo hubiera). Muestra un ejemplo sencillo del uso de las interfaces [**ICaptureGraphBuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) e [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) para crear un gráfico de captura.

Para obtener una aplicación de captura de ejemplo que admita más características, vea [Ejemplo de AmCap.](amcap-sample.md)

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos DirectShow SDK, instale la versión más reciente de [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este ejemplo se instala en la siguiente ruta de acceso: *\[ SDK \] Root* Samples Multimedia DirectShow Capture \\ \\ \\ \\ \\ PlayCap.
