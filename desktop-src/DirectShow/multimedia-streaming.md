# <a name="multimedia-streaming"></a>Multimedia Streaming

> [!Note]  
> Estas API están en desuso. Las aplicaciones deben usar el [**filtro Sample Grabber**](sample-grabber-filter.md) o implementar un filtro personalizado para obtener datos de un DirectShow gráfico de filtros.

 

Streaming multimedia es un conjunto de API que permiten a una aplicación obtener datos de audio y vídeo de un gráfico de filtro DirectShow sin escribir un filtro de DirectShow personalizado.
