# <a name="avencmpvqscaletype-property"></a>Propiedad AVEncMPVQScaleType

Especifica si la escala del cuantificador es lineal o no lineal. Esta propiedad se aplica a los codificadores de vídeo MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPVQScaleType**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es miembro de la [**enumeración eAVEncMPVQScaleType.**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvqscaletype)
