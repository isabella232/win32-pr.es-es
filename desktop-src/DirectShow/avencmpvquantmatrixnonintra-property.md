# <a name="avencmpvquantmatrixnonintra-property"></a>AvEncMPVQuantMatrixNonIntra, propiedad

Especifica la matriz de cuantificación de luma para bloques de macro no dentro. Esta propiedad se aplica a los codificadores de vídeo MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPVQuantMatrixNonIntra**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es una cadena que contiene los coeficientes de 64 8 bits codificados como una cadena de valores hexadecimales (128 caracteres).
