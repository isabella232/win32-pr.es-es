# <a name="working-with-sources"></a>Trabajar con orígenes

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

Un objeto de origen representa un único flujo multimedia de un archivo de origen. Los orígenes proporcionan la materia sin procesar de un proyecto. Todo lo demás de la escala de tiempo existe para proporcionar estructura (pistas, composiciones y grupos) o para modificar orígenes (efectos y transiciones).

En este artículo se describen varios detalles relacionados con los orígenes de Microsoft® DirectShow® Editing Services (DES). Para obtener información sobre los pasos básicos necesarios para agregar un origen a la escala de tiempo, vea [Construir una escala de tiempo.](constructing-a-timeline.md)

Este artículo contiene las secciones siguientes.

-   [Establecer propiedades en orígenes](setting-properties-on-sources.md)
-   [Uso del detector de medios](using-the-media-detector.md)
-   [Uso del localizador de medios](using-the-media-locator.md)
-   [Otros objetos de origen](other-source-objects.md)
