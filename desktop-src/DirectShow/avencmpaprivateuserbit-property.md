# <a name="avencmpaprivateuserbit-property"></a>AvEncMPAPrivateUserBit, propiedad

Establece el valor del bit de usuario privado en la secuencia MPEG-1. Esta propiedad se aplica a los codificadores de audio MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPAPrivateUserBit**

## <a name="property-value"></a>Valor de propiedad

Esta propiedad puede tener el valor 0 o 1.
