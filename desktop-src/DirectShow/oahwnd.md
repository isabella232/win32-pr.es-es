# <a name="oahwnd"></a>OAHWND

El `OAHWND` tipo de datos define un identificador de ventana que pueden usar los clientes de automatización OLE, como Microsoft Visual Basic 6,0.


```C++
typedef LONG_PTR OAHWND;
```
