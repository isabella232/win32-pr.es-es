# <a name="amcap-sample"></a>Ejemplo de AmCap

## <a name="description"></a>Descripción

Aplicación de captura de vídeo.

En esta aplicación de ejemplo se muestran las siguientes tareas relacionadas con la captura de audio y vídeo:

-   Captura en un archivo
-   Versión preliminar en directo
-   Asignación del archivo de captura
-   Visualización de páginas de propiedades de dispositivo
-   Enumeración de dispositivos
-   Control stream

AMCap admite la entrada de secuencias del programa MPEG-2, por ejemplo, desde los tuners de TELEVISIÓN análogos que transmiten contenido MPEG-2. Se DirectShow descodificador MPEG-2 compatible con la configuración para descodificar las secuencias.

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos DirectShow SDK, instale la versión más reciente [de Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este ejemplo se instala en la siguiente ruta de acceso: \[ *SDK Root* \] \\ Samples Multimedia DirectShow \\ Capture \\ \\ \\ AmCap.
