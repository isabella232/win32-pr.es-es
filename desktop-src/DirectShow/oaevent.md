# <a name="oaevent"></a>OAEVENT

El **tipo de datos OAEVENT** define un identificador de eventos que pueden usar los clientes de OLE Automation, como Microsoft Visual Basic 6.0.


```C++
typedef LONG_PTR OAEVENT;
```
