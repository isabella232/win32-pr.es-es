# <a name="avenccommonpassstart-property"></a>Propiedad AVEncCommonPassStart

Inicia el primer paso de codificación.

Esta propiedad es de solo escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncCommonPassStart**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es el paso de codificación que se va a iniciar. Para obtener el intervalo de valores admitidos, consulte la propiedad [**AVEncCommonMultipassMode del**](avenccommonmultipassmode-property.md) codificador.
