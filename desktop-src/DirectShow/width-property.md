# <a name="width-property-directshow"></a>Propiedad Width (DirectShow)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `Width` propiedad establece o recupera el ancho del rectángulo.

``` syntax
[ iWidth = ] DVDRect.Width
```

## <a name="return-value"></a>Valor devuelto

Devuelve el ancho, en píxeles, del rectángulo como un entero.

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura y escritura sin ningún valor predeterminado.
