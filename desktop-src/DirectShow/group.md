# <a name="group-directshow"></a>Grupo (DirectShow)

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El objeto de grupo representa una composición de nivel superior. Cada grupo define una secuencia de vídeo o audio en el proyecto final. Para crear este objeto, llame al [**método IAMTimeline::CreateEmptyNode.**](iamtimeline-createemptynode.md)

El objeto de grupo expone las interfaces siguientes:

-   [**IAMTimelineComp**](iamtimelinecomp.md)
-   [**IAMTimelineGroup**](iamtimelinegroup.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
