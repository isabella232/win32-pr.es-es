# <a name="avencddsurrounddownmixlevel-property"></a>Propiedad AVEncDDSurroundDownMixLevel

Especifica el nivel de mezcla inferior Surround, en decibelios (dB), en una secuencia de audio Dolby Digital. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDSurroundDownMixLevel**
