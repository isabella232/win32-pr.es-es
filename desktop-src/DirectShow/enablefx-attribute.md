# <a name="enablefx-attribute"></a>atributo enablefx

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El `enablefx` atributo especifica si los efectos están habilitados. Si el valor es **TRUE**, el motor de representación representa los efectos. De lo contrario, no la tiene. El valor predeterminado es **TRUE.**

## <a name="possible-values"></a>Valores posibles

Los valores siguientes se definen como TRUE: y, Y, t, T, 1. Los valores siguientes se definen como FALSE: n, N, f, F, 0 (cero).

## <a name="applies-to"></a>Se aplica a

[**línea de tiempo**](timeline-element.md)
