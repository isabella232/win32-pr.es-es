# <a name="avencvideooutputchromasubsampling-property"></a>Propiedad AVEncVideoOutputChromaSubsampling

Especifica el siting del objeto para el vídeo codificado. La sima de los colores define las posiciones de las muestras de muestreo con respecto a las muestras de luma.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoOutputChromaSubsampling**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es un OR bit a bit de marcas de la [**enumeración eAVEncVideoChromaSubsampling.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideochromasubsampling)
