# <a name="cbaserendererisstreaming-method"></a>Método CBaseRenderer.IsStreaming

El `IsStreaming` método consulta si el filtro está transmitiendo datos.

## <a name="syntax"></a>Sintaxis


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve la [**marca CBaseRenderer::m \_ bStreaming.**](cbaserenderer-m-bstreaming.md)
