# <a name="cgenericlistgetheadposition-method"></a>Método CGenericList.GetHeadPosition

El `GetHeadPosition` método recupera la posición del primer elemento de la lista.

## <a name="syntax"></a>Sintaxis


```C++
POSITION GetHeadPosition();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve un valor POSITION o **NULL si** la lista está vacía.
