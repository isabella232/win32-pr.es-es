# <a name="ec_built"></a>EC \_ BUILT

El control de vídeo envía el evento **EC \_ BUILT** [cuando](/previous-versions/windows/desktop/mstv/video-control-overview) se ha creado completamente un gráfico. Este evento no se reenvía a las aplicaciones.
