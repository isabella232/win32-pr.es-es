# <a name="avencvideofieldswap-property"></a>Propiedad AVEncVideoFieldSwap

Invierte el orden de los campos entrelazados en el vídeo de origen.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoFieldSwap**

## <a name="remarks"></a>Observaciones

Si el valor es **VARIANT \_ TRUE,** el codificador intercambia el orden de campo.
