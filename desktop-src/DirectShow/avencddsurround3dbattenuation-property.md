# <a name="avencddsurround3dbattenuation-property"></a>Propiedad AVEncDDSurround3dBAttenuation

Especifica si los canales surround de una secuencia de audio Dolby Digital se atenuan 3 decibelios (dB) antes de la codificación. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDSurround3dBAttenuation**
