# <a name="avencvideoheaderminutes-property"></a>AvEncVideoHeaderMinutes, propiedad

Especifica el número de minuto inicial en el encabezado de grupo de imágenes (GOP).

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoHeaderMinutes**

## <a name="property-value"></a>Valor de propiedad

El intervalo válido de esta propiedad es de 0 a 59, ambos inclusive.
