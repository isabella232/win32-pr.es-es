# <a name="am_rate_simpleratechange-property"></a>Propiedad AM \_ RATE \_ SimpleRateChange

Esta propiedad se usa para enviar cambios de velocidad al descodificador. El tipo de datos de esta propiedad es una estructura **\_ SimpleRateChange** de AM, que proporciona la nueva velocidad de reproducción y el tiempo de presentación cuando la nueva velocidad entra en vigor.



| Etiqueta | Value |
|-------------------|-----------------------------------------------------|
| GUID del conjunto de propiedades | AM \_ KSPROPSETID \_ TSRateChange                       |
| Id. de propiedad       | AM \_ RATE \_ SimpleRateChange                          |
| Tipo de datos         | [**AM \_ SimpleRateChange**](/previous-versions/windows/desktop/api/Dvdmedia/ns-dvdmedia-am_simpleratechange) |
