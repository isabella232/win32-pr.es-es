# <a name="avencvideoforcesourcescantype-property"></a>Propiedad AVEncVideoForceSourceScanType

Especifica si los fotogramas de entrada son progresivas o entrelazadas.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoForceSourceScanType**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es miembro de la [**enumeración eAVEncVideoSourceScanType.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideosourcescantype)
