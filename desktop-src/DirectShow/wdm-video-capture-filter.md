# <a name="wdm-video-capture-filter"></a>Filtro de captura de vídeo de WDM

El filtro captura de vídeo de WDM controla los dispositivos de captura análogos que usan Modelo de controlador de Windows (WDM).

Este filtro es realmente un complemento KsProxy en modo kernel. Proporciona páginas de propiedades e interfaces COM para controladores WDM que controlan dispositivos de captura análogos (también denominados descodificadores de vídeo análogos). Una aplicación puede tratarla simplemente como un filtro. Para agregar este filtro a un gráfico de filtros, use el [enumerador de dispositivos del sistema](system-device-enumerator.md). Devuelve un moniker único para cada dispositivo que usa este complemento. Para obtener más información, [vea Enumerating Devices and Filters (Enumeración de dispositivos y filtros)](enumerating-devices-and-filters.md) y How Hardware Devices Participate in the Filter Graph (Cómo participan los dispositivos [de hardware en el gráfico de filtros).](how-hardware-devices-participate-in-the-filter-graph.md) Al igual que con cualquier filtro basado en KsProxy, el nombre descriptivo del filtro dependerá del controlador subyacente.

No todos los dispositivos que se exponen a través del filtro de captura de vídeo de WDM implementarán todas las interfaces que se enumeran a continuación. La aplicación puede usar **QueryInterface para** determinar qué interfaces son compatibles con un dispositivo determinado.
