> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

## <a name="syntax"></a>Sintaxis


```C++
HRESULT PasteXMLFile(
   IUnknown *pTimeline,
   double   dStart,
   BSTR     FileName
);
```
