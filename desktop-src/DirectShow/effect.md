# <a name="effect"></a>Efecto

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El objeto de efecto genera un efecto de audio o vídeo. Para crear este objeto, llame al [**método IAMTimeline::CreateEmptyNode.**](iamtimeline-createemptynode.md)

El objeto de efecto expone las interfaces siguientes:

-   [**IAMTimelineEffect**](iamtimelineeffect.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
-   **IPropertyPage**
