# <a name="cdeferredcommandgettime-method"></a>Método CDeferredCommand.GetTime

El `GetTime` método recupera la hora a la que se ejecutará el método.

## <a name="syntax"></a>Sintaxis


```C++
CRefTime GetTime();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve un [**objeto CRefTime**](creftime.md) que contiene una hora de referencia.
