# <a name="dmo-structures"></a>DMO Estructuras

En esta sección se describen las estructuras del objeto multimedia DirectX (DMO).



| Estructura                                                   | Descripción                                                 |
|-------------------------------------------------------------|-------------------------------------------------------------|
| [**\_DMO TIPO DE \_ MEDIO**](/previous-versions/windows/desktop/api/mediaobj/ns-mediaobj-dmo_media_type)                  | Describe el formato de los datos utilizados por una secuencia en un DMO. |
| [**\_DMO BÚFER DE \_ DATOS \_ DE SALIDA**](/previous-versions/windows/desktop/api/Mediaobj/ns-mediaobj-dmo_output_data_buffer) | Describe un búfer de salida utilizado por un DMO.                   |
| [**\_DMO PARTIAL \_ MEDIATYPE**](/previous-versions/windows/desktop/api/Dmoreg/ns-dmoreg-dmo_partial_mediatype)    | Describe parcialmente un tipo de medio utilizado por un DMO.             |
| [**SEGMENTO \_ DE SOBRE DE \_ MP**](/previous-versions/windows/desktop/api/Medparam/ns-medparam-mp_envelope_segment)        | Define un segmento de sobre utilizado por un parámetro multimedia.      |
| [**MP \_ PARAMINFO**](/previous-versions/windows/desktop/api/Medparam/ns-medparam-mp_paraminfo)                       | Contiene información sobre un parámetro multimedia.               |
