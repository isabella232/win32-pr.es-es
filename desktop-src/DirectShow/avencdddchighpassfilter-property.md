# <a name="avencdddchighpassfilter-property"></a>Propiedad AVEncDDCHighPassFilter

Especifica si se aplica un filtro de paso alto de bloqueo de controlador de dominio a una secuencia de audio Dolby Digital. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDCHighPassFilter**
