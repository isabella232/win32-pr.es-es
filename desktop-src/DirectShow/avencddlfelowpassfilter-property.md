# <a name="avencddlfelowpassfilter-property"></a>Propiedad AVEncDDLFELowPassFilter

Especifica si se aplica un filtro de paso bajo al canal de efecto de baja frecuencia (LFE) en una secuencia de audio Dolby Digital. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDLFELowPassFilter**
