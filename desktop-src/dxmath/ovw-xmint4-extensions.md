# <a name="xmint4-extensions"></a>Extensiones de XMINT4

[**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) describe una estructura que contiene un vector 4D que consta de cuatro valores enteros.

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
