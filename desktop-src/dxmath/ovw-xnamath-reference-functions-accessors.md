# <a name="directxmath-library-vector-accessor-functions"></a>Funciones del accessor de vectores de directXMath Library

Enumera las funciones del accessor vectorial proporcionadas por la biblioteca DirectXMath.

Los accessors de vector directXMath permiten a los desarrolladores escribir código que obtiene y establece componentes individuales de un vector de una manera portátil y óptima.
