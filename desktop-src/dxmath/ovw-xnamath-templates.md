# <a name="directxmath-library-template-functions"></a>Funciones de plantilla de la biblioteca DirectXMath

Enumera las funciones de plantilla proporcionadas por DirectXMath.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                  | Descripción                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMMax**](xmmax-template.md)<br/>                             | Compara dos instancias de tipo de datos numéricos, o dos instancias de un objeto que admite una sobrecarga de <, y devuelve la más grande de las dos instancias. El tipo de datos de los argumentos y el valor devuelto es el mismo.<br/>  |
| [**XMMin**](xmmin-template.md)<br/>                             | Compara dos instancias de tipo de datos numérico, o dos instancias de un objeto que admite una sobrecarga de <, y devuelve la más pequeña de las dos instancias. El tipo de datos de los argumentos y el valor devuelto es el mismo.<br/> |
| [**XMVectorPermute**](xmvectorpermute-template.md)<br/>         | Permuta los componentes de dos vectores para crear un nuevo vector.<br/>                                                                                                                                                                 |
| [**XMVectorRotateLeft**](xmvectorrotateleft-template.md)<br/>   | Gira el vector a la izquierda por un número determinado de elementos de 32 bits.<br/>                                                                                                                                                                  |
| [**XMVectorRotateRight**](xmvectorrotateright-template.md)<br/> | Gira el vector a la derecha en un número determinado de elementos de 32 bits.<br/>                                                                                                                                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft-template.md)<br/>     | Desplaza un vector a la izquierda por un número determinado de elementos de 32 bits, rellenando los elementos desocupados con elementos de un segundo vector.<br/>                                                                                                    |
| [**XMVectorSwzzle**](xmvectorswizzle-template.md)<br/>         | Desliza un vector.<br/>                                                                                                                                                                                                             |
| [**XMVectorInsert**](xmvectorinsert-template.md)<br/>           | Gira un vector a la izquierda por un número determinado de componentes de 32 bits e inserta los elementos seleccionados de ese resultado en otro vector.<br/>                                                                                                  |
