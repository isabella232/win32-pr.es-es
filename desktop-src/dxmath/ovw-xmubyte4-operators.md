# <a name="xmubyte4-operators"></a>Operadores XMUBYTE4

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección

|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmubyte4-operator-assign)<br/> | Asigna los datos del componente vectorial de una instancia de `XMUBYTE4` a la instancia actual de `XMUBYTE4` . <br/> |
