# <a name="xmxdecn4-extensions"></a>Extensiones XMXDECN4

[**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) define un vector 4D para almacenar valores con signo y normalizados como componentes x-, y-y- con signo de 10 bits y valores normalizados sin signo como un componente w de 2 bits sin signo.

> [!Note]  
> Las características enumeradas aquí solo están disponibles al desarrollar con C++.
