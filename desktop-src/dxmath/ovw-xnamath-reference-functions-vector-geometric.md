# <a name="geometric-vector-functions"></a>Funciones vectoriales geométricas

Enumera las funciones vectoriales geométricas.

## <a name="in-this-section"></a>En esta sección



| Tema                                                           | Descripción                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorBaryCentric**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentric)<br/>   | Devuelve un punto en coordenadas baricéntricas, utilizando los vectores de posición especificados.<br/>                                                                                                                     |
| [**XMVectorBaryCentricV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentricv)<br/> | Devuelve un punto en coordenadas baricéntricas, utilizando los vectores de posición especificados.<br/>                                                                                                                     |
| [**XMVectorCatmullRom**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullrom)<br/>     | Realiza una Catmull-Rom interpolación mediante los vectores de posición especificados.<br/>                                                                                                                           |
| [**XMVectorCatmullRomV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullromv)<br/>   | Realiza una Catmull-Rom interpolación mediante los vectores de posición especificados.<br/>                                                                                                                           |
| [**XMVectorHermite**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermite)<br/>           | Realiza una interpolación spline de Hermite, utilizando los vectores especificados.<br/>                                                                                                                                 |
| [**XMVectorHermiteV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermitev)<br/>         | Realiza una interpolación spline de Hermite, utilizando los vectores especificados.<br/>                                                                                                                                 |
| [**XMVectorInBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinbounds)<br/>         | Comprueba si los componentes de un vector determinado están dentro de los límites establecidos.<br/>                                                                                                                                 |
| [**XMVectorInBoundsR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinboundsr)<br/>       | Comprueba si los componentes de un vector determinado están dentro de determinados límites y establece un valor de comparación que se puede examinar mediante funciones como [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVectorLerp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerp)<br/>                 | Realiza una interpolación lineal entre dos vectores.<br/>                                                                                                                                                  |
| [**XMVectorLerpV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerpv)<br/>               | Realiza una interpolación lineal entre dos vectores.<br/>                                                                                                                                                  |
