# <a name="directxmath-3d-vector-transformation-functions"></a>Funciones de transformación de vectores 3D de DirectXMath

Enumera las funciones de transformación de vectores 3D.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                               | Descripción                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Gira un vector 3D usando el inverso de un cuaternión.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Project un vector 3D desde el espacio del objeto al espacio de pantalla.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Proyecta una secuencia de vectores 3D desde el espacio del objeto en el espacio de la pantalla.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Gira un vector 3D mediante un cuaternión.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Transforma un vector 3D mediante una matriz.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Transforma un vector 3D mediante una matriz determinada, proyectando el resultado de nuevo en w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Transforma una secuencia de vectores 3D por una matriz determinada, proyectando los vectores resultantes de forma que sus coordenadas w sean iguales a 1,0.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Transforma el vector 3D normal por la matriz especificada.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Transforma una secuencia de vectores normales 3D por una matriz determinada.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Transforma una secuencia de vectores 3D por una matriz determinada.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Proyecta un vector 3D desde el espacio de pantalla al espacio de objetos.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Transforma una secuencia de vectores 3D del espacio de la pantalla al espacio del objeto.<br/>                                                                  |
