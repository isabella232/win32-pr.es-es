# <a name="xmxdec4-operators"></a>Operadores XMXDEC4

La [**estructura XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                            | Descripción                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la instancia en un formato `XMXDEC4` empaquetado. <br/> |
| [**operator =**](xmxdec4-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMXDEC4` . <br/>                                       |
