# <a name="about-communications-resources"></a>Acerca de los recursos de comunicaciones

Las funciones de entrada y salida (E/S) de archivo [**(CreateFile**](/windows/desktop/api/fileapi/nf-fileapi-createfilea), [**CloseHandle**](/windows/desktop/api/handleapi/nf-handleapi-closehandle), [**ReadFile,**](/windows/desktop/api/fileapi/nf-fileapi-readfile) [**ReadFileEx,**](/windows/desktop/api/fileapi/nf-fileapi-readfileex) [**WriteFile**](/windows/desktop/api/fileapi/nf-fileapi-writefile)y [**WriteFileEx)**](/windows/desktop/api/fileapi/nf-fileapi-writefileex)proporcionan la interfaz básica para abrir y cerrar un identificador de recursos de comunicaciones y para realizar operaciones de lectura y escritura. Las funciones de comunicaciones proporcionan acceso a los recursos de comunicaciones. En esta introducción se describe el uso de funciones de E/S y comunicaciones de archivos.
