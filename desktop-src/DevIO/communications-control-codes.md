# <a name="communications-control-codes"></a>Códigos de control de comunicaciones

El código de control siguiente se usa con la entrada y salida del dispositivo.



| Código de control                                                        | Operación                                                                           |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**IOCTL \_ SERIAL \_ LSRMST \_ INSERT**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_serial_lsrmst_insert) | Habilita o deshabilita la colocación de datos de estado de línea y módem en el flujo de datos. |
