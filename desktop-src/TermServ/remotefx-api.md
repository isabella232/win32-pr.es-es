# <a name="remotefx-media-redirection-api"></a>RemoteFX API de redireccionamiento multimedia

La REMOTEFX Media Redirection API se usa en una sesión de Escritorio remoto para identificar las áreas del servidor que muestran contenido de cambio rápido, como vídeo. Este contenido se puede codificar en vídeo y enviarse al cliente en formato codificado.
