# <a name="remote-desktop-virtualization-enumerations"></a>Escritorio remoto enumeraciones de virtualización

La API Escritorio remoto Virtualization admite los siguientes tipos de enumeración.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**TIPO \_ DE CAMBIO DE \_ DESTINO**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_change_type)
</dt> <dd>

Especifica el tipo de cambio que se produjo en un destino.

</dd> <dt>

[**ESTADO DE \_ DESTINO**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_state)
</dt> <dd>

Indica el estado de un destino.

</dd> <dt>

[**TIPO DE \_ DESTINO**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_type)
</dt> <dd>

Indica si un destino pertenece a un grupo o una granja.

</dd> <dt>

[**ESTADO DE \_ TSSESSION**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssession_state)
</dt> <dd>

Indica el estado de una sesión.

</dd> <dt>

[**TSSD \_ AddrV46Type**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssd_addrv46type)
</dt> <dd>

Indica el tipo de dirección IP.

</dd> <dt>

[**TS \_ SB \_ SORT \_ BY**](/windows/desktop/api/sbtsv/ne-sbtsv-ts_sb_sort_by)
</dt> <dd>

Especifica el criterio de ordenación. Se usa como parámetro en el [**método EnumerateTargets.**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets)

</dd> </dl>
