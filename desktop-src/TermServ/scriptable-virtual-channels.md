# <a name="scriptable-virtual-channels"></a>Canales virtuales que se pueden incluir en scripts

Para obtener información sobre el uso de canales virtuales que pueden incluirse en scripts, vea Implementar canales virtuales que se pueden [incluir en scripts mediante Conexión web a Escritorio remoto](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md).

Para obtener información sobre los canales virtuales, [vea Servicios de Escritorio remoto Virtual Channels](terminal-services-virtual-channels.md).

**Windows XP:** Windows XP incluye Conexión web a Escritorio remoto, que es una versión actualizada de Terminal Services Cliente avanzado (TSAC). Conexión web a Escritorio remoto admite [Protocolo de escritorio remoto](remote-desktop-protocol.md) (RDP) 5.1 y es compatible con versiones anteriores de RDP 5.0.
