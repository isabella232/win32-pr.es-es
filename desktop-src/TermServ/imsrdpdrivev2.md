# <a name="imsrdpdrivev2-interface"></a>Interfaz IMsRdpDriveV2

Contiene información sobre un objeto de unidad. Se trata de una mejora de la [**interfaz IMsRdpDrive.**](imsrdpdrive.md)

## <a name="members"></a>Miembros

La **interfaz IMsRdpDriveV2** hereda de [**IMsRdpDrive.**](imsrdpdrive.md) **IMsRdpDriveV2** también tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

La **interfaz IMsRdpDriveV2** tiene estas propiedades.



| Propiedad                                                              | Tipo de acceso          | Descripción                                                |
|:----------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------|
| [**DriveLetterIndex**](imsrdpdrivev2-driveletterindex.md)<br/> | Solo lectura<br/> | Contiene el índice de la letra de la unidad.<br/> |
