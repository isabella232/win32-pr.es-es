# <a name="iremotedesktopclienteventsondialogdismissed-method"></a>IRemoteDesktopClientEvents:: OnDialogDismissed (método)

Se llama después de que se descarte un cuadro de diálogo que muestra el control de cliente.

## <a name="syntax"></a>Sintaxis


```C++
void OnDialogDismissed();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
