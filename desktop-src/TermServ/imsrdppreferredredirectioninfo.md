# <a name="imsrdppreferredredirectioninfo-interface"></a>Interfaz IMsRdpPreferredRedirectionInfo

Proporciona una propiedad para controlar mediante un servidor de redirección.

## <a name="members"></a>Miembros

La interfaz **IMsRdpPreferredRedirectionInfo** hereda de la interfaz [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IMsRdpPreferredRedirectionInfo** también tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

La interfaz **IMsRdpPreferredRedirectionInfo** tiene estas propiedades.



| Propiedad                                                                                               | Tipo de acceso           | Descripción                                                          |
|:-------------------------------------------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------|
| [**UseRedirectionServerName**](imsrdppreferredredirectioninfo-useredirectionservername.md)<br/> | Lectura/escritura<br/> | Obtiene y establece si se utiliza el nombre del servidor de redirección.<br/> |
