# <a name="imsrdpclientnonscriptable3devicecollection-property"></a>IMsRdpClientNonScriptable3::D eviceCollection

Recupera la colección de objetos Plug and Play (PnP) que se redirigirán.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_DeviceCollection(
  [out] IMSRdpDeviceCollection **ppDeviceCollection
);
```



## <a name="property-value"></a>Valor de propiedad

Recupera la colección de objetos de dispositivo PnP de tipo [**IMSRdpDevice**](imsrdpdevice.md).
