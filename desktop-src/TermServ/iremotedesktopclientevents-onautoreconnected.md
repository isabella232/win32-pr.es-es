# <a name="iremotedesktopclienteventsonautoreconnected-method"></a>IRemoteDesktopClientEvents::OnAutoReconnected (método)

Se llama cuando el control de cliente se ha vuelto a conectar automáticamente a una sesión remota.

## <a name="syntax"></a>Sintaxis


```C++
void OnAutoReconnected();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
