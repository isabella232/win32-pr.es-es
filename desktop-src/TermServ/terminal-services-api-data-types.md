# <a name="remote-desktop-services-api-data-types"></a>Servicios de Escritorio remoto tipos de datos de API

Servicios de Escritorio remoto define los siguientes tipos de datos.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**IDENTIFICADOR \_ DE TLS**](tls-handle.md)
</dt> <dd>

Representa un identificador para un Escritorio remoto de licencias.

</dd> <dt>

[**WTSLISTENERNAME**](wtslistenername.md)
</dt> <dd>

Representa el nombre de un agente Servicios de Escritorio remoto escuchas en un servidor Escritorio remoto de sesión de escritorio remoto ( Host de sesión de Escritorio remoto).

</dd> </dl>
