# <a name="imsrdpclientadvancedsettingsinputeventsatonce-property"></a>IMsRdpClientAdvancedSettings:: InputEventsAtOnce (propiedad)

Esta propiedad no es compatible.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT put_InputEventsAtOnce(
  [in]  LONG inputEventsAtOnce
);

HRESULT get_InputEventsAtOnce(
  [out] LONG *pinputEventsAtOnce
);
```



## <a name="property-value"></a>Valor de propiedad

Nuevo número de eventos de entrada. El valor predeterminado es 10.

## <a name="error-codes"></a>Códigos de error

Devuelve **S \_ false**.
