# <a name="imsrdpclient5remoteprogram-property"></a>Propiedad IMsRdpClient5::RemoteProgram

Recupera un objeto que admite la [**interfaz ITSRemoteProgram.**](itsremoteprogram.md)

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_RemoteProgram(
  [out] ITSRemoteProgram **ppRemoteProgram
);
```



## <a name="property-value"></a>Valor de propiedad

Puntero [**de interfaz ITSRemoteProgram.**](itsremoteprogram.md)
