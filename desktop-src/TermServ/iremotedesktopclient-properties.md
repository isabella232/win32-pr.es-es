# <a name="iremotedesktopclient-properties"></a>Propiedades de IRemoteDesktopClient

La [**interfaz IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient) expone las siguientes propiedades.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Propiedad Actions**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_actions)
</dt> <dd>

Recupera el objeto actions para el cliente Protocolo de escritorio remoto de contenedor de aplicaciones (RDP).

</dd> <dt>

[**Configuración propiedad**](iremotedesktopclient-settings.md)
</dt> <dd>

Recupera el objeto de configuración para el cliente Protocolo de escritorio remoto de contenedor de aplicaciones (RDP).

</dd> <dt>

[**TouchPointer, propiedad**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_touchpointer)
</dt> <dd>

Contiene el [**objeto RemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) para el Protocolo de escritorio remoto contenedor de aplicaciones (RDP).

</dd> </dl>
