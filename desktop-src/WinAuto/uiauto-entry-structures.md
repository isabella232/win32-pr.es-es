# <a name="ui-automation-structures"></a>Automatización de la interfaz de usuario estructuras

En esta sección se describen las estructuras usadas por Automatización de la interfaz de usuario cliente y Automatización de la interfaz de usuario proveedor para Microsoft Win32.

## <a name="in-this-section"></a>En esta sección



| Estructura                                                                            | Descripción                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/desktop/api/UIAutomationClient/ns-uiautomationclient-extendedproperty)<br/>                       | Contiene información sobre una propiedad extendida. <br/>                                  |
| [**UIAutomationEventInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo)<br/>       | Contiene información sobre un evento personalizado.<br/>                                         |
| [**UIAutomationMethodInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo)<br/>     | Contiene información sobre un método que es compatible con un patrón de control personalizado.<br/> |
| [**UIAutomationParameter**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationparameter)<br/>       | Contiene información sobre un parámetro de un patrón de control personalizado.<br/>                |
| [**UIAutomationPatternInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo)<br/>   | Contiene información sobre un patrón de control personalizado.<br/>                               |
| [**UIAutomationPropertyInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo)<br/> | Contiene información sobre una propiedad personalizada.<br/>                                      |
| [**UiaChangeInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiachangeinfo)<br/>                                    | Contiene datos sobre un Automatización de la interfaz de usuario cambio que se produjo.<br/>                          |
| [**UiaPoint**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiapoint)<br/>                                 | Contiene las coordenadas de un punto. <br/>                                              |
| [**UiaRect**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiarect)<br/>                                   | Contiene la posición y el tamaño de un rectángulo, en coordenadas de pantalla.<br/>              |
