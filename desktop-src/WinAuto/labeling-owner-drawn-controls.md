# <a name="labeling-owner-drawn-controls"></a>Etiquetado de Owner-Drawn controles

Los desarrolladores de aplicaciones deben definir una etiqueta, también conocida como texto de título, para todos los controles dibujados por el propietario, incluso si la etiqueta no está visible en la pantalla. La etiqueta debe ir inmediatamente antes del control que está etiquetando y debe estar en el siguiente elemento en el orden de tabulación.

Por ejemplo, si define una etiqueta al crear un botón dibujado por el propietario en el que se muestra un gráfico en lugar del texto de título normal, las aplicaciones cliente obtienen la etiqueta y la usan para identificar el control.
