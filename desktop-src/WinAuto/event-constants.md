# <a name="event-constants-winuserh"></a>Constantes de evento (Winuser.h)

En este tema se describen los eventos generados por el sistema operativo y por las aplicaciones de servidor. Las constantes se enumeran en orden alfabético.

Antes de usar estos eventos, las aplicaciones cliente deben usar [Accessible Event Watcher](accessible-event-watcher.md) para comprobar que los elementos de la interfaz de usuario usan estos eventos.

Para obtener más información sobre los eventos en general, vea What Are WinEvents? (¿Qué son [los eventos WinEvent?)](what-are-winevents.md) y System Level and Object Level Events (Eventos de nivel de sistema y de nivel de [objeto).](system-level-and-object-level-events.md) Para obtener más información sobre los eventos enviados por el sistema, vea [Apéndice A: Referencia Interfaz de usuario elementos admitidos](appendix-a--supported-user-interface-elements-reference.md).
