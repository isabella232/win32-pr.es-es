# <a name="dynamic-annotation"></a>Anotación dinámica

Anteriormente, la anotación dinámica solo estaba disponible para Microsoft Active Accessibility y clientes. Ahora la técnica está disponible para todos los Automatización de la interfaz de usuario, así como para algunas implementaciones de servidor MSAA Automatización de la interfaz de usuario proveedor. Además, las anotaciones realizadas en controles personalizados mediante anotaciones dinámicas (las tres técnicas) son accesibles para Automatización de la interfaz de usuario clientes.

Para más información, consulte [API de anotación dinámica.](dynamic-annotation-api.md)
