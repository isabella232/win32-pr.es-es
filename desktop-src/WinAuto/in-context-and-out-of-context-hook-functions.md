# <a name="in-context-and-out-of-context-hook-functions"></a>In-Context y funciones de enlace fuera de contexto

Al registrar una función de enlace con [**SetWinEventHook,**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)los clientes especifican si la función de enlace está en contexto *o* fuera *de contexto.* Estos términos describen la ubicación de memoria de la función de enlace en relación con el espacio de direcciones del servidor.
