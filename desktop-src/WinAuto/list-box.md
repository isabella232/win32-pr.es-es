# <a name="list-box-msaa-ui-element-reference"></a>Cuadro de lista (Referencia del elemento de interfaz de usuario de MSAA)

> [!Note]  
> En este tema se describen **los objetos List Box** para fines de referencia de elementos de la interfaz de usuario de MSAA. Aquí no se describe **cómo crear objetos de Cuadro** de lista en varios marcos de interfaz de usuario. Consulte la documentación de referencia de API para el marco de interfaz de usuario que está usando.

 

Los cuadros de lista muestran una lista desde la que un usuario puede seleccionar uno o varios elementos. Se crea un cuadro de lista con una barra de desplazamiento vertical, una barra de control horizontal o ambas.

El nombre de clase de ventana de un cuadro de lista es "LISTBOX".

## <a name="iaccessible-methods"></a>Métodos IAccessible

Los cuadros de lista admiten los [**siguientes métodos IAccessible:**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)



| Método                                                                    | Comentarios                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | El propio objeto de cuadro de lista no tiene una **propiedad DefaultAction;** sin embargo, [**accDoDefaultAction hace**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) doble clic en los elementos del cuadro de lista.                                                                                    |
| [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 | El [**método accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest) devuelve cero si el punto probado está dentro del cuadro de lista, pero no en un elemento del cuadro de lista. Si el punto probado está en un elemento del cuadro de lista, **accHitTest** devuelve el índice del elemento de cuadro de lista. |
| [**accLocation**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                            |
| [**accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)               |                                                                                                                                                                                                                                                                            |
| [**accSelect**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)                   |                                                                                                                                                                                                                                                                            |
