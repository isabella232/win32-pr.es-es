# <a name="unexpectedtesterror"></a>UnexpectedTestError

## <a name="text"></a>Texto

Excepción :( {0} )

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

AccChecker ha producido una excepción no relacionada con las comprobaciones. Para notificar el problema, vea el cuadro de diálogo AccChecker **About (Información** de AccChecker) para obtener información de contacto.
