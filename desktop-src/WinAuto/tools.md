# <a name="tools"></a>Herramientas

Puede usar las siguientes herramientas para probar Microsoft Active Accessibility implementaciones. Para obtener más información sobre las herramientas de prueba, vea [Herramientas de prueba.](testing-tools.md)



| Herramienta                                                     | Descripción                                                                                                                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Accessible Event Watcher](accessible-event-watcher.md) | Registra [WinEvents](winevents-infrastructure.md) a medida que se producen dentro del sistema.                                                                                                         |
| [Inspeccionar](inspect-objects.md)                           | Identifica problemas con la implementación de propiedades y métodos en objetos accesibles de Microsoft Active Accessibility y en elementos de automatización de Microsoft Automatización de la interfaz de usuario. |
