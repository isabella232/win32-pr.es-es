# <a name="tree-view-control-msaa-ui-element-reference"></a>Control Vista de árbol (referencia de elemento de la interfaz de usuario de MSAA)

> [!Note]  
> En este tema se describen **los objetos de control de** vista de árbol para fines de referencia de elementos de la interfaz de usuario de MSAA. Aquí no se describe **cómo crear objetos de control** de vista de árbol en varios marcos de interfaz de usuario. Consulte la documentación de referencia de API para el marco de interfaz de usuario que está usando.

 

Un control de vista de árbol muestra una lista jerárquica de elementos mediante líneas para conectar elementos relacionados en una jerarquía. Cada elemento consta de una etiqueta y un mapa de bits opcional. Windows El Explorador usa un control de vista de árbol para mostrar directorios.

El nombre de clase de ventana para un control de vista de árbol es WC TREEVIEW, que se define como \_ "SysTreeView" en Commctrl.h.

## <a name="notes"></a>Notas

La **propiedad accLocation** de los elementos de vista de árbol devuelve la ubicación solo del texto del elemento. Las coordenadas devueltas por **accLocation para** los elementos de vista de árbol no incluyen el icono situado a la izquierda del texto.
