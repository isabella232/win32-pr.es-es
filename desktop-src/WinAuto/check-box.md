# <a name="check-box-msaa-ui-element-reference"></a>Casilla (Referencia del elemento de la interfaz de usuario de MSAA)

> [!Note]  
> En este tema se describen **los objetos Check Box** para fines de referencia de elementos de la interfaz de usuario de MSAA. Aquí no se **describe cómo crear objetos Check Box** en varios marcos de interfaz de usuario. Consulte la documentación de referencia de API para el marco de interfaz de usuario que está usando.

 

Las casillas se usan para habilitar o deshabilitar una o varias características u opciones de un conjunto, normalmente dentro de un cuadro de diálogo. Normalmente, una casilla contiene una caja pequeña con texto adyacente. Cuando se selecciona una opción, aparece una marca de verificación en el cuadro.

El nombre de clase de ventana de una casilla es "BUTTON".
