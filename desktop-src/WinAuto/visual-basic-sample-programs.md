# <a name="visual-basic-sample-programs"></a>Visual Basic programas de ejemplo

El libro *Advanced microsoft Visual Basic 6,0, 2ª edición* (Mandelbrot Set International, Ltd.) incluye un capítulo en el que se explica Microsoft Active Accessibility. El capítulo proporciona código de ejemplo que muestra cómo usar los elementos clave de la API de Microsoft Active Accessibility. (Microsoft Press, ISBN 1-57231-893-7) (Es posible que este libro no esté disponible en algunos idiomas y países).
