# <a name="active-accessibility-objects"></a>Active Accessibility objetos

En Microsoft Active Accessibility terminología, hay objetos *accesibles y* *elementos simples*. Aunque la mayoría de las aplicaciones contienen ambos, los objetos accesibles son más comunes que los elementos simples. En esta sección se definen y analizan los objetos accesibles y los elementos simples, y se proporcionan ejemplos de interfaz de usuario adecuados.
