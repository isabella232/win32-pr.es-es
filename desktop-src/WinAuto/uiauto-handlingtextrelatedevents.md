# <a name="handling-text-related-events"></a>Controlar eventos relacionados con texto

En este tema se describen los eventos que los controles basados en texto provoca cuando se cambia, selecciona o deselecciona el contenido.


Los controles basados en texto genera un [**evento \_ \_ TextChangedEventId**](uiauto-event-ids.md) de texto UIA cada vez que cambia el contenido textual. Genera el evento [**\_ \_ TextSelectionChangedEventId de UIA**](uiauto-event-ids.md) cada vez que se selecciona o anula la selección del contenido textual. Para obtener más información, [vea Automatización de la interfaz de usuario información](uiauto-eventsoverview.md) general sobre eventos y suscripción a Automatización de la interfaz de usuario [eventos](uiauto-eventsforclients.md).
