# <a name="custom-properties-events-and-control-patterns"></a>Propiedades personalizadas, eventos y patrones de control

Microsoft Automatización de la interfaz de usuario API especifica un conjunto básico predefinido de propiedades, patrones de control y eventos. Sin embargo, las aplicaciones no se limitan al uso de estas especificaciones predefinidas. Las Automatización de la interfaz de usuario extensibilidad permiten a terceros introducir propiedades, eventos y patrones de control personalizados y acordados mutuamente para admitir nuevos elementos de la interfaz de usuario y escenarios de aplicación. Automatización de la interfaz de usuario proveedores y clientes pueden empezar a usar las propiedades personalizadas, los eventos y los patrones de control inmediatamente sin necesidad de actualizar el marco de trabajo de automatización de la interfaz de usuario principal.
