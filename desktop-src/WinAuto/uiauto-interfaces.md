# <a name="interfaces-for-providers"></a>Interfaces para proveedores

En esta sección se describen las interfaces básicas implementadas por Automatización de la interfaz de usuario para aplicaciones De Microsoft Win32.

## <a name="in-this-section"></a>En esta sección



| Interfaz                                                                                                    | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessibleEx**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessibleex)<br/>                                                     | Expone métodos a los que Llama Microsoft Automatización de la interfaz de usuario para recuperar información adicional sobre un control que admite Microsoft Active Accessibility.<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAccessibleHostingElementProviders**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessiblehostingelementproviders)<br/>                  | Un Microsoft Active Accessibility implementa esta interfaz cuando el objeto es la raíz de un árbol de accesibilidad que incluye controles de Microsoft ActiveX sin ventanas que implementan Automatización de la interfaz de usuario. Dado que Microsoft Active Accessibility y Automatización de la interfaz de usuario usan interfaces diferentes, esta interfaz permite a un cliente detectar la lista de controles de ActiveX sin ventanas hospedados que admiten Automatización de la interfaz de usuario en caso de que el cliente necesite tratarlos de forma diferente.<br/> |
| [**IProxyProviderWinEventHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventhandler)<br/>                     | Expone un método implementado por proveedores de proxy para controlar WinEvents.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProxyProviderWinEventSink**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventsink)<br/>                           | Expone los métodos utilizados por los proveedores de proxy para generar eventos.<br/>                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IRawElementProviderAdviseEvents**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovideradviseevents)<br/>                 | Expone métodos a los que se llama para notificar al elemento raíz de un fragmento cuando una aplicación cliente de Automatización de la interfaz de usuario comienza o termina de escuchar eventos en ese fragmento.<br/>                                                                                                                                                                                                                                                                                                |
| [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment)<br/>                         | Expone métodos y propiedades en elementos de la interfaz de usuario que forman parte de una estructura de más de un nivel de profundidad, como un cuadro de lista o un elemento de lista. Implementado por Automatización de la interfaz de usuario proveedor.<br/>                                                                                                                                                                                                                                                                                          |
| [**IRawElementProviderFragmentRoot**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragmentroot)<br/>                 | Expone métodos y propiedades en el elemento raíz de un fragmento.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IRawElementProviderHostingAccessibles**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderhostingaccessibles)<br/> | Esta interfaz la implementa un proveedor Automatización de la interfaz de usuario cuando el proveedor es la raíz de un árbol de accesibilidad que incluye controles sin ventanas que admiten Microsoft Active Accessibility. Dado Automatización de la interfaz de usuario y Microsoft Active Accessibility usan interfaces diferentes, esta interfaz permite a un cliente detectar la lista de controles Microsoft Active Accessibility hospedados en caso de que necesite tratarlos de forma diferente.<br/>                                 |
| [**IRawElementProviderHwndOverride**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderhwndoverride)<br/>                 | Expone un método que permite cambiar la posición de los elementos basados en ventanas dentro del árbol de Automatización de la interfaz de usuario fragmento.<br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple)<br/>                             | Define métodos y propiedades que exponen elementos de interfaz de usuario simples.<br/>                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2)<br/>                           | Extiende la interfaz [**IRawElementProviderSimple para**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple) habilitar la invocación mediante programación de menús contextuales.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple3**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple3)<br/>                           | Extiende la interfaz [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2) para habilitar la recuperación de metadatos sobre cómo la tecnología accesible debe decir el tipo de contenido preferido.<br/>                                                                                                                                                                                                                                                                    |
| [**IRawElementProviderWindowlessSite**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderwindowlesssite)<br/>         | Un ActiveX de control implementa esta interfaz para permitir que un control Automatización de la interfaz de usuario habilitado para ActiveX para expresar su accesibilidad. Esta interfaz permite que el contenedor de controles proporcione un puntero [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment) para los elementos primarios o relacionados del control ActiveX sin ventana y proporcione un identificador de tiempo de ejecución que sea único para el sitio de control.<br/>                                                           |
