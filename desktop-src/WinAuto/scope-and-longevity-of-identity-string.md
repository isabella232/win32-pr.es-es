# <a name="scope-and-longevity-of-identity-string"></a>Ámbito y longevidad de la cadena de identidad

Una cadena de identidad identifica una instancia determinada de un elemento de interfaz de usuario. Varias instancias de un elemento accesible tendrán cadenas de identidad diferentes; por lo tanto, no suele ser útil conservar una cadena de identidad.
