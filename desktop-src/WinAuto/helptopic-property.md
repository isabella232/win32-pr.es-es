# <a name="helptopic-property"></a>Propiedad HelpTopic

La **propiedad HelpTopic** obtiene los detalles usados con la función [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) de Microsoft Win32 para mostrar un tema de Ayuda asociado al objeto . No todos los objetos admiten esta propiedad.

La **propiedad HelpTopic** se recupera mediante una llamada a [**IAccessible::get \_ accHelpTopic**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic).
