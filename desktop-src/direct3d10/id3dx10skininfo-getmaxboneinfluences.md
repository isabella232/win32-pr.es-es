# <a name="id3dx10skininfogetmaxboneinfluences-method"></a>Método ID3DX10SkinInfo::GetMaxMaxMaxInfluences

Obtenga el número de vértices que puede influir al máximo un tómelo.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetMaxBoneInfluences();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

El número de vértices que puede influir al máximo un pórtico.
