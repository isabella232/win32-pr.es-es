# <a name="d3dxplane-structure-d3dx10mathh"></a>Estructura D3DXPLANE (D3DX10Math.h)

Describe un plano.

## <a name="syntax"></a>Sintaxis


```C++
typedef struct D3DXPLANE {
  FLOAT a;
  FLOAT b;
  FLOAT c;
  FLOAT d;
} D3DXPLANE, *LPD3DXPLANE;
```
