# <a name="ui_pkey_minvalue"></a>UI \_ PKEY \_ MinValue

Identifica la propiedad \_ MinValue de ui PKEY. \_

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Observaciones

Una aplicación usa ui PKEY MinValue para consultar \_ el valor mínimo del control \_ [Spinner.](windowsribbon-controls-spinner.md)
