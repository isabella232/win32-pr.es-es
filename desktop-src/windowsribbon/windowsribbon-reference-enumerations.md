# <a name="constants-and-enumerations"></a>Constantes y enumeraciones

Documentación de referencia para las constantes y enumeraciones Windows marco de la cinta de opciones.

### <a name="constants-and-enumerations"></a>Constantes y enumeraciones




| Constante | Descripción | 
|----------|-------------|
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-all-commands"><strong>UI_ALL_COMMANDS</strong></a><br /> | Especifica una constante que identifica la colección de comandos declarada en el archivo de recursos de marcado.<br /> | 
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-collection-invalidindex"><strong>UI_COLLECTION_INVALIDINDEX</strong></a><br /> | Especifica una constante que identifica un índice no válido en una colección.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_collectionchange"><strong>UI_COLLECTIONCHANGE</strong></a><br /> | Especifica valores que identifican los tipos de cambios que se pueden realizar en una colección.<br /> | 
| <a href="/previous-versions/windows/desktop/legacy/hh448736(v=vs.85)"><strong>UI_COMMANDCATEGORY</strong></a><br /> | Especifica valores que indican si un comando se definió en tiempo de compilación o en tiempo de ejecución.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype"><strong>UI_COMMANDTYPE</strong></a><br /> | Especifica valores que identifican el tipo de comando asociado a un control Ribbon.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_contextavailability"><strong>UI_CONTEXTAVAILABILITY</strong></a><br /> | Especifica valores que identifican la disponibilidad de una <a href="windowsribbon-element-ribbon-contextualtabs.md"><strong>pestaña contextual.</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_controldock"><strong>UI_CONTROLDOCK</strong></a><br /> | Especifica valores que identifican el estado de acoplamiento de la barra de herramientas de acceso rápido (QAT). <br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventlocation"><strong>UI_EVENTLOCATION</strong></a><br /> | Identifica las ubicaciones donde se pueden originar los eventos asociados a un control de cinta de opciones.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventtype"><strong>UI_EVENTTYPE</strong></a><br /> | Identifica los tipos de eventos asociados a una cinta <a href="windowsribbon-element-ribbon.md"><strong>de opciones</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_executionverb"><strong>UI_EXECUTIONVERB</strong></a><br /> | Especifica valores que identifican los id. de ejecución que se asignan a las acciones que un usuario puede iniciar en un <a href="windowsribbon-element-command.md"><strong>comando</strong></a>. <br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontdeltasize"><strong>UI_FONTDELTASIZE</strong></a><br /> | Especifica valores que identifican si el tamaño de fuente de una ejecución de texto resaltado se debe incrementar o disminuir.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontproperties"><strong>UI_FONTPROPERTIES</strong></a><br /> | Especifica valores que identifican el estado de propiedad de fuente de <a href="windowsribbon-element-fontcontrol.md"><strong>un FontControl,</strong></a>como <strong>Tachado.</strong><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontunderline"><strong>UI_FONTUNDERLINE</strong></a><br /> | Especifica valores que identifican el estado de subrayado de <a href="windowsribbon-element-fontcontrol.md"><strong>un FontControl</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontverticalposition"><strong>UI_FONTVERTICALPOSITION</strong></a><br /> | Especifica valores que identifican el estado de alineación vertical de <a href="windowsribbon-element-fontcontrol.md"><strong>un FontControl</strong></a>.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_invalidations"><strong>UI_INVALIDATIONS</strong></a><br /> | Especifica valores que identifican el aspecto de un comando que se invalidará.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_ownership"><strong>UI_OWNERSHIP</strong></a><br /> | Especifica valores que identifican las condiciones de propiedad en las que el marco de la cinta de opciones crea una imagen.<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode"><strong>UI_SWATCHCOLORMODE</strong></a><br /> | Especifica si una muestra tiene el modo normal o monocromático.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_swatchcolortype"><strong>UI_SWATCHCOLORTYPE</strong></a><br /> | Especifica los valores que identifican cómo se rellena una muestra de color en un selector <strong></strong>de colores <a href="windowsribbon-element-dropdowncolorpicker.md"><strong>DropDownColorPicker</strong></a> o <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a> <strong>(color</strong> de texto o resaltado de texto).<br /><blockquote>[!Note]<br />Estas son solo recomendaciones. La aplicación puede asociar cualquier tipo de relleno a estos valores.</blockquote><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewtype"><strong>UI_VIEWTYPE</strong></a><br /> | Especifica valores que identifican la vista del marco de la cinta de opciones.<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewverb"><strong>UI_VIEWVERB</strong></a><br /> | Especifica valores que identifican el tipo de acción que se completará en una vista del marco de la cinta de opciones.<br /> |
