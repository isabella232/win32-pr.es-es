# <a name="ui_pkey_tooltipdescription"></a>UI \_ PKEY \_ TooltipDescription

Identifica la propiedad UI \_ PKEY \_ TooltipDescription.

```
propertyDescription
   name = UI_PKEY_TooltipDescription
   shellPKey = UI_PKEY_TooltipDescription
   formatID = 00000005-7363-696e-8441798acf5aebb7
   propID = 5
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Comentarios

Una aplicación usa ui PKEY TooltipDescription para consultar la descripción asociada a una interfaz de usuario \_ \_ [ \_ PKEY \_ TooltipTitle](windowsribbon-reference-properties-uipkey-tooltiptitle.md).

El valor de propiedad es una cadena restringida a cualquier secuencia de caracteres, incluidos los espacios en blanco y los caracteres de salto de línea.

> [!Note]  
> Use la referencia de caracteres XML de juego de caracteres universales (UCS) `&#xA;` para especificar un salto de línea.

 

La longitud máxima de ui \_ PKEY \_ TooltipDescription está sin enlazar.

No se admite la alineación derecha.
