# <a name="ui_pkey_contextavailable"></a>UI \_ PKEY \_ ContextAvailable

Identifica la propiedad \_ ContextAvailable de PKEY \_ de la interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_ContextAvailable
   shellPKey = UI_PKEY_ContextAvailable
   formatID = 00001100-7363-696e-8441798acf5aebb7
   propID = 1100
   typeInfo
      type = UI_CONTEXTAVAILABILITY
```

## <a name="remarks"></a>Observaciones

La interfaz de usuario PKEY ContextAvailable la usa una aplicación para consultar la disponibilidad de un \_ \_ control Tab [Group](windowsribbon-controls-tabgroup.md) en función del contexto de la aplicación actual.
