# <a name="ui_pkey_fontproperties"></a>UI \_ PKEY \_ FontProperties

Identifica la propiedad \_ PKEY FontProperties de la interfaz de \_ usuario.

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Observaciones

Ui \_ PKEY FontProperties es un almacén de propiedades utilizado por una \_ aplicación para consultar todas las propiedades de fuente.
