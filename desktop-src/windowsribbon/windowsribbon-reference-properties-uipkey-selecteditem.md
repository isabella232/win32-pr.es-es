# <a name="ui_pkey_selecteditem"></a>UI \_ PKEY \_ SelectedItem

Identifica la propiedad \_ PKEY SelectedItem de la \_ interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Observaciones

La interfaz de usuario \_ PKEY \_ SelectedItem la usa una aplicación para consultar el elemento seleccionado en un control de la galería.

El valor de propiedad es el índice del elemento seleccionado en una colección.
