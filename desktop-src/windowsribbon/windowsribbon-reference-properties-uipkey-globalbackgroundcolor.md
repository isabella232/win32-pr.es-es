# <a name="ui_pkey_globalbackgroundcolor"></a>UI \_ PKEY \_ GlobalBackgroundColor

Identifica la propiedad \_ GlobalBackgroundColor de PKEY de la \_ interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Observaciones

La \_ interfaz de usuario PKEY GlobalBackgroundColor la usa una aplicación para consultar \_ el color de fondo global de una cinta personalizada.

**Introducido en Windows 8:** [ui \_ PKEY \_ GlobalHighlightColor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) no se puede establecer independientemente de la interfaz de usuario \_ PKEY \_ GlobalBackgroundColor.
