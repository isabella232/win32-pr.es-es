# <a name="ui_pkey_booleanvalue"></a>UI \_ PKEY \_ BooleanValue

Identifica la propiedad BooleanValue de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY BooleanValue para consultar el estado de alternancia de una [casilla](windowsribbon-controls-checkbox.md), un [botón de alternancia](windowsribbon-controls-togglebutton.md)y el control de botón de una [**SplitButtonGallery**](windowsribbon-element-splitbuttongallery.md).
