# <a name="server-side-pipe-implementation"></a>Server-Side de canalización

Los programas de servidor para aplicaciones distribuidas que usan canalizaciones no necesitan implementar ninguna función de inserción, extracción o alocado. Deben contener procedimientos que los clientes pueden invocar de forma remota para iniciar transferencias de datos. En los temas siguientes, en esta sección se explica cómo se pueden implementar los procedimientos remotos del servidor.

-   [Implementar canalizaciones de entrada en el servidor](implementing-input-pipes-on-the-server.md)
-   [Implementar canalizaciones de salida en el servidor](implementing-output-pipes-on-the-server.md)
