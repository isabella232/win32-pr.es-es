# <a name="i-rpc"></a>I (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H I J K L [M](l-glos.md) [N](m-glos.md) [](n-glos.md) [O](o-glos.md) [P P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_idempotent_glos"></span><span id="_RPC_IDEMPOTENT_GLOS"></span>**idempotente**
</dt> <dd>

Llamada a procedimiento remoto que no cambia un estado y devuelve la misma información cada vez que se llama con los mismos parámetros de entrada.

</dd> <dt>

<span id="_rpc_input_parameter_glos"></span><span id="_RPC_INPUT_PARAMETER_GLOS"></span>**parámetro de entrada**
</dt> <dd>

Parámetro que contiene los datos que la aplicación cliente transmite a la aplicación de servidor. El **atributo in** designa un parámetro de entrada.

</dd> <dt>

<span id="_rpc_idl_glos"></span><span id="_RPC_IDL_GLOS"></span>**Lenguaje de definición de interfaz (IDL)**
</dt> <dd>

Lenguaje estándar OSF-DCE para especificar la interfaz para las llamadas a procedimiento remoto. Vea también [*MIDL*](m-glos.md).

</dd> <dt>

<span id="_rpc_ip_glos"></span><span id="_RPC_IP_GLOS"></span>**Protocolo de Internet (IP)**
</dt> <dd>

Protocolo de comunicaciones de capa de red sin conexión. Consulte también [*datagrama*](d-glos.md).

</dd> <dt>

<span id="_rpc_ipx_glos"></span><span id="_RPC_IPX_GLOS"></span>**Internetwork Packet Exchange (IPX)**
</dt> <dd>

Protocolo de comunicación NetWare deCard que usa sockets [*de datagramas*](d-glos.md) para enrutar paquetes de información a través de redes de área local y redes de área extensa.

</dd> <dt>

<span id="_rpc_intranet_glos"></span><span id="_RPC_INTRANET_GLOS"></span>**Intranet**
</dt> <dd>

Red privada que usa productos y tecnologías de Internet (por ejemplo, servidores web), pero no está disponible para usuarios externos de Internet.

</dd> </dl>
