<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**Protocolo de datagramas de usuario (UDP)**
</dt> <dd>

Transporte de red que usa sockets [*de datagramas*](d-glos.md) sin conexión y se encuentra en capas sobre el protocolo de Internet [*(IP).*](i-glos.md)

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**Desvincular**
</dt> <dd>

Proceso de terminación de la conexión lógica entre un cliente y un servidor.

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**puntero único**
</dt> <dd>

Puntero que puede ser null o apuntar a datos existentes, cuyo valor puede cambiar durante una llamada a procedimiento remoto. No se puede crear un alias para un puntero único. El \[ [atributo](/windows/desktop/Midl/unique) \] único designa un puntero único. Vea también [*puntero completo ,*](f-glos.md)puntero de [*referencia*](r-glos.md).

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**Identificador único universal (UUID)**
</dt> <dd>

(o [**GUID)**](/windows/win32/api/guiddef/ns-guiddef-guid)valor de 128 bits usado en la comunicación entre procesos para identificar entidades como interfaces de cliente y servidor, vectores de punto de entrada de administrador y objetos RPC. Vea también [uuidgen](/windows).

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**desmarque**
</dt> <dd>

Proceso de desempaquetar parámetros que se han enviado a través de los límites del proceso.

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**uuidgen**
</dt> <dd>

Programa de utilidad, proporcionado con el Kit de desarrollo de software de plataforma (SDK), que usa un valor de tiempo y el identificador de la tarjeta de red de la máquina para generar [UUID que](/windows) se garantiza que sean únicos.

</dd> </dl>
