# <a name="receiving-cancellations"></a>Recepción de cancelaciones

La aplicación de servidor puede llamar a [**RpcServerTestCancel con**](/windows/desktop/api/Rpcdce/nf-rpcdce-rpcservertestcancel) el identificador de enlace de la llamada en cuestión para averiguar si el cliente ha solicitado la cancelación de la llamada asincrónica. Devolverá RPC \_ S \_ OK si el cliente canceló la llamada.
