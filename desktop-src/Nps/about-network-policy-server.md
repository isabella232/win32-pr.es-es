# <a name="about-network-policy-server"></a>Acerca del servidor de directivas de red

> [!Note]  
> El nombre del Servicio de autenticación de Internet (IAS) se ha cambiado a Servidor de directivas de red (NPS) a partir Windows Server 2008. El contenido de este tema se aplica a IAS y NPS.

 

Servidor de directivas de red es la implementación de Microsoft de un servidor RADIUS y un proxy. En las secciones siguientes se proporciona información general sobre NPS y RADIUS.

-   [Registro con el servidor de directivas de red](/windows/desktop/Nps/ias-radius-accounting-packets)
-   [Trabajar con un servidor de estado](/windows/desktop/Nps/ias-working-with-a-state-server)
