# <a name="configuration-information-not-related-to-printcapabilities"></a>Información de configuración no relacionada con PrintCapabilities

Este tema no es actual. Para obtener la información más reciente, vea [La especificación de esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Además de proporcionar selecciones para los atributos de dispositivo definidos en PrintCapabilities, PrintTicket permite que los clientes puedan agregar información adicional en forma de elementos Property. El esquema de impresión define una serie de instancias de propiedad estándar y el proveedor de interfaz también puede introducir instancias de propiedad privadas. Por ejemplo, si los miembros de una organización envían la mayoría de sus trabajos de impresión a una instalación de lote central, pueden especificar para cada trabajo una instancia de propiedad privada que contenga una dirección de devolución. Esta propiedad puede facilitar la entrega del trabajo completado al originador.
