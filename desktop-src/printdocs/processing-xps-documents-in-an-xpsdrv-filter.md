# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Procesamiento de documentos XPS en un filtro XPSDrv

XpS OM se puede usar para acceder a los componentes de un documento XPS desde un filtro de controlador de impresora XPSDrv.

Para obtener más información sobre los filtros XPS y los controladores de impresora XPSDrv, vea [XpS Filters](/windows-hardware/drivers/print/xps-filters) in the Windows Driver Kit (Filtros XPS en Windows Driver Kit).
