# <a name="parameter-constructs"></a>Construcciones de parámetros

Este tema no es actual. Para obtener la información más reciente, vea [La especificación de esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Los parámetros usados en el esquema de impresión describen concisamente un intervalo aceptable de valores, en lugar de una enumeración discreta de valores. El marco de esquema de impresión define tres elementos específicos de los parámetros: los elementos ParameterDef describen el parámetro y sus características, los elementos ParameterInit definen una instancia o un valor determinados para un parámetro y los elementos ParameterRef hacen referencia a un elemento ParameterInit específico que se define en otro lugar.
