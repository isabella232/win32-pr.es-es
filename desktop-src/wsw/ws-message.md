# <a name="ws_message"></a>WS \_ MESSAGE

Tipo opaco utilizado para hacer referencia a un [objeto de](message.md) mensaje.


```C++
typedef struct _WS_MESSAGE WS_MESSAGE;
```



## <a name="remarks"></a>Comentarios

Este objeto no es seguro para subprocesos. Para obtener más información, vea [Seguridad para subprocesos.](thread-safety.md)
