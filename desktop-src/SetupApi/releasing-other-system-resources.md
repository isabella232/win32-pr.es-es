# <a name="releasing-other-system-resources"></a>Liberar otros recursos del sistema

Si se usó un archivo de registro o una lista de origen, los recursos deben liberarse llamando a [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) o [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
