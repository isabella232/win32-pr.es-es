## <a name="remarks"></a>Comentarios

**Los valores FOLDERTYPEID** son GUID y se declaran en Shlguid.h.

En la práctica, cualquier **valor FOLDERTYPEID** se puede aplicar a cualquier carpeta independientemente del contenido de esa carpeta. **FOLDERTYPEID es** solo una plantilla de vista, que proporciona el diseño y las columnas adecuados para el contenido esperado de la carpeta. No hay ninguna comprobación del contenido de la carpeta con respecto al **valor FOLDERTYPEID** para garantizar la idoneidad.

Estos y otros tipos de carpetas se definen en el Registro en la subclave **FolderTypes,** que se encuentra como se muestra aquí:

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
