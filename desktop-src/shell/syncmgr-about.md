# <a name="about-synchronization-manager"></a>Acerca del Administrador de sincronización

Synchronization Manager proporciona una tecnología centralizada y estándar para sincronizar archivos para su uso sin conexión en un equipo local. El usuario puede preparar un equipo para su uso sin conexión actualizando los archivos necesarios desde el almacenamiento de red. A continuación, cuando el equipo vuelve a estar en línea, los cambios sin conexión se pueden volver a copiar en la red. SyncMgr proporciona una interfaz de usuario común que todas las aplicaciones pueden compartir para sincronizar sus archivos.

Los archivos se transfieren independientemente del protocolo. Por ejemplo, un programa de correo electrónico puede transferir sus mensajes mediante SMTP, NMTP o POP3, mientras que un explorador puede usar HTTP y una base de datos puede usar RPC. El protocolo es transparente para SyncMgr.

SyncMgr también es independiente del almacenamiento y puede transferir archivos, mensajes de correo electrónico, páginas HTML o información de base de datos de forma transparente.
