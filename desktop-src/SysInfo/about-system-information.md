# <a name="about-system-information"></a>Acerca de Información del sistema

Las funciones descritas en esta introducción recuperan una variedad de datos. La información de configuración del sistema describe el hardware del equipo, el nombre del equipo y el nombre de usuario. La información de configuración del sistema operativo describe la versión del sistema operativo, los directorios clave y las variables de entorno. Los parámetros del sistema describen los atributos del sistema. Las métricas del sistema proporcionan las dimensiones de los elementos del área de presentación.

En las secciones siguientes se describe cómo obtener estos tipos de información:

-   [Configuración de hardware](hardware-configuration.md)
-   [Nombres de equipos](computer-names.md)
-   [Versión de sistema operativo](operating-system-version.md)
-   [Configuración del sistema operativo](operating-system-configuration.md)
