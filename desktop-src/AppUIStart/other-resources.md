<dl> <dt>

<span id="_Windows_Interface_Guidelines_for_Software_Design"></span><span id="_windows_interface_guidelines_for_software_design"></span><span id="_WINDOWS_INTERFACE_GUIDELINES_FOR_SOFTWARE_DESIGN"></span>*Windows interfaces para el diseño de software*
</dt> <dd>

(Microsoft Press, 1995; ISBN 1-55615-679-0)

En este libro se explican todas las reglas de interfaz de usuario clave que se deben seguir, con una gran cantidad de ejemplos de cómo crear cuadros de diálogo, asistentes, barras de herramientas y menús contextuales.

</dt> <dd>

(AP Professional, 1994; ISBN 0125184069)

Este libro se centra en las técnicas para medir eficazmente la facilidad de uso de una interfaz. Explica muchas técnicas de pruebas diferentes para comparar diferentes ideas de diseño, incluidas sugerencias para mantener los costos de las pruebas de facilidad de uso. En este libro se tratan las técnicas de análisis que se pueden aplicar a sitios web, páginas web o cualquier tipo de software. Es excelente para las fases iniciales de aprendizaje sobre los usuarios y las fases intermedias de las pruebas de iteración y facilidad de uso.

(Currency/Doubleday, 1988; ISBN: 0385267746)

Entre los ejemplos se incluyen electrodomésticos, puertas de oficina y máquinas de cocina.
