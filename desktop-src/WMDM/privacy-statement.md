# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Declaración de privacidad (Windows Media Administrador de dispositivos 11 SDK)

Los desarrolladores que Windows aplicaciones Administrador de dispositivos media deben comunicar claramente a sus usuarios cómo podría verse afectada su privacidad. Para ayudarle a crear una declaración de privacidad del usuario final, Microsoft ha publicado directrices de privacidad en la [Windows web de privacidad de los medios](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx).
