# <a name="interfaces-for-service-providers-and-applications"></a>Interfaces para proveedores de servicios y aplicaciones

Los proveedores de servicios y las aplicaciones que usan el SDK de Windows Media Administrador de dispositivos pueden usar las interfaces siguientes.



| Interfaz                                                | Descripción                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Registra las entradas en un archivo de registro común.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Proporciona una comunicación segura y cifrada entre módulos de Windows Media Administrador de dispositivos. |
