# <a name="functions-windows-touch-gestures"></a>Funciones (Windows gestos táctiles)

Esta sección contiene funciones para Windows gestos táctiles.

Las siguientes funciones se usan para Windows gestos táctiles.



| Función                                                 | Descripción                                                                                             |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**CloseGestureInfoHandle**](/windows/desktop/api/winuser/nf-winuser-closegestureinfohandle) | Cierra los recursos asociados a un identificador recuperado manualmente a una estructura de información de gestos.    |
| [**GetGestureExtraArgs**](/windows/desktop/api/winuser/nf-winuser-getgestureextraargs)       | Recupera información adicional sobre un gesto de su [**identificador GESTUREINFO.**](/windows/win32/api/winuser/ns-winuser-gestureinfo)    |
| [**GetGestureInfo**](/windows/desktop/api/winuser/nf-winuser-getgestureinfo)                 | Recupera manualmente una estructura de información de gestos dado un identificador a la estructura de información de gestos. |
| [**GetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-getgestureconfig)             | Recupera la información de configuración de gestos de una ventana.                                           |
| [**SetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-setgestureconfig)             | Establece la configuración de gestos de una ventana.                                                            |
