# <a name="structures-windows-touch-gestures"></a>Estructuras (Windows gestos táctiles)

Esta sección contiene estructuras para Windows gestos táctiles.

En esta sección se especifican las estructuras siguientes.



| Estructura                                          | Descripción                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | Se usa para establecer la configuración de qué mensajes de gestos están habilitados o deshabilitados.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Contiene información sobre un gesto.                                                    |
| [**GESTURENOTIFYSTRUCT**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Contiene información sobre una [**notificación \_ GESTURENOTIFY de WM.**](wm-gesturenotify.md) |
