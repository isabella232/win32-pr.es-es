# <a name="iscommandblocked-method-of-the-win32_tpm-class"></a>Método IsCommandBlocked de la clase Tpm \_ win32

## <a name="syntax"></a>Sintaxis


```mof
uint32 IsCommandBlocked(
  [in]  uint32 CommandOrdinal,
  [out] uint32 IsCommandBlocked
);
```

## <a name="remarks"></a>Observaciones

Este método lee las siguientes claves del Registro:

-   directiva de grupo lista de comandos de TPM bloqueados:

    **KEY \_ Directivas \_ de** software de MÁQUINA LOCAL \\  \\  \\ **Microsoft** \\ **Tpm** \\ **BlockedCommands** \\ **List**

-   Lista predeterminada de comandos de TPM bloqueados:

    **KEY \_ SOFTWARE DE \_ MÁQUINA LOCAL** \\  \\ **Microsoft** \\ **Tpm** \\ **BlockedCommands** \\ **List**

-   directiva de grupo configuración para omitir la lista predeterminada de comandos de TPM bloqueados:

    **KEY \_ Directivas \_ de** software de MÁQUINA LOCAL \\  \\  \\ **Microsoft** \\ **Tpm** \\ **BlockedCommands** \\ **IgnoreDefaultList**

-   Lista local de comandos de TPM bloqueados:

    **KEY \_ SISTEMA \_ DE MÁQUINA LOCAL** \\  \\ **CurrentControlSet** \\ **Services** \\ **SharedAccess** \\ **Parameters** \\ **Tpm** \\ **BlockedCommands** \\ **List**

-   directiva de grupo configuración para omitir la lista local de comandos de TPM bloqueados:

    **KEY \_ Directivas \_ de** software de MÁQUINA LOCAL \\  \\  \\ **Microsoft** \\ **Tpm** \\ **BlockedCommands** \\ **IgnoreLocalList**

El valor devuelto del método se puede derivar de la siguiente manera:

-   Suponga que *IsCommandBlockedTemp* es un valor de 3 bits, establecido inicialmente en 0.
-   Si la configuración de directiva de grupo para omitir la lista predeterminada de comandos de TPM bloqueados no existe o es 0, y un valor en la clave del Registro que corresponde a la lista predeterminada de comandos de TPM bloqueados es *CommandOrdinal*, el bit menos significativo de *IsCommandBlockedTemp* se establece en 1.
-   Si la configuración de directiva de grupo para omitir la lista local de comandos de TPM bloqueados no existe o es 0, y un valor de la clave del Registro que corresponde a la lista local de comandos de TPM bloqueados es *CommandOrdinal*, el segundo bit menos significativo de *IsCommandBlockedTemp* se establece en 1.
-   Si un valor de las claves del Registro que corresponde directiva de grupo la lista de comandos de TPM bloqueados es *CommandOrdinal,* el bit más significativo de *IsCommandBlockedTemp* se establece en 1.
-   *IsCommandBlocked* devuelve *IsCommandBlockedTemp.*
