# <a name="removeblockedcommand-method-of-the-win32_tpm-class"></a>Método RemoveBlockedCommand de la clase Tpm \_ win32

El **método RemoveBlockedCommand** de la clase [**Tpm de Win32 \_**](win32-tpm.md) quita un comando tpm de la lista local de comandos bloqueados para que no se ejecuten en el equipo.

Es posible que tenga que configurar directiva de grupo para desbloquear un comando. Por ejemplo, si desea permitir un comando que está bloqueado de forma predeterminada, primero debe configurar directiva de grupo para omitir la lista predeterminada de comandos bloqueados. Use el [**método IsCommandBlocked**](iscommandblocked-win32-tpm.md) para determinar cómo se bloquea un comando, ya sea a través de la lista local o a través de la directiva de grupo.

> [!Caution]  
> El sistema operativo impide que un conjunto predefinido de comandos se ejecute de forma predeterminada. Los cambios en este valor predeterminado pueden afectar a la seguridad, privacidad o funcionalidad del equipo.

 

## <a name="syntax"></a>Sintaxis


```mof
uint32 RemoveBlockedCommand(
  [in] uint32 CommandOrdinal
);
```
