# <a name="wpd-samples"></a>Ejemplos de WPD

El SDK Windows proporciona dos ejemplos: [WpdApiSample](wpdapisample-sample-application.md) y [WpdServiceApiSample](wpdapisample-sample-service-application.md).

WpdApiSample es una aplicación de escritorio de línea de comandos que permite al usuario realizar varias funciones, como enumerar los dispositivos conectados, explorar los objetos de estos dispositivos y enumerar las propiedades de los objetos.

WpdServiceApiSample es una aplicación de escritorio de línea de comandos que admite un servicio de dispositivo Contacts y permite al usuario realizar funciones como: enumerar servicios de dispositivo conectados, recuperar propiedades y formatos para un servicio determinado e invocar métodos en ese servicio.
