# <a name="selecting-a-device-wpd-api"></a>Selección de un dispositivo (API de WPD)

Una vez que la aplicación enumera los dispositivos conectados, es posible que desee ofrecer al usuario la capacidad de seleccionar un dispositivo determinado y establecer una conexión para operaciones adicionales. En la aplicación de ejemplo, `ChooseDevice` la función del módulo DeviceEnumeration.cpp realiza estas tareas.

La `ChooseDevice` función usa el código de enumeración de dispositivos que ya hemos visto para enumerar los dispositivos disponibles. (Consulte la sección [Enumeración de dispositivos](enumerating-devices.md) para obtener más información).
