# <a name="moduleadvtuisequence-table"></a>ModuleAdvtUISequence (tabla)

No cree esta tabla. El Windows no usa la [tabla AdvtUISequence](advtuisequence-table.md). La tabla AdvtUISequence no debe existir o debe dejarse en blanco en la base de datos de instalación. Tampoco cree la tabla ModuleAdvtUISequence.
