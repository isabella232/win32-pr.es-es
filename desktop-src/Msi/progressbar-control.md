# <a name="progressbar-control"></a>Control ProgressBar

El control ProgressBar muestra un gráfico de barras que cambia de longitud a medida que recibe mensajes de progreso. Este control se suscribe a [SetProgress](setprogress-controlevent.md) ControlEvent. Puede suscribirse a un control ControlEvent denominado después de la acción que se está supervisando.

Para obtener información relacionada, [vea Crear un control ProgressBar y](authoring-a-progressbar-control.md)Agregar acciones [personalizadas a ProgressBar](adding-custom-actions-to-the-progressbar.md).

## <a name="control-attributes"></a>Atributos de control

Para cambiar el valor de un atributo mediante un evento, suscriba el control a un control ControlEvent en la [tabla EventMapping](eventmapping-table.md) y enumézcalo en la columna Attribute .
