# <a name="pidkey-property"></a>PidKEY, propiedad

La **propiedad PIDKEY** contiene la parte del id. de producto especificado por el usuario. La [acción ValidateProductID valida](validateproductid-action.md) el id. de producto especificado comparando la **propiedad PIDKEY** con la [**propiedad PIDTemplate.**](pidtemplate.md)

La **propiedad PIDKEY** se puede establecer en la línea de comandos o a través de la interfaz de usuario. Esta propiedad contiene solo la parte del id. de producto especificada por el usuario.
