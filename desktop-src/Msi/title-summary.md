# <a name="title-summary-property"></a>Propiedad Resumen de título

La **propiedad Resumen de** título describe brevemente el tipo del paquete del instalador. Para esta propiedad se pueden usar frases como "Base de datos de instalación", "Transformar" o "Revisión".

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Instalador 5.0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Instalador 4.0 o Windows Instalador 4.5 en Windows Server 2008 o Windows Vista. Windows Instalador en Windows Server 2003 o Windows XP<br/> |
