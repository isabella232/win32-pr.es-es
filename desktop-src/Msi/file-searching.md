# <a name="file-searching"></a>Búsqueda de archivos

Durante la instalación, a menudo es necesario buscar un archivo específico. El instalador tiene cuatro acciones de búsqueda estándar para cumplir este propósito. La [acción AppSearch](appsearch-action.md) usa la [tabla AppSearch](appsearch-table.md) para determinar si existe un archivo en el sistema de un usuario. Durante una instalación de actualización, la acción [CCPSearch](ccpsearch-action.md) y la acción [RMCCPSearch](rmccpsearch-action.md) pueden usar sus tablas correspondientes para comprobar si el sistema del usuario tiene una versión anterior de un producto.
