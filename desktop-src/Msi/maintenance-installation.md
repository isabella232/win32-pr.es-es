# <a name="maintenance-installation"></a>Instalación de mantenimiento

Una vez que una aplicación se ha instalado correctamente, es posible que el usuario quiera volver a la instalación para agregar o quitar características. Esto se conoce como instalación de mantenimiento.

[Las transformaciones](transforms.md) que se han aplicado a una instalación deben estar disponibles durante las instalaciones de mantenimiento posteriores.
