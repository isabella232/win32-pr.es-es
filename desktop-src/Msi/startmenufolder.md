# <a name="startmenufolder-property"></a>Propiedad StartMenuFolder

El instalador establece la **propiedad StartMenuFolder** en la ruta de acceso completa de la carpeta Menú Inicio. De forma predeterminada, esta propiedad se establece en la carpeta del usuario actual. Si existe un perfil "Todos los usuarios" y se establece la propiedad [**ALLUSERS,**](allusers.md) esta propiedad se establece en la carpeta del perfil "Todos los usuarios".
