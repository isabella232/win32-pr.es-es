# <a name="m-windows-installer"></a>M (Windows instalador)

<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**aplicación administrada**
</dt> <dd>

Los privilegios del sistema, también conocidos como privilegios elevados, se usan para instalar la aplicación. Una aplicación se administra en un sistema si se trata de una instalación por máquina en Windows 2000, o si la aplicación se asigna o publica mediante directiva de grupo en Windows 2000.

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**Combinar**
</dt> <dd>

Combina dos bases de datos de instalación para formar una base de datos única. Para obtener más información, vea [Combinar bases de datos.](merging-databases.md)

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**módulo merge**
</dt> <dd>

Base de datos combinada con un paquete de instalación para entregar nuevos componentes a una aplicación. Los módulos de combinación no se pueden instalar por sí solos. Para obtener más información, vea [Módulos de mezcla](merge-modules.md).

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Instalador de Windows Microsoft**
</dt> <dd>

Servicio de instalador del lado cliente para administrar la instalación de aplicaciones en plataformas de 32 bits. La aplicación debe encapsularse en un [*paquete*](p-gly.md). Para obtener más información, vea [About Windows Installer](about-windows-installer.md).

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**.msi archivo**
</dt> <dd>

Archivo de almacenamiento estructurado en COM que contiene las instrucciones y los datos necesarios para instalar una aplicación. Cada [*paquete*](p-gly.md) contiene al menos un .msi archivo. El .msi contiene la base de datos [*del instalador*](i-gly.md), un flujo de información de resumen [*y,*](s-gly.md)posiblemente, una o varias [*transformaciones*](t-gly.md) y [*archivos de origen internos*](i-gly.md). Para obtener más información, [vea Windows Installer File Extensions](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**Archivo .msm**
</dt> <dd>

[Módulo merge](merge-modules.md). Para obtener más información, [vea Windows Installer File Extensions](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**Archivo .msp**
</dt> <dd>

[Paquete de revisión](patch-packages.md). Para obtener más información, [vea Windows Installer File Extensions](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**Archivo .mst**
</dt> <dd>

Transformar paquete. Para obtener más información, [vea Merges and Transforms](merges-and-transforms.md). Para obtener más información, [vea Windows Installer File Extensions](windows-installer-file-extensions.md).

</dd> </dl>
