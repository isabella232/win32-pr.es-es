# <a name="ice16"></a>ICE16

ICE16 valida que el valor de la propiedad [**ProductName**](productname.md) de la tabla [Property](property-table.md) no tiene más de 63 caracteres de longitud. No existe ningún límite en la longitud de la clave del Registro para DisplayName.

## <a name="result"></a>Resultado

ICE16 publica un mensaje de error si el valor ProductName establecido en la tabla Property tiene más de 63 caracteres.
