# <a name="authoring-modulecomponents-tables"></a>Creación de tablas moduleComponents

Enumera todos los componentes entregados por el módulo de combinación en la [tabla ModuleComponents](modulecomponents-table.md). Tenga en cuenta que los componentes del módulo de mezcla también se especifican en la tabla Component . Para obtener más información, vea [Crear tablas de componentes de módulo de mezcla](authoring-merge-module-component-tables.md).
