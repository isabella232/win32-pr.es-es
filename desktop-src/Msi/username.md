# <a name="username-property"></a>USERNAME, propiedad

La **propiedad USERNAME** es un usuario que realiza la instalación.

## <a name="default-value"></a>Valor predeterminado

El valor predeterminado se puede colocar en la tabla Property. Si no se establece la propiedad **USERNAME,** se establece automáticamente mediante valores del Registro.

## <a name="remarks"></a>Observaciones

Establezca la [**propiedad NOUSERNAME**](nousername.md) para suprimir la configuración automática de **USERNAME**.
