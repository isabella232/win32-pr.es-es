# <a name="configurableitemhelplocation-property"></a>Propiedad ConfigurableItem. HelpLocation

La propiedad **HelpLocation** devuelve el valor de la columna HelpLocation de la [tabla ModuleConfiguration](moduleconfiguration-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = ConfigurableItem.HelpLocation
```



## <a name="property-value"></a>Valor de propiedad

## <a name="c"></a>C++

Consulte [**Get \_ HelpLocation**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helplocation) function.
