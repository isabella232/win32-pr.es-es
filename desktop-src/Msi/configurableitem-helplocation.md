# <a name="configurableitemhelplocation-property"></a>ConfigurableItem.HelpLocation, propiedad

La **propiedad HelpLocation** devuelve el valor de la columna HelpLocation de la [tabla ModuleConfiguration](moduleconfiguration-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = ConfigurableItem.HelpLocation
```
