# <a name="nocompanyname-property"></a>Propiedad NOCOMPANYNAME

Establezca la **propiedad NOCOMPANYNAME** en 1 para suprimir el valor automático de la [**propiedad COMPANYNAME**](companyname.md) por parte del instalador. Esta propiedad la usan las aplicaciones que necesitan recopilar el nombre de la empresa en la primera ejecución.

## <a name="default-value"></a>Valor predeterminado

Sin establecer. El instalador establece automáticamente [**la propiedad COMPANYNAME**](companyname.md) mediante valores del Registro.

Windows Instalador en Windows Server 2003 o Windows XP.
