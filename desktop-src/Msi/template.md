# <a name="template"></a>Plantilla

El tipo de datos Template es una cadena de texto que puede contener propiedades que se incluyen entre \[ \] corchetes. El tipo de plantilla permite todos los [formatos de](formatted.md) tipo Con formato, más \[ *1,* \] donde *1* es un número.
