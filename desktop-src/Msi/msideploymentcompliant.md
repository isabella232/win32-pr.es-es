# <a name="msideploymentcompliant-property"></a>Propiedad MSIDEPLOYMENTCOMPLIANT

La **propiedad MSIDEPLOYMENTCOMPLIANT** se puede establecer para indicar al instalador que el paquete se ha escrito y probado para cumplir con el [*Control*](u-gly.md) de cuentas de usuario (UAC) en Windows Vista. Si no se establece esta propiedad, el instalador determinará si el paquete cumple con UAC en Windows Vista.

Para obtener más información sobre UAC y Windows Installer, vea Using Windows Installer with UAC (Uso del instalador de Windows con [UAC)](using-windows-installer-with-uac.md) y [Guidelines for Packages (Directrices para paquetes).](guidelines-for-packages.md)

**Windows Installer 3.1, Windows Installer 3.0 y Windows Installer 2.0:** Esta propiedad se omite. Esta propiedad solo la usa Windows Installer 4.0 en Windows Vista.
