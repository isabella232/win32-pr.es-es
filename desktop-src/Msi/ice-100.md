# <a name="ice-100"></a>ICE 100

ICE100 comprueba la creación de la tabla [MsiEmbeddedUI](msiembeddedui-table.md) y la tabla [MsiEmbeddedChainer](msiembeddedchainer-table.md) .

## <a name="result"></a>Resultado

ICE100 expone los siguientes errores.



| Error ICE100                                                  | Descripción                                                                                                                                                                                                                                             |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Demasiados archivos dll marcados como UI: \[ 1\]                             | La columna Attributes de la tabla [MsiEmbeddedUI](msiembeddedui-table.md) contiene demasiados archivos dll marcados como dll de la interfaz de usuario. Solo se debe marcar una DLL como una DLL de interfaz de usuario.                                                               |
| La DLL de la interfaz de usuario no puede tener un filtro de mensajes nulo o cero: \[ 1\]            | Un archivo DLL marcado como una DLL de interfaz de usuario en la columna Attributes de la tabla [MsiEmbeddedUI](msiembeddedui-table.md) tiene el valor null en el campo MessageFilter. Si una fila hace referencia a un archivo DLL de la interfaz de usuario, el valor del campo MessageFilter no debe ser null. |
| Los archivos dll de recursos deben tener filtros de mensajes nulos/cero: \[ 1\]      | Un archivo DLL no marcado como DLL de interfaz de usuario en la tabla [MsiEmbeddedUI](msiembeddedui-table.md) debe tener el valor null en el campo MessageFilter. Este campo debe ser null si una fila hace referencia a un archivo de recursos y el valor de Attributes es NULL.            |
| Varios archivos dll marcados con este nombre de archivo: \[ 2 \] (índice: \[ 1 \] ) | Varios archivos dll de la tabla [MsiEmbeddedUI](msiembeddedui-table.md) tienen el mismo nombre de archivo. La columna FileName no debe contener nombres duplicados.                                                                                                    |



 

ICE100 envía la siguiente advertencia.



| ADVERTENCIA de ICE100                                                                                          | Descripción                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Existen varios encadenadores en la tabla MsiEmbeddedChainer. Asegúrese de que solo hay una para ejecutarse. | Hay varias entradas en la tabla [MsiEmbeddedChainer](msiembeddedchainer-table.md) . Solo se puede iniciar un encadenador insertado. Si hay varias entradas que se pueden ejecutar, no se define la entrada que se ejecutará. Si el usuario está seguro de que solo la condición de una entrada puede resolverse en true, esta advertencia se puede omitir. |
