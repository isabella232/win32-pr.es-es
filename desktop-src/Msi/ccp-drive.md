# <a name="ccp_drive-property"></a>Propiedad CCP \_ DRIVE

La **propiedad CCP \_ DRIVE** se establece en la ruta de acceso raíz del volumen extraíble que va a buscar [RMCCPSearch.](rmccpsearch-action.md) La acción RMCCPSearch usa firmas de archivo para validar que los productos aptos están instalados en un sistema antes de realizar una instalación de actualización.

Esta propiedad se establece normalmente a través de la interfaz de usuario o una acción personalizada. Esta propiedad debe establecerse antes de ejecutar la acción [RMCCPSearch.](rmccpsearch-action.md)
