# <a name="replacedinusefiles-property"></a>Propiedad ReplacedInUseFiles

La **propiedad ReplacedInUseFiles** se establece si el instalador escribe en un archivo que se mantiene en uso. Esta propiedad se usa mediante acciones personalizadas para detectar que se requiere un reinicio para completar la instalación.

Establezca durante las [acciones InstallExecute](installexecute-action.md) [e InstallFinalize.](installfinalize-action.md)

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Instalador 5.0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Instalador 4.0 o Windows Installer 4.5 en Windows Server 2008 o Windows Vista. Windows Instalador en Windows Server 2003 o Windows XP. Consulte el [Windows installer Run-Time para](windows-installer-portal.md) obtener información sobre los requisitos mínimos de Windows Service Pack que requiere una Windows Installer.<br/> |
