# <a name="filesinuse-dialog"></a>Cuadro de diálogo FilesInUse

Un cuadro de diálogo FilesInUse alerta al usuario de los procesos que actualmente ejecutan archivos que la instalación debe sobrescribir o eliminar. Esto ofrece al usuario la oportunidad de apagar estos procesos y evitar tener que reiniciar el equipo para completar la sustitución o eliminación de estos archivos. Vea [Reinicios del sistema.](system-reboots.md)

Este cuadro de diálogo se creará según sea necesario en la [acción InstallValidate](installvalidate-action.md).
