# <a name="system16folder-property"></a>Propiedad System16Folder

Windows El instalador establece la **propiedad System16Folder** en la ruta de acceso completa a la carpeta para archivos DLL del sistema de 16 bits. Tenga en cuenta que la propiedad **System16Folder** no está definida para las aplicaciones de 64 bits Windows porque este sistema no admite aplicaciones o componentes de 16 bits.
