# <a name="last-printed-summary-property"></a>Última propiedad Resumen impreso

En un paquete de instalación, **la** propiedad Resumen de última impresión se puede establecer en la fecha y hora durante una [instalación](administrative-installation.md) administrativa para registrar cuándo se creó la imagen administrativa. En instalaciones no administrativas, esta propiedad es la misma que la propiedad Resumen de [**fecha y hora de**](create-time-date-summary.md) creación.

En una transformación, esta propiedad debe ser NULL.

En un paquete de revisión, esta propiedad debe ser NULL.
