# <a name="internet-download-bootstrapping"></a>Arranque de descarga de Internet

Con Windows Installer, se incluye un ejecutable de arranque configurable (Setup.exe) y una herramienta de configuración ([Msistuff.exe](msistuff-exe.md)) en Windows [SDK Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md). Mediante el Msistuff.exe para configurar los recursos de Setup.exe, los desarrolladores pueden crear fácilmente una instalación web de un paquete Windows Installer.

La versión mínima del instalador requerida por el ejecutable de arranque Windows installer versión 2.0. Las aplicaciones pueden obtener la versión del instalador mediante [**DllGetVersion**](/windows/win32/api/shlwapi/nc-shlwapi-dllgetversionproc). Para obtener más información, vea [Determinar la versión Windows instalador](determining-the-windows-installer-version.md)

El ejecutable de arranque proporcionado con Windows SDK del instalador hace lo siguiente:

-   Llama [**a WinVerifyTrust para**](/windows/win32/api/wintrust/nf-wintrust-winverifytrust) comprobar la firma digital del .msi archivo. Windows La versión del instalador 2.0 y versiones posteriores proporciona un paquete de interfaz de sujeto (SIP) para habilitar la firma de Windows instalador.
-   Si es necesario, actualiza la versión del instalador Windows en la máquina.

## <a name="setupexe-command-line-parameters"></a>Setup.exe de línea de comandos

En la tabla siguiente se enumeran las opciones de comando que se pueden usar con el Setup.exe que se incluye con los componentes del SDK de Windows para Windows [desarrolladores del instalador](platform-sdk-components-for-windows-installer-developers.md). Cuando se invoca sin ninguna opción de línea de comandos, Setup.exe el comportamiento especificado a través de los recursos configurados. Las opciones de línea de comandos son mutuamente excluyentes: solo se puede usar una de ellas a la vez.



| Opción | Uso                                                                      | Significado                                                                                                                                                                                                                                                                                                         |
|--------|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **/a** | setup.exe **/a**                                                           | Inicia una instalación administrativa. Si la propiedad ISETUPPROPNAME DATABASE está configurada en Setup.exe, esta opción se puede usar para realizar una \_ instalación administrativa del producto. Esta opción de línea de comandos no se admite si el recurso ISETUPPROPNAME \_ PATCH está configurado.<br/> |
| **/a** | setup.exe **/a<** *ruta de acceso completa a una instalación administrativa existente>* | Si la propiedad ISETUPPROPNAME PATCH está configurada en Setup.exe, esta opción se puede usar para aplicar revisiones a una instalación administrativa \_ existente del producto. Esta opción de línea de comandos no se admite si el recurso ISETUPPROPNAME \_ DATABASE está configurado.                                               |
| **/v** | setup.exe **/v***<full path to a file>*                             | Comprueba la firma en el archivo mediante WinVerifyTrust. Cuando se usa esta opción, no se muestra ninguna interfaz de usuario. En este caso, el resultado de la comprobación de confianza es el valor devuelto obtenido de la ejecución de Setup.exe.                                                                                                 |
| **/?** |                                                                            | Muestra un cuadro de diálogo de ayuda que indica los argumentos válidos. Nota: El cuadro de diálogo de ayuda también se muestra si se proporcionan argumentos no válidos.                                                                                                                                                                               |
