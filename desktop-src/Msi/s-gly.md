<dl> <dt>

<span id="_msi_sequence_tables_gly"></span><span id="_MSI_SEQUENCE_TABLES_GLY"></span>**tablas de secuencia**
</dt> <dd>

Tablas de la base [*de datos del instalador*](i-gly.md) que muestran las acciones secuencialmente. Para obtener más información, [vea Usar una tabla de secuencia.](using-a-sequence-table.md)

</dd> <dt>

<span id="_msi_source_list_gly"></span><span id="_MSI_SOURCE_LIST_GLY"></span>**lista de orígenes**
</dt> <dd>

Especifica las ubicaciones en las que el instalador va a buscar archivos [*de código fuente externos.*](e-gly.md)

</dd> <dt>

<span id="_msi_standard_action_gly"></span><span id="_MSI_STANDARD_ACTION_GLY"></span>**acción estándar**
</dt> <dd>

Acción integrada [*de*](a-gly.md) la Windows Installer. Para obtener más información, vea [Acciones estándar.](standard-actions.md)

</dd> <dt>

<span id="_msi_standard_user_gly"></span><span id="_MSI_STANDARD_USER_GLY"></span>**usuario estándar**
</dt> <dd>

Usuario sin privilegios de administrador.

</dd> <dt>

<span id="_msi_structured_query_language_gly"></span><span id="_MSI_STRUCTURED_QUERY_LANGUAGE_GLY"></span>**Lenguaje de consulta estructurado (SQL)**
</dt> <dd>

Lenguaje utilizado para procesar datos en una base de datos relacional. Puede usar SQL para realizar consultas en la base de datos del instalador. Para obtener más información, [vea Trabajar con consultas](working-with-queries.md).

</dd> <dt>

<span id="_msi_summary_information_stream_gly"></span><span id="_MSI_SUMMARY_INFORMATION_STREAM_GLY"></span>**secuencia de información de resumen**
</dt> <dd>

Ubicación de almacenamiento en [*.msi archivo para*](m-gly.md) obtener información que se puede ver con Microsoft Explorador de Windows. Para obtener más información, [vea Acerca del flujo de información de resumen](about-the-summary-information-stream.md).

</dd> <dt>

<span id="_msi_synchronous_execution_gly"></span><span id="_MSI_SYNCHRONOUS_EXECUTION_GLY"></span>**ejecución sincrónica**
</dt> <dd>

Cuando el instalador espera a que se complete el subproceso independiente de una acción personalizada antes de continuar con el subproceso de la instalación actual. Para obtener más información, vea [Acciones personalizadas sincrónicas y asincrónicas.](synchronous-and-asynchronous-custom-actions.md)

</dd> <dt>

<span id="_msi_system_policy_using_windows_installer__gly"></span><span id="_MSI_SYSTEM_POLICY_USING_WINDOWS_INSTALLER__GLY"></span>**directiva del sistema**
</dt> <dd>

Lo usan los administradores del sistema para controlar las configuraciones de usuario y equipo desde una única ubicación en una red. Las directivas del sistema propagan la configuración del Registro a un gran número de equipos sin necesidad de que el administrador tenga conocimientos detallados del registro. La directiva del sistema del instalador se puede configurar mediante el editor de directiva de grupo (GPE), incluido en Windows 2000. Para obtener más información, vea [Directiva del sistema](system-policy.md).

</dd> </dl>
