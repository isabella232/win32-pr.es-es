# <a name="ice-105"></a>ICE 105

Valida el paquete con un conjunto de requisitos para la instalación en el contexto por usuario. ICE 105 también puede validar paquetes de doble propósito. Un paquete de doble propósito permite al usuario seleccionar si desea instalar la aplicación en el contexto por usuario o por equipo. Para obtener información sobre cómo desarrollar un paquete de doble propósito, vea [Creación de paquetes únicos.](single-package-authoring.md)

ICE 105 realiza la siguiente validación del paquete.

-   Comprueba que la [tabla CustomAction](customaction-table.md) no contiene acciones personalizadas marcadas para ejecutarse con privilegios elevados. Para obtener más información sobre las acciones personalizadas con privilegios elevados, vea [Custom Action Security](custom-action-security.md).
-   Comprueba que la [tabla Directory](directory-table.md) no incluye ninguna de las siguientes propiedades de carpeta del sistema.
    -   [**AdminToolsFolder**](admintoolsfolder.md)
    -   [**CommonAppDataFolder**](commonappdatafolder.md)
    -   [**FontsFolder**](fontsfolder.md)
    -   [**System16Folder**](system16folder.md)
    -   [**System64Folder**](system64folder.md)
    -   [**SystemFolder**](systemfolder.md)
    -   [**TempFolder**](tempfolder.md)
    -   [**WindowsFolder**](windowsfolder.md)
    -   [**WindowsVolume**](windowsvolume.md)
-   Comprueba que el paquete no instala un ensamblado de Common Language Runtime en la caché global de ensamblados (GAC). Para obtener más información sobre cómo instalar ensamblados en la caché global de ensamblados, vea Agregar ensamblados [a](adding-assemblies-to-a-package.md) un paquete e Instalación de ensamblados de [Common Language Runtime.](installation-of-common-language-runtime-assemblies.md)
-   Comprueba la [tabla ODBCDataSource](odbcdatasource-table.md) para comprobar que el paquete no instala ningún origen de datos.
-   Comprueba la [tabla ServiceInstall](serviceinstall-table.md) para comprobar que el paquete no instala ningún servicio.
-   Comprueba que la [tabla del](registry-table.md) Registro no escribe ninguna entrada en la **clave HKEY LOCAL \_ \_ MACHINE.**

**[Windows instalador 4.5 o anterior:](not-supported-in-windows-installer-4-5.md)** No se admite. Este ICE está disponible a partir de Windows Installer 5.0.
