# <a name="setting-public-property-values-on-the-command-line"></a>Establecer valores de propiedad pública en la línea de comandos

Para establecer una propiedad pública en un valor de cadena literal, incluya la cadena entre comillas.

PROPERTY = "string"

Las comillas solo son necesarias si la cadena contiene espacios. Para borrar una propiedad pública en una línea de comandos (estableciendo en Null), establezca el valor de la propiedad en una cadena vacía. En este caso, se requieren comillas.

PROPERTY=""..
