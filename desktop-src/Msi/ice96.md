# <a name="ice96"></a>ICE96

ICE96 comprueba que la acción [PublishFeatures](publishfeatures-action.md) y la [acción PublishProduct](publishproduct-action.md) se han especificado en la [tabla AdvtExecuteSequence](advtexecutesequence-table.md). Un paquete no se puede anunciar sin estas acciones.

## <a name="result"></a>Resultado

ICE96 publica las siguientes advertencias.



| Advertencia de ICE96                                                            | Descripción                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| La acción PublishFeatures es necesaria en la tabla AdvtExecuteSequence. | No se puede anunciar un paquete sin la acción PublishFeatures incluida en la tabla AdvtExecuteSequence. |
| La acción PublishProduct es necesaria en la tabla AdvtExecuteSequence.  | No se puede anunciar un paquete sin la acción PublishProduct incluida en la tabla AdvtExecuteSequence.  |
