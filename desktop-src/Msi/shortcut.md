# <a name="shortcut"></a>Acceso directo

El tipo de datos Acceso directo se usa normalmente en la columna Destino de la tabla [Acceso directo](shortcut-table.md). Si contiene corchetes ( \[ \] ), el destino de acceso directo se evalúa como una cadena con formato. De lo contrario, el acceso directo se evalúa como [identificador y](identifier.md) debe ser una clave externa válida en la [tabla Característica](feature-table.md).
