# <a name="featureinfodescription-property"></a>Propiedad FeatureInfo.Description

La **propiedad** Description es una propiedad de solo lectura que devuelve la descripción de la característica en la columna Descripción de la [tabla Feature](feature-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = FeatureInfo.Description
```
