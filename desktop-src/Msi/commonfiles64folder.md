# <a name="commonfiles64folder-property"></a>CommonFiles64Folder, propiedad

El instalador establece la **propiedad CommonFiles64Folder** en la ruta de acceso completa de la carpeta Common Files de 64 bits predefinida. La propiedad [**CommonFilesFolder**](commonfilesfolder.md) existente se establece en la carpeta de 32 bits correspondiente.

## <a name="remarks"></a>Comentarios

El instalador establece esta propiedad en archivos de 64 Windows. Esta propiedad no se usa en archivos de 32 Windows. Al usar archivos de 64 Windows, el valor puede ser "C: \\ Archivos de programa common \\ files".
