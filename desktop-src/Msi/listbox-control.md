# <a name="listbox-control"></a>Control ListBox

El control ListBox es un cuadro de lista normal que permite al usuario realizar una selección única a partir de una lista de valores predeterminados. Los valores posibles se leen de la [tabla Listbox](listbox-table.md). Para asociar una propiedad de cadena o entero, escriba el nombre de la propiedad en la columna Propiedad de la [tabla Control](control-table.md).

## <a name="control-attributes"></a>Atributos de control

Puede usar los atributos siguientes con este control. Para cambiar el valor de un atributo mediante un evento, suscriba el control a un control ControlEvent en la [tabla EventMapping](eventmapping-table.md) y enumézcalo en la columna Atributo . Escriba el identificador de ControlEvent en la columna Evento.

## <a name="remarks"></a>Observaciones

Este control se puede crear a partir de la clase LISTBOX mediante la [**función CreateWindowEx.**](/windows/win32/api/winuser/nf-winuser-createwindowexa) Tiene los **estilos WS \_ TABSTOP,** **WS \_ GROUP** y **WS \_ CHILD.** Si el bit Estilo de control ordenado está en, el control se crea con los estilos NOTIFY de LBS, **WS \_ VSCROLL** y **WS \_ BORDER;** de lo contrario, el control se crea con el estilo **ESTÁNDAR de \_** **LBS. \_**
