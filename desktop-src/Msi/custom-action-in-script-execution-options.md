# <a name="custom-action-in-script-execution-options"></a>Opciones de ejecución de In-Script acción personalizada

Puede usar las siguientes marcas de opción para especificar la ejecución en script de acciones personalizadas. Estas opciones copian el código de acción en el script de ejecución, reversión o confirmación. Para establecer una opción, agregue el valor de esta tabla al valor del campo Tipo de la [tabla CustomAction](customaction-table.md).

Tenga en cuenta **que msidbCustomActionTypeInScript** debe incluirse con cada una de estas opciones.
