# <a name="entity-relationship-diagram-legend"></a>Leyenda del diagrama de relaciones de entidad

Un círculo negro que conduce a un rombo abierto indica una relación uno a varios entre la clave principal de la primera tabla y la clave externa de la segunda tabla. Para obtener un ejemplo de esto, vea la ilustración del grupo [tablas del Registro](registry-tables-group.md). Extension es una clave principal de la tabla Extension y una clave externa de la tabla Verbo. Por lo tanto, una extensión se puede registrar para tener varios verbos, pero un verbo determinado solo se puede asociar a una extensión.
