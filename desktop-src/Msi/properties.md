# <a name="properties-windows-installer"></a>Propiedades (Windows Installer)

Las propiedades son variables globales que Windows Installer usa durante la instalación. En las secciones siguientes se describen las propiedades utilizadas por el instalador:

-   [Acerca de las propiedades](about-properties.md)
-   [Utilizar propiedades](using-properties.md)
-   [Referencia de propiedades](property-reference.md)

El término "propiedad" también hace referencia a un atributo de un objeto de automatización. Vea [interfaz de automatización](automation-interface.md).
