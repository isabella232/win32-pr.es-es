# <a name="ice-103"></a>ICE 103

ICE 103 comprueba los eventos de control [MsiPrint](msiprint-controlevent.md) y [MsiLaunchApp.](msilaunchapp-controlevent.md)

ICE 103 comprueba que el evento de control [MsiPrint](msiprint-controlevent.md) solo se usa en un cuadro de diálogo con un control [ScrollableText.](scrollabletext-control.md)

**[Windows Instalador 4.5 o anterior:](not-supported-in-windows-installer-4-5.md)** No se admite. Este ICE está disponible a partir de Windows Installer 5.0.
