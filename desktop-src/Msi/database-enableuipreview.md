# <a name="databaseenableuipreview-method"></a>Método Database.EnableUIPreview

El **método EnableUIPreview** del objeto [**Database**](database-object.md) facilita la creación de cuadros de diálogo y paneles al proporcionar la compatibilidad necesaria para ver los cuadros de diálogo de la interfaz de usuario almacenados en la base de datos del instalador. El método inicia el modo de vista previa devolviendo un objeto **Database de vista** previa. El modo de vista previa finaliza cuando se libera el objeto de vista previa.

## <a name="syntax"></a>Sintaxis


```JScript
Database.EnableUIPreview()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
