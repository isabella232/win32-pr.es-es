# <a name="clientproductcode-property"></a>Propiedad Client.ProductCode

Producto cliente del componente en cuestión. Esta propiedad llama a [**la función MsiEnumClientsEx.**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa)

**[Windows Installer 4.5 o versiones anteriores:](not-supported-in-windows-installer-4-5.md)** No se admite. Esta propiedad está disponible a partir de Windows Installer 5.0.
