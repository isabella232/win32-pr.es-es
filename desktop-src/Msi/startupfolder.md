# <a name="startupfolder-property"></a>Propiedad StartupFolder

El instalador establece la propiedad **StartupFolder** en la ruta de acceso completa de la carpeta de inicio. De forma predeterminada, esta propiedad se establece en la carpeta del usuario actual. Si existe un perfil "todos los usuarios" y se establece la propiedad [**AllUsers**](allusers.md) , esta propiedad se establece en la carpeta del perfil "todos los usuarios".
