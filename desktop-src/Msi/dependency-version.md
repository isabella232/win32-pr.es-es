# <a name="dependencyversion-property"></a>Dependency.Version, propiedad

La propiedad **Version** de solo lectura devuelve la versión del módulo que requiere la combinación actual.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Dependency.Version
```



## <a name="property-value"></a>Valor de propiedad
