# <a name="companyname-property"></a>Propiedad COMPANYNAME

La **propiedad COMPANYNAME** es la organización o empresa del usuario que realiza la instalación.

## <a name="default-value"></a>Valor predeterminado

Se puede colocar un valor predeterminado en la [tabla Property](property-table.md). Si no se establece la propiedad **COMPANYNAME,** el instalador la establece automáticamente mediante valores del Registro.

## <a name="remarks"></a>Comentarios

Establezca la [**propiedad NOCOMPANYNAME**](nocompanyname.md) para suprimir la configuración automática de **COMPANYNAME.**
