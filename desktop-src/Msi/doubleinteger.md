# <a name="doubleinteger"></a>DoubleInteger

El tipo de datos DoubleInteger es un valor entero de cuatro bytes. A menos que se restrinja lo contrario, el intervalo de valores válidos es de – 2.147.483.647 a + 2.147.483.647.
