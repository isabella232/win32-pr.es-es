# <a name="windows-installer-and-logo-requirements"></a>Windows Requisitos del instalador y del logotipo

Certified for Microsoft Windows Logo identifica los productos que se han comprobado a través de pruebas independientes para cumplir con la especificación de la aplicación para Windows. Microsoft desarrolló esta especificación en colaboración con los clientes y otros desarrolladores para proporcionar un mapa de ruta para crear aplicaciones confiables y fáciles de administrar. Los proveedores de software que cumplen con la especificación son aptos para el logotipo de Certified for Microsoft Windows y, a continuación, licencian el logotipo para su uso en el empaquetado de sus productos, publicidad, material coy otros materiales de marketing.

Para obtener más información sobre Windows Vista, Windows XP y Windows Server 2008 para las [aplicaciones, vea el Microsoft Partner Network]( https://partner.microsoft.com).

Para obtener más información sobre Windows Server 2003 y Windows 2000 para las aplicaciones, vea [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

Si va a crear un paquete de instalación para la aplicación, puede aprovechar las funcionalidades de Microsoft Windows Installer para ayudar a satisfacer varios de los requisitos de certificación. En la tabla siguiente se proporcionan vínculos a la información de Windows installer pertinente para algunos de los requisitos de certificación. Tenga en cuenta que no todos los requisitos de certificación que se deban en application specification dependen de Windows Installer.
