# <a name="installing-a-com-application-with-the-windows-installer"></a>Instalación de una aplicación COM+ con Windows Installer

Las siguientes acciones y tablas para instalar aplicaciones COM+ están disponibles en Windows Installer:

-   [Complus Table](complus-table.md)
-   [Acción RegisterComPlus](registercomplus-action.md)
-   [Acción UnregisterComPlus](unregistercomplus-action.md)

Para obtener información sobre cómo instalar aplicaciones COM+, consulte la sección Implementación de aplicaciones [COM+](../cossdk/deploying-com--applications.md) en la documentación de [COM+ (Servicios de](../cossdk/component-services-portal.md) componentes).
