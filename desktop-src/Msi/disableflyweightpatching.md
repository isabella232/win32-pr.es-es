# <a name="disableflyweightpatching"></a>DisableFlyWeightPatching

Si esta directiva de sistema por máquina está [](patch-optimization.md) establecida en 1 (uno), todas las opciones de optimización de revisiones se desactivarán durante la instalación.

Las [opciones de optimización](patch-optimization.md) de revisiones y la directiva DisableFlyWeightPatching están disponibles a partir Windows installer versión 3.0.

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ Local \_ MACHINE** \\ **Software** \\ **Policies** \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de datos

**REG \_ DWORD**
