# <a name="condition"></a>Condición

El tipo de datos Condition es una cadena de texto que contiene una instrucción condicional válida que se puede evaluar como true o false. Para obtener información sobre la sintaxis de las instrucciones condicionales, vea [Sintaxis de instrucciones condicionales.](conditional-statement-syntax.md)
