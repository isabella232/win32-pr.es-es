# <a name="icem01"></a>ICEM01

ICEM01 valida que el mecanismo ICE funciona. Este ICE usa la [**propiedad Time**](time.md) para obtener la hora y devuelve la hora del sistema o Ninguna.

Los ICE del módulo de mezcla se almacenan en un archivo .uu. del módulo de mezcla denominado Mergemod.uu y no en el archivo .uu. que contiene los ICE usados para la validación del paquete.

## <a name="result"></a>Resultado

ICEM01 publica un mensaje que indica la hora a la que el instalador llamó a ICEM01. Este ICE nunca debe devolver un error.
