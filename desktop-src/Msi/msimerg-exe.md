# <a name="msimergexe"></a>Msimerg.exe

Msimerg.exe es una utilidad de línea de comandos que usa [**MsiDatabaseMerge**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabasemergea) para [combinar una](merges-and-transforms.md) base de datos de referencia en una base de datos base.

Esta herramienta solo está disponible en los componentes del [SDK de Windows para Windows programadores del instalador](platform-sdk-components-for-windows-installer-developers.md).

## <a name="syntax"></a>Sintaxis

**Msimerg** *{base de datos}{base de datos de referencia}*

Si hay un conflicto de combinación entre las dos bases de datos, la información de conflicto se coloca en la \_ tabla MergeErrors.

## <a name="command-line-options"></a>Opciones de la línea de comandos

Ninguno.
