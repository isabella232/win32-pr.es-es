# <a name="installation-context"></a>Contexto de instalación

Windows El instalador puede instalar un paquete en un equipo en dos contextos de instalación: por equipo y por usuario. Se requiere una instalación por máquina del paquete para permitir que todos los usuarios del equipo accedan a la aplicación y la usen. Dado que una instalación por máquina realiza cambios en el sistema que afectan a todos los usuarios, es posible que los usuarios estándar con privilegios limitados no puedan instalar un paquete en el contexto por equipo sin obtener primero el permiso.

Puede especificar el contexto de instalación mediante la creación del paquete para la instalación por usuario o por equipo y mediante las propiedades [**ALLUSERS**](allusers.md) [**y MSIINSTALLPERUSER.**](msiinstallperuser.md) En función de estas propiedades, Windows Installer redirige automáticamente los valores de propiedades de carpeta y registros a ubicaciones para el contexto por usuario o por equipo.

> [!Note]  
> La propiedad [**MSIINSTALLPERUSER,**](msiinstallperuser.md) disponible a partir de Windows Installer 5.0 y Windows Server 2008 R2 y Windows 7, puede facilitar el desarrollo de un único paquete capaz de instalarse en el contexto por equipo o por usuario. Para obtener información sobre el desarrollo de un paquete de doble propósito que ofrece al usuario la capacidad de elegir el contexto de instalación en el momento de la instalación, vea [Single Package Authoring](single-package-authoring.md). Windows El instalador omite la **propiedad MSIINSTALLPERUSER** si el valor de [**ALLUSERS**](allusers.md) no es 2. Windows El instalador siempre restablece el valor de **ALLUSERS** a 1 cuando se instala en el contexto por equipo y restablece el valor de **ALLUSERS** a una cadena vacía ("") cuando se instala en el contexto por usuario.

 

## <a name="shortcut-redirection"></a>Redirección de accesos directos

En la tabla siguiente se comparan las ubicaciones de los accesos directos para los contextos de instalación por equipo y por usuario.



| Per-Machine contexto de instalación (ALLUSERS=1)                                                             | Per-User contexto de instalación (ALLUSERS="")                                                                                     |
|-----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| Las aplicaciones aparecen en Agregar o quitar programas en Panel de control para todos los usuarios del equipo. <br/> | Las aplicaciones solo aparecen en Agregar o quitar programas en Panel de control para los usuarios que han instalado las aplicaciones. <br/> |
| Los accesos directos se instalan en el perfil Todos los usuarios.<br/>                                              | Los accesos directos solo se instalan en el perfil de ese usuario.<br/>                                                                 |
| Los iconos y transformaciones se almacenan en %WINDOWS% \\ Installer \\ {ProductCode}.<br/>                        | Los iconos y transformaciones se almacenan en %USERPROFILE% \\ Application Data Microsoft Installer \\ \\ \\ {ProductCode GUID}<br/>         |
