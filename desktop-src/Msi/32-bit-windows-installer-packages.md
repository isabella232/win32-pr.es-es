# <a name="32-bit-windows-installer-packages"></a>Paquetes del instalador de Windows de 32 bits

Un paquete de 32 bits consta solo de componentes de [Windows Installer](windows-installer-components.md) de 32 bits y debe [](template-summary.md) tener el valor "Intel" especificado en el campo de plataforma de la propiedad Resumen de plantilla.

Para obtener más información, vea Windows Installer en sistemas operativos de [64](windows-installer-on-64-bit-operating-systems.md) bits y paquetes del instalador [de Windows de 64 bits.](64-bit-windows-installer-packages.md)
