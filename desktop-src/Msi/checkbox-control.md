# <a name="checkbox-control"></a>Control CheckBox

Este control CheckBox \_ es una casilla de verificación de dos estados. Para asociar un entero o una propiedad de cadena a este control, escriba el nombre de propiedad en la columna Propiedad de la [tabla Control](control-table.md). El estado seleccionado del cuadro establece la propiedad en el valor especificado en la columna Valor de la tabla [CheckBox](checkbox-table.md) o en el valor inicial de la propiedad especificada en la [tabla Property](property-table.md). Si la propiedad no tiene ningún valor inicial, el estado activado lo establece en 1. El estado no seleccionado establece la propiedad en NULL.

Los controles CheckBox solo se pueden usar para publicar controles [AddLocal ControlEvent,](addlocal-controlevent.md) [AddSource ControlEvent,](addsource-controlevent.md) [Remove ControlEvent,](remove-controlevent.md) [DoAction ControlEvent](doaction-controlevent.md)o [SetProperty ControlEvent.](setproperty-controlevent.md)

## <a name="remarks"></a>Observaciones

Este control se puede crear a partir de la clase BUTTON mediante [**la función CreateWindowEx.**](/windows/win32/api/winuser/nf-winuser-createwindowexa) Tiene los estilos **BS \_ CHECKBOX**, **WS \_ TABSTOP,** **WS \_ GROUP,** **WS \_ CHILD** y **BS \_ MULTILINE.**
