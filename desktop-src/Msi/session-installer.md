# <a name="sessioninstaller-property"></a>Propiedad Session.Installer

La propiedad Installer de **solo** lectura del [**objeto Session**](session-object.md) devuelve el objeto **Installer** activo.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Session.Installer
```



## <a name="property-value"></a>Valor de propiedad
