# <a name="idcompositionrotatetransformsetangle-methods"></a>Métodos IDCompositionRotateTransform::SetAngle

Cambia o anima el valor de la propiedad Angle de una transformación de rotación. La propiedad Angle especifica el ángulo de rotación, en grados. El valor predeterminado es cero.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                     | Descripción                                          |
|:-----------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAngle(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(float))                                     | Cambia el valor de la propiedad Angle.<br/>  |
| [**SetAngle(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(idcompositionanimation)) | Anima el valor de la propiedad Angle.<br/> |
