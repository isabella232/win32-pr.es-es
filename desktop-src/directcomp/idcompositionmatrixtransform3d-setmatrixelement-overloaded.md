# <a name="idcompositionmatrixtransform3dsetmatrixelement-methods"></a>Métodos IDCompositionMatrixTransform3D::SetMatrixElement

Cambia o anima el valor de un elemento de la matriz de esta transformación 3D.

| Método                                                                                                                                 | Descripción                                                                      |
|:---------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------|
| [**SetMatrixElement(int, int, float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionmatrixtransform3d-setmatrixelement(int_int_float))                                     | Cambia el valor de un elemento de la matriz de esta transformación 3D.<br/>  |
| [**SetMatrixElement(int, int, IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionmatrixtransform3d-setmatrixelement(int_int_idcompositionanimation)) | Anima el valor de un elemento de la matriz de esta transformación 3D.<br/> |
