# <a name="tf_-constants"></a>Constantes \_ \* de TF

Las llamadas a funciones atom de TSF usan estas constantes para administrar los procesos de entrada de subprocesos.



| Constante o valor                                                                                                                                                                                                                                                                       | Descripción                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**TF \_ PROCESAR \_ TEXTO ATOM(**</dt> <dt> \_ CTF PROCESS ATOM \_ \_ \_ )</dt> </dl>                               | Se usa para notificar a las funciones atom que se ha iniciado un proceso de entrada de subproceso.<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**TF \_ ENABLE \_ PROCESS \_ ATOM**</dt> <dt>TEXT( \_ CTF ENABLE PROCESS ATOM \_ \_ \_ \_ )</dt> </dl> | Se usa para notificar a TSF, a través de funciones atom, que se ha habilitado un proceso de entrada de subproceso.<br/> |
