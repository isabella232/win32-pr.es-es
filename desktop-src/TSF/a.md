# <a name="a-text-services-framework"></a>A (Text Services Framework)

<dl> <dt>

<span id="_tsf_anchor_gly"></span><span id="_TSF_ANCHOR_GLY"></span>**Ancla**
</dt> <dd>

Uno de un conjunto de elementos que marcan el inicio y el final de un intervalo de texto. Vea [Delimitadores.](ranges.md)

</dd> <dt>

<span id="_tsf_application_character_position_gly"></span><span id="_TSF_APPLICATION_CHARACTER_POSITION_GLY"></span>**posición del carácter de aplicación (ACP)**
</dt> <dd>

Posición de carácter de base cero, dentro de una secuencia de texto continua, relativa al inicio de la secuencia. Una ACP se indica con un valor entero largo. Vea [Almacenes de texto.](text-stores.md)

</dd> </dl>
