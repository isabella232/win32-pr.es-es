# <a name="d-text-services-framework"></a>D (Text Services Framework)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**bloqueo de documento**
</dt> <dd>

Estado del documento en el que el administrador de TSF puede leer o modificar texto sin que el texto se cambie por ningún otro medio. Un bloqueo de documento puede ser de solo lectura o de lectura y escritura.

</dd> </dl>
