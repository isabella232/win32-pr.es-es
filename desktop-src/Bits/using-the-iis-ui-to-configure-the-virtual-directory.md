# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Uso de la interfaz de usuario de IIS para configurar el directorio virtual

Para usar la interfaz de usuario de IIS para  configurar el directorio virtual para las cargas de BITS, seleccione Propiedades en el menú contextual del directorio virtual. Presione la **pestaña Extensión bits** para acceder a la página de propiedades de BITS (la extensión del servidor BITS ISAPI debe estar instalada). Cambie los valores de propiedad predeterminados según corresponda y habilite el directorio para las cargas. no puede cargar archivos hasta que el directorio esté habilitado. Para más información sobre cómo establecer los valores de propiedad, consulte la Ayuda en línea.
