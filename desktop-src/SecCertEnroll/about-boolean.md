# <a name="boolean-certificate-enrollment-api"></a>BOOLEAN (API de inscripción de certificados)

Un valor booleano puede ser **TRUE** o **FALSE.** **FALSE** se codifica como un triplete TLV en el que el **campo Valor** es cero (0). **TRUE** se codifica como un valor distinto de cero. La **etiqueta** para el **tipo de** datos BOOLEAN es 0x01.
