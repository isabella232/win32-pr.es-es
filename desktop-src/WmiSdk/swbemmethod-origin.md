# <a name="swbemmethodorigin-property"></a>SWbemMethod.Origin, propiedad

La **propiedad Origin** del objeto [**SWbemMethod**](swbemmethod.md) devuelve el nombre de la clase en la que se introdujo este método. Para las clases con jerarquías de herencia profunda, a menudo es conveniente saber qué métodos se declararon en qué clases. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemMethod.Origin As String
```
