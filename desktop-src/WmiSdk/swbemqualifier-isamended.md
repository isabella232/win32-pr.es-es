# <a name="swbemqualifierisamended-property"></a>Propiedad SWbemQualifier. IsAmended

La propiedad **IsAmended** del objeto [**SWbemQualifier**](swbemqualifier.md) es un valor booleano que indica si este calificador se ha adaptado mediante una operación de combinación.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemQualifier.IsAmended As Boolean
```
