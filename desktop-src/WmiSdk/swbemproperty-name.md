# <a name="swbempropertyname-property"></a>SWbemProperty.Name propiedad

La **propiedad Name** del objeto [**SWbemProperty**](swbemproperty.md) devuelve el nombre de esta propiedad WMI. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemProperty.Name As String
```
