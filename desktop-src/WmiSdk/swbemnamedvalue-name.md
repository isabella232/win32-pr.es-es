# <a name="swbemnamedvaluename-property"></a>SWbemNamedValue.Name propiedad

La **propiedad Name** del objeto [**SWbemNamedValue**](swbemnamedvalue.md) es una cadena única que no tiene en cuenta mayúsculas de minúsculas que se usa al obtener un elemento **SWbemNamedValue.** Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemNamedValue.Name As String
```
