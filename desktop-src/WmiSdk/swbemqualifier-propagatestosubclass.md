# <a name="swbemqualifierpropagatestosubclass-property"></a>Propiedad SWbemQualifier. PropagatesToSubClass

La propiedad **PropagatesToSubClass** del objeto [**SWbemQualifier**](swbemqualifier.md) es un valor booleano que indica si este calificador se puede propagar a una subclase.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemQualifier.PropagatesToSubClass
```
