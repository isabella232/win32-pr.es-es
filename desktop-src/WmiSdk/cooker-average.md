# <a name="cooker_average"></a>COOKER \_ AVERAGE

La fórmula de \_ tipo de contador COOKER AVERAGE proporciona el punto medio estadístico de los datos de un contador. La fórmula para este tipo de contador suma observaciones repetidas de una propiedad en una instancia [**\_ de PerfRawData de Win32**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) y divide la suma por el número de observaciones. Este tipo de contador solo se define dentro de WMI y no está disponible para las tecnologías de supervisión de rendimiento, como contadores de rendimiento [versión 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Para obtener más información sobre la fórmula de tipo de contador, vea [Tipos de contador](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obtener más información sobre los proveedores de alto rendimiento y el scripting de WMI, vea [Tipos de contadores de rendimiento de WMI.](wmi-performance-counter-types.md)
