# <a name="swbemlasterrorproperties_-property"></a>Propiedad SWbemLastError.Properties \_

La **\_ propiedad Properties** del objeto [**SWbemLastError**](swbemlasterror.md) devuelve un objeto [**SWbemPropertySet**](swbempropertyset.md) que es una colección de las propiedades WMI de la clase o instancia actual. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemLastError.Properties_ As Object
```
