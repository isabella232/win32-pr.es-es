# <a name="swbemobjectpathserver-property"></a>SWbemObjectPath. Server (propiedad)

La propiedad **Server** del objeto [**SWbemObjectPath**](swbemobjectpath.md) contiene el nombre del servidor que forma parte de la ruta de acceso del objeto.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemObjectPath.Server As String
```
