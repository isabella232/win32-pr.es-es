# <a name="swbemprivilegeidentifier-property"></a>Propiedad SWbemPrivilege. Identifier

La propiedad **Identifier** de un objeto [**SWbemPrivilege**](swbemprivilege.md) es un entero [WbemPrivilegeEnum](/windows/desktop/api/Wbemdisp/ne-wbemdisp-wbemprivilegeenum) que representa el privilegio que se va a establecer o recuperar. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemPrivilege.Identifier As Integer
```
