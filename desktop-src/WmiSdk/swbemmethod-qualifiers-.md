# <a name="swbemmethodqualifiers_-property"></a>Propiedad SWbemMethod.Qualifiers \_

La **propiedad \_ Qualifiers** del objeto [**SWbemMethod**](swbemmethod.md) devuelve un objeto [**SWbemQualifierSet**](swbemqualifierset.md) que es la colección de calificadores para este método. Esta propiedad es de solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemMethod.Qualifiers_ As Object
```
