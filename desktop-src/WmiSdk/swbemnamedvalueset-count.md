# <a name="swbemnamedvaluesetcount-property"></a>SWbemNamedValueSet.Count, propiedad

Use la **propiedad Count** del [**objeto SWbemNamedValueSet**](swbemnamedvalueset.md) para determinar cuántos elementos hay en la colección. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemNamedValueSet.Count As Integer
```
