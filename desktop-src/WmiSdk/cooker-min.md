# <a name="cooker_min"></a>COOKER \_ MIN

La fórmula de tipo de contador COOKER MIN identifica el valor más pequeño de un conjunto de observaciones de una propiedad en una instancia \_ [**de \_ Win32 PerfRawData.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) Este valor mínimo calculado se coloca en una propiedad de una [**instancia \_ de PerfFormattedData de Win32.**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) Este tipo de contador solo se define dentro de WMI y no está disponible para las tecnologías de supervisión de rendimiento, como contadores de rendimiento [versión 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Para obtener más información sobre la fórmula de tipo de contador, vea [Tipos de contador](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obtener más información sobre los proveedores de alto rendimiento y scripting de WMI, vea [Tipos de contadores de rendimiento de WMI.](wmi-performance-counter-types.md)
