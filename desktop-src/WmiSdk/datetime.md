# <a name="datetime"></a>DATETIME

El tipo de datos DATETIME es una cadena BSTR de VT de longitud fija que representa una fecha y hora \_ o un intervalo específicos. Para obtener más información sobre DATETIME como una fecha, vea [Formato de fecha y hora](date-and-time-format.md). Para obtener más información sobre DATETIME como un intervalo, vea [Formato de intervalo](interval-format.md).
