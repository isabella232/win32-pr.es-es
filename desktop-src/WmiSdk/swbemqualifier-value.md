# <a name="swbemqualifiervalue-property"></a>SWbemQualifier.Value, propiedad

La **propiedad Value** del objeto [**SWbemQualifier**](swbemqualifier.md) contiene el valor variant de este calificador. Esta es la propiedad predeterminada de este objeto .

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemQualifier.Value As Variant
```
