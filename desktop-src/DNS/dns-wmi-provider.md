# <a name="dns-wmi-provider"></a>Proveedor WMI de DNS

El proveedor WMI de DNS permite a los administradores y programadores configurar registros de recursos (RR) del Sistema de nombres de dominio (DNS) y servidores DNS mediante Windows Management Instrumentation (WMI).
