<dl> <dt>

<span id="_dns_relative_distinguished_name_gly"></span><span id="_DNS_RELATIVE_DISTINGUISHED_NAME_GLY"></span>**nombre distintivo relativo**
</dt> <dd>

Nombre que identifica de forma única un host dentro de su propio dominio, pero no en toda la jerarquía DNS. Por ejemplo, server1 es un nombre distintivo relativo, mientras que server1.products.Microsoft.com es un nombre de dominio completo (FQDN).

</dd> <dt>

<span id="_dns_resolver_gly"></span><span id="_DNS_RESOLVER_GLY"></span>**Resolución**
</dt> <dd>

Componente de software incluido en la pila Windows Protocolo de control de transmisión/Protocolo de Internet (TCP/IP) que se comunica con los servidores DNS para crear y resolver consultas de resolución de nombres.

</dd> <dt>

<span id="_dns_resource_record_rr__gly"></span><span id="_DNS_RESOURCE_RECORD_RR__GLY"></span>**registro de recursos (RR)**
</dt> <dd>

Unidad de información en el DNS. Hay muchos tipos de registros de recursos.

</dd> <dt>

<span id="_dns_root_gly"></span><span id="_DNS_ROOT_GLY"></span>**Raíz**
</dt> <dd>

Nivel superior de una jerarquía determinada.

</dd> </dl>
