# <a name="embedding-information-chunks-in-an-avi-file"></a>Insertar fragmentos de información en un archivo AVI

Puede insertar fragmentos de información, como texto o datos personalizados, en un archivo AVI mediante el mensaje [**WM CAP FILE SET \_ \_ \_ \_ INFOCHUNK**](wm-cap-file-set-infochunk.md) (o la macro [**capFileSetInfoChunk).**](/windows/desktop/api/Vfw/nf-vfw-capfilesetinfochunk) También puede usar este mensaje para borrar fragmentos de información de un archivo AVI.
