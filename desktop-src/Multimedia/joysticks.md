# <a name="joysticks"></a>Joysticks

En esta información general se describen las funciones y los mensajes que admiten guiones, así como otros dispositivos de entrada auxiliares que realicen un seguimiento de las posiciones dentro de un sistema de coordenadas absoluto, como una pantalla táctil, la digitalización de tabletas y un lápiz claro. Las funcionalidades extendidas también proporcionan compatibilidad con los aceleradores, los yochos de vuelo y otros dispositivos que usan hasta seis ejes de movimiento, un casco de punto de vista y 32 botones.

-   [Acerca de los contrabandos](about-joysticks.md)
-   [Uso de contrabandos](using-joysticks.md)
-   [Referencia de los resaltes](joystick-reference.md)

La API multimedia multimedia se ha reemplazado por DirectInput, que tiene más funcionalidades para los dispositivos de entrada y de comentarios de fuerza de hoy en día. Para más información, consulte la documentación de DirectX.
