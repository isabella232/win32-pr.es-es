# <a name="joysticks"></a>Joysticks

En esta información general se describen las funciones y los mensajes que admiten los televisores, así como otros dispositivos auxiliares de entrada que realicen un seguimiento de las posiciones dentro de un sistema de coordenadas absoluto, como una pantalla táctil, una tableta digitalizadora y un lápiz claro. Las funcionalidades extendidas también proporcionan compatibilidad con cascos de punta, yochos de vuelo y otros dispositivos que usan hasta seis ejes de movimiento, un casco de punto de vista y 32 botones.

-   [Acerca de los contrabandos](about-joysticks.md)
-   [Uso de Contrabandos](using-joysticks.md)
-   [Referencia de los puertos](joystick-reference.md)

DirectInput ha reemplazado la API multimedia multimedia, que tiene más funcionalidades para los dispositivos de entrada y de comentarios de fuerza actuales. Para más información, consulte la documentación de DirectX.
