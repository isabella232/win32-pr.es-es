# <a name="the-scope-resolution-operator-in-c"></a>Operador de resolución de ámbito en C++

Dos puntos (::) se usan en C++ como operador *de resolución de* ámbito. Este operador proporciona más libertad para asignar un nombre a las variables, ya que permite distinguir entre variables con el mismo nombre. Por ejemplo, *MyFile::Read* hace referencia al método *Read* de la clase *MyFile* de objetos , en lugar de *YourFile::Read*, que hace referencia a un método *Read* en la *clase YourFile.*
