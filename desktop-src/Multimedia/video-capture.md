# <a name="video-capture-windows-multimedia"></a>Captura de vídeo (Windows Multimedia)

Puede incorporar fácilmente funcionalidades de captura de vídeo a la aplicación mediante la clase de ventana AVICap. AVICap proporciona a las aplicaciones una interfaz sencilla basada en mensajes para acceder al hardware de adquisición de audio de forma de onda y vídeo y controlar el proceso de captura de vídeo de streaming en el disco.

-   [Uso de captura de vídeo](using-video-capture.md)
-   [Referencia de captura de vídeo](video-capture-reference.md)
