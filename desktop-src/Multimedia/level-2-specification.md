# <a name="level-2-specification"></a>Especificación de nivel 2

La especificación de nivel 2 se desarrolló para fomentar la adopción de funcionalidades multimedia mejoradas. Esta especificación se basa en los requisitos establecidos en la especificación de nivel 1 y es un superconjunto de ella. La especificación de nivel 2 define la funcionalidad mínima del sistema para las funcionalidades multimedia mejoradas. No es una recomendación para una configuración del sistema.
