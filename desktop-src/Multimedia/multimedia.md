# <a name="multimedia"></a>Contenido multimedia

Este glosario contiene definiciones de los términos usados en la documentación multimedia de Windows.

<dl> <dt>

<span id="_win32_adaptive_differential_pulse_code_modulation_adpcm_gloss"></span><span id="_WIN32_ADAPTIVE_DIFFERENTIAL_PULSE_CODE_MODULATION_ADPCM_GLOSS"></span>**Modulación de código de impulso diferencial adaptable (ADPCM)**
</dt> <dd>

Técnica de compresión de audio.

</dd> <dt>

<span id="_win32_adpcm_gloss"></span><span id="_WIN32_ADPCM_GLOSS"></span>**ADPCM**
</dt> <dd>

Consulte [modulación de código de impulso diferencial adaptable (ADPCM)](/windows).

</dd> <dt>

<span id="_win32_break_key_gloss"></span><span id="_WIN32_BREAK_KEY_GLOSS"></span>**romper clave**
</dt> <dd>

En la interfaz de control de medios (MCI), pulsación de tecla que interrumpe una operación de espera. De forma predeterminada, MCI define esta clave como CTRL + INTER. Una aplicación puede volver a definir esta clave mediante el mensaje de comando de interrupción de MCI \_ .

</dd> <dt>

<span id="_win32_cd_rom_extended_architecture_cd_xa_gloss"></span><span id="_WIN32_CD_ROM_EXTENDED_ARCHITECTURE_CD_XA_GLOSS"></span>**Arquitectura extendida de CD-ROM (CD-XA)**
</dt> <dd>

Extensión del estándar de CD-ROM que permite el almacenamiento de datos de audio comprimidos junto con otros datos en un disco compacto. Este estándar también define la forma en que se leen los datos de un disco. Las señales de audio se combinan con datos de texto y gráficos en una sola pista para que se puedan leer prácticamente al mismo tiempo.

</dd> <dt>

<span id="_win32_channel_gloss"></span><span id="_WIN32_CHANNEL_GLOSS"></span>**canal** de
</dt> <dd>

Un método, proporcionado por la interfaz digital de instrumentos musicales (MIDI), para enviar mensajes a un dispositivo individual dentro de una instalación de MIDI. Hay 16 números de canal MIDI. Los dispositivos de una configuración de MIDI se pueden dirigir para responder solo a los mensajes marcados con un número de canal específico del dispositivo.

</dd> <dt>

<span id="_win32_channel_map_gloss"></span><span id="_WIN32_CHANNEL_MAP_GLOSS"></span>**mapa de canal**
</dt> <dd>

Un mapa de canal, proporcionado por el asignador de MIDI, que puede redirigir los mensajes de la interfaz digital de instrumentos musicales (MIDI) de un canal a otro. Vea también [Mapeador MIDI](/windows), [interfaz digital de instrumentos musicales (MIDI)](/windows).

</dd> <dt>

<span id="_win32_chunk_gloss"></span><span id="_WIN32_CHUNK_GLOSS"></span>**fragmento** de
</dt> <dd>

El bloque de creación básico de un archivo de formato de archivo de intercambio de recursos (RIFF), que consta de un identificador (denominado identificador de fragmento), una variable de tamaño de fragmento y un área de datos de fragmentos de tamaño variable.

</dd> <dt>

<span id="_win32_command_message_gloss"></span><span id="_WIN32_COMMAND_MESSAGE_GLOSS"></span>**mensaje de comando**
</dt> <dd>

En la interfaz de control de medios (MCI), es una constante simbólica que representa un comando único para un dispositivo MCI. Los mensajes de comandos tienen estructuras de datos asociadas que proporcionan información que necesita un dispositivo para llevar a cabo una solicitud.

</dd> <dt>

<span id="_win32_command_string_gloss"></span><span id="_WIN32_COMMAND_STRING_GLOSS"></span>**cadena de comandos**
</dt> <dd>

En la interfaz de control de medios (MCI), cadena de caracteres terminada en null que representa un comando para un dispositivo MCI. La cadena de texto contiene toda la información que un dispositivo MCI necesita para llevar a cabo una solicitud. MCI analiza la cadena de texto y la convierte en un mensaje de comando y una estructura de datos equivalentes que envía a un controlador de dispositivo MCI.

</dd> <dt>

<span id="_win32_compact_disc_digital_audio_cd_da_gloss"></span><span id="_WIN32_COMPACT_DISC_DIGITAL_AUDIO_CD_DA_GLOSS"></span>**disco compacto: audio digital (CD-da)**
</dt> <dd>

Un formato de almacenamiento de datos óptico que proporciona el almacenamiento de hasta 73 minutos de datos de audio digital de alta calidad en un disco compacto. También conocido como audio de libro rojo.

</dd> <dt>

<span id="_win32_compact_disc_read_only_memory_cd_rom_gloss"></span><span id="_WIN32_COMPACT_DISC_READ_ONLY_MEMORY_CD_ROM_GLOSS"></span>**disco compacto: memoria de solo lectura (CD-ROM)**
</dt> <dd>

Tecnología de almacenamiento de datos óptico que permite almacenar grandes cantidades de datos en un disco compacto.

</dd> <dt>

<span id="_win32_compound_device_gloss"></span><span id="_WIN32_COMPOUND_DEVICE_GLOSS"></span>**dispositivo compuesto**
</dt> <dd>

Un dispositivo de media control Interface (MCI) que requiere un elemento de dispositivo, normalmente un archivo de datos. Un ejemplo de dispositivo compuesto es el controlador de audio de forma de onda de MCI. Vea también [elemento Device](/windows).

</dd> <dt>

<span id="_win32_control_change_gloss"></span><span id="_WIN32_CONTROL_CHANGE_GLOSS"></span>**cambio de control**
</dt> <dd>

Consulte [control MIDI-Change Message](/windows).

</dd> <dt>

<span id="_win32_device_element_gloss"></span><span id="_WIN32_DEVICE_ELEMENT_GLOSS"></span>**elemento Device**
</dt> <dd>

Datos necesarios para el funcionamiento de los dispositivos compuestos de la interfaz de control de medios (MCI). Normalmente, el elemento de dispositivo es un archivo de datos de entrada o de salida.

</dd> <dt>

<span id="_win32_division_type_gloss"></span><span id="_WIN32_DIVISION_TYPE_GLOSS"></span>**tipo de división**
</dt> <dd>

La técnica que se usa para representar el tiempo entre los eventos de la interfaz digital de instrumentos musicales (MIDI) en un secuenciador MIDI.

</dd> <dt>

<span id="_win32_file_element_gloss"></span><span id="_WIN32_FILE_ELEMENT_GLOSS"></span>**File, elemento**
</dt> <dd>

Un archivo completo incluido en un archivo compuesto de formato de archivo de intercambio de recursos (RIFF).

</dd> <dt>

<span id="_win32_fm_synthesizer_gloss"></span><span id="_WIN32_FM_SYNTHESIZER_GLOSS"></span>**Sintetizador FM**
</dt> <dd>

Vea [sintetizador de frecuencia (FM)](/windows).

</dd> <dt>

<span id="_win32_fourcc_four_character_code_gloss"></span><span id="_WIN32_FOURCC_FOUR_CHARACTER_CODE_GLOSS"></span>**FourCC (código de cuatro caracteres)**
</dt> <dd>

Código que se usa para identificar fragmentos de formato de archivo de intercambio de recursos (RIFF). Un FOURCC es una cantidad de 32 bits representada como una secuencia de uno a cuatro caracteres ASCII alfanuméricos, que se rellenan a la derecha con caracteres en blanco.

</dd> <dt>

<span id="_win32_frequency_modulation_fm_synthesizer_gloss"></span><span id="_WIN32_FREQUENCY_MODULATION_FM_SYNTHESIZER_GLOSS"></span>**sintetizador de frecuencia (FM)**
</dt> <dd>

Un sintetizador que crea sonidos mediante la combinación de la salida de los osciladores digitales mediante una técnica de modulación de frecuencia.

</dd> <dt>

<span id="_win32_general_midi_gloss"></span><span id="_WIN32_GENERAL_MIDI_GLOSS"></span>**MIDI general**
</dt> <dd>

Una especificación de sintetizador creada por la Asociación de fabricantes de MIDI (MMA) que define una configuración común y un conjunto de funcionalidades para los sintetizadores de interfaz digital de instrumentos musicales (MIDI) del consumidor.

</dd> <dt>

<span id="_win32_hms_time_format_gloss"></span><span id="_WIN32_HMS_TIME_FORMAT_GLOSS"></span>**Formato de hora de HMS**
</dt> <dd>

Formato de hora que usa la interfaz de control de medios (MCI) para expresar la hora en horas, minutos y segundos. El formato de hora de HMS se usa principalmente en dispositivos de videodisco.

</dd> <dt>

<span id="_win32_ima_gloss"></span><span id="_WIN32_IMA_GLOSS"></span>**Certificado IMA**
</dt> <dd>

Consulte la [Asociación interactiva de multimedia (IMA)](/windows) y la [Asociación de MIDI Internacional (IMA)](/windows).

</dd> <dt>

<span id="_win32_interactive_multimedia_association_ima_gloss"></span><span id="_WIN32_INTERACTIVE_MULTIMEDIA_ASSOCIATION_IMA_GLOSS"></span>**Interactive Multimedia Association (IMA)**
</dt> <dd>

Asociación profesional de empresas, instituciones y personas implicadas en la producción y el uso de tecnología multimedia interactiva.

</dd> <dt>

<span id="_win32_international_midi_association_ima_gloss"></span><span id="_WIN32_INTERNATIONAL_MIDI_ASSOCIATION_IMA_GLOSS"></span>**Asociación MIDI Internacional (IMA)**
</dt> <dd>

La organización sin ánimo de lucro que circula información acerca de la especificación de interfaz digital de instrumentos musicales (MIDI).

</dd> <dt>

<span id="_win32_list_chunk_gloss"></span><span id="_WIN32_LIST_CHUNK_GLOSS"></span>**Lista de fragmentos**
</dt> <dd>

Un fragmento de formato de archivo de intercambio de recursos (RIFF) con un identificador de fragmento de lista. Los fragmentos de lista contienen una serie de subfragmentos.

</dd> <dt>

<span id="_win32_list_type_gloss"></span><span id="_WIN32_LIST_TYPE_GLOSS"></span>**tipo de lista**
</dt> <dd>

Un código de cuatro caracteres (FOURCC) que identifica el tipo de datos contenido en un fragmento de formato de archivo de intercambio de recursos (RIFF) con un identificador de fragmento de lista. Por ejemplo, un fragmento de lista con un tipo de lista de información contiene una lista de información sobre un archivo, como la fecha de creación y el autor.

</dd> <dt>

<span id="_win32_mci_gloss"></span><span id="_WIN32_MCI_GLOSS"></span>**MCI**
</dt> <dd>

Vea [interfaz de control de medios](/windows).

</dd> <dt>

<span id="_win32_media_control_interface_mci_gloss"></span><span id="_WIN32_MEDIA_CONTROL_INTERFACE_MCI_GLOSS"></span>**Media control Interface (MCI)**
</dt> <dd>

Software de control de alto nivel que proporciona una interfaz independiente del dispositivo para dispositivos multimedia y archivos multimedia. MCI incluye una interfaz de mensajes de comandos y una interfaz de cadena de comandos.

</dd> <dt>

<span id="_win32_midi_gloss"></span><span id="_WIN32_MIDI_GLOSS"></span>**MIDI**
</dt> <dd>

Consulte [interfaz digital de instrumentos musicales](/windows).

</dd> <dt>

<span id="_win32_midi_control_change_message_gloss"></span><span id="_WIN32_MIDI_CONTROL_CHANGE_MESSAGE_GLOSS"></span>**Control MIDI: cambiar mensaje**
</dt> <dd>

Un mensaje de interfaz digital de instrumentos musicales (MIDI) enviado a un sintetizador para cambiar la configuración de control de sintetizador diferente. Un ejemplo de un mensaje de cambio de control es el mensaje del controlador de volumen, que cambia el volumen de un canal MIDI específico.

</dd> <dt>

<span id="_win32_midi_manufacturers_association_mma_gloss"></span><span id="_WIN32_MIDI_MANUFACTURERS_ASSOCIATION_MMA_GLOSS"></span>**Asociación de fabricantes de MIDI (MMA)**
</dt> <dd>

Organización colectiva compuesta por fabricantes de instrumentos de instrumental digital de instrumentos musicales (MIDI) y compañías de software MIDI. MMA funciona con el Comité estándar de MIDI para mantener la especificación MIDI.

</dd> <dt>

<span id="_win32_midi_mapper_gloss"></span><span id="_WIN32_MIDI_MAPPER_GLOSS"></span>**Asignador MIDI**
</dt> <dd>

Software de sistemas Windows que modifica los mensajes de salida de la interfaz digital de instrumentos musicales (MIDI) y los redirige a un dispositivo de salida MIDI mediante los valores almacenados en un mapa de configuración de MIDI. El asignador MIDI puede cambiar el canal de destino y el dispositivo de salida de un mensaje, así como modificar los mensajes de cambio de programa, los valores de volumen y los valores de clave.

</dd> <dt>

<span id="_win32_midi_mapping_gloss"></span><span id="_WIN32_MIDI_MAPPING_GLOSS"></span>**Asignación MIDI**
</dt> <dd>

Proceso de traducción y redirección de mensajes de interfaz digital de instrumentos musicales (MIDI) según los datos definidos en una configuración de mapa MIDI.

</dd> <dt>

<span id="_win32_midi_program_change_message_gloss"></span><span id="_WIN32_MIDI_PROGRAM_CHANGE_MESSAGE_GLOSS"></span>**Programa MIDI: mensaje de cambio**
</dt> <dd>

Un mensaje de interfaz digital de instrumentos musicales (MIDI) enviado a un sintetizador para cambiar la revisión en un canal MIDI específico.

</dd> <dt>

<span id="_win32_midi_sequence_gloss"></span><span id="_WIN32_MIDI_SEQUENCE_GLOSS"></span>**Secuencia de MIDI**
</dt> <dd>

Datos de interfaz digital de instrumentos musicales (MIDI) que pueden ser reproducidos por un secuenciador MIDI.

</dd> <dt>

<span id="_win32_midi_sequencer_gloss"></span><span id="_WIN32_MIDI_SEQUENCER_GLOSS"></span>**Secuenciador MIDI**
</dt> <dd>

Programa que crea o reproduce canciones almacenadas como archivos de interfaz digital de instrumentos musicales (MIDI). Cuando un secuenciador reproduce archivos MIDI, envía datos MIDI desde el archivo a un sintetizador MIDI, lo que produce los sonidos. Windows proporciona un secuenciador MIDI, accesible a través de la interfaz de control de medios (MCI), que reproduce archivos MIDI. Vea también [interfaz de control de medios (MCI)](/windows).

</dd> <dt>

<span id="_win32_midi_setup_map_gloss"></span><span id="_WIN32_MIDI_SETUP_MAP_GLOSS"></span>**Mapa de configuración de MIDI**
</dt> <dd>

Un conjunto completo de datos para el asignador de interfaz digital de instrumentos musicales (MIDI) que se va a usar al redirigir los mensajes MIDI. Solo puede haber un mapa de configuración en vigor en un momento dado, pero el usuario puede tener varias asignaciones de instalación disponibles y puede elegir entre ellas mediante la opción del panel de control del asignador MIDI.

</dd> <dt>

<span id="_win32_mma_gloss"></span><span id="_WIN32_MMA_GLOSS"></span>**MMA**
</dt> <dd>

Consulte [Asociación de fabricantes de MIDI](/windows).

</dd> <dt>

<span id="_win32_msf_time_format_gloss"></span><span id="_WIN32_MSF_TIME_FORMAT_GLOSS"></span>**Formato de hora de MSF**
</dt> <dd>

Formato de hora que usa la interfaz de control de medios (MCI) para expresar el tiempo en minutos, segundos y fotogramas. El número de fotogramas en un segundo depende del tipo de dispositivo que se esté usando; los dispositivos de audio de disco compacto usan 75 fotogramas por segundo. El formato de hora de MSF se usa principalmente en dispositivos de audio de disco compacto.

</dd> <dt>

<span id="_win32_musical_instrument_digital_interface_midi_gloss"></span><span id="_WIN32_MUSICAL_INSTRUMENT_DIGITAL_INTERFACE_MIDI_GLOSS"></span>**Interfaz digital de instrumentos musicales (MIDI)**
</dt> <dd>

Protocolo estándar para la comunicación entre instrumentos musicales y equipos.

</dd> <dt>

<span id="_win32_parts_per_quarter_note_ppqn_gloss"></span><span id="_WIN32_PARTS_PER_QUARTER_NOTE_PPQN_GLOSS"></span>**elementos por nota de cuarto (PPQN)**
</dt> <dd>

Formato de hora que se usa para las secuencias de interfaz digital de instrumentos musicales (MIDI). PPQN es el formato de hora más común que se usa con archivos MIDI estándar.

</dd> <dt>

<span id="_win32_patch_gloss"></span><span id="_WIN32_PATCH_GLOSS"></span>**revisión**
</dt> <dd>

Una configuración determinada de un sintetizador de interfaz digital de instrumentos musicales (MIDI) que produce un sonido determinado, normalmente un sonido que simula un instrumento musical específico. Las revisiones también se denominan programas. Un mensaje de cambio de programa MIDI cambia la configuración de revisión en un sintetizador. Patch también hace referencia a la conexión o conexiones entre dispositivos MIDI. Consulte también [interfaz digital de instrumentos musicales (MIDI)](/windows).

</dd> <dt>

<span id="_win32_patch_caching_gloss"></span><span id="_WIN32_PATCH_CACHING_GLOSS"></span>**almacenamiento en caché de revisiones**
</dt> <dd>

Técnica que permite a algunos controladores de dispositivos de sintetizador de interfaz digital de instrumentos musicales (MIDI) internos cargar previamente los datos de revisión, lo que reduce el retraso entre el momento en que el sintetizador recibe un mensaje de cambio de programa MIDI y el momento en que se reproduce una nota con la nueva revisión. El almacenamiento en caché de revisiones también garantiza que estén disponibles las revisiones necesarias (el sintetizador podría cargar solo un subconjunto de sus revisiones). Consulte también [interfaz digital de instrumentos musicales (MIDI)](/windows), [patch](/windows).

</dd> <dt>

<span id="_win32_pitch_scale_factor_gloss"></span><span id="_WIN32_PITCH_SCALE_FACTOR_GLOSS"></span>**factor de escala de paso**
</dt> <dd>

En audio de forma de onda, la cantidad por la que un controlador de audio de forma de onda escala el paso. Un factor de escala de dos da como resultado un aumento de un octavo en el paso. El escalado de tono requiere hardware especializado. La velocidad de reproducción y la velocidad de muestreo no cambian.

</dd> <dt>

<span id="_win32_playback_rate_scale_factor_gloss"></span><span id="_WIN32_PLAYBACK_RATE_SCALE_FACTOR_GLOSS"></span>**factor de escala de velocidad de reproducción**
</dt> <dd>

En audio de forma de onda, cantidad por la que el controlador de audio de forma de onda escala la velocidad de reproducción. El escalado de reproducción se logra a través de software. la velocidad de muestra no cambia, pero el controlador se interpola omitiendo o sintetizando ejemplos. Por ejemplo, si se cambia la velocidad de reproducción en un factor de dos, el controlador omite cada ejemplo.

</dd> <dt>

<span id="_win32_polyphony_gloss"></span><span id="_WIN32_POLYPHONY_GLOSS"></span>**polifalsoy**
</dt> <dd>

Número máximo de notas que un dispositivo de salida de interfaz digital de instrumentos musicales (MIDI) puede reproducir simultáneamente.

</dd> <dt>

<span id="_win32_ppqn_gloss"></span><span id="_WIN32_PPQN_GLOSS"></span>**PPQN**
</dt> <dd>

Vea [Parts per Quarter Note](/windows).

</dd> <dt>

<span id="_win32_preimaging_gloss"></span><span id="_WIN32_PREIMAGING_GLOSS"></span>**creación de imágenes preimaging**
</dt> <dd>

Proceso de creación de un fotograma de película en un búfer de memoria antes de que se muestre.

</dd> <dt>

<span id="_win32_red_book_audio_gloss"></span><span id="_WIN32_RED_BOOK_AUDIO_GLOSS"></span>**Audio de libro rojo**
</dt> <dd>

Vea [disco compacto: audio digital (CD-das)](/windows).

</dd> <dt>

<span id="_win32_resource_interchange_file_format_riff_gloss"></span><span id="_WIN32_RESOURCE_INTERCHANGE_FILE_FORMAT_RIFF_GLOSS"></span>**Formato de archivo de intercambio de recursos (Riff)**
</dt> <dd>

Especificación de archivo etiquetado que se usa para definir formatos estándar para archivos multimedia. La estructura de archivos etiquetados ayuda a evitar problemas de compatibilidad que suelen producirse cuando las definiciones de formato de archivo cambian con el tiempo. Dado que cada fragmento de datos del archivo se identifica mediante un encabezado estándar, una aplicación que no reconoce un elemento de datos determinado puede omitir la información desconocida. Vea también [formato de archivo etiquetado](/windows).

</dd> <dt>

<span id="_win32_riff_gloss"></span><span id="_WIN32_RIFF_GLOSS"></span>**Riff**
</dt> <dd>

Consulte [formato de archivo de intercambio de recursos (Riff)](/windows).

</dd> <dt>

<span id="_win32_riff_chunk_gloss"></span><span id="_WIN32_RIFF_CHUNK_GLOSS"></span>**Fragmento riff**
</dt> <dd>

Un fragmento con el formato de [archivo de intercambio de recursos (Riff)](/windows) que incluye un código de identificación y cero o más subfragmentos, cuyo contenido depende del tipo de formulario.

</dd> <dt>

<span id="_win32_riff_file_gloss"></span><span id="_WIN32_RIFF_FILE_GLOSS"></span>**Archivo riff**
</dt> <dd>

Archivo cuyo formato cumple con uno de los formularios del [formato de archivo de intercambio de recursos publicado (Riff)](/windows) . Entre los ejemplos de archivos RIFF se incluyen archivos de onda para datos de audio de onda, archivos RMID para secuencias de interfaz digital de instrumentos musicales (MIDI) y archivos RDIB para mapas de bits independientes del dispositivo.

</dd> <dt>

<span id="_win32_riff_form_gloss"></span><span id="_WIN32_RIFF_FORM_GLOSS"></span>**Formulario riff**
</dt> <dd>

Especificación de formato de archivo basada en el estándar de [formato de archivo de intercambio de recursos (Riff)](/windows) .

</dd> <dt>

<span id="_win32_sample_gloss"></span><span id="_WIN32_SAMPLE_GLOSS"></span>**ejemplo** de
</dt> <dd>

Parte discreta de los datos de forma de onda representada por un único valor numérico. El muestreo es el proceso de convertir datos analógicos en datos digitales tomando muestras de la forma de onda analógica a intervalos regulares.

</dd> <dt>

<span id="_win32_sampling_rate_gloss"></span><span id="_WIN32_SAMPLING_RATE_GLOSS"></span>**velocidad de muestreo**
</dt> <dd>

La velocidad a la que un controlador de audio de forma de onda realiza una conversión de audio a digital o digital a audio. Para el disco compacto (CD-das), la velocidad de muestreo es de 44,1 kHz. Véase también [disco compacto-audio digital](/windows)

</dd> <dt>

<span id="_win32_sequence_gloss"></span><span id="_WIN32_SEQUENCE_GLOSS"></span>**secuencia** de
</dt> <dd>

Consulte [secuencia de MIDI](/windows).

</dd> <dt>

<span id="_win32_sequencer_gloss"></span><span id="_WIN32_SEQUENCER_GLOSS"></span>**Sequencer**
</dt> <dd>

Consulte [MIDI Sequencer](/windows).

</dd> <dt>

<span id="_win32_simple_device_gloss"></span><span id="_WIN32_SIMPLE_DEVICE_GLOSS"></span>**dispositivo simple**
</dt> <dd>

Un dispositivo de media control Interface (MCI) que no requiere un elemento de dispositivo (archivo de datos) para la reproducción. El controlador de audio de disco compacto de MCI es un ejemplo de dispositivo simple.

</dd> <dt>

<span id="_win32_smpte_gloss"></span><span id="_WIN32_SMPTE_GLOSS"></span>**SMPTE**
</dt> <dd>

Consulte [sociedad de la imagen de movimiento y los ingenieros de televisión](/windows).

</dd> <dt>

<span id="_win32_smpte_division_type_gloss"></span><span id="_WIN32_SMPTE_DIVISION_TYPE_GLOSS"></span>**Tipo de división SMPTE**
</dt> <dd>

Uno de los cuatro formatos de temporización de SMPTE. La hora SMPTE se expresa en horas, minutos, segundos y fotogramas. El tipo de división SMPTE especifica el valor de fotogramas por segundo correspondiente a una hora de SMPTE determinada.
