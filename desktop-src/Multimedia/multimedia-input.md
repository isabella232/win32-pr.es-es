# <a name="multimedia-input"></a>Entrada multimedia

Windows proporciona funciones que permiten a una aplicación recibir entradas multimedia. Las funciones de entrada multimedia se introdujeron en funciones de 16 bits Windows. Muchas de sus características se han reemplazado por las características de DirectX. Para más información, consulte la documentación de DirectX.

Los temas siguientes se tratan en la documentación de entrada multimedia.

-   [E/S de archivos multimedia](multimedia-file-i-o.md)
-   [Joysticks](joysticks.md)
-   [Temporizadores multimedia](multimedia-timers.md)
