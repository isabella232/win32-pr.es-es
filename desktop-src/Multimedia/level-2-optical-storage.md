# <a name="level-2-optical-storage"></a>Nivel 2 De Storage

El dispositivo de almacenamiento óptico de rendimiento mínimo es una unidad de CD-ROM de doble velocidad que cumple los criterios siguientes:

-   Una tasa de transferencia de datos sostenida de 300 kilobytes por segundo
-   Un uso de ancho de banda de CPU del 40 % o menos al mantener una tasa de transferencia de datos sostenida de 150 kilobytes por segundo o un ancho de banda de CPU del 60 % o menos al mantener una tasa de transferencia de datos sostenida de 300 kilobytes por segundo
-   Un tiempo medio de búsqueda de 400 milisegundos o menos
-   Una clasificación de tiempo medio entre errores de 10 000 horas
-   Cd-ROM XA listo (compatible con el modo 1, modo 2 compatible con el formulario 1, compatible con el modo 2)
-   Compatible con varias sesiones
-   Controlador MSCDEX-2.2 o equivalente que implementa las funciones de audio extendidas

El ancho de banda de CPU recomendado debe alcanzarse mediante un tamaño de bloque de lectura de al menos 16 000 y un tiempo de clientes potenciales que no sea superior al tiempo necesario para cargar un bloque de lectura de datos en el búfer de CD-ROM.
