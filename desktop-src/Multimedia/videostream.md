# <a name="videostream"></a>Videostream

Las aplicaciones pueden usar funciones de devolución de llamada de secuencia de vídeo durante la captura de streaming para procesar un fotograma de vídeo capturado. La ventana de captura llama a una función de devolución de llamada de secuencia de vídeo justo antes de escribir cada fotograma capturado en el disco.
