# <a name="midi-services"></a>Servicios DE MIDI

La mayoría de las aplicaciones podrán usar el secuenciador MCI MIDI o los búferes de flujo (y la función [**midiStreamOut)**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamout) para implementar toda la funcionalidad de MIDI que necesitan. Los desarrolladores graves de MIDI ( aquellos que producen herramientas de creación o secuenciación de MIDI) pueden usar una combinación de las funcionalidades de flujo y los servicios DE MIDI o usar solo los servicios DE LÍNEA. En los temas siguientes se proporciona información general sobre el uso de los servicios DE MIDI.

-   [Consulta de dispositivos MIDI](querying-midi-devices.md)
-   [Apertura y cierre de controladores de dispositivos](opening-and-closing-device-drivers.md)
-   [Asignación y preparación de bloques de datos DE MIDI](allocating-and-preparing-midi-data-blocks.md)
-   [Administración de bloques de datos DE MIDI](managing-midi-data-blocks.md)
-   [Solicitar formatos de hora](requesting-time-formats.md)
-   [Control de errores con funciones MIDI](handling-errors-with-midi-functions.md)
