# <a name="the-wait-notify-and-test-flags"></a>Las marcas Wait, Notify y Test

La mayoría de los comandos de MCI incluyen marcas que modifican el comando. Las marcas "wait" (MCI \_ WAIT) y "notify" (MCI NOTIFY) son comunes \_ a cada comando. La marca "test" (MCI \_ TEST) está disponible para los dispositivos de vídeo digital y VCR. En los temas siguientes se describe el uso de estas marcas.

-   [La marca wait](the-wait-flag.md)
-   [La marca Notify](the-notify-flag.md)
-   [La marca de prueba](the-test-flag.md)
