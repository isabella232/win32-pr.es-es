# <a name="wavestream"></a>Wavestream

Las aplicaciones usan las funciones de devolución de llamada de wavestream durante la captura de streaming para procesar búferes de audio. La ventana de captura llama a una función de devolución de llamada de flujo de onda justo antes de escribir cada búfer de audio en el disco.
