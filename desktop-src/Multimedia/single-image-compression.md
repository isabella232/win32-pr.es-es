# <a name="single-image-compression"></a>Single-Image compresión

Puede usar la función [**ICImageCompress**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) para comprimir una sola imagen. Esta función devuelve un identificador del mapa de bits independiente del dispositivo (DIB) comprimido. La DIB comprimida se empaqueta con el formato \_ CF DIB.
