# <a name="drawdib"></a>DrawDib

Las funciones DrawDib proporcionan funcionalidades de dibujo de imágenes de alto rendimiento para mapas de bits independientes del dispositivo (DIB). Las funciones DrawDib admiten DIB de profundidades de imagen de 8 bits, 16 bits, 24 bits y 32 bits.

Las funciones DrawDib escriben directamente en la memoria de vídeo. No se basan en las funciones de la interfaz de dispositivo gráfico (GDI).
