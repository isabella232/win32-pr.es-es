# <a name="eaphost-supplicant-configuration-functions"></a>Funciones de configuración de EAPHost Supplicant

Las funciones de configuración de EAP Supplicant API son las siguientes.



| Función                                                                                                           | Descripción                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EapHostPeerConfigBlob2Xml**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerconfigblob2xml)                                                 | Convierte el blob de configuración en XML.                                                                                                                                                                                                                                              |
| [**EapHostPeerConfigXml2Blob**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerconfigxml2blob)                                                 | Convierte XML en el blob de configuración.                                                                                                                                                                                                                                            |
| [**EapHostPeerCredentialsXml2Blob**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeercredentialsxml2blob)                                       | Genera las credenciales BLOB.                                                                                                                                                                                                                                                      |
| [**EapHostPeerFreeErrorMemory**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerfreeerrormemory)                                               | Libera la memoria asignada a una estructura [**\_ DE ERROR de EAP.**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_error)                                                                                                                                                                                                              |
| [**EapHostPeerFreeMemory**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerfreememory)                                                         | Libera la memoria devuelta por las API de configuración. Esta función no debe usarse para liberar memoria de errores. Para liberar memoria de errores, use la [**función EapHostPeerFreeEapError**](/previous-versions/windows/desktop/api/eappapis/nf-eappapis-eaphostpeerfreeeaperror) o [**EapHostPeerFreeErrrorMemory.**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerfreeerrormemory) |
| [**EapHostPeerGetMethodProperties**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeergetmethodproperties)                                           | Recupera las propiedades de un método EAP según la conexión y los datos de usuario.                                                                                                                                                                                                            |
| [**EapHostPeerGetMethods**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeergetmethods)                                                         | Enumera todos los métodos EAP instalados y disponibles para su uso, incluidos los métodos EAP heredados.                                                                                                                                                                                            |
| [**EapHostPeerInvokeConfigUI**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerinvokeconfigui)                                                     | Inicia la interfaz de usuario de configuración del método EAP especificado.                                                                                                                                                                                                                 |
| [**EapHostPeerInvokeIdentityUI**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerinvokeidentityui)                                                 | Inicia la interfaz de usuario de identidad.                                                                                                                                                                                                                                                  |
| [**EapHostPeerInvokeInteractiveUI**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerinvokeinteractiveui)                                       | Proporciona interactividad de credenciales al usuario, como una tarjeta inteligente y un PIN, por ejemplo.                                                                                                                                                                                            |
| [**EapHostPeerQueryCredentialInputFields**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerquerycredentialinputfields)                             | Permite al usuario determinar qué tipo de credenciales requieren los métodos para realizar la autenticación. También obtiene los campos que se mostrarán en la interfaz de usuario.                                                                                                       |
| [**EapHostPeerQueryInteractiveUIInputFields**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerqueryinteractiveuiinputfields)                       | Obtiene los campos de entrada para los componentes de interfaz de usuario interactivos que se deben generar en el suplicante.                                                                                                                                                                                       |
| [**EapHostPeerQueryUIBlobFromInteractiveUIInputFields**](/previous-versions/windows/desktop/api/eapmethodpeerapis/nf-eapmethodpeerapis-eappeerqueryuiblobfrominteractiveuiinputfields)       | Convierte la información del usuario en un BLOB de usuario que pueden consumir las funciones en tiempo de ejecución de EAPHost.                                                                                                                                                                                          |
| [**EapHostPeerQueryUserBlobFromCredentialInputFields**](/previous-versions/windows/desktop/api/eaphostpeerconfigapis/nf-eaphostpeerconfigapis-eaphostpeerqueryuserblobfromcredentialinputfields) | Obtiene el blob de credenciales que se podría usar para iniciar la autenticación una vez que se han recibido las entradas del usuario desde la interfaz de usuario de inicio de sesión único.                                                                                                                                 |
