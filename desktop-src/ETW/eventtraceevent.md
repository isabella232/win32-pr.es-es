# <a name="eventtraceevent-class"></a>Clase EventTraceEvent

Clase primaria para eventos de encabezado de registro. Esta clase siempre es la primera clase de seguimiento de eventos enviada a un consumidor (este evento no se envía a consumidores en tiempo real).

La sintaxis siguiente se simplifica a partir del código MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[Guid("{68fdd900-4a3e-11d1-84f4-0000f80464e3}")]
class EventTraceEvent : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Miembros

La **clase EventTraceEvent** no define ningún miembro.
