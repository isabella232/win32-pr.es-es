# <a name="webdav"></a>WebDAV

## <a name="purpose"></a>Propósito

Web Distributed Authoring and Versioning (WebDAV) es una extensión del Protocolo de transferencia de hipertexto (HTTP) que define cómo se realizan las funciones de archivo básicas, como copiar, mover, eliminar y crear mediante HTTP. La API de WebDAV es un conjunto de funciones de Win32 para crear y administrar conexiones a servidores WebDAV y realizar operaciones de E/S de archivos en archivos remotos en servidores WebDAV. Las funciones de API de WebDAV están diseñadas para [](/windows/desktop/WNet/windows-networking-functions) usarse junto con otras funciones de API win32, como Windows de red y funciones [de administración de archivos.](/windows/desktop/FileIO/file-management-functions)

## <a name="developer-audience"></a>Audiencia de desarrolladores

La API de WebDAV está diseñada para que la usen los desarrolladores de C y C++.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La API de WebDAV está en Windows Vista y versiones posteriores. Para obtener información sobre qué versiones de sistema operativo son necesarias para usar un elemento de programación determinado, vea la sección Requisitos de la página de referencia de ese elemento.
