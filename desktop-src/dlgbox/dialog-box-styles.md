# <a name="dialog-box-styles"></a>Estilos de cuadro de diálogo

En la tabla siguiente se enumeran los estilos de cuadro de diálogo que puede especificar al crear un cuadro de diálogo. Puede usar estos estilos en llamadas a las funciones [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) y [**CreateWindowEx,**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) en el miembro de estilo de las estructuras [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) y [**DLGTEMPLATEEX**](dlgtemplateex.md) y en la instrucción de una definición de cuadro de diálogo en un archivo de recursos. 



| Constante                                                                                                                                                               | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**DS \_ 3DLOOK**</dt> </dl>                      | Obsoleto. El sistema aplica automáticamente el aspecto tridimensional a los cuadros de diálogo creados por las aplicaciones.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**ABSALIGN de DS \_**</dt> </dl>                | Indica que las coordenadas del cuadro de diálogo son coordenadas de pantalla. Si no se especifica este estilo, las coordenadas son coordenadas de cliente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**CENTRO de \_ DS**</dt> </dl>                      | Centra el cuadro de diálogo en el área de trabajo del monitor que contiene la ventana de propietario. Si no se especifica ninguna ventana de propietario, el cuadro de diálogo se centra en el área de trabajo de un monitor determinado por el sistema. El área de trabajo es el área que la barra de tareas o las barras de aplicaciones no ocultan.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**DS \_ CENTERMOUSE**</dt> </dl>       | Centra el cuadro de diálogo en el cursor del mouse.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**DS \_ CONTEXTHELP**</dt> </dl>       | Incluye un signo de interrogación en la barra de título del cuadro de diálogo. Cuando el usuario hace clic en el signo de interrogación, el cursor se transforma en un signo de interrogación con un puntero. Si el usuario hace clic en un control en el cuadro de diálogo, el control recibe un [**mensaje DE AYUDA \_ DE WM.**](../shell/wm-help.md) El control debe pasar el mensaje al procedimiento del cuadro de diálogo, que debe llamar a la función mediante el **comando HELP \_ WM \_ HELP.** La aplicación de ayuda muestra una ventana emergente que normalmente contiene ayuda para el control.<br/> Tenga en cuenta **que DS \_ CONTEXTHELP** es solo un marcador de posición. Cuando se crea el cuadro de diálogo, el sistema busca **DS \_ CONTEXTHELP** y, si está ahí, agrega **WS \_ EX \_ CONTEXTHELP** al estilo extendido del cuadro de diálogo. **WS \_ EX \_ CONTEXTHELP no** se puede usar con los estilos **WS \_ MAXIMIZEBOX** **o WS \_ MINIMIZEBOX.**<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**DS \_ CONTROL**</dt> </dl>                   | Crea un cuadro de diálogo que funciona bien como una ventana secundaria de otro cuadro de diálogo, de forma muy parecido a una página de una hoja de propiedades. Este estilo permite al usuario establecer tabulaciones entre las ventanas de control de un cuadro de diálogo secundario, usar sus teclas de aceleración, y así sucesivamente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**DS \_ FIXEDSYS**</dt> </dl>                | Hace que el cuadro de diálogo use **la FUENTE FIJA DEL SISTEMA \_ \_ en** lugar de la **fuente del sistema \_ predeterminada.** Se trata de una fuente monoespacial compatible con la fuente System en versiones de 16 bits de Windows anteriores a la 3.0.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**DS \_ LOCALEDIT**</dt> </dl>             | Solo se aplica a aplicaciones de 16 bits. Este estilo dirige los controles de edición en el cuadro de diálogo para asignar memoria desde el segmento de datos de la aplicación. De lo contrario, los controles de edición asignan almacenamiento desde un objeto de memoria global.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**DS \_ MODALFRAME**</dt> </dl>          | Crea un cuadro de diálogo con un marco de cuadro de diálogo modal que se puede combinar con una barra de título y un menú de ventana especificando los estilos **WS \_ CAPTION** y **WS \_ SYSMENU.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**DS \_ NOFAILCREATE**</dt> </dl>    | Crea el cuadro de diálogo aunque se produzcan errores, por ejemplo, si no se puede crear una ventana secundaria o si el sistema no puede crear un segmento de datos especial para un control de edición.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**DS \_ NOIDLEMSG**</dt> </dl>             | Suprime [**los mensajes \_ ENTERIDLE**](wm-enteridle.md) de WM que el sistema enviaría de otro modo al propietario del cuadro de diálogo mientras se muestra el cuadro de diálogo.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**DS \_ SETFONT**</dt> </dl>                   | Indica que el encabezado de la plantilla de cuadro de diálogo (estándar o extendido) contiene datos adicionales que especifican la fuente que se usará para el texto en el área cliente y los controles del cuadro de diálogo. Si es posible, el sistema selecciona una fuente según los datos de fuente especificados. El sistema pasa un identificador a la fuente al cuadro de diálogo y a cada control enviándoles el [**mensaje \_ WM SETFONT.**](/windows/desktop/winmsg/wm-setfont) Para obtener descripciones del formato de estos datos de fuente, vea [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) y [**DLGTEMPLATEEX**](dlgtemplateex.md).<br/> Si no **se especifica DS \_ SETFONT** ni **DS \_ SHELLFONT,** la plantilla de cuadro de diálogo no incluye los datos de fuente.<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**DS \_ SETFOREGROUND**</dt> </dl> | Hace que el sistema use la [**función SetForegroundWindow**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) para poner el cuadro de diálogo en primer plano. Este estilo es útil para los cuadros de diálogo modales que requieren atención inmediata del usuario, independientemente de si la ventana propietaria es la ventana de primer plano.<br/> El sistema restringe qué procesos pueden establecer la ventana de primer plano. Para obtener más información, vea [Foreground and Background Windows](/windows/desktop/winmsg/window-features).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**SHELLFONT de DS \_**</dt> </dl>             | Indica que el cuadro de diálogo debe usar la fuente del sistema. El **miembro de tipo** de letra de la plantilla de cuadro de diálogo extendido debe establecerse en Dlg de Shell de MS. De lo contrario, este estilo no tiene ningún efecto. También se recomienda usar el recurso [DIALOGEX](../menurc/dialogex-resource.md), en lugar del [recurso DIALOG](../menurc/dialog-resource.md). Para obtener más información, vea [Fuentes de cuadro de diálogo](about-dialog-boxes.md).<br/> El sistema selecciona una fuente con los datos de fuente especificados en los miembros **de** puntos , **peso** y **cursiva.** El sistema pasa un identificador a la fuente al cuadro de diálogo y a cada control enviándoles el [**mensaje \_ WM SETFONT.**](/windows/desktop/winmsg/wm-setfont) Para obtener descripciones del formato de estos datos de fuente, vea [**DLGTEMPLATEEX**](dlgtemplateex.md). <br/> Si no **se especifica DS \_ SHELLFONT** ni **DS \_ SETFONT,** la plantilla de cuadro de diálogo extendido no incluye los datos de fuente.<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**DS \_ SYSMODAL**</dt> </dl>                | Este estilo está obsoleto y se incluye por compatibilidad con versiones de 16 bits de Windows. Si especifica este estilo, el sistema crea el cuadro de diálogo con el **estilo \_ \_ TOPMOST de WS EX.** Este estilo no impide que el usuario acceda a otras ventanas del escritorio.<br/> No combine este estilo con el **estilo \_ DS CONTROL.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
