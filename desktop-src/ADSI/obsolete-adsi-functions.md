# <a name="obsolete-adsi-functions"></a>Funciones ADSI obsoletas

Las siguientes funciones se expusieron en las versiones anteriores del SDK de ADSI, pero ya no se usan en el SDK.

-   **ADsDecodeBinaryData**
-   **AdsFreeAdsValues**
-   **ADsFreeAllErrorRecords**
-   **AdsTypeToPropVariant**
-   **AssertADsMemLeaks**
-   **DumpMemoryTracker**
-   **InitADsMem**
-   **PropVariantToAdsType**
