# <a name="memory-management-memory-management"></a>Administración de memoria (Administración de memoria)

El administrador de memoria implementa la memoria virtual, proporciona un conjunto básico de servicios como archivos asignados a memoria, memoria de copia en escritura, compatibilidad con memoria grande y compatibilidad subyacente con el administrador de caché.
