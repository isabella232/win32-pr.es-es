# <a name="tapi-21"></a>TAPI 2.1

TAPI 2.1 proporciona una adición a la especificación anterior de TAPI 2.0: funcionalidad de cliente/servidor. Con TAPI 2.1 y versiones posteriores, el hardware de telefonía se puede instalar en un servidor de telefonía al que se puede acceder desde cualquier equipo de una red. Un nuevo conjunto de funciones permite escribir archivos DLL de administración para controlar quién tiene acceso a los recursos remotos.

**Windows Me/98/95:** TAPI 2.1 proporciona compatibilidad con TSP de 32 bits. Los archivos binarios del sistema TAPI 2.1, cuando se instalan, admiten:

-   Aplicaciones TAPI 1.3, TAPI 1.4 y TAPI 2.x de 32 bits
-   TSPs tapi 2.x de 32 bits
-   Aplicaciones TAPI 1.3 y TAPI 1.4 de 16 bits
-   TAPI 1.3 y TAPI 1.4 TSPs de 16 bits
