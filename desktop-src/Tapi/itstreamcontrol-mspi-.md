# <a name="itstreamcontrol-mspi"></a>ITStreamControl (MSPI)

La interfaz [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) se expone en el objeto de llamada si hay un MSP asociado. Los métodos de esta interfaz permiten a una aplicación crear y quitar secuencias.

La interfaz [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) se implementa mediante un MSP. Consulte **ITStreamControl** en la sección interfaz MSP para obtener más información sobre esta interfaz.
