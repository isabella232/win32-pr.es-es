# <a name="tapiprotocol_-constants"></a>Constantes TAPIPROTOCOL \_

TAPI 3 define los protocolos siguientes y los CLID están en Tapi3.lib. Los proveedores de servicios pueden definir protocolos adicionales.

<dl> <dt>

<span id="CLSID_TAPIPROTOCOL_PSTN"></span><span id="clsid_tapiprotocol_pstn"></span>**CLSID TAPIPROTOCOL \_ RTC**
</dt> <dd> <dl> <dt>



Protocolo de red telefónica pública conmutada.


</dt> </dl> </dd> <dt>

<span id="CLSID_TAPIPROTOCOL_H323"></span><span id="clsid_tapiprotocol_h323"></span>**CLSID TAPIPROTOCOL \_ H323**
</dt> <dd> <dl> <dt>



Protocolo H323.


</dt> </dl> </dd> <dt>

<span id="CLSID_TAPIPROTOCOL_Multicast"></span><span id="clsid_tapiprotocol_multicast"></span><span id="CLSID_TAPIPROTOCOL_MULTICAST"></span>**Multidifusión DE CLSID TAPIPROTOCOL \_**
</dt> <dd> <dl> <dt>



Protocolo de multidifusión.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

**Las versiones BSTR** de las constantes de protocolo se declaran para el uso de Visual Basic aplicaciones.

-   BSTR TAPIPROTOCOL \_ String \_ PSTN
-   BSTR TAPIPROTOCOL \_ String \_ H323
-   Multidifusión de cadenas BSTR TAPIPROTOCOL \_ \_
