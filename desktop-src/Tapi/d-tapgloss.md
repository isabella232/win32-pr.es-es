# <a name="d-telephony-api"></a>D (API de telefonía)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) D [E](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L M](l-tapgloss.md) [N](m-tapgloss.md) O [P P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**Canal D**
</dt> <dd>

Canal de datos de la interfaz ISDN. Es uno de los tres canales estándar en una línea de ISDN de velocidad básica. En una línea BRI-ISDN, un canal D puede llevar información de señalización y datos de paquetes de baja velocidad a una velocidad de 16 000 bps. En una línea PRI- ISDN, el canal D se ejecuta a 64 000 bps. Consulte [*Basic Rate Interface (BRI-ISDN)*](b-tapgloss.md), canal [*B*](b-tapgloss.md), Primary Rate [*Interface (PRI-ISDN).*](p-tapgloss.md)

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**device (clase)**
</dt> <dd>

Un grupo de dispositivos físicos relacionados o controladores de dispositivos a través de los cuales las aplicaciones envían y reciben la información o los datos que conste una llamada. Para obtener una lista, vea [CLASES DE DISPOSITIVOS TAPI.](tapi-device-classes.md)

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**DID**
</dt> <dd>

Vea *direct inward dialing (DID) (Marcado directo hacia dentro [DID]).*

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**marcado directo hacia dentro (DID)**
</dt> <dd>

Una capacidad proporcionada por prácticamente todos los PBX modernos que permite que un llamador externo llegue a una extensión interna marcando un número de teléfono estándar de siete dígitos sin tener que pasar a través de un operador o un ayudante. Consulte [*intercambio de ramas privadas (SWITCH).*](p-tapgloss.md)

</dd> </dl>
