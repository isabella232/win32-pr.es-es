# <a name="csinglefilterterminal"></a>CSingleFilterTerminal

**CSingleFilterTerminal es** una clase base de terminal adicional aplicable a terminales estáticos y dinámicos. Se deriva de [**CBaseTerminal**](cbaseterminal.md) y hace que la implementación sea más específica suponiendo que el terminal tiene un único DirectShow filtro. Cuando se cumple esta condición, derivar una implementación de terminal de **CSingleFilterTerminal** es mucho más fácil que derivar de **CBaseTerminal**.

Definido en MSPterm.h.
