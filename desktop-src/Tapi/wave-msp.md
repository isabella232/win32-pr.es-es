# <a name="wave-msp"></a>Wave MSP

El MSP de Wave (Wavemsp.dll) se adapta a los TSP heredados escritos antes de la introducción de los MSP en la arquitectura de streaming basada en MSP de TAPI 3. Algunos de los TSP heredados exponen identificadores de dispositivos de onda, que eran un medio para que la aplicación controlara el streaming en TAPI 2.1. Tapi3.dll usa wave MSP para cualquier TSP que no tenga su propio MSP pero que tenga un dispositivo de onda.

Wave MSP se instala con Windows 2000.
