# <a name="linetermdev_-constants"></a>Constantes \_ LINETERMDEV

Las constantes de marca de bits **LINETERMDEV \_** describen diferentes tipos de dispositivos terminales.

<dl> <dt>

<span id="LINETERMDEV_HEADSET"></span><span id="linetermdev_headset"></span>**CASCO \_ LINETERMDEV**
</dt> <dd> <dl> <dt>



El terminal es un casco.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_PHONE"></span><span id="linetermdev_phone"></span>**TELÉFONO \_ LINETERMDEV**
</dt> <dd> <dl> <dt>



El terminal es un conjunto de teléfonos.


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_SPEAKER"></span><span id="linetermdev_speaker"></span>**LINETERMDEV \_ SPEAKER**
</dt> <dd> <dl> <dt>



El terminal es un altavoz externo y un micrófono.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

Sin extensibilidad. Los 32 bits están reservados.

Estas constantes se usan para caracterizar el dispositivo terminal de una línea y ayudar a una aplicación a determinar la naturaleza de un dispositivo terminal.
