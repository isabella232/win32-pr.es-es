# <a name="call-and-media-controls"></a>Llamada y controles multimedia

El siguiente material proporciona una revisión general de los objetos, las interfaces y los métodos de controles multimedia y de llamadas TAPI 3 para crear aplicaciones de comunicaciones de usuario final y servidor. [Acerca de los controles de llamada](about-call-and-media-controls.md) y multimedia describe los objetos principales y contiene un diagrama con hipervínculos que muestra objetos e interfaces. [El uso de controles de llamada](using-call-and-media-controls.md) y multimedia cubre información más específica útil para escribir aplicaciones, como los archivos .lib necesarios para compilar y algunos ejemplos de código.

Para obtener más información, vea la [Referencia de controles de llamada y multimedia](call-and-media-controls-reference.md).
