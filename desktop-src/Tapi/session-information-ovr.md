# <a name="session-information"></a>Información de la sesión

Las aplicaciones TAPI deben recopilar y procesar una variedad de información de sesión para funcionar correctamente. Por ejemplo, se produce un error en una operación de respuesta a menos que la sesión esté en el estado "oferta".

**TAPI 2.x:** Las aplicaciones usan principalmente la [**función lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) para acceder a la información de sesión. Las funciones adicionales que manipulan la información de sesión son [**lineGetCallStatus,**](/windows/win32/api/tapi/nf-tapi-linegetcallstatus) [**lineSetCallParams,**](/windows/win32/api/tapi/nf-tapi-linesetcallparams) [**lineSetCallData,**](/windows/win32/api/tapi/nf-tapi-linesetcalldata) [**lineSetCallTreatment,**](/windows/win32/api/tapi/nf-tapi-linesetcalltreatment) [**lineSetCallPrivilege,**](/windows/win32/api/tapi/nf-tapi-linesetcallprivilege) [**lineGetCallStatus**](/windows/win32/api/tapi/nf-tapi-linegetcallstatus)y [**lineSendUserUserInfo.**](/windows/win32/api/tapi/nf-tapi-linesenduseruserinfo)

**TAPI 3.x:** Las aplicaciones usan principalmente la [**interfaz ITCallInfo**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallinfo) para acceder a la información de la sesión.
