<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**Telefonía web**
</dt> <dd>

Una plataforma de aplicaciones que admite sitios web de doble acceso (accesibles por teléfono o mediante un explorador de Internet) y aplicaciones IVR basadas en HTML. La plataforma incluye un motor en tiempo de ejecución integrado en el servidor, una interfaz de administración basada en COM para instalar y configurar aplicaciones de telefonía web y acceso al objeto de llamada TAPI 3.x y a la API de scripting, para implementar operaciones de telefonía.

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**Motor de telefonía web (WTE)**
</dt> <dd>

Proporciona compatibilidad con aplicaciones IVR basadas en HTML y sitios web de acceso dual (un sitio web al que se puede acceder por teléfono y a través de un explorador gráfico). Consulte *Telefonía web*.

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**Windows Arquitectura de servicios abiertos (WOSA)**
</dt> <dd>

Arquitectura de software que proporciona una única interfaz de nivel de sistema para conectar aplicaciones front-end con servicios back-end. Windows La telefonía forma parte de WOSA.

</dd> </dl>
