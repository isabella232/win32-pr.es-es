# <a name="htapicall"></a>HTAPICALL

El tipo de datos **HTAPICALL** representa el identificador opaco de TAPI para una estructura de datos de llamada. TAPI debe resolver un valor de este tipo en una referencia a la instancia de la estructura de datos adecuada. El proveedor de servicios no debe intentar hacer referencia a través de este como si fuera un puntero, realizar suposiciones sobre sus valores o interpretar su representación de forma que no sea pasar su valor a TAPI en los momentos adecuados.
