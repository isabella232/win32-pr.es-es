# <a name="htapiphone"></a>HTAPIPHONE

El **tipo de datos HTAPIPHONE** representa el identificador opaco de TAPI para una estructura de datos de teléfono. TAPI debe resolver un valor de este tipo en una referencia a la instancia de estructura de datos adecuada. El proveedor de servicios no debe intentar hacer referencia a través de esto como si fuera un puntero, realizar suposiciones sobre sus valores o interpretar su representación de cualquier manera que no sea pasar su valor a TAPI en los momentos adecuados.
