# <a name="itemconnectstatus-property"></a>Propiedad Item.ConnectStatus

Recupera el estado de conexión del dispositivo. Esta propiedad solo se aplica a los elementos de tipo dispositivo (elementos raíz). Los valores posibles son "connected", "disconnected" o **NULL** (si esta propiedad no se aplica al elemento). Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.ConnectStatus
```



## <a name="property-value"></a>Valor de propiedad

Cadena que recibe el estado de conexión.
