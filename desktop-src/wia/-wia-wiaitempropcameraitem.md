# <a name="camera-wia-item-property-constants"></a>Constantes de propiedad del elemento WIA de la cámara

Las siguientes **constantes de** palabra forman el conjunto de propiedades válidas Windows de la cámara de adquisición de imágenes (WIA).

> [!Note]  
> WIA no admite cámaras en Windows Vista ni versiones posteriores. Para esas versiones de Windows, use la API de dispositivo portátil (WPD) de Windows descrita en el Kit de desarrollo de controladores (DDK) de Windows para adquirir imágenes de las cámaras.

 

El prefijo "WIA IPC" indica una propiedad item para las cámaras y es la convención de \_ nomenclatura que se usa en \_ C/C++. Con fines de scripting, estas constantes usan el prefijo "CameraPicture" y forman parte del tipo enumerado [WiaItemPropertyId.](-wia-wiaitempropertyid.md) El nombre de miembro correspondiente de esa enumeración de script aparece entre paréntesis junto al nombre de constante de C/C++ en la lista siguiente.
