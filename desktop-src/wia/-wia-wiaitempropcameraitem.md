# <a name="camera-wia-item-property-constants"></a>Constantes de propiedad de elementos WIA de cámara

Las siguientes **constantes de** palabra forman el conjunto de propiedades válidas Windows elemento de cámara de adquisición de imágenes (WIA).

> [!Note]  
> WIA no admite cámaras en Windows Vista o versiones posteriores. Para esas versiones de Windows, use la API Windows Portable Device (WPD) descrita en el Kit de desarrollo de controladores (DDK) de Windows para adquirir imágenes de las cámaras.

 

El prefijo "WIA IPC" indica una propiedad item para cámaras y es la convención de nomenclatura \_ \_ usada en C/C++. Con fines de scripting, estas constantes usan el prefijo "CameraPicture" y forman parte del tipo enumerado [WiaItemPropertyId.](-wia-wiaitempropertyid.md) El nombre de miembro correspondiente de esa enumeración de script aparece entre paréntesis junto al nombre de constante de C/C++ en la lista siguiente.
