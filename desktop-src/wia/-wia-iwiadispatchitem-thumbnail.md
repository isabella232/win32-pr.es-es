# <a name="itemthumbnail-property"></a>Propiedad Item.Thumbnail

Ruta de acceso y nombre de archivo de la imagen en miniatura. Esta propiedad es **NULL** si el elemento no admite miniaturas o si no se puede crear una ruta de acceso. Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>Valor de propiedad

Cadena que recibe la ruta de acceso y el nombre de archivo.
