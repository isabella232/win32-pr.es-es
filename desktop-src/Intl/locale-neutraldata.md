# <a name="locale_neutraldata"></a>LOCALE \_ NEUTRALDATA

**Windows 7 y versiones posteriores:** Datos de configuración regional neutros, es decir, datos definidos solo por idioma. Los datos de país o región usan el valor predeterminado.
