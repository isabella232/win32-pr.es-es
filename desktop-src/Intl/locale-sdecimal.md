# <a name="locale_sdecimal"></a>LOCALE \_ SDECIMAL

Caracteres usados para el separador decimal, por ejemplo, "." en "3,14" o "," en "3,14". El número máximo de caracteres permitido para esta cadena es cuatro, incluido un carácter nulo de terminación.
