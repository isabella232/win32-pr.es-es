# <a name="locale_ildate"></a>LOCALE \_ ILDATE

Especificador de ordenación de formato de fecha larga. Es preferible que la aplicación use la constante [ \_ SLONGDATE de LOCALE](locale-slongdate.md) en lugar de LOCALE \_ ILDATE.



| Value | Significado        |
|-------|----------------|
| 0     | Mes día-año |
| 1     | Día-mes-año |
| 2     | Año-mes-día |
