# <a name="locale_sshortestpm"></a>LOCALE \_ SSHORTESTPM

Cadena para el indicador de PM más corto posible para una configuración regional. En inglés, podría ser tan corto como "p"; sin embargo, otros lenguajes podrían tener más puntos de código.
