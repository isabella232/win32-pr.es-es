# <a name="locale_scurrency"></a>ESCURRENCIA \_ DE CONFIGURACIÓN REGIONAL

Cadena utilizada como símbolo monetario local. El número máximo de caracteres permitido para esta cadena es 13, incluido un carácter nulo de terminación. Por ejemplo, en la Estados Unidos, este símbolo es "$".
