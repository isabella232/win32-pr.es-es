# <a name="locale_scurrency"></a>configuración regional \_ SCURRENCY

Cadena usada como símbolo monetario local. El número máximo de caracteres permitido para esta cadena es 13, incluido un carácter nulo de terminación. Por ejemplo, en el Estados Unidos, este símbolo es "$".
