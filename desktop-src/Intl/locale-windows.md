# <a name="locale_windows"></a>VENTANAS DE CONFIGURACIÓN \_ REGIONAL

**Windows Vista y versiones posteriores:** Enumera todas las configuraciones regionales que vienen con el sistema operativo, incluidas las configuraciones regionales de reemplazo, pero excluyendo las ordenaciones alternativas. Para obtener más información, vea [Configuraciones regionales personalizadas.](custom-locales.md)
