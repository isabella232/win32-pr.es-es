# <a name="locale_skeyboardstoinstall"></a>LOCALE \_ SKEYBOARDSTOINSTALL

**Windows Vista y versiones posteriores:** Una lista delimitada por punto y coma de teclados que se instalarán potencialmente para la configuración regional y que el Windows. No hay ningún límite en el número de caracteres permitidos para esta cadena. Para recuperar el nombre del identificador de configuración regional de entrada activa (anteriormente denominado diseño de teclado), la aplicación puede llamar a la [**función GetKeyboardLayoutName.**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea)
