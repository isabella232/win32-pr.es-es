# <a name="locale_ineutral"></a>CONFIGURACIÓN REGIONAL \_ INEUTRAL

**Windows 7 y versiones posteriores:** Tipo de configuración regional. Establezca esta constante en 0 para una configuración regional específica o en 1 para una configuración regional neutra.
