# <a name="locale_idigits"></a>configuración regional \_ IDIGITS

Número de dígitos fraccionarios colocados después del separador decimal. El número máximo de caracteres permitido para esta cadena es dos, incluido un carácter nulo de terminación. Por ejemplo, 2 para 5,00, 1 para 5,0.
