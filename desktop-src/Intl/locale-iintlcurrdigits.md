# <a name="locale_iintlcurrdigits"></a>CONFIGURACIÓN REGIONAL \_ IINTLCURRDIGITS

\[LOCALE \_ IINTLCURRDIGITS no se admite y puede modificarse o no estar disponible en el futuro. En su lugar, [use LOCALE \_ ICURRDIGITS](locale-icurrdigits.md).\]

Número de dígitos fraccionarios para el formato monetario internacional. El número máximo de caracteres permitidos para esta cadena es tres, incluido un carácter nulo de terminación. Por ejemplo, el número de dígitos fraccionarios para el formato monetario internacional de dólares estadounidenses es 2, como en 345,25 USD.
