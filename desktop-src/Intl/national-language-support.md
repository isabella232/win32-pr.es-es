# <a name="national-language-support"></a>Compatibilidad con idiomas nacionales

Las funciones de compatibilidad con idiomas nacionales (NLS) ayudan a las aplicaciones a satisfacer las diferentes necesidades específicas del idioma y la configuración regional de usuarios de todo el mundo. En esta introducción se describen las funciones nls y se explica cómo usarlas en las aplicaciones.
