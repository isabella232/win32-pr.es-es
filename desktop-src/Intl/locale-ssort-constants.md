# <a name="locale_ssort_constants"></a>CONSTANTES \_ DE SSORT DE CONFIGURACIÓN \_ REGIONAL

En este tema se definen las constantes \_ SSORT de LOCALE \* usadas por NLS.



| Value               | Significado                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SSORTLOCALE | **Windows 7 y versiones posteriores:** Nombre de la configuración regional que se usará para ordenar o usar el comportamiento de mayúsculas y minúsculas.                                                                                                                                |
| LOCALE \_ SSORTNAME   | **Windows Me/98, Windows 2000:** Nombre localizado completo de la ordenación del identificador de configuración regional especificado, en función del idioma del shell. Esta constante se usa para determinar el comportamiento de las mayúsculas y minúsculas y la ordenación. |
