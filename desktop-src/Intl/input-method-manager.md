# <a name="input-method-manager"></a>Administrador de métodos de entrada

El Administrador de métodos de entrada (IMM) es una tecnología utilizada por una aplicación para comunicarse con un editor de métodos de entrada (IME), que se ejecuta como un servicio. El IME permite a los usuarios del equipo escribir caracteres complejos y símbolos, como caracteres kanji japoneses, mediante un teclado estándar.

En esta sección se describe la API de IMM y se explica cómo usar la funcionalidad para crear y administrar ventanas de IME. Se incluyen las secciones siguientes:

-   [Acerca del Administrador de métodos de entrada](about-input-method-manager.md)
-   [Usar el Administrador de métodos de entrada](using-input-method-manager.md)
-   [Referencia del Administrador de métodos de entrada](input-method-manager-reference.md)
