# <a name="locale_sopentypelanguagetag"></a>LOCALE \_ SOPENTYPELANGUAGETAG

**Windows 7 y versiones posteriores:** Etiqueta de lenguaje OpenType que se usa para recuperar características tipográficas culturalmente adecuadas de una fuente. Para obtener más información, vea [**OPENTYPE \_ TAG**](opentype-tag.md).
