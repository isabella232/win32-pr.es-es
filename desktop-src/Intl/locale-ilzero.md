# <a name="locale_ilzero"></a>configuración regional \_ ILZERO

Especificador para ceros a la izquierda en campos decimales.



| Value | Significado          |
|-------|------------------|
| 0     | Sin ceros a la izquierda |
| 1     | Ceros a la izquierda    |
