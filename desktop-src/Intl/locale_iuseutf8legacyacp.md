# <a name="locale_iuseutf8legacyacp"></a>CONFIGURACIÓN REGIONAL \_ IUSEUTF8LEGACYACP

**A partir de la siguiente versión de Windows:** Se recomienda que las aplicaciones usen Unicode en su lugar. En las máquinas que el usuario ha configurado para usar una página de códigos UTF-8, LOCALE_IUSEUTF8LEGACYACP es la página de códigos Windows heredada que se habría usado en una máquina Windows heredada que no es UTF-8.
