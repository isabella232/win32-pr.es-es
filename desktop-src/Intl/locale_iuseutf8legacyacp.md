# <a name="locale_iuseutf8legacyacp"></a>Configuración regional \_ IUSEUTF8LEGACYACP

**A partir de la siguiente versión de Windows:** En su lugar, se recomienda que las aplicaciones usen Unicode. En los equipos que el usuario ha configurado para usar una página de códigos UTF-8, LOCALE_IUSEUTF8LEGACYACP es la página de códigos heredada de Windows que se habría usado en un equipo Windows heredado que no es UTF-8.
