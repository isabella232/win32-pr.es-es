# <a name="locale_allow_neutral_names"></a>CONFIGURACIÓN REGIONAL \_ PERMITIR \_ NOMBRES \_ NEUTROS

**Windows 7 y versiones posteriores:** Permitir la devolución de nombres neutros o LCID al convertir entre nombres de configuración regional e identificadores de configuración regional.
