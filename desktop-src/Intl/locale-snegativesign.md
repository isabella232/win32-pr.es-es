# <a name="locale_snegativesign"></a>configuración regional \_ SNEGATIVESIGN

Valor de cadena para el signo negativo, por ejemplo, "-" para la configuración regional en inglés (Estados Unidos). El número máximo de caracteres permitido para esta cadena es cinco, incluido un carácter nulo de terminación.
