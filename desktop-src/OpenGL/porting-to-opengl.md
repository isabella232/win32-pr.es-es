# <a name="porting-to-opengl"></a>Trasladar a OpenGL

OpenGL está diseñado para ofrecer compatibilidad entre hardware y sistemas operativos. Este diseño facilita a los programadores el portabilidad de programas OpenGL de un sistema a otro. Aunque cada sistema operativo tiene requisitos únicos, gran parte del código OpenGL de los programas actuales se puede usar tal cual. Para trasladar la aplicación de OpenGL a Microsoft Windows, tendrá que modificar los programas para que funcionen con los sistemas de ventanas de Windows.

En general, las aplicaciones se trasladan a OpenGL para Windows desde una de estas dos plataformas:

-   Aplicaciones OpenGL desarrolladas para el sistema de ventanas X y la biblioteca X (Xlib)
-   Aplicaciones de contabilidad de IRIS
