# <a name="generating-texture-coordinates"></a>Generar coordenadas de textura

En lugar de proporcionar explícitamente coordenadas de textura, puede hacer que OpenGL las genere como una función de otros datos de vértice mediante [glTexGen \* ](gltexgen-functions.md). Una vez especificadas o generadas las coordenadas de textura, la matriz de textura las transforma. Esta matriz se controla con las mismas funciones que se usan para las transformaciones de matriz (vea [Transformaciones de matriz).](matrix-transformations.md)
