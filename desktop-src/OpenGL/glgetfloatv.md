# <a name="glgetfloatv-function"></a>Función glGetFloatv

La **función glGetFloatv** devuelve el valor o los valores de un parámetro seleccionado.

## <a name="syntax"></a>Sintaxis


```C++
void WINAPI glGetFloatv(
   GLenum  pname,
   GLfloat *params
);
```

## <a name="remarks"></a>Comentarios

Esta función devuelve valores para variables de estado simples en OpenGL. El *parámetro pname* es una constante simbólica que indica la variable de estado que se va a devolver y *params* es un puntero a una matriz del tipo indicado en la que se colocarán los datos devueltos.

La conversión de tipos se realiza *si params* tiene un tipo diferente del valor de la variable de estado que se solicita. Si llama a [**glGetBooleanv**](glgetbooleanv.md), un valor de punto flotante o entero se convierte en GL FALSE si y solo \_ si es cero. De lo contrario, se convierte a GL \_ TRUE.

Si llama a [**glGetIntegerv**](glgetintegerv.md), los valores booleanos se devuelven como GL TRUE o GL FALSE, y la mayoría de los valores de punto flotante se redondean \_ al valor entero más \_ cercano. Sin embargo, los colores y normales de punto flotante se devuelven con una asignación lineal que asigna 1,0 al valor entero más positivo que se puede representar y 1,0 al valor entero que se puede representar más negativo.

Si llama **a glGetFloatv** o [**glGetDoublev,**](glgetdoublev.md)los valores booleanos se devuelven como GL TRUE o GL FALSE, y los valores enteros se convierten en valores de punto \_ \_ flotante.

Puede consultar muchos de los parámetros booleanos más fácilmente [**con glIsEnabled.**](glisenabled.md)
