# <a name="opengl-on-windows"></a>OpenGL en Windows

La implementación de Microsoft de OpenGL en el sistema operativo Microsoft Windows es una implementación de la interfaz de software de gráficos tridimensional (3D) OpenGL estándar del sector con la que los programadores crean imágenes de color 3D animadas y fijas de alta calidad. En esta introducción se describe Windows implementación de OpenGL.

Para obtener información sobre OpenGL ES que se ejecuta Windows, vea [MSOpenTech ANGLE](https://github.com/MSOpenTech/angle/wiki).
