# <a name="porting-other-glx-code"></a>Porte de otro código GLX

Además de las funciones Xlib y GLX descritas en las secciones anteriores, el programa probablemente contenga algunas de las otras funciones GLX o Xlib enumeradas en Traducción de la biblioteca [GLX](translating-the-glx-library.md). Vuelva a escribir el código del sistema de la ventana X Windows, sustituyendo las funciones adecuadas.
