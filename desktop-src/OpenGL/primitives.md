# <a name="primitives-opengl"></a>Primitivas (OpenGL)

Durante la siguiente fase de procesamiento, las primitivas se convierten en fragmentos de píxeles en los pasos siguientes:

-   Las primitivas se recortan adecuadamente.
-   Los ajustes correspondientes necesarios se realizan en los datos color y Texture, y las coordenadas pertinentes se transforman en coordenadas de ventana.
-   La rasterización convierte los primitivos recortados en fragmentos de píxeles.

Para obtener más información, vea los temas siguientes:

-   [Recorte](clipping.md)
-   [Transformar a coordenadas de ventana](transforming-to-window-coordinates.md)
-   [Rasterización](rasterizing.md)
-   [Referencia de primitivas](primitives-reference.md)
