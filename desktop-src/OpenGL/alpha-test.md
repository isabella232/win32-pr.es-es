# <a name="alpha-test"></a>Prueba alfa

La prueba alfa (realizada solo en modo RGBA) descarta un fragmento, en función del resultado de una comparación entre el valor alfa del fragmento y un valor de referencia constante. Especifique la función de comparación y el valor de referencia [**con glAlphaFunc.**](glalphafunc.md)
