# <a name="xaudio2-macros"></a>Macros XAudio2

Esta sección contiene información sobre las macros proporcionadas por la API XAudio2 de Microsoft.



| Macro                          | Descripción                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Macro de asignación de memoria utilizada por métodos IXAPO que deben asignar estructuras de tamaño arbitrario que posteriormente se devuelven a la aplicación.<br/> |
| [**XAPOFree**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Macro usada para liberar memoria asignada con la macro XAPOAlloc.<br/>                                                                                  |
