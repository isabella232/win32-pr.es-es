# <a name="control-point-programming-considerations"></a>Consideraciones sobre la programación de puntos de control

Los desarrolladores que crean aplicaciones basadas en UPnP (o puntos de control) deben usar estas aplicaciones desde un cliente de APARTMENTTHREADED de coinit \_ . Existen problemas conocidos al usar la API de punto de control de un \_ cliente multiproceso de coinit que se ejecuta bajo estrés.
