# <a name="file-management-local-file-systems"></a>Administración de archivos (sistemas de archivos locales)

Un *objeto de archivo* proporciona una representación de un recurso (un dispositivo físico o un recurso ubicado en un dispositivo físico) que se puede administrar mediante el sistema de e/s. Al igual que otros objetos, permiten compartir el recurso, tienen nombres, están protegidos por la seguridad basada en objetos y admiten la sincronización. El sistema de e/s también permite leer o escribir en el recurso.
