# <a name="txf-control-codes"></a>Códigos de control TxF

\[Microsoft recomienda encarecidamente a los desarrolladores que utilicen medios alternativos para satisfacer las necesidades de la aplicación. Muchos de los escenarios para los que se desarrolló TxF se pueden lograr mediante técnicas más sencillas y disponibles. Además, es posible que TxF no esté disponible en versiones futuras de Microsoft Windows. Para obtener más información y alternativas a TxF, vea [Alternativas al uso de NTFS transaccional.](deprecation-of-txf.md)\]

NTFS transaccional (TxF) proporciona los siguientes códigos de control.

## <a name="in-this-section"></a>En esta sección



| Código de control                                                                                                 | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ TXFS \_ CREATE \_ MINIVERSION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_create_miniversion)<br/>                         | Crea una [nueva miniversión](glossary.md) para el archivo especificado. <br/> Las miniversiones permiten hacer referencia a una instantánea del archivo durante una transacción. Las miniversiones se descartan cuando se confirma o se revierte una transacción.<br/>                                                                                                                                                                      |
| [**OBTENER \_ INFORMACIÓN DE \_ \_ METADATOS DE FSCTL TXFS \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_metadata_info)<br/>                          | Recupera los metadatos ntfs con transacción (TxF) de un archivo y el **GUID** de la transacción que ha bloqueado el archivo especificado (si el archivo está bloqueado). <br/>                                                                                                                                                                                                                                                                         |
| [**FSCTL \_ TXFS \_ GET \_ TRANSACTED \_ VERSION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_transacted_version)<br/>                | Devuelve una [**estructura TXFS \_ GET \_ TRANSACTED \_ VERSION.**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_get_transacted_version) La estructura identifica la versión más reciente del archivo especificado, el número de versión del identificador. <br/>                                                                                                                                                                                                            |
| [**FSCTL \_ TXFS \_ LIST \_ TRANSACTION \_ LOCKED \_ FILES**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transaction_locked_files)<br/> | Devuelve una lista de todos los archivos actualmente bloqueados por la transacción especificada. Si el valor devuelto es **ERROR \_ MORE \_ DATA**, devuelve la longitud del búfer necesario para contener la lista completa de archivos en el momento de esta llamada.<br/>                                                                                                                                                                                           |
| [**FSCTL \_ TXFS \_ LIST \_ TRANSACTIONS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transactions)<br/>                           | Devuelve una lista de todas las transacciones implicadas actualmente en el administrador de recursos especificado.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**FSCTL \_ TXFS \_ MODIFY \_ RM**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_modify_rm)<br/>                                           | Establece el modo de registro y la información de parámetros de registro para un administrador de recursos secundario (RM).<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**INFORMACIÓN DE RM DE CONSULTA DE FSCTL \_ \_ \_ \_ TXFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_query_rm_information)<br/>                    | Recupera información de un administrador de recursos (RM).<br/>                                                                                                                                                                                                                                                                                                                                                                           |
| [**INFORMACIÓN DE \_ COPIA DE SEGURIDAD DE LECTURA DE FSCTL \_ \_ \_ TXFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information)<br/>              | Devuelve información específica de NTFS transaccional (TxF) para el archivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                |
| [**INFORMACIÓN DE SAVEPOINT de FSCTL \_ \_ \_ TXFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information)<br/>                   | El código de control [**FSCTL \_ TXFS \_ SAVEPOINT \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information) controla la configuración, el borrado y la re rolling back al punto de guardado especificado.<br/> Para realizar esta operación, llame a la [**función DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con los parámetros siguientes.<br/>                                                                                                                 |
| [**FSCTL \_ TXFS \_ TRANSACTION \_ ACTIVE**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_transaction_active)<br/>                         | Devuelve un valor booleano que indica si hubo transacciones activas en el volumen asociado cuando se tomó la instantánea. Esta llamada solo es válida para volúmenes de instantáneas de solo lectura.<br/>                                                                                                                                                                                                                                   |
| [**INFORMACIÓN DE \_ COPIA DE SEGURIDAD DE ESCRITURA DE FSCTL \_ \_ \_ TXFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_write_backup_information)<br/>            | Escribe información específica de NTFS transaccional (TxF) en un archivo especificado. El **miembro Buffer** de la estructura [**TXFS WRITE BACKUP \_ \_ \_ INFORMATION**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_write_backup_information) debe ser el miembro **Buffer** de la estructura [**TXFS READ BACKUP \_ INFORMATION \_ \_ \_ OUT**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_read_backup_information_out) devuelta por [**FSCTL \_ TXFS READ BACKUP \_ \_ \_ INFORMATION**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information).<br/> |
