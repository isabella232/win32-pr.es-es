# <a name="lan_profile-schema-elements"></a>Elementos de esquema de perfil de LAN \_

Un perfil de red cableada contiene los siguientes elementos de esquema. Todos los elementos con nombre están en el espacio de nombres `https://www.microsoft.com/networking/LAN/profile/v1` .

En la lista siguiente se muestran los elementos definidos en el orden en que los elementos aparecen en un perfil. Se aplica el orden de los elementos. Esta lista no muestra todos los elementos posibles que pueden aparecer en un perfil, ya que los elementos se pueden agregar en **xs:any puntos de** inserción.

-   [**LANProfile**](lan-profileschema-lanprofile-element.md)
    -   [**MSM (LANProfile)**](lan-profileschema-msm-lanprofile-element.md)
        -   [**seguridad (MSM)**](lan-profileschema-security-msm-element.md)
            -   [**OneXEnforced (seguridad)**](lan-profileschema-onexenforced-security-element.md)
            -   [**OneXEnabled (seguridad)**](lan-profileschema-onexenabled-security-element.md)
