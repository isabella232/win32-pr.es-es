# <a name="wireless-zero-configuration-functions"></a>Funciones de configuración inalámbrica cero

\[La interfaz de programación Configuración inalámbrica cero ya no se admite a partir de Windows Vista y Windows Server 2008. En su lugar, use [la API Wi-Fi nativa,](native-wifi-reference.md)que proporciona una funcionalidad similar. Para obtener más información, vea [Acerca de la API Wi-Fi nativa.](about-the-native-wifi-api.md)\]

Las siguientes funciones permiten a los desarrolladores de software usar la funcionalidad proporcionada por el módulo Configuración inalámbrica cero.

-   [**WZCEapolGetInterfaceParams**](wzceapolgetinterfaceparams.md)
-   [**WZCEnumInterfaces**](wzcenuminterfaces.md)
-   [**WZCQueryInterface**](wzcqueryinterface.md)
-   [**WZCRefreshInterface**](wzcrefreshinterface.md)
