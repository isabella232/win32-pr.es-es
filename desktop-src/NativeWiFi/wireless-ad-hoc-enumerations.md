# <a name="wireless-ad-hoc-enumerations"></a>Enumeraciones ad hoc inalámbricas

La interfaz ad hoc inalámbrica define y usa las enumeraciones siguientes.

-   [**ALGORITMO DE \_ AUTENTICACIÓN DE ADHOC \_ DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_auth_algorithm)
-   [**ALGORITMO DE CIFRADO \_ ADHOC DOT11 \_ \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_cipher_algorithm)
-   [**MOTIVO DE ERROR \_ DE DOT11 ADHOC \_ CONNECT \_ \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_connect_fail_reason)
-   [**ESTADO DE CONEXIÓN \_ DE RED DE DOT11 ADHOC \_ \_ \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_network_connection_status)

> [!Note]  
> Es posible que el modo ad hoc no esté disponible en versiones futuras de Windows. A partir de Windows 8.1 y Windows Server 2012 R2, use Wi-Fi Direct en [su lugar.](about-the-wi-fi-direct-api.md)
