# <a name="id3dx11threadpumpgetworkitemcount-method"></a>Método ID3DX11ThreadPump::GetWorkItemCount

> [!Note]  
> La biblioteca de utilidades D3DX (D3DX 9, D3DX 10 y D3DX 11) está en desuso para Windows 8 y no es compatible con las aplicaciones de Windows Store.

 

Obtiene el número de elementos de trabajo del bombeo de subprocesos.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **UINT**](/windows/desktop/WinProg/windows-data-types)**

Número de elementos de trabajo en cola en el bombeo de subprocesos.
