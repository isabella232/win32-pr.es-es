# <a name="id3dx11effectvectorvariable-interface"></a>Interfaz ID3DX11EffectVectorVariable

Una interfaz vector-variable accede a un vector de cuatro componentes.

## <a name="members"></a>Members

La **interfaz ID3DX11EffectVectorVariable** hereda de [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectVectorVariable** también tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La **interfaz ID3DX11EffectVectorVariable** tiene estos métodos.



| Método                                                                         | Descripción                                                                         |
|:-------------------------------------------------------------------------------|:------------------------------------------------------------------------------------|
| [**GetBoolVector**](id3dx11effectvectorvariable-getboolvector.md)             | Obtiene un vector de cuatro componentes que contiene datos booleanos.<br/>                  |
| [**GetBoolVectorArray**](id3dx11effectvectorvariable-getboolvectorarray.md)   | Obtiene una matriz de vectores de cuatro componentes que contienen datos booleanos.<br/>        |
| [**GetFloatVector**](id3dx11effectvectorvariable-getfloatvector.md)           | Obtiene un vector de cuatro componentes que contiene datos de punto flotante.<br/>           |
| [**GetFloatVectorArray**](id3dx11effectvectorvariable-getfloatvectorarray.md) | Obtiene una matriz de vectores de cuatro componentes que contienen datos de punto flotante.<br/> |
| [**GetIntVector**](id3dx11effectvectorvariable-getintvector.md)               | Obtiene un vector de cuatro componentes que contiene datos enteros.<br/>                  |
| [**GetIntVectorArray**](id3dx11effectvectorvariable-getintvectorarray.md)     | Obtiene una matriz de vectores de cuatro componentes que contienen datos enteros.<br/>        |
| [**SetBoolVector**](id3dx11effectvectorvariable-setboolvector.md)             | Establezca un vector de cuatro componentes que contenga datos booleanos.<br/>                  |
| [**SetBoolVectorArray**](id3dx11effectvectorvariable-setboolvectorarray.md)   | Establezca una matriz de vectores de cuatro componentes que contengan datos booleanos.<br/>        |
| [**SetFloatVector**](id3dx11effectvectorvariable-setfloatvector.md)           | Establezca un vector de cuatro componentes que contenga datos de punto flotante.<br/>           |
| [**SetFloatVectorArray**](id3dx11effectvectorvariable-setfloatvectorarray.md) | Establezca una matriz de vectores de cuatro componentes que contengan datos de punto flotante.<br/> |
| [**SetIntVector**](id3dx11effectvectorvariable-setintvector.md)               | Establezca un vector de cuatro componentes que contenga datos enteros.<br/>                  |
| [**SetIntVectorArray**](id3dx11effectvectorvariable-setintvectorarray.md)     | Establezca una matriz de vectores de cuatro componentes que contengan datos enteros.<br/>        |



 

## <a name="remarks"></a>Observaciones

> [!Note]  
> El SDK de DirectX no proporciona archivos binarios compilados para efectos. Debe usar el origen Effects 11 para compilar la aplicación de tipo effects. Para obtener más información sobre el uso del origen de Efectos 11, vea [Diferencias entre los efectos 10 y los efectos 11.](d3d11-graphics-programming-guide-effects-differences.md)
