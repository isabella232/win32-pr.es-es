# <a name="winlogon-notification-package-reference"></a>Referencia del paquete de notificación de Winlogon

En los temas siguientes se proporciona información de referencia para los [*paquetes de notificación de Winlogon:*](../secgloss/w-gly.md)

-   [Entradas del Registro](registry-entries.md) enumera los valores de clave y la configuración que se deben especificar en el registro del sistema para registrar un paquete de notificación y sus funciones de controlador de eventos.
-   [Prototipo de función del controlador de](event-handler-function-prototype.md) eventos detalla el prototipo de las funciones del controlador de eventos. Cada función de controlador de eventos del paquete de notificación debe coincidir con este prototipo.
