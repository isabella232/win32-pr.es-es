# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Especificaciones de IETF para Microsoft Digest SSP

Autenticación HTTP: la autenticación básica y de acceso implícita (RFC 2617) está disponible en [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

Puede usar digest como borrador de Internet del mecanismo SASL en [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Para obtener más información sobre SASL, consulte también la especificación de nivel de seguridad y autenticación simple (RFC 2222) disponible en [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
