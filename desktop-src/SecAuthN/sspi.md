# <a name="sspi"></a>SSPI

En esta sección se presenta información sobre [*la interfaz del proveedor de soporte técnico*](../secgloss/s-gly.md) de seguridad (SSPI). SSPI permite a una aplicación utilizar diversos modelos de seguridad disponibles en un equipo o una red sin cambiar la interfaz para el sistema de seguridad. Entre los temas se incluyen los siguientes:

-   [Modelo SSPI](sspi-model.md)
-   [Opciones de SSPI para aplicaciones distribuidas](sspi-options-for-distributed-applications.md)
-   [Semántica de contexto de SSPI](sspi-context-semantics.md)
-   [Paquetes SSP proporcionados por Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Paquetes de seguridad personalizados](custom-security-packages.md)
-   [Información de error extendida](extended-error-information.md)
