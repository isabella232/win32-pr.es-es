# <a name="tasksettingsxmltext-property"></a>Propiedad TaskSettings.XmlText

Para el scripting, obtiene o establece una definición con formato XML de la configuración de la tarea.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
TaskSettings.XmlText As String
```



## <a name="property-value"></a>Valor de propiedad

Definición con formato XML de la configuración de la tarea.
