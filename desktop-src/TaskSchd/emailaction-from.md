# <a name="emailactionfrom-property"></a>Propiedad EmailAction.From

\[Este objeto ya no se admite. Use IExecAction con el cmdlet [**Send-MailMessage de**](/powershell/module/microsoft.powershell.utility/send-mailmessage) PowerShell como solución alternativa.\]

Para el scripting, obtiene o establece la dirección de correo electrónico desde la que desea enviar el correo electrónico.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Syntax


```VB
EmailAction.From As String
```



## <a name="property-value"></a>Valor de propiedad

Dirección de correo electrónico desde la que desea enviar el correo electrónico.
