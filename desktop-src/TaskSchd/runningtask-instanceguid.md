# <a name="runningtaskinstanceguid-property"></a>Propiedad RunningTask.InstanceGuid

Para el scripting, obtiene el identificador GUID de esta instancia de la tarea.

## <a name="syntax"></a>Sintaxis


```VB
RunningTask.InstanceGuid As string
```



## <a name="property-value"></a>Valor de propiedad

Identificador GUID de esta instancia de la tarea. El servicio de Programador de tareas genera un identificador cada vez que se ejecuta la tarea.
