# <a name="emailactioncc-property"></a>EmailAction.Cc propiedad

\[Este objeto ya no se admite. Use IExecAction con el cmdlet [**Send-MailMessage de**](/powershell/module/microsoft.powershell.utility/send-mailmessage) PowerShell como solución alternativa.\]

Para el scripting, obtiene o establece la dirección de correo electrónico o las direcciones que desea cc en el mensaje de correo electrónico.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Syntax


```VB
EmailAction.Cc As String
```



## <a name="property-value"></a>Valor de propiedad

Dirección de correo electrónico o direcciones que desea cc en el mensaje de correo electrónico.
