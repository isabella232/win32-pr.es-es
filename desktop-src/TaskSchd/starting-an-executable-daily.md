# <a name="starting-an-executable-daily"></a>Iniciar un ejecutable a diario

La escritura de una tarea que inicia un archivo ejecutable a diario se realiza mediante la definición de un desencadenador diario y una acción ejecutable.

## <a name="daily-triggers"></a>Desencadenadores diarios

Los desencadenadores diarios usan su límite de inicio para activar el desencadenador y especificar la hora del día en que se ejecuta la tarea. Una vez activado el desencadenador, el intervalo de desencadenadores se usa para indicar si la tarea se ejecuta diariamente, cada dos días, cada tercer día, o así sucesivamente.
