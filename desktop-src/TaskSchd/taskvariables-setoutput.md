# <a name="taskvariablessetoutput-method"></a>Método TaskVariables.SetOutput

Para el scripting, establece las variables de salida de una tarea. Este método no se implementa.

## <a name="syntax"></a>Sintaxis


```VB
TaskVariables.SetOutput( _
  ByVal input _
)
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*entrada* \[ En\]
</dt> <dd>

Variables de salida de una tarea.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
