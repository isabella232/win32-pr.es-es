# <a name="runningtaskname-property"></a>RunningTask.Name propiedad

Para scripting, obtiene el nombre de la tarea.

## <a name="syntax"></a>Sintaxis


```VB
RunningTask.Name As string
```



## <a name="property-value"></a>Valor de propiedad
