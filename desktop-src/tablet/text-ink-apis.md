# <a name="text-ink-reference"></a>Referencia de Entrada manuscrita de texto

Además de admitir las interfaces OLE estándar, el objeto de entrada de lápiz de texto (tInk) también admite [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo).

Para obtener información sobre cómo especificar propiedades de presentación para un objeto de entrada de lápiz de texto, vea [**ESTRUCTURA INKMETRIC.**](/windows/win32/api/msinkaut/ns-msinkaut-inkmetric)
