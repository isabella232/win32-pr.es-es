# <a name="bitmap-element"></a>Bitmap (Elemento)

Contiene los datos codificados en Base64 para un mapa de bits contenido en el archivo Journal como una imagen de fondo.

## <a name="definition"></a>Definición

``` syntax
<xs:element name="Bitmap" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Elementos primarios

[**Imagen**](image-element.md)

## <a name="child-elements"></a>Elementos secundarios

Ninguno.

## <a name="attributes"></a>Atributos

Ninguno.

## <a name="element-information"></a>Información de elemento



|                  | Value                                      |
|------------------|--------------------------------------------|
| **Tipo de elemento** | xs:base64Binary                            |
| **Espacio de nombres**    | urn:schemas-microsoft-com:tabletpc:richink |
| **Nombre del esquema**  | Lector de diario                             |
