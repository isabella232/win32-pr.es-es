# <a name="about-language-models"></a>Acerca de los modelos de lenguaje

Los reconocedores de escritura a mano usan un conjunto definido de reglas para un idioma determinado como contexto principal para reconocer palabras. Estas reglas se denominan modelos de lenguaje. El reconocedor implementa el modelo de lenguaje y un desarrollador no puede cambiarlo directamente mediante el Kit de desarrollo de software (SDK) de Microsoft Windows XP Tablet PC Edition. Sin embargo, hay maneras de refinar el contexto para campos específicos de una aplicación. Estas técnicas incluyen:

-   Ámbitos de entrada
-   Factoids
-   Diccionarios
-   Listas de frases
