# <a name="docimage-element"></a>Elemento DocImage

Si la nota de diario que se está leyendo se creó mediante el controlador Escritura de notas de Journal impresora, la imagen de documento en segundo plano se mantiene en este elemento.

## <a name="definition"></a>Definición

``` syntax
<xs:element name="DocImage" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Elementos primarios

[**JournalPage**](journalpage-element.md)

## <a name="child-elements"></a>Elementos secundarios

Ninguno.

## <a name="attributes"></a>Atributos

Ninguno.

## <a name="element-information"></a>Información de elemento



|  Elemento     | Value                                                     |
|--------------|--------------------------------------------|
| Tipo de elemento | **xs:base64Binary**                        |
| Espacio de nombres    | urn:schemas-microsoft-com:tabletpc:richink |
| Nombre del esquema  | Lector de diario                             |
