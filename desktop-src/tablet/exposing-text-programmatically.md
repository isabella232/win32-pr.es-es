# <a name="exposing-text-programmatically"></a>Exponer texto mediante programación

Exponer texto mediante programación proporciona una gran ventaja para las ayuda de accesibilidad, ya que proporciona una interpretación mucho más confiable y, en algunos casos, más completa de la información.

Hay varios métodos básicos para exponer objetos para exponer texto mediante programación. Sin embargo, ningún método es efectivo con todas las ayuda de accesibilidad, por lo que también es necesario dibujar texto en la pantalla para que se pueda reconocer. Para obtener información sobre cómo exponer grandes cantidades de texto, vea [Using Active Accessibility to Expose Text](using-active-accessibility-to-expose-text.md).
