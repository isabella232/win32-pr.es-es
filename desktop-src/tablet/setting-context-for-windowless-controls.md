# <a name="setting-context-for-windowless-controls"></a>Establecer el contexto para los controles sin ventanas

Los controles sin ventana pueden llamar a la función [**SetInputScope**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) siempre que el foco se mueva entre los elementos del control sin ventana; sin embargo, esto no se recomienda por motivos de rendimiento.
