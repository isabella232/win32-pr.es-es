# <a name="winbio_component-constants"></a>Constantes WINBIO \_ COMPONENT

Las siguientes constantes se pueden usar al llamar a [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit) o [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged) para especificar el tipo de adaptador que se usa.



| Constante                                                                                                                                                                                        | Descripción                             |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------|
| <span id="WINBIO_COMPONENT_SENSOR"></span><span id="winbio_component_sensor"></span><dl> <dt>**SENSOR DE COMPONENTE WINBIO \_ \_**</dt> </dl>    | Especifica un adaptador de sensor.<br/>  |
| <span id="WINBIO_COMPONENT_ENGINE"></span><span id="winbio_component_engine"></span><dl> <dt>**MOTOR DE \_ COMPONENTES \_ WINBIO**</dt> </dl>    | Especifica un adaptador de motor.<br/> |
| <span id="WINBIO_COMPONENT_STORAGE"></span><span id="winbio_component_storage"></span><dl> <dt>**ALMACENAMIENTO DE \_ COMPONENTES WINBIO \_**</dt> </dl> | Especifica un adaptador de almacenamiento.<br/> |
