# <a name="experiences"></a>Experiencias

> [!NOTE]
> Esta guía de diseño se creó para Windows 7 y no se ha actualizado para las versiones más recientes de Windows. Gran parte de las instrucciones se siguen aplicando en principio, pero la presentación y los ejemplos no reflejan nuestra [guía de diseño actual.](/windows/uwp/design/)

Las experiencias son las experiencias comunes y los casos de uso de todas las aplicaciones, como la configuración, la primera ejecución y la impresión. Obtenga información sobre los procedimientos recomendados para crear estas experiencias y comunicar su marca de aplicación.

## <a name="in-this-section"></a>En esta sección



| Tema                                                | Descripción                                                                                                                                                                                                                        |
|------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Personal de marca de software](exper-branding.md)<br/>   | La personal de marca es el posicionamiento emocional de un producto que perciben sus clientes. La personalización de marca correcta requiere la creación habilidosa de una imagen de producto y no se consigue solo a través de logotipos de productos y combinaciones de colores. <br/> |
| [Configuración](exper-setup.md)<br/>                  | A los usuarios no les gusta instalar software, por lo que las experiencias de configuración modernas deben ser sencillas, eficaces y sin problemas. <br/>                                                                                                     |
| [Primera experiencia](exper-first-exper.md)<br/> | En la primera experiencia ideal, los usuarios instalan el programa y lo usan de forma productiva inmediatamente, sin responder a un montón de preguntas ni aprender muchas cosas. <br/>                                                   |
| [Impresión](exper-printing.md)<br/>            | La impresión es la experiencia del usuario en papel. Es fácil pasar por alto, pero es una parte importante de la experiencia general del usuario. <br/>                                                                                           |
