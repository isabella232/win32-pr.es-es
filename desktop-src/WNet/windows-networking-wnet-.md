# <a name="windows-networking-wnet"></a>Windows Redes (WNet)

Las Windows de red de red (WNet) permiten implementar funcionalidades de red en la aplicación sin tener que hacer concesiones para un proveedor de red determinado o una implementación de red física. Esto se debe a que las funciones de WNet son independientes de la red.
