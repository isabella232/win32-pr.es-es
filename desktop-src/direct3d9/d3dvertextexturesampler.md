# <a name="d3dvertextexturesampler"></a>D3DVERTEXTEXTURESAMPLER

Constantes de muestreador de textura de vértice.

Estas constantes identifican los muestreadores de textura utilizados por los sombreadores de vértices.



| \#Definir                 | Descripción        |
|--------------------------|--------------------|
| D3DVERTEXTEXTURESAMPLER0 | (D3DDMAPSAMPLER+1) |
| D3DVERTEXTEXTURESAMPLER1 | (D3DDMAPSAMPLER+2) |
| D3DVERTEXTEXTURESAMPLER2 | (D3DDMAPSAMPLER+3) |
| D3DVERTEXTEXTURESAMPLER3 | (D3DDMAPSAMPLER+4) |



 

## <a name="d3ddmapsampler"></a>D3DDMAPSAMPLER

Se trata de una constante interna utilizada por el muestreador de textura en el teselador.



| \#Definir       | Descripción                                           |
|----------------|-------------------------------------------------------|
| D3DDMAPSAMPLER | 256, que es el número máximo de muestreadores de textura. |



 

## <a name="constant-information"></a>Información constante



| Requisito                         | Value            |
|--------------------------|-------------|
| Encabezado                   | d3d9types.h |
| Sistema operativo mínimo | Windows 98  |
