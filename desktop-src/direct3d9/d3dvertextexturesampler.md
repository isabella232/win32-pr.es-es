# <a name="d3dvertextexturesampler"></a>D3DVERTEXTEXTURESAMPLER

Constantes de muestra de textura de vértices.

Estas constantes identifican los muestreadores de textura utilizados por los sombreadores de vértices.



|                          |                    |
|--------------------------|--------------------|
| \#define                 | Descripción        |
| D3DVERTEXTEXTURESAMPLER0 | (D3DDMAPSAMPLER + 1) |
| D3DVERTEXTEXTURESAMPLER1 | (D3DDMAPSAMPLER + 2) |
| D3DVERTEXTEXTURESAMPLER2 | (D3DDMAPSAMPLER + 3) |
| D3DVERTEXTEXTURESAMPLER3 | (D3DDMAPSAMPLER + 4) |



 

## <a name="d3ddmapsampler"></a>D3DDMAPSAMPLER

Se trata de una constante interna usada por la muestra de textura en del teselador.



|                |                                                       |
|----------------|-------------------------------------------------------|
| \#define       | Descripción                                           |
| D3DDMAPSAMPLER | 256, que es el número máximo de muestras de textura. |
