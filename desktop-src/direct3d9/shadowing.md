# <a name="shadowing-direct3d-9"></a>Sombreado (Direct3D 9)

Los búferes de galería de símbolos son una técnica de búfer de profundidad que se puede actualizar a medida que se representa la geometría y se usa de nuevo como máscara para dibujar más geometría. Los efectos comunes incluyen reflejos, sombras (una técnica avanzada), disoluciones, entre otros. El [ejemplo ShadowVolume usa](https://msdn.microsoft.com/library/Ee418792(v=VS.85).aspx) búferes de galería de símbolos para implementar sombras en tiempo real. La galería de símbolos de dos lados [(Direct3D 9)](two-sided-stencil.md)contiene información adicional.
