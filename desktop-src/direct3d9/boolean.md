Define un tipo booleano simple.

``` syntax
template Boolean
{
    < 537da6a0-ca37-11d0-941c-0080c80cfa7b >
    DWORD truefalse;

```

Donde:

-   truefalse: true para un valor booleano; en caso contrario, false.
