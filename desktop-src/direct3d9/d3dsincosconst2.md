# <a name="d3dsincosconst2"></a>D3DSINCOSCONST2

Crea la segunda constante (en el registro src3) para la instrucción [del sombreador de píxeles sincos - ps.](../direct3dhlsl/sincos---ps.md)

``` syntax
#define D3DSINCOSCONST2 -0.020833334f, -0.12500000f, 1.0f, 0.50000000f
```

## <a name="return-value"></a>Valor devuelto

Devuelve la segunda constante necesaria para [sincos: ps](../direct3dhlsl/sincos---ps.md).
