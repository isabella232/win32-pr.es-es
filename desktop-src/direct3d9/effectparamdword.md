# <a name="effectparamdword"></a>EffectParamDWord

Define un efecto DWORD. Esta plantilla ha reemplazado a [**la plantilla EffectDWord.**](effectdword.md)

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Donde:

-   ParamName: nombre del parámetro.
-   Valor: valor del parámetro.
