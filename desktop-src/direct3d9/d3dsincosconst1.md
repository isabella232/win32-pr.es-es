# <a name="d3dsincosconst1"></a>D3DSINCOSCONST1

Crea la primera constante (en el registro src2) para la instrucción [del sombreador de píxeles sincos - ps.](../direct3dhlsl/sincos---ps.md)

``` syntax
#define D3DSINCOSCONST1  -1.5500992e-006f, -2.1701389e-005f,  0.0026041667f, 0.00026041668f
```

## <a name="return-value"></a>Valor devuelto

Devuelve la primera constante necesaria para [sincos : ps](../direct3dhlsl/sincos---ps.md).
