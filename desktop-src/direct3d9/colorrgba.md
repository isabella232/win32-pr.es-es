# <a name="colorrgba"></a>ColorRGBA

Define un objeto de color con un componente alfa. Se usa para el color de la superficie en la definición de la plantilla de material.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
