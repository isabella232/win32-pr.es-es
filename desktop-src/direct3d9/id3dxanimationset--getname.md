# <a name="id3dxanimationsetgetname-method"></a>Método ID3DXAnimationSet::GetName

Obtiene el nombre del conjunto de animaciones.

## <a name="syntax"></a>Sintaxis


```C++
LPCSTR GetName();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **LPCSTR**](../winprog/windows-data-types.md)**

Nombre del conjunto de animaciones.
