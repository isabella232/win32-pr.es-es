# <a name="d3dusage"></a>D3DUSAGE

Opciones de uso que identifican cómo se van a usar los recursos.

En la tabla siguiente se resumen las opciones de uso disponibles.



| Constante                                                                                                                                                                                                                                          | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="D3DUSAGE_AUTOGENMIPMAP"></span><span id="d3dusage_autogenmipmap"></span><dl> <dt>**D3DUSAGE \_ AUTOGENMIPMAP**</dt> </dl>                                                          | El recurso generará automáticamente mapas MIP. Vea [Generación automática de mapas Mip (Direct3D 9).](automatic-generation-of-mipmaps.md) No se admite la generación automática de mapas mip para texturas de volumen y superficies o texturas de galería de símbolos de profundidad. Este uso no es válido para un recurso en la memoria del sistema (**D3DPOOL \_ SYSTEMMEM**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="D3DUSAGE_DEPTHSTENCIL"></span><span id="d3dusage_depthstencil"></span><dl> <dt>**D3DUSAGE \_ DEPTHSTENCIL**</dt> </dl>                                                             | El recurso será un búfer de galería de símbolos de profundidad. **D3DUSAGE \_ DEPTHSTENCIL solo** se puede usar con **D3DPOOL \_ DEFAULT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_DMAP"></span><span id="d3dusage_dmap"></span><dl> <dt>**D3DUSAGE \_ DMAP**</dt> </dl>                                                                                     | El recurso será un mapa de desplazamiento.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_DONOTCLIP"></span><span id="d3dusage_donotclip"></span><dl> <dt>**D3DUSAGE \_ DONOTCLIP**</dt> </dl>                                                                      | Se establece para indicar que el contenido del búfer de vértices nunca requerirá recorte. Al representar con búferes que tienen esta marca establecida, el estado de representación de CLIPPING de **D3DRS \_** debe establecerse en false.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_DYNAMIC"></span><span id="d3dusage_dynamic"></span><dl> <dt>**D3DUSAGE \_ DYNAMIC**</dt> </dl>                                                                            | Se establece para indicar que el búfer de vértices requiere el uso de memoria dinámica. Esto es útil para los controladores porque les permite decidir dónde colocar el búfer. En general, los búferes de vértice estáticos se colocan en la memoria de vídeo y los búferes de vértices dinámicos se colocan en la memoria de AGP. Tenga en cuenta que no hay ningún uso estático independiente. Si no especifica **D3DUSAGE \_ DYNAMIC,** el búfer de vértices se hace estático. **D3DUSAGE \_ DYNAMIC** se aplica estrictamente a través de las marcas de bloqueo **D3DLOCK \_ DISCARD** y **D3DLOCK \_ NOOVERWRITE.** Como resultado, **D3DLOCK \_ DISCARD** y **D3DLOCK \_ NOOVERWRITE** solo son válidos en búferes de vértice creados con **D3DUSAGE \_ DYNAMIC**. No son marcas válidas en búferes de vértice estáticos. Para obtener más información, vea [Administración de recursos (Direct3D 9).](managing-resources.md) <br/> Para obtener más información sobre el uso de búferes de vértices [dinámicos, vea Optimizaciones de rendimiento (Direct3D 9).](performance-optimizations.md)<br/> **D3DUSAGE \_ DYNAMIC** y **D3DPOOL \_ MANAGED** son incompatibles y no deben usarse juntos. Consulte [**D3DPOOL**](./d3dpool.md).<br/> Las texturas pueden **especificar D3DUSAGE \_ DYNAMIC.** Sin embargo, las texturas administradas no **pueden usar D3DUSAGE \_ DYNAMIC**. Para obtener más información sobre las texturas dinámicas, vea [Usar texturas dinámicas.](performance-optimizations.md) <br/> |
| <span id="D3DUSAGE_NONSECURE"></span><span id="d3dusage_nonsecure"></span><dl> <dt>**D3DUSAGE \_ NO SEGURO**</dt> </dl>                                                                      | Permitir que una aplicación no segura que tenga el identificador compartido abra una superficie compartida creada por una aplicación segura. <br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="D3DUSAGE_NPATCHES"></span><span id="d3dusage_npatches"></span><dl> <dt>**D3DUSAGE \_ NPATCHES**</dt> </dl>                                                                         | Se establece para indicar que el búfer de vértices se va a usar para dibujar N revisiones.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_POINTS"></span><span id="d3dusage_points"></span><dl> <dt>**PUNTOS D3DUSAGE \_**</dt> </dl>                                                                               | Se establece para indicar que el vértice o búfer de índice se usará para los sprites de punto de dibujo. El búfer se cargará en la memoria del sistema si se necesita el procesamiento de vértices de software para emular sprites de punto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="D3DUSAGE_RENDERTARGET"></span><span id="d3dusage_rendertarget"></span><dl> <dt>**D3DUSAGE \_ RENDERTARGET**</dt> </dl>                                                             | El recurso será un destino de representación. **D3DUSAGE \_ RENDERTARGET** solo se puede usar con **D3DPOOL \_ DEFAULT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="D3DUSAGE_RTPATCHES"></span><span id="d3dusage_rtpatches"></span><dl> <dt>**D3DUSAGE \_ RTPATCHES**</dt> </dl>                                                                      | Se establece para indicar que el búfer de vértices se va a usar para dibujar primitivas de orden alto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="D3DUSAGE_SOFTWAREPROCESSING"></span><span id="d3dusage_softwareprocessing"></span><dl> <dt>**D3DUSAGE \_ SOFTWAREPROCESSING**</dt> </dl>                                           | Si se usa esta marca, el procesamiento de vértices se realiza en software. Si no se usa esta marca, el procesamiento de vértices se realiza en hardware.<br/> La marca **D3DUSAGE \_ SOFTWAREPROCESSING** se puede establecer cuando el procesamiento de vértices de software o de modo mixto **(D3DCREATE \_ MIXED \_ VERTEXPROCESSING**  /  **D3DCREATE \_ SOFTWARE \_ VERTEXPROCESSING)** está habilitado para ese dispositivo. **D3DUSAGE \_ SOFTWAREPROCESSING** debe establecerse para que los búferes se usen con el procesamiento de vértices de software en modo mixto, pero no se debe establecer para obtener el mejor rendimiento posible cuando se usa el procesamiento de índices de hardware en modo mixto **(D3DCREATE \_ HARDWARE \_ VERTEXPROCESSING).** Sin embargo, establecer **D3DUSAGE \_ SOFTWAREPROCESSING** es la única opción cuando se usa un solo búfer con el procesamiento de vértices de hardware y software. **D3DUSAGE \_ SOFTWAREPROCESSING se** permite para dispositivos mixtos y de software.<br/> **D3DUSAGE \_ SOFTWAREPROCESSING se** usa con [**CheckDeviceFormat**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdeviceformat) para averiguar si se puede usar un formato de textura determinado como textura de vértice durante el procesamiento de vértices de software. Si es posible, la textura debe crearse en **D3DPOOL \_ SCRATCH.**<br/>                                                                                                                                                                 |
| <span id="D3DUSAGE_TEXTAPI"></span><span id="d3dusage_textapi"></span><dl> <dt>**TEXTAPI de D3DUSAGE \_**</dt> </dl>                                                                            | Esta marca de uso debe especificarse para los búferes de vértices y las superficies de origen, que se usan en las llamadas [**a ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects). Las extures creadas con esta marca de uso no se pueden usar para el filtrado de texturas. Los búferes de vértice, creados con esta marca de uso, no se pueden usar como orígenes de flujo de entrada. <br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_WRITEONLY"></span><span id="d3dusage_writeonly"></span><dl> <dt>**D3DUSAGE \_ WRITEONLY**</dt> </dl>                                                                      | Informa al sistema de que la aplicación solo escribe en el búfer de vértices. El uso de esta marca permite al controlador elegir la mejor ubicación de memoria para una representación y operaciones de escritura eficaces. Se producirá un error al intentar leer desde un búfer de vértices creado con esta funcionalidad. Los búferes creados **con D3DPOOL \_ DEFAULT** que no especifican **D3DUSAGE \_ WRITEONLY** pueden sufrir una grave penalización de rendimiento. **D3DUSAGE \_ WRITEONLY solo** afecta al rendimiento de los búferes **D3DPOOL \_ DEFAULT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_RESTRICTED_CONTENT"></span><span id="d3dusage_restricted_content"></span><dl> <dt>**CONTENIDO RESTRINGIDO DE D3DUSAGE \_ \_**</dt> </dl>                                          | Establecer esta marca indica que el recurso puede contener contenido protegido.<br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE"></span><span id="d3dusage_restrict_shared_resource"></span><dl> <dt>**RESTRICCIÓN DE RECURSOS COMPARTIDOS DE D3DUSAGE \_ \_ \_**</dt> </dl>                       | Al establecer esta marca se indica que se debe restringir el acceso al recurso compartido. <br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE_DRIVER"></span><span id="d3dusage_restrict_shared_resource_driver"></span><dl> <dt>**D3DUSAGE \_ RESTRICT \_ SHARED \_ RESOURCE \_ DRIVER**</dt> </dl> | Establecer esta marca indica que el controlador debe restringir el acceso al recurso compartido. El autor de la llamada debe crear un canal autenticado con el controlador. A continuación, el controlador debe permitir el acceso a los procesos que intentan abrir ese recurso compartido.<br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
