## <a name="remarks"></a>Comentarios

Los errores se representan mediante valores negativos y no se pueden combinar.

Otros códigos de retorno se incluyen en [S \_ PRESENT](device-state-return-codes.md).

El código de la \_ instalación FACD3D se usa para generar códigos de error, como en las macros siguientes.


```
// Direct3D Errors
#define D3DERR_UNSUPPORTEDFACTORVALUE  MAKE_D3DHRESULT(2079)
#define D3DERR_WASSTILLDRAWING         MAKE_D3DHRESULT(540)
    
#define _FACD3D                  0x876   // Direct3D facility code
#define MAKE_D3DHRESULT( code )  MAKE_HRESULT( 1, _FACD3D, code )
#define MAKE_D3DSTATUS( code )   MAKE_HRESULT( 0, _FACD3D, code )
```
