# <a name="id3dxprtcompbuffergetnumpca-method"></a>Método ID3DXPRTCompBuffer::GetNumPCA

Recupera el número de vectores de base de análisis de componentes principales (PCA) que se usarán en cada clúster.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetNumPCA();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Devuelve el número de vectores de base de análisis de componentes principales (PCA) que se usarán en cada clúster.
