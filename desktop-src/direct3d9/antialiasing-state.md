# <a name="antialiasing-state-direct3d-9"></a>Estado de suavizado de contorno (Direct3D 9)

El [suavizado de contorno (Direct3D 9)](antialiasing.md) es un método para que los bordes y las líneas aparezcan más suaves en la pantalla. De forma predeterminada, Direct3D no realiza el suavizado de contorno. Para habilitar el suavizado de contorno completo de la escena, establezca el estado de representación de D3DRS \_ MULTISAMPLEANTIALIAS en **true**. Para deshabilitarlo, establezca D3DRS \_ MULTISAMPLEANTIALIAS en **false**.
