# <a name="floatkeys"></a>FloatKeys

Define una matriz de números de punto flotante (floats) y el número de flotantes de esa matriz. Se usa para definir conjuntos de claves de animación.

``` syntax
template FloatKeys
{
    < 10DD46A9-775B-11cf-8F52-0040333594A3 >
    DWORD nValues;
    array float values[nValues];
} 
```

-   nValues: número de flotantes.
-   values \[ nValues: \] matriz de valores float.
