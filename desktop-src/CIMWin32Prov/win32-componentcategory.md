# <a name="win32_componentcategory-class"></a>\_Clase Win32 ComponentCategory

La [clase WMI](/windows/desktop/WmiSdk/retrieving-a-class) **\_ ComponentCategory de Win32** representa una categoría de componente. Las categorías de componentes son grupos de clases del modelo de objetos componentes (COM) con un conjunto de funcionalidades definido compartido entre ellas. Un cliente que usa estas interfaces consulta el registro para el título de categoría y el identificador único denominado **CategoryID**, que se crea a partir de un identificador único global (GUID).

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{0F73ED5A-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_ComponentCategory : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Status;
  string   CategoryId;
  string   Name;
};
```
