# <a name="win32_comclass-class"></a>Clase COMClass de Win32 \_

La clase WMI abstracta **\_ COMClass** [de](/windows/desktop/WmiSdk/retrieving-a-class) Win32 representa las propiedades de un componente del Modelo de objetos componentes (COM).

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract, UUID("{0F73ED50-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMClass : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
