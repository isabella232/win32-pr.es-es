# <a name="win32_comclass-class"></a>\_Clase de Comclase Win32

La [clase WMI](/windows/desktop/WmiSdk/retrieving-a-class) abstracta **\_ comClass de Win32** representa las propiedades de un componente del modelo de objetos componentes (com).

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract, UUID("{0F73ED50-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMClass : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
