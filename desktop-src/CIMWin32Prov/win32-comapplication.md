# <a name="win32_comapplication-class"></a>\_Clase Comapplication de Win32

La [clase WMI](/windows/desktop/WmiSdk/retrieving-a-class) abstracta **\_ comapplication de Win32** representa una aplicación com (modelo de objetos componentes). En este contexto, una aplicación COM es una agrupación lógica de clases COM.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract, UUID("{0F73ED4F-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMApplication : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
