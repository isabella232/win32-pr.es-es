# <a name="win32_comapplication-class"></a>Clase COMApplication de Win32 \_

La clase WMI abstracta **\_ COMApplication** [de](/windows/desktop/WmiSdk/retrieving-a-class) Win32 representa una aplicación de Modelo de objetos componentes (COM). En este contexto, una aplicación COM es una agrupación lógica de clases COM.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract, UUID("{0F73ED4F-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_COMApplication : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
