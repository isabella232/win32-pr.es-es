# <a name="temporary-creation-flags"></a>Marcas de creación temporal

La marca de creación temporal indica que el archivo subyacente se destruirá automáticamente cuando se libera el objeto de almacenamiento raíz. Esta funcionalidad es muy útil para crear archivos temporales.
