# <a name="mf_sa_d3d12_heap_flags-attribute"></a>Atributo MF \_ SA \_ D3D12 \_ HEAP \_ FLAGS

Contiene un valor que especifica las opciones del montón usadas para los recursos de Direct3D en la secuencia.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor de este atributo es una combinación OR bit a bit de valores de la [D3D12_HEAP_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_heap_flags) enumeración .
