# <a name="codecapi_avencvideominqp-property"></a>Propiedad CODECAPI \_ AVEncVideoMinQP

Especifica el parámetro de cuantificación mínimo (QP) para la codificación de vídeo.

## <a name="data-type"></a>Tipo de datos

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoMinQP**

## <a name="property-value"></a>Valor de propiedad

El valor es el QP mínimo que se usa en la codificación QP de variable.
