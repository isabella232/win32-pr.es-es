# <a name="mf_devicestream_max_frame_buffers-attribute"></a>Atributo \_ MF DEVICESTREAM \_ MAX FRAME \_ \_ BUFFERS

Especifica el número máximo de fotogramas que el origen de captura de vídeo almacenará en búfer para esta secuencia.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------------------------------------|
| Cliente mínimo compatible<br/> | \[Windows 8 solo aplicaciones de escritorio\]<br/>                                         |
| Servidor mínimo compatible<br/> | \[Windows Server 2012 solo aplicaciones de escritorio\]<br/>                               |
