# <a name="d3dauthenticatedconfigure_protection"></a>D3DAUTHENTICATEDCONFIGURE \_ PROTECTION

Habilita o deshabilita la protección del dispositivo.



| Requisito | Value |
|--------------|-----------------------------------------------------------------------------------------------------|
| GUID de comando | **D3DAUTHENTICATEDCONFIGURE \_ PROTECTION**                                                           |
| Datos de entrada   | [**D3DAUTHENTICATEDCHANNEL \_ CONFIGUREPROTECTION**](d3dauthenticatedchannel-configureprotection.md) |



 

## <a name="remarks"></a>Observaciones

Este comando es válido para todos los tipos de canal.
