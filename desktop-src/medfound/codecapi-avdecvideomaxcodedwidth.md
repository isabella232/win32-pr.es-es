# <a name="codecapi_avdecvideomaxcodedwidth-property"></a>Propiedad CODECAPI \_ AVDecVideoMaxCodedWidth

Establece el ancho máximo de imagen que un descodificador de vídeo aceptará como tipo de entrada.

## <a name="data-type"></a>Tipo de datos

**LONG** (**VT \_ I4**)

## <a name="property-guid"></a>GUID de propiedad

CODECAPI \_ AVDecVideoMaxCodedWidth
