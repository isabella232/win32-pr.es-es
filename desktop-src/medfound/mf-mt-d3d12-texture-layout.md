# <a name="mf_mt_d3d12_texture_layout-attribute"></a>Atributo MF \_ MT \_ D3D12 \_ TEXTURE \_ LAYOUT

Indica las opciones de diseño de textura que se usaron para crear los recursos de Direct3D asociados.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor de este atributo corresponde a un miembro de la enumeración [D3D12_TEXTURE_LAYOUT](/windows/win32/api/d3d12/ne-d3d12-d3d12_texture_layout) datos.
