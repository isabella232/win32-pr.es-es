# <a name="codecapi_avdecvideomaxcodedheight-property"></a>Propiedad CODECAPI \_ AVDecVideoMaxCodedHeight

Establece el alto máximo de imagen que aceptará un descodificador de vídeo como tipo de entrada.

## <a name="data-type"></a>Tipo de datos

**LONG** (**VT \_ I4**)

## <a name="property-guid"></a>GUID de propiedad

CODECAPI \_ AVDecVideoMaxCodedHeight
