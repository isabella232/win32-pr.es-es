# <a name="topoedit-main-window"></a>Ventana principal de TopoEdit

Después de iniciar TopoEdit, aparece la ventana que se muestra en la siguiente captura de pantalla.

![captura de pantalla que muestra la ventana principal de topoedit](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Los elementos de la ventana principal TopoEdit son:

-   **Panel Topología**. Aparece a la izquierda de la ventana. Esta área se usa para agregar nodos de topología, ver una topología y conectar los nodos de topología. Para obtener información sobre el uso de este panel, vea [Building Topologies by Using TopoEdit](building-topologies-by-using-topoedit.md).
-   **Panel Atributos**. Aparece a la derecha de la ventana. En este panel se muestran los atributos del nodo de topología seleccionado actualmente. Para obtener información sobre los atributos que se muestran en este panel, vea [Ver información de topología.](viewing-topology-information.md)
-   Menús. Vea [TopoEdit Menus (Menús de TopoEdit).](topoedit-menus.md)
-   Barra de herramientas. Vea [TopoEdit Toolbar](topoedit-toolbar.md).
