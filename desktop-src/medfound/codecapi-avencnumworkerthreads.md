# <a name="codecapi_avencnumworkerthreads-property"></a>Propiedad CODECAPI \_ AVEncNumWorkerThreads

Establece el número de subprocesos de trabajo utilizados por un codificador de vídeo.

## <a name="data-type"></a>Tipo de datos

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncNumWorkerThreads**

## <a name="remarks"></a>Observaciones

Si el valor es cero, el codificador selecciona el número de subprocesos.
