# <a name="mf_mt_d3d12_cpu_readback-attribute"></a>Atributo READBACK de CPU MF \_ MT \_ D3D12 \_ \_

Indica si se requiere acceso a la CPU para los recursos de Direct3D asociados. 

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor predeterminado de este atributo es 0 (FALSE).
