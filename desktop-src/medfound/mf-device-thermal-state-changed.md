# <a name="mf_device_thermal_state_changed-attribute"></a>Atributo MF \_ DEVICE \_ MF STATE \_ \_ CHANGED

Representa un evento que señala un cambio de estado térmico en el dispositivo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

El valor de este atributo especifica el valor térmico del dispositivo en porcentaje.
