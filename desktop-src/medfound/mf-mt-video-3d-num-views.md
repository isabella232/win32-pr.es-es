# <a name="mf_mt_video_3d_num_views-attribute"></a>Atributo MF \_ MT \_ VIDEO \_ 3D \_ NUM \_ VIEWS

Número de vistas en una secuencia de vídeo 3D.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Una secuencia de vídeo 3D típica contiene dos vistas, izquierda y derecha. Actualmente, el valor de este atributo debe ser 1 o 2.
