# <a name="mf_mt_dv_aaux_src_pack_1-attribute"></a>Atributo MF \_ MT \_ DV \_ \_ AATTRIBUTE SRC \_ PACK \_ 1

Paquete de origen auxiliar de audio (A ODBC) para el segundo bloque de audio en un tipo de medio de vídeo digital (DV).

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Este atributo corresponde al miembro **dwDVAAttributeSrc1** de la estructura [**DirectShow DVINFO.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

La constante GUID para este atributo se exporta desde mfuuid.lib.
