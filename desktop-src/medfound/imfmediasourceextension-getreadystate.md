# <a name="imfmediasourceextensiongetreadystate-method"></a>MÉTODO IMFMediaSourceExtension::GetReadyState

Obtiene el estado listo del origen de medios.

## <a name="syntax"></a>Sintaxis


```C++
MF_MSE_READY GetReadyState();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Estado listo del origen de medios.
