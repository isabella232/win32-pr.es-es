# <a name="mf_mt_max_luminance_level-attribute"></a>Atributo MF \_ MT \_ MAX \_ LUMINANCE \_ LEVEL

Especifica el nivel máximo de luminosidad del contenido en Nits. Este atributo tiene la misma semántica que **MaxCLL** tal como se define en el estándar CEA-861.3.

## <a name="data-type"></a>Tipo de datos

**UINT32**
