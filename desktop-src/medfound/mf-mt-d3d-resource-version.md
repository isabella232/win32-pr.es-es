# <a name="mf_mt_d3d_resource_version-attribute"></a>Atributo MF \_ MT \_ D3D \_ RESOURCE \_ VERSION

Especifica la versión de Direct3D de los recursos almacenados en el flujo de datos asociado al tipo de medio. 

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor de este atributo corresponde a un miembro de la [enumeración MF_MT_D3D_RESOURCE_VERSION_ENUM](/windows/win32/api/mfd3d12/ne-mfd3d12-mf_mt_d3d_resource_version_enum) datos.
