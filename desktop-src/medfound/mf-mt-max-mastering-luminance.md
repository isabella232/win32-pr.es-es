# <a name="mf_mt_max_mastering_luminance-attribute"></a>Atributo MF \_ MT \_ MAX \_ MASTERING \_ LUMINANCE

Especifica la luminosidad máxima de la pantalla en la que se autorizó el contenido, en Nits. Este atributo tiene la misma semántica que la luminosidad de **\_ \_ masterización \_** de pantalla máxima tal y como se define en el estándar CEA-861.3.

## <a name="data-type"></a>Tipo de datos

**UINT32**
