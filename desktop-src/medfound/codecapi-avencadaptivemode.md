# <a name="codecapi_avencadaptivemode-property"></a>Propiedad CODECAPI \_ AVEncAdaptiveMode

Especifica el modo de codificación adaptable para un codificador de vídeo.

## <a name="data-type"></a>Tipo de datos

**ULONG** (VT \_ UI4)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncAdaptiveMode**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es un **OR** bit a bit de valores de la [**enumeración eAVEncAdaptiveMode.**](/windows/win32/api/codecapi/ne-codecapi-eavencadaptivemode)
