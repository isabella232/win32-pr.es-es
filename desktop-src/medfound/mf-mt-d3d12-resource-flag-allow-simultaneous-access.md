# <a name="mf_mt_d3d12_resource_flag_allow_simultaneous_access-attribute"></a>Mf \_ MT \_ D3D12 \_ RESOURCE FLAG ALLOW SIMULTANEOUS ACCESS \_ \_ \_ \_ attribute

Indica si varias colas de comandos diferentes pueden acceder simultáneamente a los recursos de Direct3D de la secuencia.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor de este atributo corresponde al miembro D3D12_RESOURCE_FLAG_ALLOW_SIMULTANEOUS_ACCESS de la [enumeración D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) datos.
