# <a name="table-of-contents-parser-interfaces"></a>Interfaces de analizador de tabla de contenido

Las siguientes interfaces demostraron el acceso a los servicios del analizador de tabla de contenido.



| Interfaz                                | Descripción                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoc)                     | Representa una tabla de contenido individual.                                                                           |
| [**ITocCollection**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoccollection) | Representa una colección de tablas de contenido.                                                                        |
| [**ITocEntry**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentry)           | Representa una entrada individual de una tabla de contenido.                                                                |
| [**ITocEntryList**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentrylist)   | Representa una lista de entradas de una tabla de contenido.                                                                  |
| [**ITocParser**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocparser)         | Proporciona métodos para incrustar una tabla de contenido en un archivo de vídeo y leer tablas de contenido de un archivo de vídeo. |
