# <a name="mf_mt_forward_custom_nalu-attribute"></a>Atributo \_ \_ NALU MF MT FORWARD \_ \_ CUSTOM

Especifica que el descodificador debe reenviar los tipos de unidad de capa de abstracción de red (NAL) en los ejemplos de salida.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

Si el descodificador analiza una NALU, no se reenviará.
