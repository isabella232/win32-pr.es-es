# <a name="typeuri-element"></a>elemento typeUri

Especifica un tipo que se va a incluir de un archivo XSD.

## <a name="usage"></a>Uso

``` syntax
<typeUri
  type = "character_string"
  uri = "character_string"/>
```

## <a name="attributes"></a>Atributos



| Atributo           | Tipo                         | Obligatorio       | Descripción                                                            |
|---------------------|------------------------------|----------------|------------------------------------------------------------------------|
| **type**<br/> | cadena de caracteres \_<br/> | Sí<br/> | Nombre del tipo.<br/> <br/>                           |
| **uri**<br/>  | cadena de caracteres \_<br/> | Sí<br/> | Espacio de nombres del tipo. Debe ser un URI válido.<br/> <br/> |



## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                       | Descripción                                                                       |
|-------------------------------|-----------------------------------------------------------------------------------|
| [**BTM**](xsd.md)<br/> | Especifica un archivo XSD para procesar la información de contrato.<br/> <br/> |
