# <a name="texture3dsample-methods"></a>Métodos Texture3D::Sample

Muestrea una textura.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                       | Descripción                                                                                                                               |
|:-----------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**Sample(S,float,int)**](dx-graphics-hlsl-to-sample.md)                    | Muestrea una textura.<br/>                                                                                                             |
| [**Sample(S,float,int,float)**](t3d-sample-s-float-int-float-.md)           | Muestrea una textura con un valor opcional para fijar los valores de nivel de detalle (LOD) de la muestra.<br/>                                      |
| [**Sample(S,float,int,float,uint)**](t3d-sample-s-float-int-float-uint-.md) | Muestrea una textura con un valor opcional para fijar los valores de nivel de detalle (LOD) de la muestra y devuelve el estado de la operación.<br/> |
