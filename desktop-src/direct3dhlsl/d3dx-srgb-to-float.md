# <a name="d3dx_srgb_to_float-function"></a>Función D3DX \_ SRGB \_ to \_ FLOAT

Convierte un valor SRGB en FLOAT.

## <a name="syntax"></a>Sintaxis

``` syntax
FLOAT D3DX_SRGB_to_FLOAT(
   UINT val
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*Val* 
</dt> <dd>

</dd> </dl>
