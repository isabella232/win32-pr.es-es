# <a name="shader-model-51"></a>Modelo de sombreador 5.1

Esta sección contiene las páginas de referencia del modelo de sombreador HLSL 5.1, introducido con D3D12.

El modelo de sombreador 5.1 es funcionalmente muy similar al modelo de sombreador [5;](d3d11-graphics-reference-sm5.md)el cambio principal es más flexibilidad en la selección de recursos al permitir la indexación de matrices de descriptores desde dentro de un sombreador.



| Característica                   | Funcionalidad                                                                |
|---------------------------|---------------------------------------------------------------------------|
| Conjunto de instrucciones           | [**Funciones intrínsecas HLSL**](dx-graphics-hlsl-intrinsic-functions.md)  |
| Máximo del sombreador de vértices         | Sin restricción                                                            |
| Máximo de sombreador de píxeles          | Sin restricción                                                            |
| Nuevos perfiles de sombreador agregados | cs \_ 5 \_ 1, ds \_ 5 \_ 1, gs \_ 5 \_ 1, hs \_ 5 \_ 1, ps \_ 5 \_ 1, vs \_ 5 \_ 1, rootsig \_ 1 \_ 0 |
