# <a name="hlsl-specified-root-signature"></a>Firma raíz especificada de HLSL

Una [firma raíz](/windows/desktop/direct3d12/root-signatures) (una tabla de claves de recursos y otros elementos para D3D12) se puede especificar en HLSL como una cadena.

Consulte la documentación de D3D12 en Especificación de firmas raíz en [HLSL](/windows/desktop/direct3d12/specifying-root-signatures-in-hlsl) para obtener detalles completos y código de ejemplo.
