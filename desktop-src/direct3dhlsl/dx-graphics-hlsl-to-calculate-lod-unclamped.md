# <a name="calculatelevelofdetailunclamped-directx-hlsl-texture-object"></a>CalculateLevelOfDetailUnclamped (objeto de textura HLSL de DirectX)

Calcula el LOD sin fijar el resultado.



| ret Object.CalculateLevelOfDetailUnclamped( sampler \_ state S, float x ); |
|--------------------------------------------------------------------------|



 

Esta función funciona de forma idéntica a [CalculateLevelOfDetail (objeto](dx-graphics-hlsl-to-calculate-lod.md)de textura HLSL de DirectX), salvo que no hay ninguna fijación en el LOD calculado.
