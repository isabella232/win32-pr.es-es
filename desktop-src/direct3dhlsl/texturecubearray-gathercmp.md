# <a name="texturecubearraygathercmp-methods"></a>Métodos TextureCubeArray::GatherCmp

Para cuatro valores de texel de [**textureCubeArray**](texturecubearray.md) que se usarían en una operación de filtrado bi lineal, devuelve su comparación con un valor de comparación.

Consulte la documentación [sobre](./gather4-c--sm5---asm-.md) gather4_c para obtener más información sobre la instrucción DXBC subyacente.
