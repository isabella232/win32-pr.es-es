# <a name="animation-sound-effects"></a>Efectos de sonido de animación

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Microsoft Agent también permite incluir efectos de sonido como parte de la animación de un carácter. Con el Editor de caracteres de Microsoft Agent, puede especificar el nombre de archivo del Windows estándar (. WAV) que se desencadenarán en un fotograma determinado. Tenga en cuenta que Microsoft Agent no mezcla efectos de sonido y salida hablada, por lo que la salida hablada no comienza hasta que se completa un efecto de sonido. Por lo tanto, evite cualquier efecto de sonido largo o de bucle como parte de la animación de un carácter.
