# <a name="creating-a-new-sound-file"></a>Crear un nuevo archivo de sonido

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Al iniciar el editor por primera vez, puede  crear un nuevo archivo  de sonido; para ello, elija Grabar en el menú **Audio** o haga clic en el botón Grabar de la barra de herramientas del editor de sonido y, a continuación, hable en el micrófono conectado al sistema. Haga clic en **el botón Detener** de la barra de herramientas para detener la grabación. Puede seleccionar el comando **Reproducir en** el menú Audio o en la **barra** de herramientas para ver cómo Microsoft Agent procesa el archivo de sonido sin mejoras lingüísticas. Para crear otro archivo, seleccione Nuevo en el **menú** Archivo o en la barra de herramientas.
