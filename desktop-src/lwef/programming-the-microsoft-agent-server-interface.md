# <a name="programming-the-microsoft-agent-server-interface"></a>Programación de la interfaz del servidor de Microsoft Agent

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Microsoft Agent proporciona servicios que permiten programar caracteres animados desde una aplicación. Estos servicios se implementan como un servidor de AUTOMATIZACIÓN OLE. Ole Automation permite que una aplicación controle mediante programación el objeto de otra aplicación. En este documento se supone que se entiende el modelo de objetos componentes (COM) y OLE. Para obtener una introducción de estos servicios, vea Información general sobre la [interfaz de programación](microsoft-agent-programming-interface-overview.md).
