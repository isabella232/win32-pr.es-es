# <a name="top-property-propertysheet-object"></a>Propiedad Top (objeto PropertySheet)

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve un valor entero que especifica el borde superior actual, en píxeles, de la ventana de hoja de propiedades de Microsoft Agent.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agent,. PropertySheet.Top**

</dd> </dl>

## <a name="remarks"></a>Comentarios

El servidor muestra la ventana en función de la ubicación establecida por el usuario.
