# <a name="width-property-propertysheet-object"></a>Propiedad Width (objeto PropertySheet)

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve un valor entero que especifica el ancho actual, en píxeles, de la ventana de hoja de propiedades de Microsoft Agent.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintaxis** 
</dt> <dd>

*agent!". PropertySheet.Width**

</dd> </dl>

## <a name="remarks"></a>Comentarios

El servidor muestra la ventana en función de la ubicación establecida por el usuario.
