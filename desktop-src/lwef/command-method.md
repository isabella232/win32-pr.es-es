# <a name="command-method"></a>Command (método)

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve un [**objeto Command**](/windows/desktop/lwef/the-command-object) en una [**colección Commands.**](/windows/desktop/lwef/the-commands-collection-object)

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agent***. Caracteres ("**_CharacterID_*_"). Commands.Command_* "*Name*"

</dd> </dl>

## <a name="remarks"></a>Observaciones

Puede usar este método para acceder a las [**propiedades**](/windows/desktop/lwef/the-command-object) de un objeto Command.

> [!Note]  
> Este método puede ser necesario para algunos lenguajes de programación. No es necesario para VBScript ni Visual Basic. Para obtener más información sobre cómo especificar métodos **command,** vea Propiedades del objeto Command.
