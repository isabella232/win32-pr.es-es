# <a name="jet_spacehintscbminextent-property"></a>JET_SPACEHINTS.cbMinExtent, propiedad

Obtiene o establece el valor que invalida ulGrowth si es demasiado pequeño. Este valor está en bytes.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
