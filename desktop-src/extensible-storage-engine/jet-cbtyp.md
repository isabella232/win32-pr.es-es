# <a name="jet_cbtyp"></a>JET_CBTYP


_**Se aplica a:** Windows | Windows Servidor_

## <a name="jet_cbtyp"></a>JET_CBTYP

El **JET_CBTYP** de constantes describe todos los puntos posibles de una operación que el motor de base de datos notificará a una aplicación mediante una llamada a la [JET_CALLBACK](./jet-callback-callback-function.md) de devolución de llamada. El motor de base de datos pasa una de estas constantes en el *parámetro cbtyp* de la función de devolución de llamada. El significado de los demás parámetros pasados por el  motor de base de datos en esta llamada depende de los JET_CBTYP pasados.

**Windows XP:**  El **JET_CBTYP** de constantes se introduce en Windows XP.
