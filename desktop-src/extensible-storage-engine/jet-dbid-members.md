# <a name="jet_dbid-members"></a>JET_DBID miembros

Incluir miembros protegidos  
Incluir miembros heredados  

Un JET_DBID contiene el identificador de la base de datos. Se usa un identificador de base de datos para administrar el esquema de una base de datos. También se puede usar para administrar las tablas dentro de esa base de datos.

El [JET_DBID](./jet-dbid-structure.md) expone los miembros siguientes.
