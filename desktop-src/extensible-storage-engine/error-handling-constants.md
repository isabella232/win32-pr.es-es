# <a name="error-handling-constants"></a>Constantes de control de errores


_**Se aplica a:** Windows | Windows Servidor_

## <a name="error-handling-constants"></a>Constantes de control de errores

Las siguientes constantes se usan para establecer el intervalo de los parámetros [JET_paramExceptionAction](./error-handling-parameters.md) del sistema.


| <p>Constante o valor</p> | <p>Descripción</p> | 
|-----------------------|--------------------|
| <p>JET_ExceptionMsgBox<br />0x0001</p> | <p>Muestra un cuadro de mensaje cuando se produce una excepción.</p> | 
| <p>JET_ExceptionNone<br />0x0002</p> | <p>No hace nada cuando se produce una excepción.</p> | 



### <a name="requirements"></a>Requisitos


| | | <p><strong>Cliente</strong></p> | <p>Requiere Windows Vista, Windows XP o Windows 2000 Professional.</p> | | <p><strong>Servidor</strong></p> | <p>Requiere Windows Server 2008, Windows Server 2003 o Windows 2000 Server.</p> | | <p><strong>Header</strong></p> | <p>Declarado en Esent.h.</p> |
