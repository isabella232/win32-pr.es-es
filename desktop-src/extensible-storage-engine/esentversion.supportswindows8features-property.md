# <a name="esentversionsupportswindows8features-property"></a>Propiedad EsentVersion.SupportsWindows8Features

Obtiene un valor que indica si la versión actual de ESENT admite características disponibles en la Windows 8 de ESENT.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
