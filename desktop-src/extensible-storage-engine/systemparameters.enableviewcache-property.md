# <a name="systemparametersenableviewcache-property"></a>Propiedad SystemParameters.EnableViewCache

Obtiene o establece un valor que indica si el motor de base de datos debe usar E/S de archivos asignados a memoria para los archivos de base de datos.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)  
**Ensamblado:**  Microsoft.Isam.Esent.Interop (en Microsoft.Isam.Esent.Interop.dll)
