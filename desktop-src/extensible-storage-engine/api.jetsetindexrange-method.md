# <a name="apijetsetindexrange-method"></a>Método Api.JetSetIndexRange

Limita temporalmente el conjunto de entradas de índice que el cursor puede recorrer mediante [JetMove(JET_SESID, JET_TABLEID, Int32, MoveGrbit)](./api.jetmove-method-jet-sesid-jet-tableid-int32-movegrbit-.md) a las que comienzan desde la entrada de índice actual y terminan en la entrada de índice que coincide con los criterios de búsqueda especificados por la clave de búsqueda en ese cursor y los criterios enlazados especificados. Una clave de búsqueda se debe haber construido previamente mediante [JetMakeKey(JET_SESID, JET_TABLEID, \[ \] , Int32, MakeKeyGrbit).](./api.jetmakekey-method.md) Vea también [TrySetIndexRange(JET_SESID, JET_TABLEID, SetIndexRangeGrbit).](./api.trysetindexrange-method.md)
