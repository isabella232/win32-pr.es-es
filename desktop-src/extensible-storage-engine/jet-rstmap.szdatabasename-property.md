# <a name="jet_rstmapszdatabasename-property"></a>JET_RSTMAP.szDatabaseName, propiedad

Obtiene o establece el nombre o la ruta de acceso antiguos de la base de datos. Puede ser NULL si no se modifica.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
