# <a name="windows7paramwaypointlatency-field"></a>Windows7Param. WaypointLatency, campo

Este parámetro establece el número de registros en los que esent aplazará los vaciados de la base de datos. Se puede usar para aumentar la capacidad de recuperación de la base de datos si los errores provocan la pérdida de los archivos de registro.
