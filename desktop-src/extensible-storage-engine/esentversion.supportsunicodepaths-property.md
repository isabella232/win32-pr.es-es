# <a name="esentversionsupportsunicodepaths-property"></a>Propiedad EsentVersion.SupportsUnicodePaths

Obtiene un valor que indica si la versión actual de ESENT puede usar rutas de acceso no ASCII para acceder a las bases de datos.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
