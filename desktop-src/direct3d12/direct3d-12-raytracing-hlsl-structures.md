# <a name="raytracing-hlsl-structures-direct3d-12"></a>Raytracing HLSL structures (Direct3D 12)

Las siguientes estructuras HLSL admiten la canalización de trazado de rayos de Direct3D 12.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                       | Descripción                                                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Estructura del parámetro de llamadas**](call-parameter-structure.md)<br/>                              | Estructura definida por el usuario proporcionada como argumento de entrada para una llamada a CallShader y como parámetro de entrada para el sombreador al que se puede llamar.<br/>                                                                                                                                                                                                                                              |
| [**Estructura de atributos de intersección**](intersection-attributes.md)<br/>                              | Estructura definida por el usuario que se proporciona como argumento de entrada en una llamada [**TraceRay**](traceray-function.md) y como un parámetro inout en los tipos de sombreador que pueden tener acceso a la carga del rayo.<br/>                                                                                                                                                                                                                                              |
| [**Estructura de la carga de rayos**](ray-payload.md)<br/>                              | Estructura definida por el usuario que se proporciona como argumento de entrada en una llamada [**TraceRay**](traceray-function.md) y como un parámetro inout en los tipos de sombreador que pueden tener acceso a la carga del rayo.<br/>                                                                                                                                                                                                                                              |
| [**Estructura RayDesc**](raydesc.md)<br/>                              | Marcas que se pasan a la [**función TraceRay**](traceray-function.md) para invalidar la transparencia, la selección y el comportamiento de salida anticipada.<br/>                                                                                                                                                                                                                                              |
