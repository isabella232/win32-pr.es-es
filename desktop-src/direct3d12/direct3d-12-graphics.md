# <a name="direct3d-12-graphics"></a>Gráficos de Direct3D 12

Esta guía de programación contiene información sobre cómo usar la canalización programable de Direct3D 12 para crear un motor de gráficos personalizado.

Los encabezados y bibliotecas de Direct3D 12 forman parte del SDK Windows 10. No se requiere ninguna descarga o instalación independiente para usar Direct3D 12.
