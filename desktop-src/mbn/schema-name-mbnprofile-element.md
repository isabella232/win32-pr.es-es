# <a name="name-mbnprofile-element"></a>Elemento Name (MBNProfile)

El **elemento Name (MBNProfile)** identifica el nombre del perfil.

Es una cadena de hasta 255 caracteres.

El elemento es obligatorio.

``` syntax
<xs:element name="Name"
    type="nameType"
 />
```

El **elemento Name** se define mediante el elemento [**MBNProfile.**](schema-mbnprofile-element.md)
