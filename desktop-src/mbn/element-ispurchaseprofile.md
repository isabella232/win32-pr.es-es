# <a name="span-idwwan_profile_v2element_ispurchaseprofilespanispurchaseprofile"></a><span id="WWAN_profile_v2.element_IsPurchaseProfile"></span>IsPurchaseProfile

El **elemento IsPurchaseProfile** contiene un **valor booleano** que es **true** si se trata de un perfil comprado y **false,** en caso contrario. El valor predeterminado es **false**.

## <a name="element-hierarchy"></a>Jerarquía de elemento

**&lt;IsPurchaseProfile&gt;**

## <a name="syntax"></a>Syntax

``` syntax
<IsPurchaseProfile>

  boolean

</IsPurchaseProfile>
```
