# <a name="providername-providertype-element"></a>Elemento ProviderName (providerType)

El **elemento ProviderName (providerType)** contiene el nombre de la red de telefonía móvil.

El elemento es una cadena con un máximo de 20 caracteres.

Este elemento es opcional.

``` syntax
<xs:element name="ProviderName"
    type="providerNameType"
 />
```

El tipo complejo [**providerType**](schema-providertype-complextype.md) define el elemento **ProviderName.**

|-------------------------------------|---------------------------------------------------|
