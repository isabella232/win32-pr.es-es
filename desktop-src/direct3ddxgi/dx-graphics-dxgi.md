# <a name="dxgi"></a>DXGI

Microsoft Infraestructura de gráficos de DirectX (DXGI) controla la enumeración de adaptadores gráficos, la enumeración de modos de presentación, la selección de formatos de búfer, el uso compartido de recursos entre procesos (por ejemplo, entre aplicaciones y Administrador de ventanas de escritorio (DWM)) y la presentación de fotogramas representados en una ventana o monitor para su presentación.

Direct3D 10, Direct3D 11 y Direct3D 12 usan DXGI.

Aunque la mayoría de la programación de gráficos se realiza mediante Direct3D, puede usar DXGI para presentar fotogramas en una ventana, un monitor u otro componente gráfico para la composición y la presentación finales. También puede usar DXGI para leer el contenido de un monitor.

Este conjunto de documentación contiene información sobre la programación con DXGI.
