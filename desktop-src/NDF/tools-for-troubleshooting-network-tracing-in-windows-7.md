# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>Herramientas para solucionar problemas con el seguimiento de red en Windows 7

Hay una serie de herramientas que se pueden usar para ayudar a habilitar y administrar el seguimiento de red y para ver los datos resultantes. Algunas de las herramientas que pueden resultar útiles son Netsh.exe, Monitor de red, Visor de eventos, **netsh trace convert**, Tracerpt.exe y Xperf.exe.

En esta sección se explica cómo usar Netsh.exe y Monitor de red para ayudar a solucionar problemas de seguimiento de red.
