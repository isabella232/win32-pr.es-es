# <a name="solid-foundation"></a>Solid Foundation

Windows 7 proporciona una plataforma para desarrolladores altamente productiva y herramientas que cumplen los aspectos básicos del sistema operativo. Windows 7 se basa en la plataforma Windows Vista, lo que ofrece a los desarrolladores la capacidad de compilar aplicaciones compatibles con ambas plataformas, a la vez que se limitan los problemas de compatibilidad de aplicaciones. Windows 7 mejora drásticamente el rendimiento y la administración de energía para que pueda crear aplicaciones que optimicen la experiencia móvil. Como resultado, las aplicaciones serán más atractivas visualmente, más fáciles de crear y admitir una gama más amplia de estándares internacionales. También se han realizado avances en la compatibilidad global, la accesibilidad y la implementación de aplicaciones.
