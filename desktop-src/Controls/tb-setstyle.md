# <a name="tb_setstyle-message"></a>Mensaje \_ SETSTYLE de TB

Establece el estilo de un control de barra de herramientas.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>

Valor que especifica los estilos que se establecerán para el control. Este valor puede ser una combinación de estilos de [control de barra de herramientas](toolbar-control-and-button-styles.md).

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.
