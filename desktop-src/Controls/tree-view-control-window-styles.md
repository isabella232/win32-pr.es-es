# <a name="tree-view-control-window-styles"></a>Tree-View de ventana de control

En esta sección se enumeran los estilos de ventana usados al crear controles de vista de árbol.



| Constante                                                                                                                                                                        | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TVS_CHECKBOXES"></span><span id="tvs_checkboxes"></span><dl> <dt>**CASILLAS \_ DE TVS**</dt> </dl>                | [Versión 4.70.](common-control-versions.md) Habilita las casillas de los elementos de un control de vista de árbol. Solo se muestra una casilla si una imagen está asociada al elemento. Cuando se establece en este estilo, el control usa [**drawFrameControl**](/windows/desktop/api/winuser/nf-winuser-drawframecontrol) de forma eficaz para crear y establecer una lista de imágenes de estado que contiene dos imágenes. La imagen de estado 1 es la casilla desactivada y la imagen de estado 2 es la casilla activada. Al establecer la imagen de estado en cero, se quita por completo la casilla. Para obtener más información, vea [Trabajar con índices de imagen de estado.](tree-view-controls.md)<br/> [Versión 5.80.](common-control-versions.md) Muestra una casilla incluso si no hay ninguna imagen asociada al elemento.<br/> Una vez que se crea un control de vista de árbol con este estilo, no se puede quitar el estilo. En su lugar, debe destruir el control y crear uno nuevo en su lugar. La destrucción del control de vista de árbol no destruye la lista de imágenes de estado de casilla. Debe destruirlo explícitamente. Obtenga el identificador a la lista de imágenes de estado enviando al control de vista de árbol un [**mensaje \_ GETIMAGELIST de TVM.**](tvm-getimagelist.md) A continuación, destruya la lista de imágenes [**con ImageList \_ Destroy**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_destroy).<br/> Si desea usar este estilo, debe establecer el estilo TVS CHECKBOXES con \_ [**SetWindowLong**](/windows/desktop/api/winuser/nf-winuser-setwindowlonga) después de crear el control treeview y antes de rellenar el árbol. De lo contrario, las casillas podrían aparecer desactivadas, en función de los problemas de tiempo.<br/> |

| <span id="TVS_FULLROWSELECT"></span><span id="tvs_fullrowselect"></span><dl> <dt>**TVS \_ FULLROWSELECT**</dt> </dl>       | [Versión 4.71.](common-control-versions.md) Habilita la selección de fila completa en la vista de árbol. Se resalta toda la fila del elemento seleccionado y, al hacer clic en cualquier lugar de la fila de un elemento, se selecciona. Este estilo no se puede usar junto con el estilo **\_ TVS HASLINES.** <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_HASBUTTONS"></span><span id="tvs_hasbuttons"></span><dl> <dt>**TVS \_ HASBUTTONS**</dt> </dl>                | Muestra los botones más (+) y menos (-) junto a los elementos primarios. El usuario hace clic en los botones para expandir o contraer la lista de elementos secundarios de un elemento primario. Para incluir botones con elementos en la raíz de la vista de árbol, también se debe especificar TVS \_ LINESATROOT. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_HASLINES"></span><span id="tvs_haslines"></span><dl> <dt>**TVS \_ HASLINES**</dt> </dl>                      | Usa líneas para mostrar la jerarquía de elementos. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

Este valor se omite si no se especifica \_ también TVS HASLINES. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |

| <span id="TVS_RTLREADING"></span><span id="tvs_rtlreading"></span><dl> <dt>**TVS \_ RTLREADING**</dt> </dl>                | [Versión 4.70.](common-control-versions.md) Hace que el texto se muestre de derecha a izquierda (RTL). Normalmente, las ventanas muestran texto de izquierda a derecha (LTR). Windows se puede *reflejar para* mostrar idiomas como hebreo o árabe que leen RTL. Normalmente, el texto de la vista de árbol se muestra en la misma dirección que el texto en su ventana primaria. Si se establece TVS RTLREADING, el texto de la vista de árbol se lee en la dirección opuesta del \_ texto de la ventana primaria. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

Si el mouse se usa para hacer un solo clic en el elemento seleccionado y ese elemento está cerrado, se expandirá. Si el usuario mantiene presionada la tecla CTRL al seleccionar un elemento, el elemento que se va a no seleccionar no se contraerá. <br/> [Versión 5.80.](common-control-versions.md) Hace que el elemento seleccionado se expanda y que el elemento que se está no seleccionado se contraiga tras la selección en la vista de árbol. Si el usuario mantiene presionada la tecla CTRL al seleccionar un elemento, el elemento que se va a no seleccionar no se contraerá.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
