# <a name="pager-control-styles"></a>Estilos de control de paginación

En esta sección se enumeran los estilos de ventana usados al crear controles de paginación.



| Constante                                                                                                                                                         | Descripción                                                                                                                                                                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="PGS_AUTOSCROLL"></span><span id="pgs_autoscroll"></span><dl> <dt>**PGS \_ AUTOSCROLL**</dt> </dl> | El control de paginación se desplazará cuando el usuario mantenga el mouse sobre uno de los botones de desplazamiento.<br/>                                                                                                                     |
| <span id="PGS_DRAGNDROP"></span><span id="pgs_dragndrop"></span><dl> <dt>**PGS \_ DRAGNDROP**</dt> </dl>    | La ventana independiente puede ser un destino de arrastrar y colocar. El control de paginación se desplazará automáticamente si un elemento se arrastra desde fuera del paginador sobre uno de los botones de desplazamiento.<br/>                                     |
| <span id="PGS_HORZ"></span><span id="pgs_horz"></span><dl> <dt>**PGS \_ HORZ**</dt> </dl>                   | Crea un control de paginación que se puede desplazar horizontalmente. Este estilo y el **estilo \_ PGS VERT** son mutuamente excluyentes y no se pueden combinar.<br/>                                                                 |
| <span id="PGS_VERT"></span><span id="pgs_vert"></span><dl> <dt>**PGS \_ VERT**</dt> </dl>                   | Crea un control de paginación que se puede desplazar verticalmente. Esta es la dirección predeterminada si no se especifica ningún estilo de dirección. Este estilo y el **estilo PGS \_ HORZ** son mutuamente excluyentes y no se pueden combinar.<br/> |
