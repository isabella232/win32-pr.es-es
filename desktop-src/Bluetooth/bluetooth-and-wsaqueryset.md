# <a name="bluetooth-and-wsaqueryset"></a>Bluetooth y WSAQUERYSET

La [**estructura WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) se usa en operaciones como la consulta de dispositivos, la consulta del servicio y la configuración del servicio.

En los temas siguientes se describe el uso de la estructura WSAQUERYSET en cada una de estas operaciones:

-   [Bluetooth y WSAQUERYSET para la consulta de dispositivos](bluetooth-and-wsaqueryset-for-device-inquiry.md)
-   [Bluetooth y WSAQUERYSET para la consulta de servicio](bluetooth-and-wsaqueryset-for-service-inquiry.md)
-   [Bluetooth y WSAQUERYSET para set service](bluetooth-and-wsaqueryset-for-set-service.md)

Para obtener más información y una referencia completa de [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) o para otros elementos Windows Sockets, consulte la sección Windows Sockets del Kit de desarrollo de software (SDK) de plataforma.
