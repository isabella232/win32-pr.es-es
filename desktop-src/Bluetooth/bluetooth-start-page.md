# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Propósito

Bluetooth es un protocolo estándar del sector que permite la conectividad inalámbrica para equipos, dispositivos portátiles, teléfonos móviles y otros dispositivos.

## <a name="where-applicable"></a>Donde sea aplicable

La Bluetooth de programación de aplicaciones permite a los desarrolladores usar el conocimiento de programación de red existente para desarrollar o portabilidad de aplicaciones rápidamente.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Bluetooth está diseñado para su uso por los programadores de C/C++. Algunas Bluetooth características están disponibles con Windows Sockets. Es necesario estar familiarizado con la Windows de Microsoft y Windows Sockets.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La compatibilidad Bluetooth microsoft comienza con Windows XP con Service Pack 1 (SP1).

La compatibilidad con Bluetooth 2.1 se ofrece en Windows Vista SP2 y Windows 7.
