# <a name="performance-dimensions"></a>Dimensiones de rendimiento

El rendimiento de la aplicación difiere según la perspectiva. Para un usuario, una aplicación ideal ofrecería un alto rendimiento y capacidad de respuesta. Para un administrador, una aplicación ideal consumiría la menor cantidad de recursos de red. El rendimiento también difiere en función del tipo de aplicación. El rendimiento de una aplicación de transferencia masiva de archivos puede tener implicaciones diferentes que el rendimiento de una aplicación transaccional.

En las secciones siguientes se abordan diferentes dimensiones de rendimiento y tipos de aplicación, y cómo afectan los entornos de red al rendimiento.

-   [Transaccional frente a aplicaciones de streaming](transactional-versus-streaming-applications-2.md)
