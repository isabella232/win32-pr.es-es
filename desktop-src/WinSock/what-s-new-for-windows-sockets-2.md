## <a name="updated-for-windows-8-and-windows-server-2012"></a>Actualizado para Windows 8 y Windows Server 2012

Microsoft Windows 8 y Windows Server 2012 nuevos elementos de programación Windows Sockets.

Hay disponible un conjunto de extensiones de red de alta velocidad para aumentar el rendimiento de las redes con menor latencia y vibración. Estas extensiones destinadas principalmente a las aplicaciones de servidor usan búferes de datos registrados previamente y colas de finalización para aumentar el rendimiento.

A continuación se muestra una Windows sockets agregados para admitir las extensiones de E/S registradas de redes de alta velocidad de Winsock:

-   [**RIOCloseCompletionQueue**](/previous-versions/windows/desktop/legacy/hh448837(v=vs.85))
-   [**RIOCreateCompletionQueue**](/windows/win32/api/mswsock/nc-mswsock-lpfn_riodequeuecompletion)
