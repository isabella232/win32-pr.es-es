## <a name="updated-for-windows-8-and-windows-server-2012"></a>Actualizado para Windows 8 y Windows Server 2012

Microsoft Windows 8 y Windows Server 2012 introducen nuevos elementos de programación de Windows Sockets.

Hay disponible un conjunto de extensiones de red de alta velocidad para un mayor rendimiento de red con una latencia y una vibración inferiores. Estas extensiones dirigidas principalmente a las aplicaciones de servidor usan búferes de datos previamente registrados y colas de finalización para aumentar el rendimiento.

A continuación se indican las nuevas funciones de Windows Sockets agregadas para admitir las extensiones de e/s registradas de red de alta velocidad de Winsock:

-   [**RIOCloseCompletionQueue**](/previous-versions/windows/desktop/legacy/hh448837(v=vs.85))
-   [**RIOCreateCompletionQueue**](/windows/win32/api/mswsock/nc-mswsock-lpfn_riodequeuecompletion)
