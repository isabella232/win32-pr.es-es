# <a name="winhttp-interfaces"></a>WinHTTP Interfaces

[Microsoft Windows HTTP Services (WinHTTP)](about-winhttp.md) expone las dos interfaces siguientes:

-   [**IWinHttpRequest**](iwinhttprequest-interface.md) implementa métodos WinHTTP que no implican eventos.
-   [**IWinHttpRequestEvents**](iwinhttprequestevents-interface.md) implementa eventos WinHTTP y métodos basados en eventos.
