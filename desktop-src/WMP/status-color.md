# <a name="status-color"></a>Color de estado

Se debe especificar el valor de color del texto para mostrar del estado. El valor debe ser tres enteros positivos separados por comas. Los valores representan la cantidad de rojo, verde y azul para un color de texto y van de 0 a 255.
