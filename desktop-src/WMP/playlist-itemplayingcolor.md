# <a name="playlistitemplayingcolor"></a>PLAYLIST.itemPlayingColor

El **atributo itemPlayingColor** especifica o recupera el color de resaltado que indica el elemento que se está reproduciendo actualmente en la lista de reproducción.

``` syntax
        elementID.itemPlayingColor
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura que contiene cualquier valor de color Internet Explorer microsoft. Tiene un valor predeterminado de \# "00FF00".

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
