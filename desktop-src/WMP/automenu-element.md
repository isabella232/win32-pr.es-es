# <a name="automenu-element"></a>Elemento AUTOMENU

El **elemento AUTOMENU** proporciona una manera de mostrar el **Panel de acceso** rápido en una máscara. Se trata de un menú que aparece en el modo completo del reproductor al hacer clic en la flecha situada a la derecha del **botón Reproducción** ahora. Proporciona acceso inmediato a los archivos multimedia digitales organizados de varias maneras, como en listas de reproducción o por álbum, intérprete o género. Use los **atributos izquierdo** **y superior** para especificar la ubicación en la que debe aparecer el menú cuando se llama al método **show.**

El **elemento AUTOMENU** admite el método siguiente.



| Método                    | Descripción                          |
|---------------------------|--------------------------------------|
| [show](automenu-show.md) | Muestra el **cuadro de diálogo Panel de acceso**. |



 

El **elemento AUTOMENU** admite los siguientes atributos ambientales: left y top.

> [!Note]  
> Este elemento requiere Reproductor de Windows Media serie 9 o posterior.
