# <a name="equalizersettingscurrentpresettitle"></a>EQUALIZERSETTINGS.currentPresetTitle

El **atributo currentPresetTitle** recupera el título del valor preestablecido actual.

``` syntax
        elementID.currentPresetTitle
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de solo **lectura.**

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
