# <a name="playerapplicationplayerdocked"></a>PLAYERAPPLICATION.playerDocked

El **atributo playerDocked** recupera un valor que indica si Reproductor de Windows Media está en un estado acoplado.

``` syntax
        elementID.playerDocked
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un valor booleano de **solo lectura.**



| Value | Descripción                       |
|-------|-----------------------------------|
| True  | Reproductor de Windows Media está acoplado.   |
| False | Reproductor de Windows Media está desacoplado. |



 

## <a name="remarks"></a>Observaciones

Este atributo solo se usa al controlar el Reproductor de Windows Media remoto.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media serie 9 o posterior<br/> |
