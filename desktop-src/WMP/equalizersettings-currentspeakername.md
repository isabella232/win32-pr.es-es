# <a name="equalizersettingscurrentspeakername"></a>EQUALIZERSETTINGS.currentSpeakerName

El **atributo currentSpeakerName** recupera el nombre de la configuración actual del hablante.

``` syntax
        elementID.currentSpeakerName
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de solo **lectura** que contiene uno de los valores siguientes.



| Value           | Descripción                              |
|-----------------|------------------------------------------|
| Auriculares      | Los hablantes actuales son auriculares.     |
| Oradores normales | Los hablantes actuales tienen un tamaño normal. |
| Altavoces grandes  | Los altavoces actuales son grandes.          |



 

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media serie 9 o posterior<br/> |
