# <a name="creating-the-disabled-file"></a>Creación del archivo deshabilitado

El archivo deshabilitado es similar al archivo pushed, salvo que se usa otro color (rojo) para mostrar cuando algo está deshabilitado.

La siguiente imagen es el archivo Deshabilitado.

![archivo deshabilitado](images/ceswmdis.png)

Tenga en cuenta que archivos como este se pueden recortar en los lados derecho e inferior. El tamaño real del mapa de bits no es importante, solo que los elementos se alinean con el mapa de bits Región para las pruebas de posición, tal como se mide desde las esquinas superiores izquierdas (incluidos los desplazamientos).
