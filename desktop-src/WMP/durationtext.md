# <a name="durationtext"></a>DURATIONTEXT

Se trata de un elemento **TEXT predefinido** con los siguientes valores predeterminados.

``` syntax
value="wmpprop:player.currentMedia.DurationString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Comentarios

Esto creará un elemento **TEXT** que mostrará la duración del medio que se va a reproducir. Todas las propiedades de **este elemento TEXT** se pueden invalidar si se especifican explícitamente.
