# <a name="rules-for-each-line"></a>Reglas para cada línea

Cada definición del archivo de definición de máscara debe estar en una línea independiente. Debe escribir texto en líneas mediante las siguientes reglas:

-   Cada línea debe terminar con una devolución de párrafo. Algunas líneas pueden ser más largas que la longitud de línea visible del editor de texto, pero no interrumpir las líneas. Es posible que quiera desactivar el ajuste de línea para asegurarse de que las líneas no se interrumpirán.
-   Use solo caracteres visibles como se define en la Referencia de máscara.
-   Una línea consta de uno o varios elementos, separados por espacios.
-   Puede especificar tantos espacios entre elementos como desee para mejorar la legibilidad.
-   No use caracteres de tabulación en ningún lugar del archivo.
-   Las líneas que comienzan con dos barras diagonales consecutivas (//) se omiten. Úsese estos para los comentarios. También se omiten las líneas en blanco.
-   Incluya espacios entre corchetes y el nombre de la sección.
