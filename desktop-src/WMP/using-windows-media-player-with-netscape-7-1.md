# <a name="using-windows-media-player-with-netscape-71"></a>Uso del Reproductor de Windows Media con Netscape 7.1

El explorador web Netscape 7.1 proporciona compatibilidad nativa con controles Reproductor de Windows Media ActiveX mediante el componente de ActiveX Gecko. Esto significa que puede insertar el control Reproductor de Windows Media en las páginas web de Netscape 7.1 con la misma sintaxis que usaría para Microsoft Internet Explorer. Netscape 7.1 también admite el scripting del control y el control de eventos de forma similar.

Si desea usar el control Reproductor de Windows Media en Netscape, se recomienda usar Netscape 7.1.

Para más información sobre el uso del control Reproductor de Windows Media con Netscape 7.1, consulte el sitio [web de Netscape.](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape)

Para obtener más información sobre el componente de ActiveX Gecko, consulte el [sitio StreamingMedia.com web .](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx)

La descarga completa de este SDK incluye un ejemplo que muestra la detección del explorador web. Para obtener más información sobre los ejemplos incluidos con el SDK, vea [Ejemplos.](samples.md)
