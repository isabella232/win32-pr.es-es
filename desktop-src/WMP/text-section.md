# <a name="text-section"></a>Sección de texto

No se usa ninguna sección de texto en esta máscara, pero el siguiente es un ejemplo del formato que debe usar al implementar una sección de texto:


```C++
[ Text ]

//  <Item>       <Location>     <Align>  <Font>          <Color>
//  ------       ----------     -------  ------          -------

```
