# <a name="videoonvideostart"></a>VIDEO.onvideostart

El **controlador de eventos onvideostart** controla un evento que se produce cuando se carga el vídeo y comienza a representarse.

``` syntax

onvideostart 
```

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
