# <a name="stringcollectionisidentical-method"></a>StringCollection. isIdentical (método)

El método **isIdentical** recupera un valor que indica si el objeto proporcionado es igual que el actual.

## <a name="syntax"></a>Sintaxis


```JScript
bRetVal = StringCollection.isIdentical(
  stringCollection
)
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*stringCollection* \[ de\]
</dt> <dd>

Objeto **StringCollection** que se va a comparar con el actual.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método devuelve un **valor booleano**.

## <a name="remarks"></a>Observaciones

Para usar este método, se requiere acceso de lectura a la biblioteca. Para obtener más información, vea [acceso a la biblioteca](library-access.md).
