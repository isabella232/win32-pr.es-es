# <a name="ambientattributesaccdescription"></a>AmbientAttributes.accDescription

El **atributo accDescription** especifica o recupera una descripción para cualquier elemento.

``` syntax
        elementID.accDescription
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura con un valor predeterminado de "" (cadena vacía).

## <a name="remarks"></a>Observaciones

Este atributo se usa con fines de accesibilidad. Permite que un programa de lector lea en voz alta la descripción de cualquier elemento.

Este atributo también se aplica a los elementos de botón dentro del control de grupo de botones.
