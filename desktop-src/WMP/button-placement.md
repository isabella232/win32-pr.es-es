# <a name="button-placement"></a>Colocación de botones

Aunque puede dibujar botones para que sean cualquier forma, se tratan internamente como si tengan un límite exterior rectangular. Por este motivo, evite las coordenadas de botón superpuestas porque pueden producirse resultados inesperados.
