# <a name="requirements-for-online-music-stores"></a>Requisitos de las tiendas de Música en línea

Todos los requisitos que debe cumplir la tienda en línea se detallan en el Contrato de publicación de Online Media Service para Reproductor de Windows Media. Para asegurarse de que la tienda en línea proporciona una excelente experiencia de usuario, hemos resaltado algunos requisitos clave relacionados con la personalización y la reproducción.

-   La página inicial del almacén expuesto en la Reproductor de Windows Media interfaz de usuario debe ser funcional sin necesidad de que el usuario inicie sesión.
-   Se prohíben los anuncios emergentes que inician una ventana del explorador independiente.
-   Toda la experiencia de exploración, compra y reproducción debe representarse dentro de la interfaz de usuario del Reproductor de Windows Media y no dentro de una ventana del explorador emergente independiente o un reproductor multimedia incrustado.
-   La instalación de systray de una aplicación y los procesos en segundo plano que se ejecutan continuamente están prohibidos.
-   Las acciones discretas como el inicio de sesión, la facturación, la compra de CD físicos u otras características relacionadas fuera del servicio multimedia digital principal pueden vincularse a una instancia del explorador fuera del reproductor para la característica de medios no digitales.
-   Las licencias proporcionadas por la tienda deben seguir siendo válidas independientemente de la conexión del almacén a Reproductor de Windows Media. Para más información, consulte la sección Término y terminación del Contrato de publicación de Online Media Service para Reproductor de Windows Media.

De vez en cuando, Microsoft puede actualizar los requisitos de idoneidad y incorporación y publicarlos en el sitio [web de Windows Media](https://www.microsoft.com/windows/windowsmedia/default.mspx) ( https://www.microsoft.com/windows/windowsmedia/default.mspx) .
