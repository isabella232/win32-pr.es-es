# <a name="iwmpdvdresume-method"></a>IWMPDVD::resume (método)

El **método resume** vuelve al modo de reproducción desde el modo de menú y se reanuda en la misma posición que cuando se invocó el menú.

## <a name="syntax"></a>Sintaxis


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
