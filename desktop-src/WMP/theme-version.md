# <a name="themeversion"></a>THEME.version

El **atributo** version especifica o recupera el número Reproductor de Windows Media versión para el que se edizó la máscara.

``` syntax
theme.version
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** (**float**). Especificar un valor para este atributo en tiempo de diseño o en tiempo de ejecución no tiene ningún efecto. El valor siempre es 1,0.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
