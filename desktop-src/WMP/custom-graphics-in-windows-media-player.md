# <a name="custom-graphics-in-windows-media-player"></a>Gráficos personalizados en Reproductor de Windows Media

Con Windows scripting de metarchivo multimedia, puede agregar elementos gráficos a Reproductor de Windows Media mientras se reproduce la secuencia. Hay un tipo básico de gráficos que puede agregar.



| Tipo   | Descripción                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BANDERA | Con el **elemento BANNER** puede colocar una imagen de banner (32 píxeles de alto por 194 píxeles de ancho) en Reproductor de Windows Media en la parte inferior del área de presentación del vídeo. A continuación, puede usar un elemento secundario para agregar un hipervínculo al banner. |
