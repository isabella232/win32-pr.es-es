# <a name="navigating-from-windows-media-player-features"></a>Navegar desde Reproductor de Windows Media características

Si proporciona una tienda de música en línea, puede realizar la navegación desde la vista del Centro de información, la información del álbum y las páginas HTMLView a los paneles de tareas del servicio. El proceso funciona exactamente como se describe en [Navegación entre paneles de tareas de servicio](navigating-between-service-task-panes.md). La diferencia es que puede navegar desde una característica a un panel de tareas de servicio, pero no desde un panel de tareas de servicio a una característica.
