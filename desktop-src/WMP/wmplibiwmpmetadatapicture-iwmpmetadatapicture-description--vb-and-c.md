# <a name="iwmpmetadatapicturedescription-property"></a>IWMPMetadataPicture::D escription

La `Description` propiedad obtiene la descripción de la imagen representada por el atributo de metadatos.

## <a name="syntax"></a>Sintaxis


```CSharp
public System.String Description {get; set;}
```


```VB

Public ReadOnly Property Description As System.String
```





## <a name="property-value"></a>Valor de propiedad

**System.String que** es la descripción de la imagen.

## <a name="remarks"></a>Observaciones

Antes de usar esta propiedad, debe tener acceso de lectura a la biblioteca. Para obtener más información, vea [Acceso a la biblioteca.](library-access.md)
