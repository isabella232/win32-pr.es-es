# <a name="navigation-for-type-2-online-stores"></a>Navegación para almacenes en línea de tipo 2

Las páginas web hospedadas por Reproductor de Windows Media 10 o posterior, como las de los paneles de tareas del servicio de la tienda en línea, pueden usar el método [External.NavigateTaskPaneURL](external-navigatetaskpaneurl.md) para controlar la navegación dentro de Reproductor de Windows Media. La navegación puede producirse de las maneras siguientes:

-   Entre paneles de tareas de servicio. Consulte [Navegación entre paneles de tareas de servicio](navigating-between-service-task-panes.md)
-   Desde características como la vista del Centro de información, las páginas web HTMLView y la página de información del álbum a los paneles de tareas de servicio. Consulte [Navegación desde Reproductor de Windows Media características](navigating-from-windows-media-player-features.md).
-   Entre tiendas en línea. Vea [Navegar a otras tiendas en línea.](navigating-to-other-online-stores.md)
