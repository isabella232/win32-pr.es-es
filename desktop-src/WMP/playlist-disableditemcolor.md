# <a name="playlistdisableditemcolor"></a>PLAYLIST.disabledItemColor

El **atributo disabledItemColor** especifica o recupera el color de una pista de CD deshabilitada o del contenido en línea cuando está sin conexión.

``` syntax
        elementID.disabledItemColor
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura que contiene cualquier valor de color Internet Explorer microsoft. Tiene un valor predeterminado de "graytext".

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
