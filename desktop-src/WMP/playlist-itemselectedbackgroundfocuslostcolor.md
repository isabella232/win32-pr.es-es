# <a name="playlistitemselectedbackgroundfocuslostcolor"></a>PLAYLIST.itemSelectedBackgroundFocusLostColor

El **atributo itemSelectedBackgroundFocusLostColor** especifica o recupera un valor que indica el color de texto de un elemento seleccionado en la lista de reproducción.

``` syntax
        elementID.itemSelectedBackgroundFocusLostColor
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura que representa un color. El valor predeterminado es el color Windows de la cara del botón.
