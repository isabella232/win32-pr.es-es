# <a name="download-manager"></a>Administrador de descarga

Microsoft Reproductor de Windows Media Download Manager permite a las tiendas en línea de tipo 2 controlar el proceso de descarga de contenido desde un panel de tareas Reproductor de Windows Media tienda en línea. Un panel de tareas de la tienda en línea hospeda una página web que se muestra cuando el usuario decide usar una tienda en línea.
