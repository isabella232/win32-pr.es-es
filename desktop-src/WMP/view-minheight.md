# <a name="viewminheight"></a>VIEW.minHeight

El **atributo minHeight** especifica o recupera el alto mínimo en píxeles de **VIEW** al redimensionar.

``` syntax
        elementID.minHeight
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un número de **lectura** y escritura (**long**) con un valor de cero o mayor. Tiene un valor predeterminado de cero, lo que significa que no hay ninguna restricción en el alto mínimo de **VIEW**.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Version<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
