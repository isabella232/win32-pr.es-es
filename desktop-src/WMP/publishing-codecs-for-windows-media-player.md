# <a name="publishing-codecs-for-windows-media-player"></a>Publicación de códecs para Reproductor de Windows Media

El sitio web de WMPlugins ya no se admite para publicar el códec y hacer que los usuarios puedan detectarlo Reproductor de Windows Media usuario. En su lugar, puede publicar el códec en un sitio web de su elección.
