# <a name="media-item-attributes"></a>Atributos de elementos multimedia

Cuando se observa la biblioteca en Reproductor de Windows Media, normalmente se ve una gran cantidad de información sobre un elemento multimedia. Además del nombre de una pista de audio, por ejemplo, es probable que vea el nombre del álbum en el que se encuentra la pista, los nombres del intérprete y compositor, el género de la música, y así sucesivamente.

En el modelo Reproductor de Windows Media objetos, estos elementos de metadatos se denominan atributos. El modelo de objetos incluye propiedades y métodos que puede usar para consultar elementos multimedia y listas de reproducción de atributos. A continuación, puede mostrar valores de atributo en la interfaz de usuario o el código puede realizar acciones basadas en el valor de un atributo.

En los temas siguientes se describe cómo trabajar con atributos de elementos multimedia.
