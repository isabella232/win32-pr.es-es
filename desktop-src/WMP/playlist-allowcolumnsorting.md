# <a name="playlistallowcolumnsorting"></a>PLAYLIST.allowColumnSorting

El **atributo allowColumnSorting** especifica o recupera un valor que indica si se permite ordenar el contenido de la columna.

``` syntax
        elementID.allowColumnSorting
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un valor booleano de lectura **y escritura.**



| Value | Descripción                                  |
|-------|----------------------------------------------|
| true  | Predeterminada. Se permite ordenar el contenido de la columna. |
| false | No se permite ordenar el contenido de la columna.      |



 

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
