# <a name="minimizebutton"></a>MINIMIZEBUTTON

Se trata de un button predefinido con los siguientes valores predeterminados.

``` syntax
onclick="jscript:view.minimize();"
upToolTip="Minimize"
```

## <a name="remarks"></a>Comentarios

Esto creará un control **BUTTON** que minimizará el reproductor. La información sobre herramientas se localiza. Todas las propiedades de **este BOTÓN** se pueden invalidar especificándolos explícitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|----------------------------------------------|
| Versión<br/> | Reproductor de Windows Media 7.0 o posterior<br/> |
