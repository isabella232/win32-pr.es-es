# <a name="pixelaspectratiox"></a>PixelAspectRatioX

El **atributo PixelAspectRatioX** es la parte horizontal de la relación de aspecto de un vídeo.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Observaciones

Para un vídeo con una relación de aspecto de 16:9, **PixelAspectRatioX** sería igual a "16".
