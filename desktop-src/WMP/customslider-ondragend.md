# <a name="customsliderondragend"></a>CUSTOMSLIDER.onDragEnd

El **controlador de eventos onDragEnd** controla un evento que tiene lugar cuando se libera el botón izquierdo del mouse después de una operación de arrastre.

``` syntax
onDragEnd
```
