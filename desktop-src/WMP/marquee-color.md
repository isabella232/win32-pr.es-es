# <a name="marquee-color"></a>Color de marquesina

Se debe especificar el valor de color del tipo. El valor debe especificarse mediante tres enteros positivos separados por comas. Los valores representan la cantidad de rojo, verde y azul para un color de mapa de bits y los intervalos de 0 a 255.
