# <a name="separate-window-plug-ins"></a>Complementos de ventana independientes

Los complementos de interfaz de usuario de ventana independientes se muestran en sus propias ventanas. Estas ventanas se pueden fijar en tamaño o se pueden cambiar de tamaño. Se pueden abrir varias ventanas de complemento de interfaz de usuario independientes al mismo tiempo.

Los complementos de interfaz de usuario de ventana independientes son útiles cuando se quiere  que la información o los controles que proporcione permanezcan visibles cuando el usuario salga del panel Reproducción ahora o cargue otro complemento de interfaz de usuario. Los complementos de interfaz de usuario de ventana independientes que se ejecutan cuando se cierra Reproductor de Windows Media se volverán a cargar y se mostrarán en el mismo tamaño y posición cuando se vuelva a iniciar el reproductor.
