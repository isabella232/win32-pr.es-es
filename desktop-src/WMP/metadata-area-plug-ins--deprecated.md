# <a name="metadata-area-plug-ins-deprecated"></a>Complementos del área de metadatos (en desuso)

En esta página se documenta una característica que puede no estar disponible en versiones futuras de Reproductor de Windows Media y Reproductor de Windows Media SDK.

Los complementos de interfaz de usuario del área  de metadatos se muestran en el área de metadatos del panel Reproducción ahora a la derecha del área visual principal y el panel Herramientas, y debajo de la vista de lista de reproducción (si está habilitada). Solo se puede habilitar un complemento de área de metadatos a la vez.

Los complementos de área de metadatos son útiles para mostrar información concisa sobre el elemento multimedia que se reproduce actualmente, controles simples o hipervínculos a información adicional.
