# <a name="closebutton"></a>CLOSEBUTTON

Se trata de un **button predefinido** con los siguientes valores predeterminados.


```C++
onclick="jscript:view.close();"
upToolTip="Close"

```



## <a name="remarks"></a>Observaciones

Esto crea un control **BUTTON** que cierra el reproductor. La información sobre herramientas está localizada. Todas las propiedades de **este BOTÓN** se pueden invalidar si se especifican explícitamente.
