# <a name="playerapplicationswitchtoplayerapplication-method"></a>Método PlayerApplication.switchToPlayerApplication

El **método switchToPlayerApplication** cambia un control Reproductor de Windows Media remoto al modo completo del reproductor.

## <a name="syntax"></a>Sintaxis


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

Este método solo se usa cuando se usa la comunicación remota Reproductor de Windows Media control .

**Reproductor de Windows Media 10 Mobile:** No se admite este método.
