# <a name="text-color"></a>Color de texto

Se debe especificar el valor de color del tipo. El valor debe especificarse como tres enteros positivos separados por comas. Los valores representan la cantidad de rojo, verde y azul para un color de mapa de bits, y van de 0 a 255.
