# <a name="contracts-and-licenses-for-online-music-stores"></a>Contratos y licencias para almacenes de Música en línea

En la tabla siguiente se muestra quién debe firmar qué tipos de contratos antes de que se pueda incluir una tienda en línea en Reproductor de Windows Media.




| ¿Qué está intentando hacer? | ¿Quién es usted? | Contrato o licencias que deberá firmar | 
|----------------------------|--------------|------------------------------------------|
| Cree una tienda en línea de marca que aparezca en Reproductor de Windows Media. El consumidor ve la marca de la tienda en línea. | El propietario de la marca de la tienda en línea que tiene derechos de distribución para el contenido, desarrolló la tienda para Reproductor de Windows Media y es responsable de administrar el servidor de licencias (si procede) y las actividades de atención al cliente, como facturación y soporte técnico. | El propietario de la marca firma:<br /><ul><li>Acuerdo de lista de Media Service en línea para Reproductor de Windows Media</li><li>Windows Licencia de Media Rights Manager (si se usa DRM)</li><li>Windows Licencia del SDK de formato multimedia</li><li>Reproductor de Windows Media Acuerdo de redistribución (para redistribuir Reproductor de Windows Media a través de CD, sitio web o equipo personal, y para establecer la tienda como el servicio activo inicial) (opcional)</li></ul> | 
| Ofrezca una tienda en línea de marca que se muestra en Reproductor de Windows Media.<br /> El consumidor ve la marca de la tienda en línea. <br /> | Propietario de la marca de la tienda en línea que se contrata con uno o varios proveedores de servicios para crear y operar una tienda en línea única.<br /> Estas actividades pueden incluir:<br /><ul><li>Desarrollo de las características y funcionalidades de la interfaz de usuario y la experiencia de usuario que cumplen los requisitos técnicos y Reproductor de Windows Media SDK de la interfaz de usuario</li><li>Funcionamiento del servidor de licencias (si procede)</li><li>Proporcionar actividades de atención al cliente, como facturación y soporte técnico.</li></ul> | El propietario de la marca firma:<br /><ul><li>Acuerdo de lista de Media Service en línea para Reproductor de Windows Media</li><li>Reproductor de Windows Media Acuerdo de redistribución (para redistribuir el reproductor a través de CD, sitio web o equipo personal, y para establecer la tienda como el servicio activo inicial) (opcional)</li></ul>Firma de proveedores de servicios:<br /><ul><li>Windows Licencia de Media Rights Manager (si se usa DRM)</li><li>Windows Licencia del SDK de formato multimedia</li></ul><blockquote>[!Note]<br />El operador del servidor de licencias es responsable de firmar la Windows licencia de Media Rights Manager.</blockquote><br /> | 
| Ofrezca servicios de etiqueta blanca llave en mano para los propietarios de marca y enuméralo en Reproductor de Windows Media.<br /> El consumidor ve la marca De la Tienda en línea. <br /> | Proveedor de servicios que crea y opera tiendas en línea para propietarios de marca.<br /> Estas actividades pueden incluir:<br /><ul><li>Desarrollo de características y funcionalidades de interfaz de usuario y experiencia de usuario que cumplen Reproductor de Windows Media SDK y requisitos técnicos</li><li>Funcionamiento del servidor de licencias (si procede)</li><li>Proporcionar actividades de atención al cliente, como facturación y soporte técnico.</li></ul> | Firma del proveedor de servicios:<br /><ul><li>Acuerdo de lista de Media Service en línea para Reproductor de Windows Media</li><li>Windows Licencia de Media Rights Manager (si se usa DRM)</li><li>Windows Licencia del SDK de formato multimedia</li></ul>Signos de propietario de marca:<br /><ul><li>Prueba A del Contrato de lista de Servicios multimedia en línea para Reproductor de Windows Media que el proveedor de servicios firmó.</li></ul><blockquote>[!Note]<br />El operador del servidor de licencias es responsable de firmar la Windows licencia de Media Rights Manager.</blockquote><br /> | 
| Ofrezca una tienda en línea de marca propia y haga que aparezca en Reproductor de Windows Media.<br /> Usted y la tienda en línea son los que determinen qué marca resultante aparece en Reproductor de Windows Media y el consumidor lo ve.<br /> | Terceros (por ejemplo, un OEM de PC) que negocia con una tienda en línea existente. | Firma de terceros (por ejemplo, un OEM de PC):<br /><ul><li>Reproductor de Windows Media Acuerdo de redistribución (para redistribuir el reproductor a través de CD, sitio web o equipo personal, y para establecer la tienda como el servicio activo inicial)</li></ul>Signos de propietario de marca:<br /><ul><li>Acuerdo de lista de Media Service en línea para Reproductor de Windows Media</li><li>Windows Licencia de Media Rights Manager (si se usa DRM)</li><li>Windows Licencia de Media Rights Manager (si se usa DRM)</li></ul><blockquote>[!Note]<br />Estos contratos pueden haber sido firmados anteriormente por el propietario de la marca de la tienda en línea o por su proveedor de servicios.</blockquote><br /> |
