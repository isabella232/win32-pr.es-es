# <a name="repeatbutton"></a>REPEATBUTTON

Se trata de un **button predefinido** con los siguientes valores predeterminados.

``` syntax
onclick="jscript:player.settings.setMode('loop',down);"
upToolTip="Turn Repeat On"
downToolTip="Turn Repeat Off"
down="wmpprop:player.settings.getMode('loop')"
sticky="true"
```

## <a name="remarks"></a>Observaciones

Esto crea un control **BUTTON** que activa y desactiva Repetir. La información sobre herramientas está localizada. Todas las propiedades de **este BOTÓN** se pueden invalidar si se especifican explícitamente.
