# <a name="wmcontentgroupdescription-attribute"></a>Atributo WM/ContentGroupDescription

El **atributo WM/ContentGroupDescription** es la descripción del grupo de contenido, que es una colección de elementos multimedia, como un conjunto de conversión boxeada de CD.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de audio](audio-item-attributes.md)
-   [Atributos de archivo multimedia Windows uso frecuente](commonly-used-windows-media-file-attributes.md)

## <a name="remarks"></a>Observaciones

Este atributo se almacena tanto en la biblioteca como en el archivo multimedia digital.

**ContentGroupDescription** es un alias para este atributo.

La Windows SDK de formato multimedia para este atributo es g \_ wszWMContentGroupDistribution.

Para determinar si puede cambiar el valor de este atributo, use el [método Media.isReadOnlyItem.](media-isreadonlyitem.md)
