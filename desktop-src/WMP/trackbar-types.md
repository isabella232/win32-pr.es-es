# <a name="trackbar-types"></a>Tipos de barra de seguimiento

No es necesario incluir controles deslizantes ni barras de seguimiento, pero pueden ser muy útiles para permitir que el usuario ajuste el volumen y la posición de reproducción. Hay dos tipos de barras de seguimiento:

-   Seek
-   Volumen

Para obtener más información sobre los controles deslizantes, consulte la [sección Barras de seguimiento](trackbars.md) de la referencia de máscara.
