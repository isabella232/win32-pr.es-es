# <a name="externalfilter"></a>External.filter

> [!Note]  
> En este tema se describe la funcionalidad diseñada para su uso por las tiendas en línea. No se admite el uso de esta funcionalidad fuera del contexto de una tienda en línea.

 

La **propiedad filter** recupera el filtro de búsqueda actualmente en uso por Reproductor de Windows Media.

``` syntax
window.external.filter
      
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es una cadena de solo **lectura.**

## <a name="remarks"></a>Observaciones

El filtro de búsqueda es el texto que el usuario escribe en el control de rueda de palabras del reproductor.
