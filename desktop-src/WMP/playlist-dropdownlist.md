# <a name="playlistdropdownlist"></a>PLAYLIST.dropDownList

El **atributo dropDownList** especifica o recupera un valor que indica qué elementos se muestran en el cuadro de lista desplegable de una instancia determinada del elemento **PLAYLIST.**

``` syntax
        elementID.dropDownList
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura que contiene uno de los valores siguientes.



| Value       | Descripción                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| showAlbums  | Muestra álbumes.                                                                                    |
| showAll     | Predeterminada. Muestra todos los elementos disponibles, incluidas las listas de reproducción de CD, las listas de reproducción de usuario y los valores preestablecidos de radio. |
| showCD      | Muestra la lista de reproducción de CD.                                                                           |
| showClips   | Muestra todos los clips.                                                                                 |
| showCurrent | Muestra la lista de reproducción actual sin guardar.                                                             |
| showLibrary | Muestra solo las listas de reproducción de biblioteca.                                                                    |
| showRadio   | Muestra los valores preestablecidos de radio.                                                                             |
| showQueries | Muestra las consultas.                                                                                   |
