# <a name="marquee-location"></a>Ubicación de marquesina

Debe definir la ubicación del cuadro de presentación de marquesina que desea usar. Debe escribir cuatro enteros positivos, separados por comas, que definan la izquierda, la parte superior, el ancho y el alto del cuadro de presentación de texto, en píxeles, en relación con la imagen de fondo.

Un valor típico es:


```C++
180,46,50,30 

```
