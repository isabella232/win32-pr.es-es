# <a name="sdbwritewordtag-function"></a>Función SdbWriteWORDTag

Escribe un valor **WORD en** la base de datos especificada.

## <a name="syntax"></a>Sintaxis


```C++
BOOL WINAPI SdbWriteWORDTag(
  _In_ PDB  pdb,
  _In_ TAG  tTag,
  _In_ WORD wData
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*pdb* \[ En\]
</dt> <dd>

Identificador de la base de datos shim.

</dd> <dt>

*tTag* \[ En\]
</dt> <dd>

Etiqueta de la entrada. Esta ETIQUETA debe ser de tipo **TAG \_ TYPE \_ WORD.**

</dd> <dt>

*wData* \[ En\]
</dt> <dd>

Valor.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

La función devuelve **TRUE si** se ejecuta correctamente o **FALSE** en caso de error.
