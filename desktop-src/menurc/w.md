<dl> <dt>

<span id="tools.w_1_gly"></span><span id="TOOLS.W_1_GLY"></span>**Advertencia: Carácter ASCII no equivalente al código de clave virtual**
</dt> <dd>

Existe un código de clave virtual no válido en la [**instrucción ACCELERATORS.**](accelerators-resource.md) Los valores ASCII de algunos caracteres como , ^o & no son equivalentes a los códigos de \* clave virtual para las claves correspondientes. En el caso del asterisco ( ), el código de clave virtual es equivalente al valor ASCII de 8, el carácter numérico \* en la misma clave.

</dd> <dt>

<span id="tools.w_2_gly"></span><span id="TOOLS.W_2_GLY"></span>**Advertencia: MAYÚS o CONTROL usados sin VIRTKEY**
</dt> <dd>

Las **opciones ALT,** **MAYÚS** [**y CONTROL**](control-control.md) solo se aplican a las claves virtuales de la instrucción [**ACCELERATORS.**](accelerators-resource.md) Asegúrese de que la **opción VIRTKEY** se usa con una de estas otras opciones.

</dd> <dt>

<span id="tools.w_3_gly"></span><span id="TOOLS.W_3_GLY"></span>**Escritura de resourcetypeimagelang:languagesize:size**
</dt> <dd>

(V) Microsoft Windows Resource Compiler (RC) está escribiendo el recurso especificado. El *tipo* es el tipo de recurso. Puede ser un número o una cadena. La *imagen* puede ser un número o una cadena. El *idioma* es el idioma nacional del recurso. El *tamaño* es el tamaño decimal del recurso en bytes.

</dd> </dl>
