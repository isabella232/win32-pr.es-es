# <a name="menu-statement"></a>Instrucción MENU

Define el menú de un cuadro de diálogo. Si no se da ninguna instrucción, el cuadro de diálogo no tiene ningún menú.

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*menuname*
</dt> <dd>

Nombre o identificador del menú que se va a usar.

</dd> </dl>

## <a name="examples"></a>Ejemplos

En el ejemplo siguiente se muestra el uso de la **instrucción de diálogo MENU:**

``` syntax
MENU errmenu
```
