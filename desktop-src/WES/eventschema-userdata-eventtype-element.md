# <a name="userdata-eventtype-element"></a>Elemento UserData (EventType)

Contiene los datos del evento.

``` syntax

    type="UserDataType"
 />
```

El elemento **UserData** se define mediante el tipo complejo de [**EventType**](eventschema-eventtype-complextype.md) .
