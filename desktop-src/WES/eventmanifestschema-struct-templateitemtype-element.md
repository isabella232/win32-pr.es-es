# <a name="struct-templateitemtype-element"></a>Elemento struct (TemplateItemType)

Define una estructura que incluye uno o varios elementos de datos que desea incluir con el evento .

``` syntax
<xs:element name="struct"
    type="StructDefinitionType"
 />
```

El tipo complejo [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md) define el elemento **struct.**
