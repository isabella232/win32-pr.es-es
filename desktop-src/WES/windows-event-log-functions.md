# <a name="windows-event-log-functions"></a>Windows Funciones del registro de eventos

Windows El registro de eventos define las siguientes funciones que puede usar para obtener eventos de un canal o registro de eventos y para obtener los metadatos de un proveedor y los eventos que genera.



| Función                                                                   | Descripción                                                                                                                                                                                                       |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEVOLUCIÓN DE LLAMADA \_ DE SUSCRIPCIÓN DE EVT \_**](/windows/win32/api/winevt/nc-winevt-evt_subscribe_callback)                 | Implemente esta devolución de llamada si llama a [**la función EvtSubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe) para recibir eventos que coincidan con la consulta.                                                                                    |
| [**EvtArchiveExportedLog**](/windows/desktop/api/WinEvt/nf-winevt-evtarchiveexportedlog)                     | Agrega cadenas localizadas a los eventos del archivo de registro especificado.                                                                                                                                                   |
| [**EvtCancel**](/windows/desktop/api/WinEvt/nf-winevt-evtcancel)                                             | Cancela todas las operaciones pendientes en un identificador.                                                                                                                                                                       |
| [**EvtClearLog**](/windows/desktop/api/WinEvt/nf-winevt-evtclearlog)                                         | Quita todos los eventos del canal especificado y los escribe en el archivo de registro de destino.                                                                                                                             |
| [**EvtClose**](/windows/desktop/api/WinEvt/nf-winevt-evtclose)                                               | Cierra un identificador abierto.                                                                                                                                                                                            |
| [**EvtCreateBookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtcreatebookmark)                             | Crea un marcador que identifica un evento en un canal.                                                                                                                                                         |
| [**EvtCreateRenderContext**](/windows/desktop/api/WinEvt/nf-winevt-evtcreaterendercontext)                   | Crea un contexto que especifica la información del evento que desea representar.                                                                                                                            |
| [**EvtExportLog**](/windows/desktop/api/WinEvt/nf-winevt-evtexportlog)                                       | Copia los eventos del canal o archivo de registro especificado y los escribe en el archivo de registro de destino.                                                                                                                      |
| [**EvtFormatMessage**](/windows/desktop/api/WinEvt/nf-winevt-evtformatmessage)                               | Da formato a una cadena de mensaje.                                                                                                                                                                                         |
| [**EvtGetChannelConfigProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetchannelconfigproperty)         | Obtiene la propiedad de configuración de canal especificada.                                                                                                                                                                |
| [**EvtGetEventInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventinfo)                                 | Obtiene información que identifica la consulta XML estructurada que seleccionó el evento y el canal o archivo de registro del que llegó.                                                                                 |
| [**EvtGetEventMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgeteventmetadataproperty)         | Obtiene la propiedad de metadatos de eventos especificada.                                                                                                                                                                       |
| [**EvtGetExtendedStatus**](/windows/desktop/api/WinEvt/nf-winevt-evtgetextendedstatus)                       | Obtiene un mensaje de texto que contiene la información de error extendida para el error actual.                                                                                                                           |
| [**EvtGetLogInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetloginfo)                                     | Obtiene información sobre un canal o archivo de registro.                                                                                                                                                                     |
| [**EvtGetObjectArrayProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarrayproperty)             | Obtiene una propiedad de metadatos del proveedor del objeto especificado en la matriz.                                                                                                                                         |
| [**EvtGetObjectArraySize**](/windows/desktop/api/WinEvt/nf-winevt-evtgetobjectarraysize)                     | Obtiene el número de elementos de la matriz de objetos .                                                                                                                                                              |
| [**EvtGetPublisherMetadataProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtgetpublishermetadataproperty) | Obtiene la propiedad de metadatos del proveedor especificada.                                                                                                                                                                    |
| [**EvtGetQueryInfo**](/windows/desktop/api/WinEvt/nf-winevt-evtgetqueryinfo)                                 | Obtiene información sobre una consulta que ejecutó que identifica la lista de canales o archivos de registro a los que la consulta intentó acceder y una lista de códigos de retorno que indica el éxito o error de cada acceso. |
| [**EvtNext**](/windows/desktop/api/WinEvt/nf-winevt-evtnext)                                                 | Obtiene el siguiente evento de los resultados de la consulta o suscripción.                                                                                                                                                       |
| [**EvtNextChannelPath**](/windows/desktop/api/WinEvt/nf-winevt-evtnextchannelpath)                           | Obtiene un nombre de canal del enumerador.                                                                                                                                                                          |
| [**EvtNextEventMetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtnexteventmetadata)                       | Obtiene una definición de evento del enumerador.                                                                                                                                                                     |
| [**EvtNextPublisherId**](/windows/desktop/api/WinEvt/nf-winevt-evtnextpublisherid)                           | Obtiene el identificador de un proveedor del enumerador.                                                                                                                                                            |
| [**EvtOpenChannelConfig**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelconfig)                       | Obtiene un identificador que se usa para leer o modificar la propiedad de configuración de un canal.                                                                                                                                  |
| [**EvtOpenChannelEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenchannelenum)                           | Obtiene un identificador que se usa para enumerar la lista de canales registrados en el equipo.                                                                                                                 |
| [**EvtOpenEventMetadataEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopeneventmetadataenum)               | Obtiene un identificador que se usa para enumerar la lista de eventos que define el proveedor.                                                                                                                             |
| [**EvtOpenLog**](/windows/desktop/api/WinEvt/nf-winevt-evtopenlog)                                           | Obtiene un identificador de un canal o archivo de registro que puede usar para obtener información sobre el canal o archivo de registro.                                                                                                    |
| [**EvtOpenPublisherEnum**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublisherenum)                       | Obtiene un identificador que se usa para enumerar la lista de proveedores registrados en el equipo.                                                                                                                         |
| [**EvtOpenPublisherMetadata**](/windows/desktop/api/WinEvt/nf-winevt-evtopenpublishermetadata)               | Obtiene un identificador que se usa para leer los metadatos del proveedor especificado.                                                                                                                                             |
| [**EvtOpenSession**](/windows/desktop/api/WinEvt/nf-winevt-evtopensession)                                   | Establece una conexión a un equipo remoto que se puede usar al llamar al otro Windows de eventos.                                                                                                |
| [**EvtQuery**](/windows/desktop/api/WinEvt/nf-winevt-evtquery)                                               | Ejecuta una consulta para recuperar eventos de un canal o archivo de registro que coinciden con los criterios de consulta especificados.                                                                                                               |
| [**EvtRender**](/windows/desktop/api/WinEvt/nf-winevt-evtrender)                                             | Representa un fragmento XML basado en el contexto de representación que especifique.                                                                                                                                          |
| [**EvtSaveChannelConfig**](/windows/desktop/api/WinEvt/nf-winevt-evtsavechannelconfig)                       | Guarda los cambios realizados en la configuración de un canal.                                                                                                                                                              |
| [**EvtSeek**](/windows/desktop/api/WinEvt/nf-winevt-evtseek)                                                 | Busca un evento específico en un conjunto de resultados de consulta.                                                                                                                                                                  |
| [**EvtSetChannelConfigProperty**](/windows/desktop/api/WinEvt/nf-winevt-evtsetchannelconfigproperty)         | Establece la propiedad de configuración especificada de un canal.                                                                                                                                                           |
| [**EvtSubscribe**](/windows/desktop/api/WinEvt/nf-winevt-evtsubscribe)                                       | Crea una suscripción que recibirá eventos actuales y futuros de un canal o archivo de registro que coincidan con los criterios de consulta especificados.                                                                            |
| [**EvtUpdateBookmark**](/windows/desktop/api/WinEvt/nf-winevt-evtupdatebookmark)                             | Actualiza el marcador con información que identifica el evento especificado.                                                                                                                                        |
