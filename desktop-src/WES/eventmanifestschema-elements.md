# <a name="eventmanifest-schema-elements"></a>Elementos de esquema EventManifest

Estos son los elementos que define el esquema EventManifest. Esta sección contiene los nombres de los elementos que se usan en el manifiesto; sin embargo, para obtener los detalles de cada elemento, vea el tipo complejo que contiene el elemento.



| Elemento                                                                                                                  | Descripción                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**autoBackup (ChannelLoggingType)**](eventmanifestschema-autobackup-channelloggingtype-element.md)                     | Determina si se va a crear un nuevo archivo de registro cuando el archivo de registro actual alcance su tamaño máximo.<br/>                                                                                                                                                                           |
| [**binary (TemplateItemType)**](eventmanifestschema-binary-templateitemtype-element.md)                                 | Contiene los datos binarios proporcionados por la API [de registro de](/windows/desktop/EventLog/event-logging) eventos.<br/>                                                                                                                                                                                     |
| [**bitMap (MapType)**](eventmanifestschema-bitmap-maptype-element.md)                                                   | Define una lista de pares nombre-valor que asignan valores de bits y valores de cadena.<br/>                                                                                                                                                                                                 |
| [**bufferSize (ChannelPublishingType)**](eventmanifestschema-buffersize-channelpublishingtype-element.md)               | Contiene la cantidad de memoria, en kilobytes, asignada para los búferes de sesión.<br/>                                                                                                                                                                                           |
| [**channel (ChannelListType)**](eventmanifestschema-channel-channellisttype-element.md)                                 | Define un canal en el que se pueden registrar los eventos.<br/>                                                                                                                                                                                                                          |
| [**channels (MetadataType)**](eventmanifestschema-channels-metadatatype-element.md)                                     | Define una lista de canales en los que los proveedores pueden registrar eventos.<br/>                                                                                                                                                                                                             |
| [**channels (ProviderType)**](eventmanifestschema-channels-providertype-element.md)                                     | Define una lista de canales en los que los proveedores pueden registrar eventos.<br/>                                                                                                                                                                                                             |
| [**clockType (ChannelPublishingType)**](eventmanifestschema-clocktype-channelpublishingtype-element.md)                 | Resolución del reloj que se usará al registrar la marca de tiempo para cada evento.<br/>                                                                                                                                                                                                   |
| [**controlGuid (ChannelPublishingType)**](eventmanifestschema-controlguid-channelpublishingtype-element.md)             | Identifica el GUID de sesión para una sesión ETW que contiene eventos WPP.<br/>                                                                                                                                                                                                  |
| [**data (StructDefinitionType)**](eventmanifestschema-data-structdefinitiontype-element.md)                             | Define un elemento de datos que desea incluir en la estructura .<br/>                                                                                                                                                                                                            |
| [**data (TemplateItemType)**](eventmanifestschema-data-templateitemtype-element.md)                                     | Define un elemento de datos que desea incluir con el evento .<br/>                                                                                                                                                                                                              |
| [**description (InputType)**](eventmanifestschema-description-inputtype-element.md)                                     | Descripción del tipo de entrada.<br/>                                                                                                                                                                                                                                          |
| [**event (DefinitionType)**](eventmanifestschema-event-definitiontype-element.md)                                       | Define un evento que el proveedor puede registrar.<br/>                                                                                                                                                                                                                              |
| [**event (código de operación)**](eventmanifestschema-event-opcode-element.md)                                                       | No disponible. **Windows Server 2008 y Windows Vista:** Define un evento para un código de operación específico de la tarea. No disponible a partir de Windows 7.<br/> <br/>                                                                                                                  |
| [**event (TaskEventDefinitionType)**](eventmanifestschema-event-taskeventdefinitiontype-element.md)                     | No disponible. **Windows Server 2008 y Windows Vista:** Define un evento para un evento específico de la tarea. No disponible a partir de Windows 7.<br/> <br/>                                                                                                                   |
| [**events (InstrumentationType)**](eventmanifestschema-events-instrumentationtype-element.md)                           | Contiene una lista de proveedores definidos en el manifiesto.<br/>                                                                                                                                                                                                            |
| [**events (ProviderType)**](eventmanifestschema-events-providertype-element.md)                                         | Contiene una lista de los eventos que define el proveedor.<br/>                                                                                                                                                                                                                  |
| [**importChannel (ChannelListType)**](eventmanifestschema-importchannel-channellisttype-element.md)                     | Identifica un canal definido por otro proveedor.<br/>                                                                                                                                                                                                           |
| [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)                                   | Nodo raíz del manifiesto.<br/>                                                                                                                                                                                                                                            |
| [**instrumentación (instrumentationManifest)**](eventmanifestschema-instrumentation-instrumentationmanifest-element.md) | Define los tipos de instrumentación que contiene el manifiesto (por ejemplo, instrumentación de eventos o contadores).<br/>                                                                                                                                                          |
| [**inType (InputTypeListType)**](eventmanifestschema-intype-inputtypelisttype-element.md)                               | Define un tipo de datos de entrada.<br/>                                                                                                                                                                                                                                               |
| [**inTypes (TypeListType)**](eventmanifestschema-intypes-typelisttype-element.md)                                       | Contiene una lista de definiciones de tipo de entrada.<br/>                                                                                                                                                                                                                                |
| [**keyword (KeywordListType)**](eventmanifestschema-keyword-keywordlisttype-element.md)                                 | Define una palabra clave que identifica una categoría de eventos.<br/>                                                                                                                                                                                                                   |
| [**keywords (ChannelPublishingType)**](eventmanifestschema-keywords-channelpublishingtype-element.md)                   | Máscara de bits que determina la categoría de eventos que se escriben en el canal.<br/>                                                                                                                                                                                         |
| [**keywords (MetadataType)**](eventmanifestschema-keywords-metadatatype-element.md)                                     | Define una lista de palabras clave que clasifican eventos.<br/>                                                                                                                                                                                                                        |
| [**keywords (ProviderType)**](eventmanifestschema-keywords-providertype-element.md)                                     | Define una lista de palabras clave que clasifican eventos.<br/>                                                                                                                                                                                                                        |
| [**latencia (ChannelPublishingType)**](eventmanifestschema-latency-channelpublishingtype-element.md)                     | Tiempo de espera antes de vaciar los búferes, en milisegundos.<br/>                                                                                                                                                                                                            |
| [**level (ChannelPublishingType)**](eventmanifestschema-level-channelpublishingtype-element.md)                         | Nivel de gravedad de los eventos que se escriben en el canal.<br/>                                                                                                                                                                                                                 |
| [**level (LevelListType)**](eventmanifestschema-level-levellisttype-element.md)                                         | Define un valor de gravedad que determina el nivel de detalle de los eventos durante el registro.<br/>                                                                                                                                                                                          |
| [**levels (MetadataType)**](eventmanifestschema-levels-metadatatype-element.md)                                         | Define una lista de niveles que especifican la gravedad de un evento.<br/>                                                                                                                                                                                                           |
| [**levels (ProviderType)**](eventmanifestschema-levels-providertype-element.md)                                         | Define una lista de niveles que especifican la gravedad de un evento.<br/>                                                                                                                                                                                                           |
| [**localización (instrumentationManifest)**](eventmanifestschema-localization-instrumentationmanifest-element.md)       | Define un grupo de recursos localizados a los que se hace referencia en el manifiesto.<br/>                                                                                                                                                                                               |
| [**registro (ChannelType)**](eventmanifestschema-logging-channeltype-element.md)                                         | Define las propiedades del archivo de registro que hace una copia de seguridad del canal, como su capacidad y si es secuencial o circular.<br/>                                                                                                                                             |
| [**map (BitMapType)**](eventmanifestschema-map-bitmaptype-element.md)                                                   | Define la asignación entre un valor de bit y un valor de cadena.<br/>                                                                                                                                                                                                               |
| [**map (PatternMapType)**](eventmanifestschema-map-patternmaptype-element.md)                                           | Define las expresiones regulares utilizadas para buscar una cadena correspondiente en la cadena del mensaje y modificarla.<br/>                                                                                                                                                                        |
| [**map (ValueMapType)**](eventmanifestschema-map-valuemaptype-element.md)                                               | Define la asignación entre un valor entero y un valor de cadena.<br/>                                                                                                                                                                                                          |
| [**maps (ProviderType)**](eventmanifestschema-maps-providertype-element.md)                                             | Define una lista de pares nombre-valor a los que puede hacer referencia en la sección de plantilla del manifiesto.<br/>                                                                                                                                                                        |
| [**maxBuffers (ChannelPublishingType)**](eventmanifestschema-maxbuffers-channelpublishingtype-element.md)               | Número máximo de búferes que se asignarán para la sesión.<br/>                                                                                                                                                                                                                |
| [**maxSize (ChannelLoggingType)**](eventmanifestschema-maxsize-channelloggingtype-element.md)                           | Tamaño máximo, en bytes, del archivo de registro.<br/>                                                                                                                                                                                                                              |
| [**message (tabla de mensajes)**](eventmanifestschema-message-messagetable-element.md)                                      | Contiene una referencia a una cadena en la sección localización del manifiesto.<br/>                                                                                                                                                                                             |
| [**elemento messageTable (EventsType)**](eventmanifestschema-messagetable-instrumentationtype-element.md)                | Contiene una referencia a una cadena en la sección localización del manifiesto.<br/>                                                                                                                                                                                             |
| [**metadatos (instrumentationManifest)**](eventmanifestschema-metadata-instrumentationmanifest-element.md)               | Contiene valores globales a los que puede hacer referencia en otros manifiestos.<br/>                                                                                                                                                                                                         |
| [**minBuffers (ChannelPublishingType)**](eventmanifestschema-minbuffers-channelpublishingtype-element.md)               | Número mínimo de búferes que se asignarán para la sesión.<br/>                                                                                                                                                                                                                |
| [**namedQueries (MetadataType)**](eventmanifestschema-namedqueries-metadatatype-element.md)                             | No se usa. Define una lista de consultas con nombre que consultan un valor en la cadena del mensaje de evento y realizan una acción especificada si se encuentran.<br/>                                                                                                                                        |
| [**namedQueries (ProviderType)**](eventmanifestschema-namedqueries-providertype-element.md)                             | No se usa. Define una lista de consultas con nombre que consultan un valor en la cadena del mensaje de evento y realizan una acción especificada si se encuentran.<br/>                                                                                                                                        |
| [**opcode (OpcodeListType)**](eventmanifestschema-opcode-opcodelisttype-element.md)                                     | Contiene un valor numérico que identifica la actividad o un punto dentro de una actividad que la aplicación estaba realizando al generar el evento (por ejemplo, inicialización o cierre).<br/>                                                                                |
| [**opcode (TaskEventDefinitionType)**](eventmanifestschema-opcode-taskeventdefinitiontype-element.md)                   | No disponible. **Windows Server 2008 y Windows Vista:** Contiene un código de operación específico de la tarea para un evento. Se supone que todas las definiciones de código de operación son específicas de la tarea que contiene las definiciones de eventos. No disponible a partir de Windows 7.<br/> <br/> |
| [**opcodes (MetadataType)**](eventmanifestschema-opcodes-metadatatype-element.md)                                       | Define una lista de códigos de operación que puede usar para agrupar eventos dentro de una tarea.<br/>                                                                                                                                                                                                 |
| [**opcodes (ProviderType)**](eventmanifestschema-opcodes-providertype-element.md)                                       | Define una lista de códigos de operación que puede usar para agrupar eventos dentro de una tarea.<br/>                                                                                                                                                                                                 |
| [**opcodes (TaskType)**](eventmanifestschema-opcodes-tasktype-element.md)                                               | Define una lista de códigos de operación específicos de la tarea.<br/>                                                                                                                                                                                                                                  |
| [**outType (InputType)**](eventmanifestschema-outtype-inputtype-element.md)                                             | Lista de tipos de datos de salida que determinan cómo se representan los datos.<br/>                                                                                                                                                                                                      |
| [**patternMap (PatternMapListType)**](eventmanifestschema-patternmap-patternmaplisttype-element.md)                     | Define una asignación entre dos expresiones regulares. Una expresión se usa para buscar una cadena que coincida en la cadena del mensaje y la otra para modificar la cadena antes de que el servicio la vuelva a coloca en la cadena del mensaje.<br/>                                             |
| [**patternMaps (NamedQueryType)**](eventmanifestschema-patternmaps-namedquerytype-element.md)                           | Define una lista de pares de expresiones regulares que se usan para modificar la cadena del mensaje.<br/>                                                                                                                                                                                     |
| [**provider (EventsType)**](eventmanifestschema-provider-eventstype-element.md)                                         | Define un proveedor y los metadatos que usa para definir sus eventos.<br/>                                                                                                                                                                                                    |
| [**publicación (ChannelType)**](eventmanifestschema-publishing-channeltype-element.md)                                   | Define las propiedades de registro de la sesión que usa el canal.<br/>                                                                                                                                                                                                     |
| [**resources (LocalizationType)**](eventmanifestschema-resources-localizationtype-element.md)                           | Define un grupo de tablas de cadenas que contienen las cadenas localizadas a las que se hace referencia en el manifiesto.<br/>                                                                                                                                                                  |
| [**retention (ChannelLoggingType)**](eventmanifestschema-retention-channelloggingtype-element.md)                       | Determina si el archivo de registro es un archivo de registro secuencial o circular. <br/>                                                                                                                                                                                                    |
| [**sidType (ChannelPublishingType)**](eventmanifestschema-sidtype-channelpublishingtype-element.md)                     | Determina si se debe incluir un identificador de seguridad (SID) de la entidad de seguridad con cada evento escrito en el canal.<br/>                                                                                                                                                        |
| [**string (StringTableType)**](eventmanifestschema-string-stringtabletype-element.md)                                   | Define una cadena localizada.<br/>                                                                                                                                                                                                                                               |
| [**stringTable (LocalizationType)**](eventmanifestschema-stringtable-localizationtype-element.md)                       | Define una lista de cadenas localizadas a las que puede hacer referencia en el manifiesto.<br/>                                                                                                                                                                                              |
| [**stringTable (recursos)**](eventmanifestschema-stringtable-localizationtype-element.md)                              | Define una lista de cadenas localizadas a las que puede hacer referencia en el manifiesto.<br/>                                                                                                                                                                                              |
| [**struct (TemplateItemType)**](eventmanifestschema-struct-templateitemtype-element.md)                                 | Define una estructura que incluye uno o varios elementos de datos que desea incluir con el evento .<br/>                                                                                                                                                                         |
| [**task (DefinitionType)**](eventmanifestschema-task-definitiontype-element.md)                                         | No disponible. **Windows Server 2008 y Windows Vista:** Define un evento específico de la tarea que el proveedor puede registrar. No disponible a partir de Windows 7.<br/> <br/>                                                                                                     |
| [**task (TaskListType)**](eventmanifestschema-task-tasklisttype-element.md)                                             | Define un componente o subcomponente de una aplicación.<br/>                                                                                                                                                                                                                    |
| [**tasks (ProviderType)**](eventmanifestschema-tasks-providertype-element.md)                                           | Define una lista de tareas que un proveedor puede usar para agrupar eventos.<br/>                                                                                                                                                                                                          |
| [**template (TemplateListType)**](eventmanifestschema-template-templatelisttype-element.md)                             | Plantilla que define los datos que se incluirán con un evento .<br/>                                                                                                                                                                                                                |
| [**templates (ProviderType)**](eventmanifestschema-templates-providertype-element.md)                                   | Define una lista de plantillas que especifican los datos que se incluirán con los eventos.<br/>                                                                                                                                                                                             |
| [**types (MetadataType)**](eventmanifestschema-types-metadatatype-element.md)                                           | Define una lista de tipos XML.<br/>                                                                                                                                                                                                                                              |
| [**UserData (TemplateItemType)**](eventmanifestschema-userdata-templateitemtype-element.md)                             | Fragmento XML que se usa para representar los datos del evento.<br/>                                                                                                                                                                                                                    |
| [**valueMap (MapType)**](eventmanifestschema-valuemap-maptype-element.md)                                               | Define una lista de pares nombre-valor que asignan valores enteros y valores de cadena.<br/>                                                                                                                                                                                             |
| [**xmlType (XmlTypeListType)**](eventmanifestschema-xmltype-xmltypelisttype-element.md)                                 | Define un tipo XML.<br/>                                                                                                                                                                                                                                                      |
| [**xmlTypes (TypeListType)**](eventmanifestschema-xmltypes-typelisttype-element.md)                                     | Define una lista de tipos de salida que el servicio usa para determinar cómo representar un tipo de datos de entrada.<br/>                                                                                                                                                                       |
