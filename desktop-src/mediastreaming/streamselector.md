# <a name="streamselector-class"></a>StreamSelector (clase)

Implementa la [**interfaz IStreamSelectorStatics**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-istreamselectorstatics) y permite seleccionar una secuencia.

**StreamSelector** tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La **clase StreamSelector** tiene estos métodos.



| Método                                                                      | Descripción                                                  |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------|
| [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) | Obtiene las propiedades de la secuencia de forma asincrónica.<br/> |
| [**SelectBestStreamAsync**](/previous-versions/windows/desktop/legacy/hh829002(v=vs.85))       | Consulta de forma asincrónica la mejor secuencia.<br/>       |
