# <a name="sensor-properties"></a>Propiedades del sensor

La plataforma sensor y ubicación define constantes que identifican las propiedades de los sensores. Los fabricantes de sensores también pueden definir sus propias propiedades.

La plataforma define los siguientes valores **PROPERTYKEY** para las propiedades del sensor. Estas propiedades son de solo lectura a menos que se indique lo contrario.

Cada propiedad de sensor definida por la plataforma **PROPERTYKEY** se basa en un **GUID común** denominado SENSOR PROPERTY \_ COMMON \_ \_ GUID:

{7F8383EC-D3EC-495C-A8CF-B8CF85C2920}.

> [!IMPORTANT]
> No use este valor base para definir sus propias claves de propiedad.

 

La aplicación cliente puede especificar los valores de las propiedades designadas como de lectura y escritura. Los valores de las propiedades designadas como estáticas no deben cambiar con el tiempo. Las propiedades designadas como necesarias deben ser compatibles con el sensor.
