# <a name="sensor_category_orientation"></a>ORIENTACIÓN \_ DE LA CATEGORÍA DEL \_ SENSOR

La categoría ORIENTACIÓN \_ DE CATEGORÍA DE SENSOR contiene \_ sensores que proporcionan información sobre la orientación física. Las brújulas proporcionan orientación de navegación, como las basadas en el norte magnético. Los inclinómetros miden la pendiente o elevación. Los sensores de distancia miden la proximidad de algún objeto al sensor.
