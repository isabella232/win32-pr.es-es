# <a name="rectfrectf-constructors"></a>Constructores RectF.RectF

En este tema se enumeran los constructores de la [**clase RectF.**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) Para obtener una lista de clases completa, **vea Clase RectF**.

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                                        | Descripción                                                                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RectF(PointF&,SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))         | Crea un [**objeto RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) utilizando un objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) para inicializar los miembros de datos **X** e **Y** y usa un objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) para inicializar los miembros de datos **Width** y **Height** de este rectángulo.<br/> |
| [**RectF(REAL, REAL, REAL, REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inreal_inreal_inreal_inreal)) | Crea un [**objeto RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) utilizando cuatro enteros para inicializar los miembros de datos **X**, **Y,** **Width** y **Height.**<br/>                                                                                                                                                          |
| [**RectF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))                                     | Crea un [**objeto RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) e inicializa los miembros de datos **X,** **Y,** **Width** y **Height** en cero. Éste es el constructor predeterminado.<br/>                                                                                                                                      |
