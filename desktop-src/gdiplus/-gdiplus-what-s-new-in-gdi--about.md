# <a name="whats-new-in-gdi"></a>Novedades de GDI+?

Windows GDI+ es diferente de Windows Interfaz de dispositivo gráfico (GDI) de un par de maneras. En primer lugar, GDI+ amplía las características de GDI proporcionando nuevas funcionalidades, como pinceles de degradado y combinación alfa. En segundo lugar, se ha revisado el modelo de programación para que la programación de gráficos sea más fácil y flexible.

-   [Nuevas características](-gdiplus-new-features-about.md)
-   [Cambios en el modelo de programación](-gdiplus-changes-in-the-programming-model-about.md)
