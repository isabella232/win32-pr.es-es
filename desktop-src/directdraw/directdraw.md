# <a name="directdraw"></a>DirectDraw

Esta sección contiene información sobre la programación con el componente DirectDraw de la interfaz de programación de aplicaciones (API) de DirectX.

Ya no se recomienda usar DirectDraw. Con el lanzamiento de Direct3D 9.0, toda la funcionalidad bidimensional se incluye en Direct3D, sus funciones auxiliares asociadas en D3DX y la tecnología Direct2D de DirectX 11. Sin embargo, la documentación de referencia de DirectDraw sigue estando disponible en esta sección.

Esta sección contiene la documentación [de referencia de DirectDraw.](directdraw-reference.md)
