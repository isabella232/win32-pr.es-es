# <a name="using-oleview"></a>Uso de OleView

Aunque el Editor del Registro muestra una vista sin procesar del registro y es útil para aprender la organización del registro, el uso del Editor del Registro puede llevar mucho tiempo.

OleView proporciona una vista de nivel superior de la información contenida en el Registro. En lugar de mirar una larga lista de CLID, OleView ofrece controles de árbol con nombres descriptivos.
