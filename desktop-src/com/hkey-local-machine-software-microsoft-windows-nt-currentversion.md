# <a name="hkey_local_machinesoftwaremicrosoftwindows-ntcurrentversion"></a>HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion

Las subclaves y los valores del registro asociados a la clave **HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion** contienen información relacionada con la funcionalidad de depuración de com RPC.



| Subclave                                                 | Descripción                                         |
|--------------------------------------------------------|-----------------------------------------------------|
| [**DebugObjectRPCEnabled**](debugobjectrpcenabled.md) | Opciones de configuración para la depuración remota de COM RPC. |
