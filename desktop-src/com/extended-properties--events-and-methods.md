# <a name="extended-properties-events-and-methods"></a>Propiedades, eventos y métodos extendidos

ActiveX contenedores de controles no son necesarios para admitir controles extendidos. Sin embargo, si el contenedor de controles admite propiedades extendidas, debe admitir el siguiente conjunto mínimo:

-   Visible
-   Parent
-   Valor predeterminado
-   Cancelar

Actualmente, las propiedades extendidas, los eventos y los métodos no tienen problemas estándar.
