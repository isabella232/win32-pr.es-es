# <a name="security-values"></a>Valores de seguridad

Varios valores afectan a la seguridad de las aplicaciones COM, incluido el nivel de autenticación, el nivel de suplantación, la protección, la identidad de la aplicación, la configuración de la directiva de restricción de software y el seguimiento de referencias.

Para obtener más información, vea los temas siguientes:

-   [Nivel de autenticación](authentication-level.md) describe cómo ayudar a controlar la cantidad de protección para la comunicación entre objetos COM.
-   [Delegación e suplantación](delegation-and-impersonation.md) explica cómo afecta la protección a la identidad presentada al servidor durante la suplantación y cómo el cliente concede al servidor la autoridad para actuar en su nombre.
-   [Application Identity](application-identity.md) identifica y explica las identidades que puede usar la aplicación.
-   [La directiva de restricción](software-restriction-policy.md) de software explica cómo ayudar a proteger el sistema frente a los riesgos asociados con la ejecución de código desconocido.
-   [Seguimiento de](reference-tracking.md) referencias explica cómo puede asegurarse de que los objetos no se liberan demasiado pronto.
