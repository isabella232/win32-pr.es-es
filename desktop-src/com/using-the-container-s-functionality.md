# <a name="using-the-containers-functionality"></a>Uso de la funcionalidad del contenedor

En las secciones anteriores se ha descrito parte de la compatibilidad necesaria del lado llamador que debe tener un control ActiveX para tener acceso a determinadas características de su contenedor. En la tabla siguiente se describe el uso de interfaces del lado contenedor de un control y cuándo se produciría dicho uso.



| Interfaz                                                     | Objeto contenedor    | Uso                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------|---------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)<br/>           | Sitio<br/>     | Los controles que implementan [**métodos IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject) llaman a [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite) como parte del protocolo de inserción OLE estándar, específicamente los métodos [**SaveObject**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-saveobject), [**ShowObject**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-showobject), [**OnShowWindow**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-onshowwindow) (solo si se admite un estado de activación de ventana independiente), [**RequestNewObjectLayout**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-requestnewobjectlayout)y GetContainer (si se desea la comunicación con otros controles). El [**método GetMoniker**](/windows/desktop/api/OleIdl/nf-oleidl-ioleclientsite-getmoniker) solo se usa cuando el control se puede vincular externamente, es decir, el control no está marcado con OLEMISC \_ CANTLINKASYNC.<br/> |
| [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)<br/>         | Sitio<br/>     | Los controles que tienen una activación en contexto y posiblemente un estado activo de la interfaz de usuario llamarán a los métodos [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite) (generalmente todos ellos con la excepción de [**ContextSensitiveHelp)**](/windows/desktop/api/OleIdl/nf-oleidl-iolewindow-contextsensitivehelp)como parte del protocolo de activación ole en contexto estándar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)<br/>                 | Sitio<br/>     | El control llama a [**OnDataChange**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-ondatachange) si el control admite [**IDataObject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject), [**OnViewChange**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onviewchange) si el control admite [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)y [**OnClose**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onclose), [**OnSave**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onsave)y [**OnRename**](/windows/desktop/api/ObjIdl/nf-objidl-iadvisesink-onrename) si el control admite [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject).<br/>                                                                                                                                                                                                                                                                                                           |
| [**IOleControlSite**](/windows/desktop/api/OCIdl/nn-ocidl-iolecontrolsite)<br/>         | Sitio<br/>     | Si se admite, el control llama a [**OnControlInfoChanged**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-oncontrolinfochanged) cuando cambia mnemonics, [**LockInPlaceActive**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-lockinplaceactive) y [**TransformCoords**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-transformcoords) si se desencadenan eventos (el último método solo se usa si las coordenadas se pasan como argumentos de evento), [**OnFocus**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-onfocus) y [**TranslateAccelerator**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-translateaccelerator) si el control tiene un estado activo de la interfaz de usuario y [**GetExtendedControl**](/windows/desktop/api/OCIdl/nf-ocidl-iolecontrolsite-getextendedcontrol) si el control quiere ver las propiedades de control extendido (propiedad del contenedor).<br/>                                                                       |
| **IDispatch** (propiedades de ambiente)<br/>                 | Sitio<br/>     | Se usa para acceder a las propiedades de ambiente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IPropertyNotifySink**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink)<br/> | Varía<br/>   | Un control debe generar [**OnChanged**](/windows/desktop/api/OCIdl/nf-ocidl-ipropertynotifysink-onchanged) y [**OnRequestEdit**](/windows/desktop/api/OCIdl/nf-ocidl-ipropertynotifysink-onrequestedit) para las propiedades de control marcadas como enlazables y \[ [](/windows/desktop/Midl/bindable) \] \[ [**solicitar**](/windows/desktop/Midl/requestedit) \] , respectivamente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Otras interfaces de receptor de eventos<br/>                        | Varía<br/>   | A un control que tenga interfaces salientes que no sean [**IPropertyNotifySink**](/windows/desktop/api/OCIdl/nn-ocidl-ipropertynotifysink) se le entregarán otros punteros de interfaz del IID correcto a las implementaciones [**IConnectionPoint::Advise**](/windows/desktop/api/OCIdl/nf-ocidl-iconnectionpoint-advise) del control (que normalmente se encuentran en sub objects del control). Un control siempre sabe cómo llamar a sus propias interfaces de eventos porque el control define esas interfaces.<br/>                                                                                                                                                                                                                                                                                                              |
| [**IOleInPlaceFrame**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)<br/>       | Frame<br/>    | Se usa cuando un control tiene un estado activo de interfaz de usuario local que requiere herramientas de nivel de marco o elementos de menú.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IOleInPlaceUIWindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)<br/> | Documento<br/> | Solo se usa cuando un control tiene un estado activo de interfaz de usuario local que requiere herramientas de interfaz de usuario de nivel de documento o de panel. Este es poco habitual.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
