# <a name="insertable-progid-key"></a>Insertable (ProgID Key)

Indica que esta clase se puede insertar en contenedores OLE 2.

## <a name="registry-entry"></a>Entrada del Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      Insertable
```

## <a name="remarks"></a>Observaciones

Se trata de una entrada necesaria para los objetos que se pueden insertar en contenedores OLE 2.
