# <a name="service-identity"></a>Identidad de servicio

La elección de esta identidad hace que la aplicación se ejecute como un servicio. Puede establecer cuentas de servicio si elige la **opción Servicios** en el **Panel de control**. Para obtener más información, vea [Installing as a Service Application](installing-as-a-service-application.md).
