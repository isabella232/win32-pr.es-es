# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>\_Marcas WS Group y WS \_ TABSTOP en controles

Un control no debe usar los \_ marcadores WS Group y WS \_ TABSTOP internamente; algunos contenedores se basan en estas marcas para administrar el control del teclado.
