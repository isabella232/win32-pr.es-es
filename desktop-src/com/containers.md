# <a name="containers"></a>Contenedores

Un ActiveX Control es un contenedor OLE que admite las siguientes características adicionales:

-   Objetos incrustados de servidores en proceso
-   Activación local
-   OLEMISC \_ ACTIVATEWHENVISIBLE
-   Control de eventos

ActiveX Los contenedores de controles deben proporcionar compatibilidad con todas estas características.
