|-------------------|--------------------------------------|
| CN                | ACS-Total-No-Of-Flows                |
| Ldap-Display-Name | aCSTotalNoOfFlows                    |
| Size              | 4 bytes                              |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.763               |
| System-Id-Guid    | 7f561280-5301-11d1-a9c5-0000f80367c1 |
