# <a name="unstructuredname-attribute"></a>atributo unstructuredName

|-------------------|--------------------------------------|
| CN                | unstructuredName                     |
| Nombre para mostrar de LDAP | unstructuredName                     |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |

| Attribute-Id      | 1.2.840.113549.1.9.2                 |
