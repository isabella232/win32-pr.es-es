# <a name="priority-attribute"></a>Atributo de prioridad

La prioridad actual (de un proceso, un trabajo de impresión, y así sucesivamente).



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Prioridad                             |
| Ldap-Display-Name | priority                             |
| Size              | 4 bytes                              |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.231               |
| System-Id-Guid    | 281416c7-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
