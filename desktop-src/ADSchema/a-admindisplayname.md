# <a name="admin-display-name-attribute"></a>Atributo Admin-Display-Name

Nombre que se va a mostrar en las pantallas de administración.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Admin-Display-Name                          |
| Ldap-Display-Name | adminDisplayName                            |
| Size              | \-                                          |
| Actualizar privilegios  | Administrador de dominio                        |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.194                      |
| System-Id-Guid    | bf96791a-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
