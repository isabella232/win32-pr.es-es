# <a name="print-rate-attribute"></a>Print-Rate atributo

Velocidad de impresión proporcionada por el controlador.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Print-Rate                           |
| Ldap-Display-Name | printRate                            |
| Size              | 4 bytes                              |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.285               |
| System-Id-Guid    | ba305f77-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
