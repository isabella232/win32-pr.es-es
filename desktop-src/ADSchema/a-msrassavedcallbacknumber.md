# <a name="msrassavedcallbacknumber-attribute"></a>Atributo msRASSavedCallbackNumber

El [**atributo msRASSavedCallbackNumber**](a-msradiusservicetype.md) se usa internamente. No modifique este valor directamente.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | msRASSavedCallbackNumber             |
| Ldap-Display-Name | msRASSavedCallbackNumber             |
| Size              | \-                                   |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1189              |
| System-Id-Guid    | db0c90c5-c1f2-11d1-él5-0080c76670c0 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
