# <a name="ms-pki-certificate-name-flag-attribute"></a>atributo MS-PKI-Certificate-Name-Flag

Contiene las marcas relacionadas con la construcción del nombre del firmante en un certificado emitido.



| Entrada | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | Marca MS-PKI-Certificate-Name-                                                                      |
| Nombre para mostrar de LDAP | Atributo mspki-Certificate-Name-Flag                                                                       |
| Tamaño              | 4 bytes                                                                                           |
| Actualizar privilegio  | Administrador de dominio                                                                              |
| Frecuencia de actualización  | Cuando se edita, crea o clona el objeto plantilla de certificado (MS-PKI-Certificate-template). |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| System-ID-GUID    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                                                              |
