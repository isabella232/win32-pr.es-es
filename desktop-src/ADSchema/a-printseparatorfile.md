# <a name="print-separator-file-attribute"></a>Atributo Print-Separator-File

Ruta de acceso del archivo de la página separadora de impresora.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Archivo de separador de impresión                        |
| Ldap-Display-Name | printSeparatorFile                          |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.230                      |
| System-Id-Guid    | 281416c6-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
