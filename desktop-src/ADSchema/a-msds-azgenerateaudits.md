# <a name="ms-ds-az-generate-audits-attribute"></a>atributo MS-DS-AZ-Generate-Audits

Un campo booleano que indica si es necesario activar las auditorías en tiempo de ejecución (incluir auditorías para las comprobaciones de acceso, etc.).



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Generate-Audits                |
| Nombre para mostrar de LDAP | msDS-AzGenerateAudits                   |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Administrador de AzRoles                           |
| Frecuencia de actualización  | Durante la inicialización o el cambio de directiva. |
| Attribute-Id      | 1.2.840.113556.1.4.1805                 |
| System-ID-GUID    | f90abab0-186c-4418-bb85-88447c87222a    |
| Sintaxis            | [**Booleano**](s-boolean.md)            |
