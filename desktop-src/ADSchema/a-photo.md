# <a name="photo-attribute"></a>atributo photo

Objeto codificado en fax G3, como se explica en la recomendación T.4, con un contenedor ASN.1 para que sea compatible con un elemento BodyPart X.400, tal como se define en X.420.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Foto                                                 |
| Ldap-Display-Name | Foto                                                 |
| Size              | \-                                                    |
| Privilegio actualizar  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-Id-Guid    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
