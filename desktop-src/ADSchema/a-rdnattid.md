# <a name="rdn-att-id-attribute"></a>Atributo RDN-ATT-ID

El RDN para el atributo que se usa para asignar un nombre a una clase.



| Entrada | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | RDN-ATT-ID                                                      |
| Nombre para mostrar de LDAP | rDNAttID                                                        |
| Tamaño              | \-                                                              |
| Actualizar privilegio  | Administrador de esquema                                            |
| Frecuencia de actualización  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.26                                           |
| System-ID-GUID    | bf967a0f-0de6-11d0-a285-00aa003049e2                            |
| Sintaxis            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
