# <a name="tree-name-attribute"></a>Tree-Name atributo

Nombre DNS del dominio en la raíz de un árbol.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Tree-Name                                   |
| Ldap-Display-Name | treeName                                    |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.660                      |
| System-Id-Guid    | 28630ebd-41d5-11d1-a9c1-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
