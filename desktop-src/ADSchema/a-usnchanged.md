# <a name="usn-changed-attribute"></a>USN-Changed atributo

Número de secuencia de actualización (USN) asignado por el directorio local para el cambio más reciente, incluida la creación. Vea también , [**creado por USN.**](a-usncreated.md)



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| Ldap-Display-Name | uSNChanged                           |
| Size              | 8 bytes                              |
| Privilegio actualizar  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se cambia el objeto.          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| System-Id-Guid    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
