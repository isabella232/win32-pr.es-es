# <a name="do-garbage-collection-extended-right"></a>Derecho extendido de recolección de elementos no utilizados

Derecho extendido para forzar al servicio de directorio a realizar la recolección de elementos no utilizados. Derecho de control para forzar al servicio de directorio a realizar la recolección de elementos no utilizados.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | Recolección de elementos no utilizados                |
| Display-Name | Realizar recolección de elementos no utilizados                |
| Rights-GUID  | fec364e0-0a98-11d1-adbb-00c04fd8d5cd |
