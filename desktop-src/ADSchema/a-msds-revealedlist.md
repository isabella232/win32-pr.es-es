# <a name="ms-ds-revealed-list-attribute"></a>atributo MS-DS-Revelad-List

Identifica las entidades de seguridad cuyas contraseñas de cuentas de equipo actuales se han replicado en el RODC.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Revelad-List                                   |
| Nombre para mostrar de LDAP | msDS-RevealedList                                     |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1940                               |
| System-ID-GUID    | cbdad11c-7fec-387b-6219-3a0627d9af81                  |
| Sintaxis            | [**Object(Access-Point)**](s-object-access-point.md) |
