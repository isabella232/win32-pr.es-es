# <a name="initial-auth-incoming-attribute"></a>Atributo Initial-Auth-Incoming

Contiene información sobre una solicitud de autenticación entrante inicial por parte de un cliente a este servidor. A continuación, este servidor envía esta solicitud al servidor de autenticación para el dominio.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Initial-Auth-Incoming                       |
| Ldap-Display-Name | initialAuthIncoming                         |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.539                      |
| System-Id-Guid    | 52458023-ca6a-11d0-afff-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
