# <a name="ms-tapi-conference-blob-attribute"></a>Atributo ms-TAPI-Conference-Blob

Blob binario de datos que describe varias propiedades de una conferencia de multidifusión TAPI. Su formato y contenido están determinados por el valor del atributo Protocol-Id en el mismo objeto. Normalmente, los datos de este BLOB se ajustan a RFC2327.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------------------------------------|
| CN                | ms-TAPI-Conference-Blob                                                                                      |
| Ldap-Display-Name | msTAPI-ConferenceBlob                                                                                        |
| Size              | Puede ser de longitud arbitraria.                                                                                  |
| Actualizar privilegios  | No se requieren privilegios especiales.                                                                              |
| Frecuencia de actualización  | Puede cambiar a discreción del propietario de una conferencia TAPI cuando es necesario cambiar algunos datos sobre la conferencia. |
| Attribute-Id      | 1.2.840.113556.1.4.1700                                                                                      |
| System-Id-Guid    | 4cc4601e-7201-4141-abc8-3e529ae88863                                                                         |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                                                        |
