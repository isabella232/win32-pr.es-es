# <a name="msmq-routing-service-attribute"></a>Atributo MSMQ-Routing-Service

Indica si este servidor proporciona servicios de enrutamiento.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Servicio de enrutamiento de MSMQ                 |
| Nombre para mostrar de LDAP | mSMQRoutingService                   |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1237              |
| System-ID-GUID    | 2df90d81-009f-11d2-aa4c-00c04fd7d83a |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
