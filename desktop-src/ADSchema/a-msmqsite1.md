# <a name="msmq-site-1-attribute"></a>Atributo MSMQ-Site-1

DN del primer sitio de un par que está conectado.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-1                             |
| Ldap-Display-Name | mSMQSite1                               |
| Size              | \-                                      |
| Actualizar privilegios  | \-                                      |
| Frecuencia de actualización  | Cada vez que se crea una línea de sitio de MSMQ. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-Id-Guid    | 9a0dc337-c100-11d1-gni5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
