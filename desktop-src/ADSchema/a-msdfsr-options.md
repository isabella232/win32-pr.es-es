# <a name="ms-dfsr-options-attribute"></a>Atributo ms-DFSR-Options

Contiene las opciones Sistema de archivos distribuido de objeto replicación dfs (DFS).

|-------------------|--------------------------------------|
| CN                | ms-DFSR-Options                      |
| Ldap-Display-Name | msDFSR-Options                       |
| Size              | \-                                   |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.17           |
| System-Id-Guid    | d6d67084-c720-417d-8647-b696237a114c |
| Syntax            | [**Enumeración**](s-enumeration.md) |
