# <a name="acs-non-reserved-max-sdu-size-attribute"></a>Atributo ACS-non-Reserved-Max-SDU-Size

El **atributo ACS-non-Reserved-Max-SDU-Size** es solo para uso interno. Basado en RFC2814.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ACS-non-Reserved-Max-SDU-Size        |
| Ldap-Display-Name | aCSNonReservedMaxSDUSize             |
| Size              | 8 bytes                              |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-Id-Guid    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
