# <a name="inter-site-topology-failover-attribute"></a>Atributo Inter-Site-Topology-Failover

Indica cuánto tiempo debe transcurrir desde el último período de vida para que el generador de topologías entre sitios se considere fallado.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Conmutación por error entre sitios         |
| Ldap-Display-Name | interSiteTopologyFailover            |
| Size              | 4 bytes                              |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| System-Id-Guid    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
