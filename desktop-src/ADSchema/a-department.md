# <a name="department-attribute"></a>Atributo Department

Contiene el nombre del departamento en el que trabaja el usuario.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Departamento                                  |
| Ldap-Display-Name | department                                  |
| Size              | \-                                          |
| Actualizar privilegios  | Administrador de dominio o propietario de la cuenta.      |
| Frecuencia de actualización  | Siempre que el departamento necesite cambiar.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-Id-Guid    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
