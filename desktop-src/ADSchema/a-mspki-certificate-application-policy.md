# <a name="ms-pki-certificate-application-policy-attribute"></a>Atributo ms-PKI-Certificate-Application-Policy

El OID de la directiva de aplicación está en un certificado.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Certificate-Application-Policy                                                      |
| Ldap-Display-Name | msPKI-Certificate-Application-Policy                                                       |
| Size              | \-                                                                                         |
| Privilegio actualizar  | Administrador de empresa                                                                   |
| Frecuencia de actualización  | Cada vez que se crea una plantilla o se editan los atributos de una plantilla existente. |
| Attribute-Id      | 1.2.840.113556.1.4.1674                                                                    |
| System-Id-Guid    | dbd90548-aa37-4202-9966-8c537ba5ce32                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
