# <a name="ms-pki-certificate-application-policy-attribute"></a>atributo MS-PKI-Certificate-Application-Policy

El OID de la Directiva de aplicación de un certificado.



| Entrada | Value |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-Application-Policy                                                      |
| Nombre para mostrar de LDAP | Atributo mspki-Certificate-Application-Policy                                                       |
| Tamaño              | \-                                                                                         |
| Actualizar privilegio  | Administrador de empresa                                                                   |
| Frecuencia de actualización  | Siempre que se crea una nueva plantilla o se editan los atributos de las plantillas existentes. |
| Attribute-Id      | 1.2.840.113556.1.4.1674                                                                    |
| System-ID-GUID    | dbd90548-aa37-4202-9966-8c537ba5ce32                                                       |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md)                                                |
