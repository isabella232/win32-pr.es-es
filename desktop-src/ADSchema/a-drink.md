# <a name="drink-attribute"></a>atributo de bebida

El tipo de atributo de bebida (bebida favorita) especifica la bebida favorita de un objeto (o persona).



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Beber                                       |
| Ldap-Display-Name | Beber                                       |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.5                   |
| System-Id-Guid    | 1a1aa5b5-262e-4df6-af04-2cf6b0d80048        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
