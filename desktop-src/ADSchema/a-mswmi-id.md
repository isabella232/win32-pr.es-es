# <a name="ms-wmi-id-attribute"></a>Atributo ms-WMI-ID

Identificador único de una instancia de objeto. Indexado en el catálogo global.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-ID                                   |
| Ldap-Display-Name | msWMI-ID                                    |
| Size              | \-                                          |
| Privilegio actualizar  | El sistema establece este valor.            |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-Id-Guid    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
