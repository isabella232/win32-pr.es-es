# <a name="ms-ds-nc-type-attribute"></a>atributo MS-DS-NC-Type

Un campo de bits que mantiene información acerca de los aspectos de una réplica de NC que es relevante para la replicación.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Tipo MS-DS-NC                        |
| Nombre para mostrar de LDAP | msDS-NcType                          |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2024              |
| System-ID-GUID    | 5a2eacd7-cc2b-48cf-9d9a-b6f1a0024de9 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
