# <a name="create-time-stamp-attribute"></a>Atributo Create-Time-Stamp

Fecha en que se creó este objeto. Este valor se replica.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------|
| CN                | Crear marca de tiempo                                             |
| Ldap-Display-Name | createTimeStamp                                               |
| Size              | 8 bytes                                                       |
| Privilegio actualizar  | El sistema establece este valor.                              |
| Frecuencia de actualización  | Cuando se crea el objeto .                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| System-Id-Guid    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
