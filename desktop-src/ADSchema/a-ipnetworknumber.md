# <a name="ipnetworknumber-attribute"></a>atributo ipNetworkNumber

Contiene un número de red IP en notación decimal con puntos, omitiendo los ceros a la izquierda.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ipNetworkNumber                      |
| Nombre para mostrar de LDAP | ipNetworkNumber                      |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-ID-GUID    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)  |
