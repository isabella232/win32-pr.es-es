# <a name="super-scope-description-attribute"></a>Atributo Super-Scope-Description

Este atributo proporciona una descripción para un superádido.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Super-Scope-Description                     |
| Ldap-Display-Name | superScopeDescription                       |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.711                      |
| System-Id-Guid    | 963d274c-48be-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
