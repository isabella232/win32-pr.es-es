# <a name="is-defunct-attribute"></a>Is-Defunct atributo

Si **es TRUE,** la clase o el atributo ya no se puede utiliza. Es posible que existan versiones anteriores de este objeto, pero no se pueden crear otras nuevas.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| Ldap-Display-Name | isDefunct                            |
| Size              | \-                                   |
| Actualizar privilegios  | Administrador de esquemas                 |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-Id-Guid    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
