# <a name="print-min-y-extent-attribute"></a>Print-min-Y-extensión (atributo)

La región de impresión vertical mínima.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Imprimir-min-Y-extensión                   |
| Nombre para mostrar de LDAP | printMinYExtent                      |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.280               |
| System-ID-GUID    | ba305f72-47e3-11d0-a1a6-00c04fd930c9 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
