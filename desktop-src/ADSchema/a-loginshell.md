# <a name="loginshell-attribute"></a>atributo loginShell

Contiene la ruta de acceso al shell de inicio de sesión.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | loginShell                           |
| Ldap-Display-Name | loginShell                           |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.4                      |
| System-Id-Guid    | a553d12c-3231-4c5e-8adf-8d189697721e |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
