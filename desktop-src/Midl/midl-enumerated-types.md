# <a name="midl-enumerated-types"></a>Tipos enumerados de MIDL

El compilador de MIDL no traduce la declaración de [**enumeración**](enum.md) en instrucciones de **\# definición** , como lo indican algunos compiladores DCE. Se reproduce como una declaración de **enumeración** en lenguaje C en el archivo de encabezado generado.
