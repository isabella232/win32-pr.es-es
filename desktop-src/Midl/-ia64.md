# <a name="ia64-switch"></a>Modificador /ia64

El **modificador /ia64** dirige al compilador MIDL a generar archivos de código auxiliar, o un archivo de biblioteca de tipos, para un entorno de arquitectura Intel de 64 bits.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Observaciones

> [!Note]  
> No es posible usar MIDL.exe generar un tlb de 64 bits en Windows 2000.
