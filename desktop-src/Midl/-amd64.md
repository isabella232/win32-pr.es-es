# <a name="amd64-switch"></a>Modificador /amd64

El **modificador /amd64** dirige al compilador MIDL a generar archivos de código auxiliar, o un archivo de biblioteca de tipos, para un entorno de 64 bits de Advanced Micro Devices.

``` syntax
midl /amd64
```

## <a name="remarks"></a>Observaciones

> [!Note]  
> El **modificador /x64** descrito por la MIDL.exe línea de comandos ha quedado en desuso. **Se debe usar /amd64** en su lugar.
