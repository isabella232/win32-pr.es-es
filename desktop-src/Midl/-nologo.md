# <a name="nologo-switch"></a>Modificador /nologo

El **modificador /nologo** deshabilita la presentación del banner de copyright.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Opciones de cambio

Este modificador no tiene parámetros.
