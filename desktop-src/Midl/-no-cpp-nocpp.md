# <a name="no_cpp-nocpp"></a>/no \_ cpp, /nocpp

El **modificador /no \_ cpp** especifica que el compilador MIDL no llama al preprocesador de C para preprocesar el archivo IDL.

El **modificador /no \_ cpp** tiene prioridad sobre los modificadores [**/cpp \_ cmd**](-cpp-cmd.md) y [**/cpp \_ opt.**](-cpp-opt.md)

## <a name="switch-options"></a>Opciones de cambio

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Ejemplo

**midl /no \_ cpp filename.idl**
