# <a name="idwritegeometrysink-interface"></a>Interfaz IDWriteGeometrySink

**IDWriteGeometrySink es** una **definición de tipo** de la interfaz [**ID2D1SimplifiedGeometrySink.**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) Consulte la página [**de referencia ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) para obtener más información.

## <a name="members"></a>Members

La **interfaz IDWriteGeometrySink** hereda de la [**interfaz IUnknown,**](/windows/win32/api/unknwn/nn-unknwn-iunknown) pero no tiene miembros adicionales.
