# <a name="nmeventdata-structure"></a>Estructura NMEVENTDATA

La **estructura NMEVENTDATA** contiene información sobre una condición de evento que se pasa a Monitor de red para insertar una línea en el visor experto.

## <a name="syntax"></a>Sintaxis


```C++
typedef struct {
  BYTE         Version;
  DWORD        EventIdent;
  DWORD        Flags;
  DWORD        Severity;
  BYTE         NumColumns;
  LPSTR        szSourceName;
  LPSTR        szEventName;
  LPSTR        szDescription;
  LPSTR        szMachine;
  JTYPE        Justification;
  LPSTR        szUrl;
  SYSTEMTIME   SysTime;
  NMCOLUMNINFO Column[];
} NMEVENTDATA, *PNMEVENTDATA;
```
