# <a name="g-network-monitor"></a>G (Monitor de red)

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**formateador genérico**
</dt> <dd>

Formateador de datos de propiedad Monitor de red proporciona. El formateador genérico puede dar formato a los datos que se muestran en el panel de detalles de la interfaz Monitor de red usuario. Si desea que Monitor de red el formateador genérico, debe especificar la función **FormatPropertyInstance** en el miembro **InstanceData** de la **estructura PROPERTYINFO** de una propiedad. Consulte también la [salida del formateador genérico.](generic-formatter-output.md)

</dd> </dl>
