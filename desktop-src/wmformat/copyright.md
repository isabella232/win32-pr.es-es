# <a name="copyright"></a>Copyright

El **atributo Copyright** contiene un mensaje de copyright para el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMCopyright

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="example"></a>Ejemplo



| Tipo de archivo | Valor de ejemplo                          |
|-----------|----------------------------------------|
| Any       | "Copyright (C) Microsoft Corporation." |



 

## <a name="remarks"></a>Observaciones

El texto de un mensaje de copyright no se ajusta a ningún estándar. Las empresas individuales pueden usar cualquier formato que sea adecuado.

Este atributo se aplica a todo el archivo.
