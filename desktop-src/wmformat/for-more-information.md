# <a name="for-more-information-windows-media-format-sdk"></a>Más información (SDK de Windows Media Format)

-   Para obtener la información más reciente sobre los problemas conocidos con Windows Media Format SDK, vea las notas de la versión publicadas en el sitio [web de MSDN Library](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fwmfsdk11_b2_release_notes_rzxb.asp%3fframe%3dtrue).
-   Para obtener características, artículos y respuestas a las preguntas más frecuentes sobre los medios de streaming en general y los SDK de Windows Media en particular, consulte el Centro para desarrolladores de Windows Media en el sitio [web de Microsoft](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Para obtener más información sobre cómo obtener una licencia para el SDK de Windows Media Format, vea la página Licencias de Windows Media en el sitio [web de Microsoft](https://msdn.microsoft.com/library/ms867178.aspx).
-   Para obtener más información sobre los grupos de noticias, visite la página Grupos de noticias y alias de correo electrónico de Windows Media en el sitio [web de Microsoft](https://windows.microsoft.com/windows/products/windows-media).
-   Para obtener descripciones detalladas y de bajo nivel de los objetos de datos que son archivos ASF, descargue la especificación de ASF desde el [sitio web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).
-   Para conocer las características, artículos, procedimientos recomendados y respuestas a las preguntas más frecuentes sobre administración de derechos digitales (DRM) y Windows Media Rights Manager, consulte los artículos técnicos de la página [Rights Management](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fdigitalrightsmanagement.asp) digital en el sitio web de Microsoft.
