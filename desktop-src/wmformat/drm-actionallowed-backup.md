# <a name="drm_actionallowed_backup"></a>Acción de \_ DRMAllowed \_ Backup

El **atributo \_ ActionAllowed \_ Backup** de DRM indica si se puede realizar una copia de seguridad de la licencia.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ ActionAllowed \_ Backup

## <a name="data-type"></a>Tipo de datos

**WMT \_ TYPE \_ BOOL**

## <a name="remarks"></a>Observaciones
