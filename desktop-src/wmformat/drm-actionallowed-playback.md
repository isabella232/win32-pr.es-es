# <a name="drm_actionallowed_playback"></a>Acción de \_ DRMAllowed \_ Playback

El **atributo \_ Acción DRMAllowed \_ Playback** indica si se permite la reproducción del contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ ActionAllowed \_ Playback

## <a name="data-type"></a>Tipo de datos

**TIPO WMT \_ \_ BOOL**

## <a name="remarks"></a>Observaciones

Se trata de una propiedad de solo lectura que se recupera mediante [**IWMDRMReader::GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
