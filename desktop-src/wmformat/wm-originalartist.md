# <a name="wmoriginalartist"></a>WM/OriginalArtist

El **atributo WM/OriginalArtist** contiene el nombre del intérprete que produjo originalmente el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMOriginalArtist

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Comentarios

El intérprete original se usa normalmente para remakes de canciones. Para estos archivos. use el [**nombre del**](author.md) autor de la banda que registró originalmente la canción.
