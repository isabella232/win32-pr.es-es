# <a name="irmgetlicense-interface"></a>Interfaz IRMGetLicense

\[**IRMGetLicense** está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, [use Microsoft PlayReady](https://www.microsoft.com/PlayReady/). \]

La **interfaz IRMGetLicense** se implementa en una biblioteca en tiempo de ejecución que se instala como parte del SDK Windows Media Format. Permite la adquisición de licencias del lado cliente. Dado que los emisores de licencias usan este método a través de aplicaciones basadas en web, se documenta en la documentación del SDK Windows Media Rights Manager.

## <a name="members"></a>Members

La **interfaz IRMGetLicense** hereda de la [**interfaz IUnknown,**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) pero no tiene miembros adicionales.
