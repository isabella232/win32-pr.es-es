# <a name="wmalbumcoverurl"></a>WM/AlbumCoverURL

El **atributo WM/AlbumCoverURL** contiene la dirección de una página web que contiene una imagen de la portada del álbum e información sobre el álbum.

## <a name="global-constant"></a>Constante global

g \_ wszWMAlbumCoverURL

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**
