## <a name="remarks"></a>Observaciones

El identificador de la interfaz para cada una de las interfaces de este SDK es el nombre de la interfaz precedida por "IID \_ ". Por ejemplo, el identificador de la interfaz **IWMReaderAdvanced** es IID \_ IWMReaderAdvanced. Para obtener un puntero a la interfaz **IWMReaderAdvanced** de un objeto de lector, use el código siguiente, donde *pReader* es un puntero a una interfaz **IWMReader** :


```C++
IWMReaderAdvanced *pReaderAdvanced;
hr = pReader->QueryInterface(IID_IWMReaderAdvanced,
                             (LPVOID*) &pReaderAdvanced);

```
