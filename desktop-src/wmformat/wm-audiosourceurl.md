# <a name="wmaudiosourceurl"></a>WM/AudioSourceURL

El **atributo WM/AudioSourceURL** contiene la dirección de una página web oficial del medio de origen. Por ejemplo, una canción de un sonido podría tener un vínculo a la página oficial de la película.

## <a name="global-constant"></a>Constante global

g \_ wszWMAudioSourceURL

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**
