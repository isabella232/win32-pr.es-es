# <a name="wmprovider"></a>WM/Provider

El **atributo WM/Provider** contiene el nombre del proveedor de contenido de metadatos. A menudo, los metadatos se obtienen a través de un servicio.

## <a name="global-constant"></a>Constante global

g \_ wszWMProvider

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

|-----------|---------------|
