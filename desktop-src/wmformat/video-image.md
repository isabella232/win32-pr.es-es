# <a name="video-image"></a>Imagen de vídeo

El códec Windows Media Video 9 Image v2 permite crear archivos de vídeo multimedia de Windows cuyo contenido consta de una serie de imágenes fijas, como fotografías digitales. Las imágenes del flujo de imágenes de vídeo se pueden manipular para generar movimientos panorámicos y zoom, así como varias transiciones. Los usuarios finales pueden usar esta funcionalidad para crear y compartir sencillas presentaciones de diapositivas que también pueden contener un sonido de música o narración de voz en exceso. Debido a la compresión superior del códec Windows Media Video Image, un archivo que contiene varias fotografías sigue siendo lo suficientemente pequeño como para enviar un correo electrónico a través de Internet.

Una secuencia de imagen de vídeo tiene su propio tipo de medio y una estructura especializada que define las distintas transiciones que son posibles al pasar de una imagen a la siguiente. Puede desplazarse y acercar la vista de las imágenes mientras las combina. A continuación, el archivo resultante se puede reproducir como cualquier otro Windows de vídeo multimedia.
