# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9.5 SDK x64 Edition

Esta documentación se aplica a la nueva edición x64 del SDK de Windows Media Format 9.5, así como a las versiones anteriores de 32 bits del SDK de Windows Media Format.

En la lista siguiente se resumen los cambios en el SDK de la versión basada en x64:

-   NO se admite DRM. Los archivos relacionados con DRM (incluidos los ejemplos) se han quitado de la instalación del SDK.
-   Windows La Administrador de dispositivos multimedia no forma parte de la instalación del SDK basada en x64.
-   Los ejemplos y encabezados que admiten Windows media audio y video codec interfaces no forman parte de la instalación del SDK basado en x64.
-   Los perfiles del sistema localizados no forman parte de la instalación del SDK basada en x64.

En las notas de la sección Referencia de programación de esta documentación se resaltan las características que no se admiten en Windows Media Format 9.5 SDK x64 Edition.
