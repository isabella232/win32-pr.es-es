# <a name="rating"></a>Rating

El atributo Rating contiene la clasificación del contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMRating

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Observaciones

Se trata de un atributo de nivel de archivo.

Las implementaciones individuales de los archivos existentes usan diferentes escalas y notaciones. Por este problema, debe evitar el uso de este atributo; no hay ninguna estandarización a partir de la cual desarrollar interfaces de usuario coherentes.
