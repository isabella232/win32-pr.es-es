# <a name="pnrp-constants"></a>Constantes PNRP

La API del proveedor de espacios de nombres PNRP utiliza las siguientes constantes.



| Constante                                                 | Descripción                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**\_marcas de nube PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Especifica la validez de un nombre de nube.                    |
| [**\_Estado de la nube PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Especifica los distintos Estados en los que puede estar una nube PNRP.     |
| [**\_criterios de resolución de PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Especifica los criterios que usa PNRP para resolver búsquedas. |
