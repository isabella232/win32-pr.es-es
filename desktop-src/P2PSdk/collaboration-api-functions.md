# <a name="collaboration-api-functions"></a>Funciones de la API de colaboración

La infraestructura de colaboración del mismo nivel admite las siguientes funciones.



| Función                                                                                       | Descripción                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerCollabAddContact**](/windows/desktop/api/P2P/nf-p2p-peercollabaddcontact)                                           | Agrega un contacto a la lista de contactos de un elemento del mismo nivel.                                                                                                                                                   |
| [**PeerCollabAsyncInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinvitecontact)                           | Envía una invitación a un contacto del mismo nivel de confianza para unirse a la actividad de colaboración del mismo nivel del remitente a través de una conexión segura.                                                                       |
| [**PeerCollabAsyncInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinviteendpoint)                         | Envía una invitación a un punto de conexión del mismo nivel especificado para unirse a la actividad de colaboración del mismo nivel del remitente. La disponibilidad de la respuesta a la invitación se actualiza a través de un evento asincrónico. |
| [**PeerCollabCancelInvitation**](/windows/desktop/api/P2P/nf-p2p-peercollabcancelinvitation)                               | Cancela una invitación enviada previamente por el autor de la llamada a un contacto.                                                                                                                               |
| [**PeerCollabCloseHandle**](/windows/desktop/api/P2P/nf-p2p-peercollabclosehandle)                                         | Cierra el identificador a una invitación de actividad de colaboración del mismo nivel.                                                                                                                                  |
| [**PeerCollabDeleteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabdeletecontact)                                     | Elimina un contacto del mismo nivel actual.                                                                                                                                                        |
| [**PeerCollabDeleteEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteendpointdata)                           | Elimina los datos del punto de conexión del mismo nivel en el nodo del mismo nivel que coincide con los datos de punto de conexión proporcionados.                                                                                                |
| [**PeerCollabDeleteObject**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteobject)                                       | Elimina un objeto del mismo nivel del punto de conexión que realiza la llamada.                                                                                                                                                |
| [**PeerCollabEnumApplications**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplications)                               | Devuelve el identificador a una enumeración que contiene las funcionalidades registradas en los puntos de conexión de un elemento del mismo nivel específico.                                                                                |
| [**PeerCollabEnumApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplicationregistrationinfo) | Obtiene el identificador de enumeración utilizado para recuperar información de la aplicación del mismo nivel.                                                                                                                   |
| [**PeerCollabEnumContacts**](/windows/desktop/api/P2P/nf-p2p-peercollabenumcontacts)                                       | Devuelve un identificador a un conjunto enumerado que contiene todos los contactos de red de colaboración del mismo nivel disponibles actualmente en el elemento del mismo nivel que realiza la llamada.                                                     |
| [**PeerCollabEnumEndpoints**](/windows/desktop/api/P2P/nf-p2p-peercollabenumendpoints)                                     | Devuelve el identificador a una enumeración que contiene los puntos de conexión asociados a un contacto del mismo nivel específico.                                                                                       |
| [**PeerCollabEnumObjects**](/windows/desktop/api/P2P/nf-p2p-peercollabenumobjects)                                         | Devuelve el identificador a una enumeración que contiene los objetos del mismo nivel asociados al punto de conexión de un elemento del mismo nivel específico.                                                                                 |
| [**PeerCollabEnumPeopleNearMe**](/windows/desktop/api/P2P/nf-p2p-peercollabenumpeoplenearme)                               | Devuelve un identificador a un conjunto enumerado que contiene todos los puntos de conexión de la red de colaboración del mismo nivel "personas cerca de mí" disponibles actualmente en la subred del elemento del mismo nivel que realiza la llamada.                     |
| [**PeerCollabExportContact**](/windows/desktop/api/P2P/nf-p2p-peercollabexportcontact)                                     | Exporta los datos de contacto asociados con un nombre del mismo nivel a un búfer de cadena de datos XML de contacto.                                                                                                       |
| [**PeerCollabGetAppLaunchInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplaunchinfo)                               | Obtiene la información de inicio de la aplicación del mismo nivel, incluido el nombre del contacto, el punto de conexión del mismo nivel y la solicitud de invitación.                                                                     |
| [**PeerCollabGetApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplicationregistrationinfo)   | Obtiene información de registro de aplicación específica.                                                                                                                                          |
| [**PeerCollabGetContact**](/windows/desktop/api/P2P/nf-p2p-peercollabgetcontact)                                           | Obtiene la información de un contacto del mismo nivel específico según el nombre del mismo nivel del contacto.                                                                                                         |
| [**PeerCollabGetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabgetendpointname)                                 | Recupera el nombre del punto de conexión actual del par que realiza la llamada previamente establecido por una llamada a PeerCollabSetEndpointName.                                                                           |
| [**PeerCollabGetEventData**](/windows/desktop/api/P2P/nf-p2p-peercollabgeteventdata)                                       | Obtiene los datos asociados a un evento de colaboración del mismo nivel que se genera en el mismo nivel.                                                                                                                 |
| [**PeerCollabGetInvitationResponse**](/windows/desktop/api/P2P/nf-p2p-peercollabgetinvitationresponse)                     | Obtiene la respuesta del mismo nivel invitado previamente para unirse a una actividad de colaboración del mismo nivel.                                                                                                        |
| [**PeerCollabGetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetpresenceinfo)                                 | Recupera la información de presencia del punto de conexión asociado a un contacto específico.                                                                                                         |
| [**PeerCollabGetSigninOptions**](/windows/desktop/api/P2P/nf-p2p-peercollabgetsigninoptions)                               | Obtiene las opciones actuales de presencia de red de colaboración del mismo nivel con sesión firmada.                                                                                                               |
| [**PeerCollabInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabinvitecontact)                                     | Envía una invitación para unirse a una actividad de colaboración del mismo nivel a un contacto de confianza. Esta llamada es sincrónica y, si se realiza correctamente, obtiene una respuesta del contacto.                               |
| [**PeerCollabInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabinviteendpoint)                                   | Envía una invitación a un punto de conexión del mismo nivel especificado para unirse a la actividad de colaboración del mismo nivel del remitente. Esta llamada es sincrónica y, si se realiza correctamente, obtiene una respuesta del punto de conexión del mismo nivel.      |
| [**PeerCollabParseContact**](/windows/desktop/api/P2P/nf-p2p-peercollabparsecontact)                                       | Analiza un búfer de cadena Unicode que contiene datos XML de contacto en una [**estructura de datos PEER \_ CONTACT.**](/windows/desktop/api/P2P/ns-p2p-peer_contact)                                                                         |
| [**PeerCollabQueryContactData**](/windows/desktop/api/P2P/nf-p2p-peercollabquerycontactdata)                               | Recupera la información de contacto del punto de conexión del mismo nivel proporcionado.                                                                                                                               |
| [**PeerCollabRefreshEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabrefreshendpointdata)                         | Actualiza el nodo del mismo nivel que realiza la llamada con nuevos datos de punto de conexión.                                                                                                                                           |
| [**PeerCollabRegisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterapplication)                         | Registra una aplicación con el equipo local para que se pueda iniciar en una actividad de colaboración del mismo nivel.                                                                                   |
| [**PeerCollabRegisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterevent)                                     | Registra una aplicación con la infraestructura de colaboración del mismo nivel para recibir devoluciones de llamada para eventos de colaboración del mismo nivel específicos.                                                                |
| [**PeerCollabSetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabsetendpointname)                                 | Establece el nombre del punto de conexión actual utilizado por la aplicación del mismo nivel.                                                                                                                             |
| [**PeerCollabSetObject**](/windows/desktop/api/P2P/nf-p2p-peercollabsetobject)                                             | Crea o actualiza un objeto de datos del mismo nivel que se usa en una red de colaboración del mismo nivel.                                                                                                                     |
| [**PeerCollabSetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabsetpresenceinfo)                                 | Actualiza la información de presencia del autor de la llamada a los contactos que la ven.                                                                                                                          |
| [**PeerCollabSignIn**](/windows/desktop/api/P2P/nf-p2p-peercollabsignin)                                                   | Inicia la sesión del mismo nivel en un proveedor de presencia de red de colaboración del mismo nivel hospedado en Internet (presencia sin servidor) o subred ("Equipos a mi alrededor").                                                          |
| [**PeerCollabSignOut**](/windows/desktop/api/P2P/nf-p2p-peercollabsignout)                                                 | Cierra un emparejamiento de un tipo específico de proveedor de presencia de red de colaboración del mismo nivel.                                                                                                            |
| [**PeerCollabShutdown**](/windows/desktop/api/P2P/nf-p2p-peercollabshutdown)                                               | Cierra la infraestructura de colaboración del mismo nivel y libera los recursos asociados a ella.                                                                                                 |
| [**PeerCollabStartup**](/windows/desktop/api/P2P/nf-p2p-peercollabstartup)                                                 | Inicializa la infraestructura de colaboración del mismo nivel.                                                                                                                                              |
| [**PeerCollabSubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabsubscribeendpointdata)                     | Crea una suscripción a un punto de conexión disponible.                                                                                                                                                |
| [**PeerCollabUnregisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterapplication)                     | Anula el registro de las aplicaciones específicas de un equipo del mismo nivel del equipo local.                                                                                                                        |
| [**PeerCollabUnregisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterevent)                                 | Anula el registro de una aplicación de la notificación sobre eventos específicos de colaboración del mismo nivel.                                                                                                          |
| [**PeerCollabUnsubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabunsubscribeendpointdata)                 | Quita una suscripción a un punto de conexión creado con PeerCollabSubscribeEndpointData.                                                                                                             |
| [**PeerCollabUpdateContact**](/windows/desktop/api/P2P/nf-p2p-peercollabupdatecontact)                                     | Actualiza un par que participa en una red de colaboración del mismo nivel con nueva información sobre un contacto del mismo nivel.                                                                                            |
