# <a name="constants-for-cryptencodeobject-and-cryptdecodeobject"></a>Constantes para CryptEncodeObject y CryptDecodeObject

Las funciones [**CryptEncodeObject,**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptencodeobject) [**CryptEncodeObjectEx,**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptencodeobjectex) [**CryptSignAndEncodeCertificate,**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptsignandencodecertificate) [**CryptDecodeObject**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptdecodeobject)y [**CryptDecodeObjectEx**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptdecodeobjectex) son funciones generalizadas de codificación y descodificación, capaces de codificar y descodificar certificados codificados con la notación de sintaxis abstracta [*One*](../secgloss/a-gly.md) (ASN.1), listas de [*revocación*](../secgloss/c-gly.md) de certificados (CRL), [](../secgloss/c-gly.md) listas de certificados de confianza (CL) y solicitudes de certificado.

En la tabla siguiente se enumeran las constantes predefinidas, las extensiones y los atributos utilizados con las operaciones de codificación y descodificación, y la estructura de datos a la que apunta el *parámetro pvStructInfo.*

> [!Note]  
> Algunas constantes predefinidas y cadenas OID tienen el mismo significado. Cuando lo hacen, se puede usar como parámetro *lpszStuctType.*
