# <a name="extensibility-cryptography"></a>Extensibilidad (criptografía)

Servicios de Certificate Server admite extensiones de certificado [*X. 509*](../secgloss/x-gly.md) . Los módulos de directivas pueden agregar o modificar extensiones y un módulo de salida puede recuperar estas extensiones.

Consulte [controladores de extensión](extension-handlers.md) para obtener más información.
