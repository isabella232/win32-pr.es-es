# <a name="oidsitem-property"></a>Propiedad OIDs.Item

\[La **propiedad Item** está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En su lugar, use [**la clase OidCollection en**](/dotnet/api/system.security.cryptography.oidcollection?view=netcore-3.1) el espacio de nombres [**System.Security.Cryptography.**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true)\]

La **propiedad Item** recupera un objeto [**OID**](oid.md) de la colección. Esta es la propiedad predeterminada.

## <a name="syntax"></a>Sintaxis


```VB
OIDs.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Valor de propiedad

El [**objeto OID**](oid.md) en el índice especificado o el **objeto OID** con el valor de puntos especificado.
