# <a name="prov_rsa_schannel"></a>PROV \_ RSA \_ SCHANNEL

El tipo de proveedor SCHANNEL RSA de PROV \_ \_ admite los protocolos RSA y [*Schannel.*](../secgloss/s-gly.md)

## <a name="algorithms-supported"></a>Algoritmos admitidos

Para obtener descripciones de cada uno de estos algoritmos, consulte el glosario.



| Propósito      | Algoritmos admitidos                                                                                                                                                                                                                                                                                                          |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Intercambio de claves | [*RSA*](../secgloss/r-gly.md)                                                                                                                                                                                                                                                                   |
| Firma    | RSA                                                                                                                                                                                                                                                                                                                           |
| Cifrado   | Cualquier CSP de SCHANNEL RSA de PROV \_ debe implementar uno o varios de los \_ algoritmos siguientes: [ *RC4*](../secgloss/r-gly.md)<br/> [*DES*](../secgloss/d-gly.md)<br/> [*Triple DES*](../secgloss/t-gly.md)<br/> |
| Aplicación de algoritmo hash      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/>                                                                                                                                                                                             |
