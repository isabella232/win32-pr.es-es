# <a name="oidscount-property"></a>Propiedad OIDs.Count

\[La **propiedad Count** está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En su lugar, use [**la clase OidCollection en**](/dotnet/api/system.security.cryptography.oidcollection?view=netcore-3.1) el espacio de nombres [**System.Security.Cryptography.**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true)\]

La **propiedad Count** recupera el número de OMISIÓN de la colección.

## <a name="syntax"></a>Sintaxis


```VB
OIDs.Count As Long
```



## <a name="property-value"></a>Valor de propiedad

Número de [**objetos OID**](oid.md) de la colección.
