# <a name="requests"></a>Requests

Servicios de certificados [*admite solicitudes de certificado basadas*](../secgloss/c-gly.md) en el formato PKCS 10 y el formato \# Keygen (Netscape). Servicios de certificados también admite solicitudes de renovación PKCS 7 y solicitudes de sintaxis de \# mensajes criptográficos (CMS).

Servicios de certificados también proporciona una [interfaz ICertRequest,](/windows/desktop/api/Certcli/nn-certcli-icertrequest) que contiene métodos para enviar una solicitud de certificado y (si la solicitud está aprobada) para recibir el certificado emitido resultante.

Hay interfaces adicionales relacionadas con la seguridad disponibles en el [Control de inscripción de certificados](certificate-enrollment-control.md).
