# <a name="firewall-dynamic-keywords-structures"></a>Estructuras de palabras clave dinámicas de firewall

Las palabras clave dinámicas del firewall contienen las siguientes estructuras.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0) | Contiene los datos devueltos al cliente cuando se llama a las API **de** enumeración. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address0) | Permite al cliente crear una dirección de palabra clave dinámica, que contiene una lista de direcciones IP. |
