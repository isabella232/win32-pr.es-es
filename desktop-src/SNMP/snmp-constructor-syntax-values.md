# <a name="snmp-constructor-syntax-values"></a>Valores de sintaxis del constructor SNMP

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, [Windows administración remota](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-Man.\]

Los valores de sintaxis del constructor SNMP describen los tipos que son compatibles con el estándar de codificación Abstract Syntax Notation One (ASN.1).

|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------|
| <span id="ASN_SEQUENCE"></span><span id="asn_sequence"></span><dl> <dt>**ASN \_ SEQUENCE**</dt> </dl>       | Indica que el mensaje es una secuencia de ASN.<br/> |
