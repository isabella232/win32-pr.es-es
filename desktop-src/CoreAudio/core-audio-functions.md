# <a name="core-audio-functions"></a>Funciones de audio principales

En esta sección se describen las funciones que usan las API de audio principales a partir de Windows 8.



| Función                                                           | Descripción                                                                                                                                                                         |
|--------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActivateAudioInterfaceAsync**](/windows/desktop/api/mmdeviceapi/nf-mmdeviceapi-activateaudiointerfaceasync) | Permite que las aplicaciones de la tienda Windows tengan acceso a las interfaces del modelo de objetos componentes (COM) preexistentes en la familia [WASAPI](wasapi.md) .<br/> Se aplica a partir de Windows 8.<br/> |
