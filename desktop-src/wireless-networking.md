# <a name="wireless-networking"></a>Redes inalámbricas

## <a name="in-this-section"></a>En esta sección



| Tema                                                                          | Descripción                                                                                                                                                                                                                                                                                                     |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Bluetooth](./bluetooth/bluetooth-start-page.md)<br/>                         | Bluetooth es un protocolo estándar del sector que habilita la conectividad inalámbrica para equipos, dispositivos de mano, teléfonos móviles y otros dispositivos.<br/>                                                                                                                                                     |
| [Asociación de datos por infrarrojos (IrDA)](/previous-versions/windows/desktop/irda/irda-start-page)<br/>            | La Asociación de datos por infrarrojos (IrDA) es un conjunto de protocolos diseñado para proporcionar conectividad inalámbrica y de línea entre dispositivos. <br/>                                                                                                                                                                      |
| [Banda ancha móvil](./mbn/mobile-broadband-networks-portal.md)<br/>            | La API de banda ancha móvil se usa para implementar la conectividad a redes móviles. Las aplicaciones no deben comunicarse directamente con estos dispositivos de banda ancha móvil. En su lugar, deben usar la API de banda ancha móvil. <br/>                                                                                  |
| [Wi-Fi nativa](./nativewifi/portal.md)<br/>                                         | El componente de configuración automática WiFi nativa configura, se conecta a y desconecta de las redes inalámbricas. Wi-Fi nativo puede almacenar perfiles en las redes con las que interactúa en forma de documentos XML. <br/>                                                                                   |
| [Windows Connect Now](./wcn/portal.md)<br/>                                   | Windows Connect Now (WCN) permite que los dispositivos móviles e incrustados, los puntos de acceso 802,11 (APs) y los equipos se conecten de forma segura, y intercambien la configuración entre sí. WCN está diseñado para el usuario doméstico o de pequeña empresa, lo que proporciona un compromiso razonable entre la facilidad de uso y la seguridad sólida. <br/> |
| [Administrador de conexiones de Windows](./wcm/windows-connection-manager-portal.md)<br/> | El administrador de conexiones de Windows (WCM) permite crear y configurar el software de Connection Manager.<br/>                                                                                                                                                                                              |
