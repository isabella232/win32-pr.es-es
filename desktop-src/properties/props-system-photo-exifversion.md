# <a name="systemphotoexifversion"></a>System.Photo.EXIFVersion

La versión del archivo de imagen intercambiable (EXIF).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.EXIFVersion
   shellPKey = PKEY_Photo_EXIFVersion
   formatID = D35F743A-EB2E-47F2-A286-844132CB1427
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
