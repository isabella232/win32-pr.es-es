# <a name="systemdeviceinterfacebluetoothmanufacturer"></a>System. DeviceInterface. Bluetooth. manufacturer

Fabricante del dispositivo Bluetooth.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.Manufacturer
   shellPKey = PKEY_DeviceInterface_Bluetooth_Manufacturer
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
