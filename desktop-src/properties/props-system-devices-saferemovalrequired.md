# <a name="systemdevicessaferemovalrequired"></a>System.Devices.SafeRemovalRequired

Indica si un dispositivo requiere una eliminación segura.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.SafeRemovalRequired
   shellPKey = PKEY_Devices_SafeRemovalRequired
   formatID = AFD97640-86A3-4210-B67C-289C41AABE55
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
