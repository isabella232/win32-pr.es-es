# <a name="systemcontactotheraddresspostofficebox"></a>System.Contact.OtherAddressPostOfficeBox

Número de la oficina de correos de otra dirección del contacto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.OtherAddressPostOfficeBox
   shellPKey = PKEY_Contact_OtherAddressPostOfficeBox
   formatID = 8B26EA41-058F-43F6-AECC-4035681CE977
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
