# <a name="systemcalendarorganizeraddress"></a>System.Calendar.OrganizerAddress

Dirección del organizador de eventos. Se trata de una dirección postal o de correo electrónico.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Calendar.OrganizerAddress
   shellPKey = PKEY_Calendar_OrganizerAddress
   formatID = 744C8242-4DF5-456C-AB9E-014EFB9021E3
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
