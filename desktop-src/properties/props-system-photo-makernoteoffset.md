# <a name="systemphotomakernoteoffset"></a>System.Photo.MakerNoteOffset

Desplazamiento de la nota de creador especificada en [System.Photo.MakerNote.](./props-system-photo-makernote.md) Consulte la especificación archivo de imagen intercambiable (EXIF) 2.2 para obtener más información. Esta propiedad no está pensada para mostrarse en el Shell, pero está disponible mediante programación para las aplicaciones.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNoteOffset
   shellPKey = PKEY_Photo_MakerNoteOffset
   formatID = 813F4124-34E6-4D17-AB3E-6B1F3C2247A1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
