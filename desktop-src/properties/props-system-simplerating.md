# <a name="systemsimplerating"></a>System. SimpleRating

Sistema de clasificación que utiliza un intervalo de valores enteros entre 0 y 5.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.SimpleRating
   shellPKey = PKEY_SimpleRating
   formatID = A09F084E-AD41-489F-8076-AA5BE3082BCA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.

Por compatibilidad con el sistema de clasificación del shell de Windows Vista, el controlador de propiedades también debe rellenar la propiedad [System. Rating](./props-system-rating.md) con la asignación tal y como se describe en esa propiedad.

Use la tabla siguiente para convertir de [System. Rating](./props-system-rating.md) a [System. SimpleRating]().



| System. Rating | System. SimpleRating |
|---------------|---------------------|
| 0             | 0                   |
| 1-12          | 1                   |
| 13-37         | 2                   |
| 38-62         | 3                   |
| 63-87         | 4                   |
| 88-99         | 5                   |
