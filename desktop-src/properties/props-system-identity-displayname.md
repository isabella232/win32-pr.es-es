# <a name="systemidentitydisplayname"></a>System. Identity. DisplayName

```
propertyDescription
   name = System.Identity.DisplayName
   shellPKey = PKEY_Identity_DisplayName
   formatID = 7D683FC9-D155-45A8-BB1F-89D19BCB792F
```
