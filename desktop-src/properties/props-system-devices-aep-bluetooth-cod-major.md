# <a name="systemdevicesaepbluetoothcodmajor"></a>System.Devices.Aep. Bluetooth. Cod.Major

Bluetooth clase de código principal del dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Major
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Major
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
