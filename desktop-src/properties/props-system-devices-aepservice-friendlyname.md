# <a name="systemdevicesaepservicefriendlyname"></a>System. Devices. AepService. FriendlyName

Nombre descriptivo del servicio de punto de conexión de Asociación de dispositivo

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.AepService.FriendlyName
   shellPKey = PKEY_Devices_AepService_FriendlyName
   formatID = 71724756-3E74-4432-9B59-E7B2F668A593
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
