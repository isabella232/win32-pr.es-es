# <a name="systemproplistfileoperationprompt"></a>System.PropList.FileOperationPrompt

Lista de propiedades que se mostrarán en el cuadro de diálogo de confirmación de la operación de archivo. No se mostrarán las propiedades con valores vacíos. Si no se especifica esta lista, se usa en su lugar la lista de propiedades infotip. Regístrese en el valor reg de "FileOperationPrompt".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.FileOperationPrompt
   shellPKey = PKEY_PropList_FileOperationPrompt
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
