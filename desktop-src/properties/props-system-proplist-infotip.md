# <a name="systemproplistinfotip"></a>System. proplist. recuadro informativo

Lista de propiedades que se van a mostrar en el recuadro informativo. No se mostrarán las propiedades con valores vacíos. Regístrese en el regvalue de "recuadro informativo".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.InfoTip
   shellPKey = PKEY_PropList_InfoTip
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
