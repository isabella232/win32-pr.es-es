# <a name="systemdevicesdnssdnetworkadapterid"></a>System. Devices. dnssd. NetworkAdapterId

GUID del adaptador de red en el que se va a buscar un servicio.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.Dnssd.NetworkAdapterId
   shellPKey = PKEY_Devices_Dnssd_NetworkAdapterId
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
