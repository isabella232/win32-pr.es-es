# <a name="systemdatemodified"></a>System.DateModified

Fecha y hora de la última modificación del elemento. El nombre descriptivo del servicio de indexación es "write".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DateModified
   shellPKey = PKEY_DateModified
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DateModified
   shellPKey = PKEY_DateModified
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 14
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
