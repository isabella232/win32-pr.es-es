# <a name="systemdeviceinterfacehidvendorid"></a>System. DeviceInterface. HID. VendorId

Identificador de proveedor de dispositivo HID.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Hid.VendorId
   shellPKey = PKEY_DeviceInterface_Hid_VendorId
   formatID = CBF38310-4A17-4310-A1EB-247F0B67593B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
