# <a name="systemproplistcontentviewmodeforbrowse"></a>System.PropList.ContentViewModeForBrowse

Lista de propiedades que se mostrarán en el modo de vista de contenido de un elemento en el contexto de exploración.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.PropList.ContentViewModeForBrowse
   shellPKey = PKEY_PropList_ContentViewModeForBrowse
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
