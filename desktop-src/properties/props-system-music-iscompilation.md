# <a name="systemmusiciscompilation"></a>Sistema. Música. IsCompilation

Indica si el archivo de música forma parte de una compilación.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Music.IsCompilation
   shellPKey = PKEY_Music_IsCompilation
   formatID = C449D5CB-9EA4-4809-82E8-AF9D59DED6D1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
