# <a name="systemdevicesaepserviceprotocolid"></a>System.Devices.AepService.ProtocolId

Identidad del protocolo sobre el que se ha detectado este servicio de punto de conexión de asociación de dispositivos

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.AepService.ProtocolId
   shellPKey = PKEY_Devices_AepService_ProtocolId
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
