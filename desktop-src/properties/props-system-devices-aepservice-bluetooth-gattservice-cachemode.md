# <a name="systemdevicesaepservicebluetoothgattservicecachemode"></a>System.Devices.AepService. Bluetooth. ServiceService.CacheMode

Establece el Bluetooth de caché de La Cache para la consulta.

## <a name="windows-10-version-1511"></a>Windows 10, versión 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.CacheMode
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_CacheMode
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
