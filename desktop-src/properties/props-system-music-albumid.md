# <a name="systemmusicalbumid"></a>System. Music. AlbumID

Esta propiedad distingue los álbumes con el mismo título de distintos intérpretes. Es la concatenación de System. Music. AlbumArtist y System. Music. álbum.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Music.AlbumID
   shellPKey = PKEY_Music_AlbumID
   formatID = 56A3372E-CE9C-11D2-9F0E-006097C686F6
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
