# <a name="systemdevicesdevicecapabilities"></a>System.Devices.DeviceCapabilities

Funcionalidades del dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.DeviceCapabilities
   shellPKey = PKEY_Devices_DeviceCapabilities
   formatID = A45C254E-DF1C-4EFD-8020-67D146A850E0
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

Los valores PKEY se definen en Propkey. h.
