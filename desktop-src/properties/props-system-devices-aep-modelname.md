# <a name="systemdevicesaepmodelname"></a>System.Devices.Aep.ModelName

Nombre del modelo del punto de conexión de asociación de dispositivos

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.Aep.ModelName
   shellPKey = PKEY_Devices_Aep_ModelName
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
