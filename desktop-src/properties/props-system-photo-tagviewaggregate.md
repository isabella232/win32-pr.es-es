# <a name="systemphototagviewaggregate"></a>System. Photo. TagViewAggregate

Agregación de solo lectura de propiedades similares a las etiquetas para su uso en la creación de vistas.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Photo.TagViewAggregate
   shellPKey = PKEY_Photo_TagViewAggregate
   formatID = B812F15D-C2D8-4BBF-BACD-79744346113F
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
