# <a name="systemgpsdestdistance"></a>System.GPS.DestDistance

Indica la distancia al punto de destino. Se calcula a partir de PKEY \_ GPS \_ DestDistanceNumerator y PKEY \_ GPS \_ DestDistanceDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versi贸n 1703, Windows 10, versi贸n 1607, Windows 10, versi贸n 1511, Windows 10, versi贸n 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

   name = System.GPS.DestDistance
   shellPKey = PKEY_GPS_DestDistance
   formatID = A93EAE04-6804-4F24-AC81-09B266452118
   propID = 100
   SearchInfo

   typeInfo
