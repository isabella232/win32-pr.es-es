# <a name="systemsearchqueryfocusedsummarywithfallback"></a>System. Search. QueryFocusedSummaryWithFallback

Resumen orientado a consultas del documento. Si no hay ninguno disponible, se devuelve el resumen. Esta propiedad la crea automáticamente el sistema de búsqueda. Utilice esta propiedad solo cuando consuma los valores de la aplicación, no para proporcionar valores a un controlador de propiedades.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Search.QueryFocusedSummaryWithFallback
   shellPKey = PKEY_Search_QueryFocusedSummaryWithFallback
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
