# <a name="systemphotofocalplaneyresolution"></a>System.Photo.FocalPlaneYResolution

Indica el número de píxeles en el alto de la imagen (dirección Y) por FocalPlaneResolutionUnit en el plano focal de la cámara. Se calcula a partir de PKEY \_ Photo \_ FocalPlaneYResolutionNumerator y PKEY \_ Photo \_ FocalPlaneYResolutionDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneYResolution
   shellPKey = PKEY_Photo_FocalPlaneYResolution
   formatID = 4FFFE4D0-914F-4AC4-8D6F-C9C61DE169B1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
