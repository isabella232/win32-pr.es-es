# <a name="systemdevicesserviceaddress"></a>System.Devices.ServiceAddress

Dirección del punto de conexión del servicio de dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.ServiceAddress
   shellPKey = PKEY_Devices_ServiceAddress
   formatID = 656A3BB3-ECC0-43FD-8477-4AE0404A96CD
   propID = 16384
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
