# <a name="systemmessageparticipants"></a>System.Message.Participants

Participantes en la comunicación.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1

```
propertyDescription
   name = System.Message.Participants
   shellPKey = PKEY_Message_Participants
   formatID = 1A9BA605-8E7C-4D11-AD7D-A50ADA18BA1B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
