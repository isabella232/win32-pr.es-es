# <a name="systemmessageflags"></a>System.Message.Flags

Marcas asociadas a mensajes de correo electrónico (que identifican que hay un recibo de lectura pendiente, por ejemplo). Los valores almacenados aquí por Outlook se definen para LAS \_ MARCAS DE MENSAJE DE LA \_ PR.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Message.Flags
   shellPKey = PKEY_Message_Flags
   formatID = A82D9EE7-CA67-4312-965E-226BCEA85023
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = false
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
