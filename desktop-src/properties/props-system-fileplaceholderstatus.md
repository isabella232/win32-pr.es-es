# <a name="systemfileplaceholderstatus"></a>System.FilePlaceholderStatus

Contiene las marcas de estado del archivo de marcador de posición.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1

```
propertyDescription
   name = System.FilePlaceholderStatus
   shellPKey = PKEY_FilePlaceholderStatus
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
