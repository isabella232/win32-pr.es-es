# <a name="systemparsingbindcontext"></a>System.ParsingBindContext

Se usa para obtener [**IBindCtx para**](/windows/win32/api/objidl/nn-objidl-ibindctx) un elemento que se va a analizar.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versi贸n 1703, Windows 10, versi贸n 1607, Windows 10, versi贸n 1511, Windows 10, versi贸n 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParsingBindContext
   shellPKey = PKEY_ParsingBindContext
   formatID = DFB9A04D-362F-4CA3-B30B-0254B17B5B84
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Any
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
