# <a name="systemcontactassistanttelephone"></a>System.Contact.AssistantTelephone

Número de teléfono del asistente del contacto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.AssistantTelephone
   shellPKey = PKEY_Contact_AssistantTelephone
   formatID = 9A93244D-A7AD-4FF8-9B99-45EE4CC09AF6
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
