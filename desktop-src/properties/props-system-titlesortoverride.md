# <a name="systemtitlesortoverride"></a>System. TitleSortOverride

Este valor de cadena opcional permite reemplazar el criterio de ordenación estándar de System. title. esto es muy importante para ordenar correctamente los archivos de música en japonés que no se pueden ordenar correctamente de forma fonética (el orden esperado por el usuario) sin este campo. También se puede usar para personalizar la ordenación en escenarios que no son de Asia oriental, como permitir que el usuario elimine artículos con fines de ordenación.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.TitleSortOverride
   shellPKey = PKEY_TitleSortOverride
   formatID = F0F7984D-222E-4AD2-82AB-1DD8EA40E57E
   propID = 300
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
