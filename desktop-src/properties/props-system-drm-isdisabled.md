# <a name="systemdrmisdisabled"></a>System. DRM. IsDisabled

Indica si DRM ha deshabilitado el archivo multimedia.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.DRM.IsDisabled
   shellPKey = PKEY_DRM_IsDisabled
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
