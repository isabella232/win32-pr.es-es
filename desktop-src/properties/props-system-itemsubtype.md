# <a name="systemitemsubtype"></a>System.ItemSubType

Describe el subtipo de un elemento. El valor está pensado para mostrarse al usuario. A diferencia de System.ItemType, que normalmente se usa para describir una clase de elementos que todos tienen el mismo formato de contenido común, System.ItemSubType puede diferir en función del contenido individual o el propósito del elemento. Por ejemplo, esta propiedad se puede usar para identificar un elemento con System.ItemType = "jpg"as System.ItemSubType = "Panorama" o System.ItemSubType = "Smart Shot".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.ItemSubType
   shellPKey = PKEY_ItemSubType
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 37
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
