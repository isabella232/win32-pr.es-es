# <a name="systemcontactbusinessaddresscountry"></a>System.Contact.BusinessAddressCountry

País de la dirección de negocio del contacto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.BusinessAddressCountry
   shellPKey = PKEY_Contact_BusinessAddressCountry
   formatID = B0B87314-FCF6-4FEB-8DFF-A50DA6AF561C
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
