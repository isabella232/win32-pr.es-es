# <a name="systemdevicesdefaulttooltip"></a>System.Devices.DefaultTooltip

Información sobre herramientas para el estado predeterminado.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.DefaultTooltip
   shellPKey = PKEY_Devices_DefaultTooltip
   formatID = 880F70A2-6082-47AC-8AAB-A739D1A300C3
   propID = 153
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
