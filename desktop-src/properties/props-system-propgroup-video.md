# <a name="systempropgroupvideo"></a>System.PropGroup.Video

Separador de grupo de propiedades que se usa en las listas de propiedades para separar los archivos de vídeo de otros tipos. No use esta propiedad para obtener o establecer valores. Solo está pensado como un marcador.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Video
   shellPKey = PKEY_PropGroup_Video
   formatID = BEBE0920-7671-4C54-A3EB-49FDDFC191EE
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
