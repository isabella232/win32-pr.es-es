# <a name="systemcontactprimaryaddressstreet"></a>System.Contact.PrimaryAddressAddressAddress

Dirección principal del contacto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.PrimaryAddressStreet
   shellPKey = PKEY_Contact_PrimaryAddressStreet
   formatID = 63C25B20-96BE-488F-8788-C09C407AD812
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
