# <a name="systemdevicesaepisconnected"></a>System.Devices.Aep.IsConnected

Si el dispositivo está conectado actualmente al sistema o no

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.Aep.IsConnected
   shellPKey = PKEY_Devices_Aep_IsConnected
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
