# <a name="systemphotoflashenergy"></a>System. Photo. FlashEnergy

Indica la energía de luz estroboscópica en el momento en que se capturó la imagen, medida en los segundos de la vela de luz. Se calcula a partir de PKEY \_ Photo \_ FLASHENERGYNUMERATOR y PKEY \_ Photo \_ FlashEnergyDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FlashEnergy
   shellPKey = PKEY_Photo_FlashEnergy
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 41483
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
