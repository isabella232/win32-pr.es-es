# <a name="systemthumbnailstream"></a>System.ThumbnailStream

Datos que representan la miniatura en formato VT STREAM, compatible con códecs Windows GDI+ y Windows, como \_ .jpg y .png.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1

```
propertyDescription
   name = System.ThumbnailStream
   shellPKey = PKEY_ThumbnailStream
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 27
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Stream
      IsInnate = false
```

## <a name="windows-8-windows-7-windows-vista"></a>Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ThumbnailStream
   shellPKey = PKEY_ThumbnailStream
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 27
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Stream
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
