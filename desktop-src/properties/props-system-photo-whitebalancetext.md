# <a name="systemphotowhitebalancetext"></a>System.Photo.WhiteBalanceText

La forma fácil de usar [de System.Photo.WhiteBalance](./props-system-photo-whitebalance.md). No está pensado para analizarse mediante programación.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.WhiteBalanceText
   shellPKey = PKEY_Photo_WhiteBalanceText
   formatID = 6336B95E-C7A7-426D-86FD-7AE3D39C84B4
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
