# <a name="icategory-properties"></a>Propiedades de ICategory

La [**interfaz ICategory**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) define las siguientes propiedades.



| Propiedad                                     | Descripción                                                                                       |
|----------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Idcategoría**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_categoryid)   | Obtiene el identificador de la categoría.                                                              |
| [**Children**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_children)       | Obtiene una colección de interfaz que contiene las categorías secundarias de esta categoría.                 |
| [**Descripción**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_description) | Obtiene la descripción de la categoría.                                                             |
| [**Imagen**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_image)             | Obtiene una interfaz que contiene información sobre la imagen asociada a la categoría . |
| [**Nombre**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_name)               | Obtiene el nombre localizado de la categoría.                                                          |
| [**Pedido**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_order)             | Obtiene el orden de presentación recomendado de esta categoría entre sus categorías del mismo nivel.                 |
| [**Parent**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_parent)           | Obtiene una interfaz que describe la categoría primaria de esta categoría.                            |
| [**Tipo**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_type)               | Obtiene el tipo de la categoría.                                                                    |
| [**Actualizaciones**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_updates)         | Obtiene una interfaz que contiene una colección de actualizaciones que pertenecen inmediatamente a la categoría .  |
