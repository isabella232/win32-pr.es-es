# <a name="iautomaticupdates-properties"></a>Propiedades de IAutomaticUpdates

La interfaz [**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) define las siguientes propiedades.



| Propiedad                                                   | Descripción                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ServiceEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Obtiene un valor booleano que indica si están disponibles todos los componentes que Actualizaciones automáticas requiere. |
| [**Configuración**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Obtiene los valores de configuración para Actualizaciones automáticas.                                                        |
