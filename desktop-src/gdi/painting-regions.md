# <a name="painting-regions"></a>Regiones de dibujo

Una aplicación rellena el interior de una región mediante el pincel seleccionado actualmente en un contexto de dispositivo por la [**función PaintRgn.**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) Esta función usa los modos de relleno de polígono actuales (alternativo y sinuoso).
