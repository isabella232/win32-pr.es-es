# <a name="framing-regions"></a>Regiones de trama

Una aplicación dibuja un borde alrededor de una región llamando a la función [**FrameRgn**](/windows/desktop/api/Wingdi/nf-wingdi-framergn) y especificando el ancho del borde y el patrón de pincel que usa el sistema al dibujar el marco.
