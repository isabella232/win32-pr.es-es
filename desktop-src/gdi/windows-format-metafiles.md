# <a name="windows-format-metafiles"></a>Windows-Format metarchivos

Los metarchivos Windows formato de microsoft están limitados en sus funcionalidades y rara vez se deben usar. Las Windows de formato de archivo se admiten para mantener la compatibilidad con versiones anteriores con las aplicaciones que se escribieron para ejecutarse como aplicaciones basadas en Windows bits. En su lugar, debe usar las funciones de formato mejorado.

Las Windows de 16 bits usan *un metarchivo* de formato Windows de 16 bits. El formato consta de un encabezado y una matriz de registros de metarchivo.

Estas son las limitaciones de este formato:

-   Un metarchivo Windows formato es dependiente de la aplicación y del dispositivo. Los cambios en los modos de asignación de la aplicación o la resolución del dispositivo afectan a la apariencia de los metarchivos creados en este formato.
-   Un metarchivo de Windows formato no contiene un encabezado completo que describa las dimensiones de imagen originales, la resolución del dispositivo en el que se creó la imagen, una descripción de texto opcional o una paleta opcional.
-   Un metarchivo Windows formato no admite las nuevas funciones de curva, ruta de acceso y transformación. Consulte la lista de funciones admitidas en la tabla siguiente.
-   Algunos Windows metarchivo con formato de archivo no se pueden escalar.
-   No se puede consultar el contexto de dispositivo de metarchivo asociado a un metarchivo de formato Windows (es decir, una aplicación no puede recuperar datos de resolución de dispositivos, métricas de fuente, entre otros).

Para convertir un metarchivo de formato Windows en un metarchivo de formato mejorado, llame a la función [**GetMetaFileBitsEx**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) para recuperar los datos del metarchivo de formato Windows y, a continuación, llame a la función [**SetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) para convertir estos datos en un metarchivo de formato mejorado. Para convertir un registro de formato mejorado en un registro Windows formato, llame a la [**función GetWinMetaFileBits.**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits)
