# <a name="windows-format-metafiles"></a>Metaarchivos de Windows-Format

Los metaarchivos con formato de Microsoft Windows están limitados en sus capacidades y raramente deben usarse. Las funciones de formato de Windows se admiten para mantener la compatibilidad con versiones anteriores de las aplicaciones que se escribieron para ejecutarse como aplicaciones basadas en Windows de 16 bits. En su lugar, debe usar las funciones de formato mejorado.

Las aplicaciones basadas en Windows de 16 bits usan un *metarchivo con formato de Windows* . El formato consta de un encabezado y una matriz de registros de metarchivo.

A continuación se indican las limitaciones de este formato:

-   Un metarchivo de formato de Windows es dependiente de la aplicación y el dispositivo. Los cambios en los modos de asignación de la aplicación o en la resolución del dispositivo afectan a la apariencia de los metaarchivos creados con este formato.
-   Un metarchivo con formato de Windows no contiene un encabezado completo que describa las dimensiones de la imagen original, la resolución del dispositivo en el que se creó la imagen, una descripción de texto opcional o una paleta opcional.
-   Un metarchivo con formato de Windows no es compatible con las nuevas funciones de curva, trazado y transformación. Consulte la lista de funciones admitidas en la tabla siguiente.
-   Algunos registros de metarchivos con formato de Windows no se pueden escalar.
-   No se puede consultar el contexto de dispositivo de metarchivo asociado a un metarchivo de formato de Windows (es decir, una aplicación no puede recuperar los datos de resolución del dispositivo, las métricas de fuente, etc.).

A continuación se muestran las únicas funciones que admiten los metaarchivos con formato de Windows.

Para convertir un metarchivo con formato de Windows en un metarchivo con formato mejorado, llame a la función [**GetMetaFileBitsEx**](/windows/desktop/api/Wingdi/nf-wingdi-getmetafilebitsex) para recuperar los datos del metarchivo Format de Windows y, a continuación, llame a la función [**SetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-setwinmetafilebits) para convertir estos datos en un metarchivo con formato mejorado. Para convertir un registro con formato mejorado en un registro con formato de Windows, llame a la función [**GetWinMetaFileBits**](/windows/desktop/api/Wingdi/nf-wingdi-getwinmetafilebits) .
