# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Internet Explorer Asistente para compatibilidad 8 (Aomorno)

El Asistente para compatibilidad de Windows Internet Explorer 8 actualiza automáticamente los sitios web para que se representen correctamente en Internet Explorer 8 sin romper su compatibilidad en exploradores anteriores. El asistente automatiza la tarea de destino de la versión del explorador, manteniendo la apariencia y la funcionalidad de las páginas. Para obtener más información sobre este asistente, [vea Internet Explorer 8 Compatibility Wizard Inicio rápido](https://www.mobilize.net/).
