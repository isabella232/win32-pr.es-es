# <a name="windows-server-2008-r2-logo-program-for-software"></a>Windows Programa de logotipos de Server 2008 R2 para software

## <a name="affected-platforms"></a>Plataformas afectadas

 **Servidores:** Windows Server 2008 R2  

### <a name="description"></a>Descripción

El Windows logotipo de Server 2008 R2 se ha retirado.

Windows Los envíos de pruebas de Server 2008 R2 2012 ya no se aceptan para las competencias de Microsoft Partner Network (MPN) ni para el logotipo de certified for Windows Server.

Windows Server 2012 R2 es actualmente la plataforma preferida para el logotipo de Certified for Windows Server. Consulte el [programa Enterprise certificación de logotipos de aplicaciones en la nube](https://azure.microsoft.com/overview/commercial-marketplace/) para obtener más información.
