# <a name="the-private-data-object-type"></a>Tipo de objeto de datos privados

Puede usar objetos de datos privados para almacenar cadenas de datos en un formato cifrado. Un sistema puede contener varios objetos Private Data. Los objetos de datos privados se proporcionan principalmente para admitir el almacenamiento de contraseñas de cuenta de servidor. La información almacenada en objetos de datos privados se cifra mediante una clave específica del sistema generada.
