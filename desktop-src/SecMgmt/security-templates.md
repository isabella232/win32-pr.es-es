# <a name="security-templates"></a>Plantillas de seguridad

Las plantillas de seguridad son una herramienta de complemento independiente que los usuarios pueden usar para definir configuraciones de seguridad independientes del equipo. Estas configuraciones se guardan como archivos .inf basados en texto.

Las plantillas de seguridad y [la configuración y el análisis de seguridad](security-configuration-and-analysis.md) son los complementos de configuración de seguridad.
