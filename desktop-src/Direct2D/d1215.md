# <a name="d1215-vertex-range-must-be-null"></a>D1215: El intervalo de vértices debe ser NULL

Cuando se usa un sombreador de vértices NULL, también se debe usar UN VALOR NULL D2D1 \_ VERTEX \_ RANGE.
