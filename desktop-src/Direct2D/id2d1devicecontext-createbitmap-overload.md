# <a name="id2d1devicecontextcreatebitmap-methods"></a>Métodos ID2D1DeviceContext::CreateBitmap

Crea un mapa de bits que se puede usar como superficie de destino, para volver a leer en la CPU o como origen de las API [**DrawBitmap**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1devicecontext-drawbitmap(id2d1bitmap_constd2d1_rect_f_float_d2d1_interpolation_mode_constd2d1_rect_f_constd2d1_matrix_4x4_f)) e [**ID2D1BitmapBrush.**](/windows/win32/api/d2d1/nn-d2d1-id2d1bitmapbrush) Además, se puede pasar información de contexto de color al mapa de bits.
