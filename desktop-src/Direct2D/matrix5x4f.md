# <a name="matrix5x4f-class"></a>Matrix5x4F (clase)

La **clase Matrix5x4F** representa una matriz 5 por 4 y proporciona métodos prácticos para crear matrices.

## <a name="members"></a>Members

La **clase Matrix5x4F** hereda de [**D2D1 \_ MATRIX \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f). **Matrix5x4F** también tiene estos tipos de miembros:

-   [Constructores](#constructors)
