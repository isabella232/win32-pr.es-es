# <a name="d1152-command-list-closed-twice"></a>D1152: Lista de comandos cerrada dos veces

Se intentó cerrar una lista de comandos por segunda vez.






 

## <a name="possible-causes"></a>Causas posibles

Después de rellenar una lista de comandos, solo debe cerrarse una vez. Direct2D considera que las llamadas posteriores a ID2D1CommandList::Close son un error.

## <a name="possible-fixes"></a>Posibles correcciones

Para evitar este mensaje, asegúrese de que una lista de comandos solo se cierra una vez. Si una aplicación usa devoluciones de llamada o rutas de acceso de código recursivas para interactuar con listas de comandos, asegúrese de que su comportamiento es correcto.
