# <a name="d1240-source-tranform-still-mapped"></a>D1240: Tranform de origen todavía asignado

El mapa de bits pasado a ID2D1SourceTransform::D raw nunca se desaprobó.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación llamó a ID2D1Bitmap1::Map, pero no llamó a ID2D1Bitmap1::Unmap en un mapa de bits pasado a ID2D1SourceTransform::D raw.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que se llama a Unmap cada vez que se llama a Map.
