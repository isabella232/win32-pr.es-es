# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: La opción Descartar mapa requiere textura dinámica

D2D1 \_ MAP OPTIONS DISCARD solo es válido con un mapa de bits creado a partir de una textura \_ \_ dinámica.






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1::Map con el atributo MAP OPTIONS DISCARD de D2D1, pero el recurso direct3D subyacente para el mapa de bits no se asigna con \_ \_ \_ D3D11 \_ USAGE \_ DYNAMIC.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación requiere acceso a un mapa de bits mediante ID2D1Bitmap1::Map, asegúrese de que el recurso subyacente de Direct3D tiene el atributo D3D11 \_ USAGE \_ DYNAMIC.
