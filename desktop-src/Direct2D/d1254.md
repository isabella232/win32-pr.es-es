# <a name="d1254-operation-on-single-threaded-domain"></a>D1254: Operación en un dominio de subproceso único

Se realizó una llamada a [**ID2D1Multithread::Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) o [**ID2D1Multithread::Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) en un dominio de generador de un solo subproceso.

Esto puede ocurrir si la aplicación creó accidentalmente el generador D2D con [**D2D1 \_ FACTORY TYPE MULTI \_ \_ \_ THREADED**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type). En este caso, se omitirá la llamada.
