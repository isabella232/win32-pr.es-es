# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: El formato DXGI no se admite en ninguna API

Direct2D no admite el formato DXGI pasado a esta API.





 

## <a name="possible-causes"></a>Causas posibles

Algunos formatos de superficie no son compatibles con las operaciones de Direct2D, independientemente de cómo se utilicen. Este mensaje indica que la aplicación debe elegir otro formato de superficie.

## <a name="possible-fixes"></a>Posibles correcciones

Consulte [Formatos de píxeles admitidos y modos alfa](supported-pixel-formats-and-alpha-modes.md) para obtener instrucciones sobre qué formatos se admiten y cuáles no. Si la aplicación debe representarse en un único canal e intenta usar un formato de solo canal rojo que no se admite, considere la posibilidad de cambiarlo para que se represente a un formato de solo canal alfa.
