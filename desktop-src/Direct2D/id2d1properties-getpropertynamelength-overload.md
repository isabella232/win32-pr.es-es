# <a name="id2d1propertiesgetpropertynamelength-methods"></a>Métodos ID2D1Properties::GetPropertyNameLength

Obtiene el número de caracteres para el nombre de propiedad especificado.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                         | Descripción                                                                                        |
|:-------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**GetPropertyNameLength(U)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getpropertynamelength(u))     | Obtiene el número de caracteres para el nombre de propiedad especificado. Se trata de una sobrecarga de plantilla.<br/> |
| [**GetPropertyNameLength(UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getpropertynamelength(uint32)) | Obtiene el número de caracteres para el nombre de propiedad especificado.<br/>                              |
