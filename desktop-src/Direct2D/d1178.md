# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: El estilo de guion personalizado requiere una matriz de guiones

La creación de un estilo de trazo con DASH \_ STYLE CUSTOM requiere una \_ matriz de guiones.






 

## <a name="possible-causes"></a>Causas posibles

Cuando la aplicación intentó crear un id2D1StrokeStyle, usó el atributo CUSTOM DE ESTILO DASH D2D1, pero proporcionó una \_ \_ \_ matriz de guiones NULL.

## <a name="possible-fixes"></a>Posibles correcciones

Si la aplicación requiere guiones personalizados, asegúrese de que se especifica una matriz de guiones válida. Vea CreateStrokeStyle para obtener un ejemplo. Si un estilo de guion integrado es aceptable, deje la matriz de guiones como NULL y use un especificador como D2D1 \_ DASH \_ STYLE \_ DASH.
