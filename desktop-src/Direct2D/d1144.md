# <a name="d1144-effect-is-not-a-target"></a>D1144: el efecto no es un destino

Error de SetTarget porque se pasó un efecto como destino.






 

## <a name="possible-causes"></a>Causas posibles

ID2D1DeviceContext:: SetTarget toma un \* parámetro ID2D1Image. Las imágenes de Direct2D pueden ser mapas de bits, listas de comandos o efectos. Direct2D solo tiene en cuenta los mapas de bits y las listas de comandos como destinos válidos.

## <a name="possible-fixes"></a>Posibles correcciones

Modifique la aplicación para usar un mapa de bits o una lista de comandos como destino de su contexto de dispositivo. Direct2D no realiza la inicialización de efectos mediante SetTarget. En su lugar, para elegir el contenido de un efecto, una aplicación crea el efecto mediante ID2D1DeviceContext:: CreateEffect y usa un GUID de efecto correspondiente.
