# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229: D1229: Llamada inmersiva a GetDesktopdpi

Se llamó a ID2D1Factory::GetDesktopDpi en una Windows Store. Esto no es recomendable. En su lugar, se debe usar DisplayProperties::LogicalDpi.






 

## <a name="possible-causes"></a>Causas posibles

Se llamó a ID2D1Factory::GetDesktopDpi en una Windows Store.

## <a name="possible-fixes"></a>Posibles correcciones

DisplayProperties::LogicalDpi debe usarse para obtener el valor de PPP en una Windows Store.
