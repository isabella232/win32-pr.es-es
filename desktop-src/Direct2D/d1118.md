# <a name="d1118-target-not-a-bitmap"></a>D1118: Destino no un mapa de bits

Error en una operación de dibujo porque el destino es NULL o una lista de comandos.





 

## <a name="possible-causes"></a>Causas posibles

Una causa probable de este mensaje es que la aplicación llamó a GetDC o ReleaseDC mientras el destino de su contexto de dispositivo se estableció en NULL o en una lista de comandos.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación usa GetDC o ReleaseDC expuesto por IGdiInteropRenderTarget, asegúrese de que el ID2D1DeviceContext correspondiente tenga un mapa de bits válido establecido como destino.
