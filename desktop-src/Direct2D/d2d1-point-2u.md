# <a name="d2d1_point_2u"></a>D2D1 \_ POINT \_ 2U

Representa un par de coordenadas X e Y en un espacio bidimensional.


```C++
typedef D2D_POINT_2U D2D1_POINT_2U;
```



## <a name="remarks"></a>Observaciones

Los puntos de Direct2D se representan mediante las estructuras [**D2D1 \_ POINT \_ 2F**](d2d1-point-2f.md) o **D2D1 \_ POINT \_ 2U.** Ambos contienen un par de coordenadas X e Y en un espacio bidimensional. La **estructura D2D1 \_ POINT \_ 2F** almacena las coordenadas como valores **FLOAT** y la estructura **D2D1 \_ POINT \_ 2U** las almacena como **valores UINT32.**

**D2D1 \_ POINT \_ 2U** es un nuevo nombre para el tipo ya definido [**D2D \_ POINT \_ 2U**](/windows/desktop/api/dcommon/ns-dcommon-d2d_point_2u).
