# <a name="d1235-target-rendered-to-from-multiple-dcs"></a>D1235: destino representado en desde varios equipos de dominio

Se intentó representar en una imagen mientras aún se estaba representando en un contexto de dispositivo anterior. Se debe llamar a EndDraw en ese contexto de dispositivo antes de representarlo con el contexto del dispositivo actual. Cuando se usa la capa de depuración, también se genera la dirección de este contexto de dispositivo.
