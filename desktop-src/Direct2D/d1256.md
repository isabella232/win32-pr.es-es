# <a name="d1256-invalid-analysis-effect-usage"></a>D1256: Uso de efectos de análisis no válidos

Un efecto de análisis se usó incorrectamente. Esto puede ocurrir cuando se usa un efecto de análisis como entrada a otro efecto o a un pincel de imagen, se dibuja en una lista de comandos o se almacena en caché su salida.
