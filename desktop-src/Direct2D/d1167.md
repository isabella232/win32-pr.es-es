# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: La opción Leer mapa requiere mapa de bits legible de CPU

D2D1 \_ MAP OPTIONS READ solo es válido en un mapa de bits legible de LA \_ \_ CPU






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1::Map con el atributo READ D2D1 MAP OPTIONS, pero el mapa de bits no es legible por \_ \_ la \_ CPU.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación requiere acceso de lectura mediante ID2D1Bitmap1::Map, cree el mapa de bits mediante la opción D2D1 \_ BITMAP \_ OPTIONS CPU \_ \_ READ.
