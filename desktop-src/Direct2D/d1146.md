# <a name="d1146-invalid-render-controls"></a>D1146: controles de representación no válidos

Los controles de representación especificados tenían parámetros no válidos y se omitirán.






 

## <a name="possible-causes"></a>Causas posibles

Este mensaje puede producirse durante ID2D1DeviceContext:: SetRenderingControls si la aplicación especifica un valor de precisión de búfer no válido o un tamaño de mosaico de cero.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que la aplicación usa un valor de precisión de búfer válido. Vea [**\_ \_ precisión del búfer de D2D1**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Asegúrese de que la aplicación especifica un tamaño de mosaico distinto de cero.
