# <a name="d2d1_matrix_4x3_f"></a>MATRIZ D2D1 \_ \_ 4X3 \_ F

Representa una matriz de 4 por 3.


```C++
typedef D2D_MATRIX_4X3_F D2D1_MATRIX_4X3_F;
```



## <a name="remarks"></a>Comentarios

**D2D1 \_ MATRIX \_ 4X3 es** un nuevo nombre para la estructura [**D2D \_ MATRIX \_ 4X3 \_ F.**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f) Para obtener una lista de los campos proporcionados por la matriz, vea [**D2D \_ MATRIX \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f).
