# <a name="id2d1devicecontextcreatebitmapfromdxgisurface-methods"></a>Métodos ID2D1DeviceContext::CreateBitmapFromDxgiSurface

Crea un mapa de bits a partir de una superficie DXGI que se puede establecer como una superficie de destino o tener información de contexto de color adicional especificada.
