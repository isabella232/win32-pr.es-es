# <a name="installing-the-direct2d-debug-layer"></a>Instalación de la capa de depuración de Direct2D

A partir Windows 8, la capa de depuración de Direct2D se incluye [con Microsoft Visual Studio Express 2012 para Windows 8](https://www.microsoft.com/visualstudio/11/downloads).

La capa de depuración de Direct2D Windows 7 ya no está disponible para su descarga.
