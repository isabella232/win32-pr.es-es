# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190: No se ha usado correctamente la marca de mapa de bits CannotDraw

D2D1 BITMAP OPTIONS CANNOT DRAW debe combinarse con \_ \_ \_ \_ D2D1 \_ BITMAP OPTIONS CPU READ o \_ \_ \_ D2D1 \_ BITMAP OPTIONS \_ \_ TARGET





 

## <a name="possible-causes"></a>Causas posibles

La aplicación especificó una combinación incompatible de las marcas BITMAP OPTIONS de D2D1. \_ \_

## <a name="possible-fixes"></a>Posibles correcciones

Agregar D2D1 \_ BITMAP OPTIONS CPU READ o \_ \_ \_ D2D1 \_ BITMAP OPTIONS TARGET a \_ \_ D2D1 \_ BITMAP \_ OPTIONS
