# <a name="d2d1_matrix_4x4_f"></a>MATRIZ D2D1 \_ \_ 4X4 \_ F

Representa una matriz de 4 por 4.


```C++
typedef D2D_MATRIX_4X4_F D2D1_MATRIX_4X4_F;
```



## <a name="remarks"></a>Observaciones

**D2D1 \_ MATRIX \_ 4X4** es un nuevo nombre para la estructura [**\_ \_ MATRIZ 4X4 \_ F de D2D.**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x4_f) Para obtener una lista de los campos proporcionados por la matriz, vea **D2D \_ MATRIX \_ 4X4 \_ F**.
