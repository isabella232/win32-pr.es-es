# <a name="d1177-unsupported-wic-pixel-format"></a>D1177: Formato de píxel wic no admitido

Direct2D no admite el formato de píxel WIC proporcionado. En concreto, Direct2D no admite mapas de bits de WIC que usen alfa recta a menos que la profundidad sea 8bpp.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación intentó interoperar un recurso de WIC cuyo formato no es compatible con Direct2D. Normalmente, estos formatos son aquellos con alfa recta y cuya profundidad no es de 8 bits por plano.

## <a name="possible-fixes"></a>Posibles correcciones

Considere la posibilidad de crear el recurso de WIC mediante un formato alternativo para la interoperabilidad con Direct2D o convertirlo.
