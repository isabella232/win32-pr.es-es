# <a name="d1230-unknown-buffer-precision-support"></a>D1230: Compatibilidad con precisión de búfer desconocida

Se realizó una llamada a IsBufferPrecisionSupported con D2D1 \_ BUFFER \_ PRECISION \_ UNKNOWN. Esta API devuelve false, aunque D2D1 BUFFER PRECISION UNKNOWN puede ser válida como \_ \_ entrada para algunas \_ API.






 

## <a name="possible-causes"></a>Causas posibles

Se realizó una llamada a IsBufferPrecisionSupported con D2D1 \_ BUFFER \_ PRECISION \_ UNKNOWN.

## <a name="possible-fixes"></a>Posibles correcciones

Para determinar si D2D1 BUFFER PRECISION UNKNOWN es una precisión de búfer válida para una API determinada, lea la documentación de esa API en lugar de intentar determinarla \_ \_ mediante \_ programación.
